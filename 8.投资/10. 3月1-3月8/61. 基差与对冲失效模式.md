## 核心概念

### 基差（Basis）

**定义**：基差 = 现货价格 - 期货价格。基差衡量的是现货与期货之间的价差，是对冲效果的关键变量。

> **Hull**："完美对冲很少存在。基差风险是使用期货对冲时必须面对的现实。"

$$\text{基差} = S - F$$

| 符号 | 含义 |
|------|------|
| $S$ | 现货价格（Spot Price） |
| $F$ | 期货价格（Futures Price） |

#### 生活中的例子

**例子1：农民卖玉米**

一个农民预计3个月后收获玉米，想锁定价格：

- **现货价格**：今天玉米现货 2000元/吨
- **期货价格**：3个月后交割的期货 2050元/吨
- **基差**：2000 - 2050 = -50元/吨

农民卖出期货对冲。3个月后：
- 如果现货跌到1800，期货也跌到1850 → 基差仍是-50 → **完美对冲**
- 如果现货跌到1800，期货跌到1900 → 基差变成-100 → **对冲不完美，多亏50元**

**例子2：航空公司买航油**

航空公司用原油期货对冲航油成本：
- 航油和原油价格高度相关，但不完全相同
- 当航油相对原油涨价时，对冲效果变差
- 这就是"交叉对冲"的基差风险

#### 基差的变化

| 情况 | 基差变化 | 对多头对冲者 | 对空头对冲者 |
|------|----------|--------------|--------------|
| 基差走强（变大） | $S$ 涨幅 > $F$ 涨幅 | 有利 | 不利 |
| 基差走弱（变小） | $S$ 涨幅 < $F$ 涨幅 | 不利 | 有利 |

**记忆口诀**：
- 空头对冲（卖期货）→ 希望基差走弱
- 多头对冲（买期货）→ 希望基差走强

---

### 为什么"理论正确但结果不理想"？

**对冲的理论假设 vs 现实**：

| 理论假设 | 现实情况 |
|----------|----------|
| 现货与期货完全相关 | 相关性 < 1，存在基差风险 |
| 可以精确匹配数量 | 合约有固定乘数，无法精确匹配 |
| 交易无成本 | 有买卖价差、手续费、滑点 |
| 可以持有到期 | 可能需要提前平仓或展期 |
| 保证金充足 | 可能面临追加保证金压力 |
| 流动性充足 | 某些合约流动性差 |

> **实务警示**："对冲不是消除风险，而是把价格风险转换成基差风险。"

## 五大对冲失效模式

### 失效模式1：基差风险

**机制**：现货与期货价格变动不同步，导致对冲效果偏离预期。

**经典案例：2008年航空公司对冲失败**

- **背景**：某航空公司用原油期货对冲航油成本
- **发生了什么**：
  - 原油价格从$140暴跌到$40
  - 航油价格也下跌，但跌幅较小（炼油产能紧张）
  - 期货空头大赚，但航油成本节省不如预期
- **结果**：对冲盈利，但航油成本下降幅度小于期货盈利，基差损失
- **教训**：交叉对冲的基差风险在极端行情下会放大

**监控指标**：

| 指标 | 计算方法 | 警戒阈值 |
|------|----------|----------|
| 基差波动率 | 基差的标准差 | > 历史均值2倍 |
| 相关系数 | 现货与期货收益率相关性 | < 0.9 |
| 基差偏离度 | 当前基差 / 历史均值 | > 150% 或 < 50% |

---

### 失效模式2：展期风险（Roll Risk）

**机制**：期货合约有到期日，长期对冲需要不断"展期"（平仓近月、开仓远月），展期时可能亏损。

**展期损益 = 新合约价格 - 旧合约价格**

| 市场结构 | 展期操作 | 展期损益 |
|----------|----------|----------|
| 期货升水（Contango） | 卖低买高 | 亏损（负展期收益） |
| 期货贴水（Backwardation） | 卖高买低 | 盈利（正展期收益） |

**经典案例：2020年原油负价格事件**

- **背景**：2020年4月，WTI原油期货跌至-$37/桶
- **发生了什么**：
  - 5月合约即将到期，持有者必须展期或实物交割
  - 存储设施满载，无人愿意接收实物
  - 多头被迫在负价格平仓
- **教训**：展期风险在极端情况下可能导致灾难性损失

**监控指标**：

| 指标 | 计算方法 | 警戒阈值 |
|------|----------|----------|
| 展期成本（年化） | (远月价格 - 近月价格) / 近月价格 × 12 | > 10% |
| 持仓集中度 | 你的持仓 / 该合约总持仓 | > 5% |
| 距到期天数 | 合约到期日 - 今天 | < 5天需警惕 |

---

### 失效模式3：流动性风险

**机制**：市场流动性不足时，无法以合理价格建仓或平仓。

**表现**：
- 买卖价差扩大
- 大单冲击成本高
- 极端情况下无法成交

**生活例子：拍卖会的流动性**

想象你在拍卖会卖一幅画：
- **流动性好**：很多买家竞价，你能卖个好价钱
- **流动性差**：只有一两个买家，你可能被迫低价出手

期货市场也一样。流动性差的合约，你的对冲成本会大幅上升。

**经典案例：2015年中国股指期货限制**

- **背景**：2015年股灾后，监管层限制股指期货交易
- **发生了什么**：
  - 开仓手数限制、手续费大幅提高
  - 期货贴水扩大到10%以上
  - 对冲基金无法有效对冲
- **结果**：很多量化对冲基金被迫清盘或暂停运作
- **教训**：监管政策变化也是流动性风险的来源

**监控指标**：

| 指标 | 计算方法 | 警戒阈值 |
|------|----------|----------|
| 买卖价差 | (卖一价 - 买一价) / 中间价 | > 0.1% |
| 日成交量 | 当日成交合约数 | < 你持仓的100倍 |
| 持仓量变化 | 今日持仓 / 昨日持仓 | < 80% |

---

### 失效模式4：保证金风险

**机制**：期货采用保证金交易，价格不利变动时需要追加保证金，资金不足可能被强制平仓。

**保证金计算**：

$$\text{维持保证金} = \text{合约价值} \times \text{保证金比例}$$

**生活例子：房贷断供**

想象你买房首付30%，贷款70%：
- 如果房价跌30%，你的首付（保证金）"没了"
- 银行可能要求你补钱（追加保证金）
- 如果补不上，银行可能收走房子（强制平仓）

期货的保证金机制类似，但波动更频繁，追加更及时。

**经典案例：LTCM的保证金危机（1998年）**

- **背景**：长期资本管理公司（LTCM）高杠杆套利
- **发生了什么**：
  - 俄罗斯债务违约，市场恐慌
  - LTCM的对冲头寸逆向波动
  - 经纪商要求追加保证金
  - 被迫在最差价格平仓，亏损加剧
- **结果**：几乎破产，美联储协调救助
- **教训**：保证金风险可以摧毁"理论正确"的策略

**监控指标**：

| 指标 | 计算方法 | 警戒阈值 |
|------|----------|----------|
| 保证金使用率 | 已用保证金 / 总资金 | > 50% |
| 可用保证金比例 | 可用资金 / 维持保证金 | < 150% |
| 最大不利波动覆盖 | 可用资金 / (合约价值 × 日波动率 × 3) | < 1 |

---

### 失效模式5：操作与模型风险

**机制**：人为错误、系统故障、模型假设错误导致对冲失效。

**常见操作错误**：

| 错误类型 | 示例 | 后果 |
|----------|------|------|
| 方向错误 | 该卖期货却买了 | 双倍风险敞口 |
| 数量错误 | 少了一个零 | 对冲比例错误 |
| 合约错误 | 买错月份/标的 | 对冲无效 |
| 时间错误 | 忘记展期 | 被迫实物交割或强平 |

**模型风险示例**：

| 假设 | 可能的现实 |
|------|-----------|
| 相关系数稳定 | 危机时相关性剧变 |
| 波动率恒定 | 波动率聚集、跳跃 |
| 正态分布 | 肥尾、黑天鹅 |
| 线性关系 | 非线性、结构突变 |

**监控指标**：

| 指标 | 方法 | 频率 |
|------|------|------|
| 对冲比例检查 | 实际持仓 vs 理论持仓 | 每日 |
| 合约到期提醒 | 距到期日天数 | 每周 |
| 模型回测 | 对冲效果 vs 预期 | 每月 |
| 压力测试 | 极端情景模拟 | 每季 |

## 投资者视角

### 为什么必须提前识别失效模式？

1. **风险预算**：对冲不是免费的，要预留"失效"的空间
2. **应急预案**：提前知道什么情况下对冲会失效，才能制定应对方案
3. **成本评估**：失效模式的发生概率 × 损失 = 真实对冲成本

### 对冲决策框架（含失效考虑）

| 决策要素 | 问题 | 你的答案 |
|----------|------|----------|
| 对冲必要性 | 敞口有多大？能承受多少损失？ | |
| 工具选择 | 哪个期货合约最匹配？基差风险多大？ | |
| 对冲比例 | 100%对冲还是部分对冲？ | |
| 资金准备 | 预留多少保证金缓冲？ | |
| 展期计划 | 何时展期？展期成本预算多少？ | |
| 止损规则 | 什么情况下放弃对冲？ | |

### 失效模式优先级排序

| 优先级 | 失效模式 | 理由 |
|--------|----------|------|
| ⭐⭐⭐ | 保证金风险 | 可能导致强制平仓，损失不可控 |
| ⭐⭐⭐ | 流动性风险 | 无法平仓时损失可能无限 |
| ⭐⭐ | 展期风险 | 长期对冲成本可能很高 |
| ⭐⭐ | 基差风险 | 影响对冲效果但通常可控 |
| ⭐ | 操作风险 | 可通过流程控制降低 |

## 关键要点

1. **基差是对冲的核心变量**：对冲把价格风险变成基差风险，基差波动决定对冲效果。

2. **展期有成本**：Contango市场下长期对冲需要支付展期成本，可能侵蚀对冲收益。

3. **流动性是隐形成本**：流动性差的合约对冲成本高，极端情况下可能无法平仓。

4. **保证金是生死线**：保证金不足导致强平是对冲失败的最常见原因，必须预留充足缓冲。

5. **提前识别>事后补救**：在建立对冲前就要识别所有失效模式，设置监控指标和应对预案。

## 延伸思考

### 完美对冲为什么不存在？

> **本质原因**：完美对冲需要两个资产的价格变动完全相同，但现实中没有两个资产是完全相同的。

| 不完美的来源 | 解释 |
|--------------|------|
| 标的不匹配 | 你对冲的是个股，期货跟踪的是指数 |
| 时间不匹配 | 你的敞口期限和期货到期日不一致 |
| 数量不匹配 | 合约乘数固定，无法精确匹配 |
| 市场摩擦 | 交易成本、买卖价差、滑点 |

### 什么时候不应该对冲？

| 情况 | 原因 |
|------|------|
| 对冲成本 > 预期损失 | 经济上不划算 |
| 失效概率很高 | 对冲可能无效 |
| 你有信息优势 | 你比市场更了解风险 |
| 敞口很小 | 不值得付出对冲的复杂性成本 |

**投资者警示**：对冲是工具，不是目的。在决定对冲前，先问自己：对冲的成本和失效风险加起来，是否真的比承担原始风险更好？
