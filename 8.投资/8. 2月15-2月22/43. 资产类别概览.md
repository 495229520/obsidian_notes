# 资产类别概览

## 核心概念

### 什么是资产类别？

**定义**：资产类别是一组具有相似特征、受相似经济因素驱动、在法律和监管上有相似地位的投资工具。理解资产类别是构建投资组合的第一步。

> **哈里·马科维茨**："分散投资是唯一的免费午餐。"
> — 《投资组合选择》（1952）

#### 生活中的例子

**例子1：餐厅的菜单分类**

想象你去一家餐厅，菜单分为：
- **主菜**（牛排、鱼、鸡肉）→ 类似**股票**：风险高、回报高、是正餐的核心
- **配菜**（沙拉、面包、汤）→ 类似**债券**：稳定、配角、降低整体风险
- **饮料**（咖啡、果汁、酒）→ 类似**大宗商品**：可选、有时提供独特价值

一顿好的饭需要搭配，一个好的投资组合也需要资产类别搭配。

**例子2：三条腿的凳子**

一个稳定的凳子需要三条腿：
- 股票 = 一条腿（增长）
- 债券 = 一条腿（稳定）
- 大宗商品/其他 = 一条腿（对冲通胀）

少一条腿，凳子就不稳；投资组合也是如此。

### 三大核心资产类别

#### 1. 股票（Equities）

**定义**：股票代表公司所有权的一部分。持有股票意味着你是公司的股东，有权分享公司利润（通过股息或股价上涨）。

> **沃伦·巴菲特**："如果你不愿意持有一只股票十年，那就不要考虑持有它十分钟。"
> — 致股东信

**核心特征**：

| 特征 | 说明 |
|------|------|
| 回报来源 | 资本增值 + 股息 |
| 风险水平 | 高（波动性大） |
| 流动性 | 通常高（公开交易市场） |
| 与经济关系 | 正相关（经济好，股票涨） |

**股票的细分类型**：

| 类型 | 特征 | 典型例子 |
|------|------|----------|
| 大盘股 | 市值>100亿美元，稳定 | 苹果、腾讯、茅台 |
| 小盘股 | 市值<20亿美元，高增长潜力 | 新兴科技公司 |
| 成长股 | 高增长预期，低/无股息 | 特斯拉、英伟达 |
| 价值股 | 低估值，高股息 | 银行、保险公司 |
| 周期股 | 随经济周期波动 | 汽车、航空 |
| 防御股 | 经济衰退时相对稳定 | 公用事业、必需消费品 |

**经典案例：标普500的长期回报**

从1926年到2023年：
- 标普500年化回报约10-11%（含股息再投资）
- 但单年波动可达-40%至+50%
- **启示**：长期持有股票可以获得可观回报，但需要承受短期波动

> **约翰·博格**："不要在干草堆中找针，买下整个干草堆。"
> — 指数基金之父

---

#### 2. 债券（Fixed Income）

**定义**：债券是一种借贷关系——你把钱借给发行方（政府或公司），发行方承诺按约定利率支付利息，并在到期时归还本金。

> **本杰明·格雷厄姆**："债券投资者最大的敌人是通货膨胀。"
> — 《聪明的投资者》

**核心特征**：

| 特征 | 说明 |
|------|------|
| 回报来源 | 固定利息（票息）+ 资本利得/损失 |
| 风险水平 | 通常低于股票 |
| 流动性 | 因债券类型而异 |
| 与利率关系 | **反向关系**（利率涨，债券价格跌） |

**债券的细分类型**：

| 类型 | 信用风险 | 收益率 | 典型例子 |
|------|----------|--------|----------|
| 国债 | 极低（无违约风险） | 最低 | 美国国债、中国国债 |
| 投资级公司债 | 低至中 | 中等 | 苹果、微软发行的债券 |
| 高收益债（垃圾债）| 高 | 高 | 评级BB以下的公司债 |
| 市政债 | 低至中 | 中等（常免税） | 地方政府债券 |
| 通胀保护债券 | 极低 | 随通胀调整 | TIPS、通胀挂钩债券 |

**债券价格与收益率的反向关系**：

```
利率上升 1% → 久期为10年的债券价格下跌约 10%
利率下降 1% → 久期为10年的债券价格上涨约 10%
```

**经典案例：2022年债券大跌**

2022年，美联储大幅加息：
- 美国长期国债下跌超过20%
- 这是40年来债券最差的年份
- **教训**：债券不是"无风险"的，利率风险同样重要

参见：[[24. 债券：价格、收益率和久期]]

#### 3. 大宗商品（Commodities）

**定义**：大宗商品是可以在市场上交易的实物资产或原材料，包括能源、金属、农产品等。

> **吉姆·罗杰斯**："大宗商品是对冲通胀的最佳工具之一。"
> — 著名商品投资者

**核心特征**：

| 特征 | 说明 |
|------|------|
| 回报来源 | 纯粹的价格变动（无利息、无股息） |
| 风险水平 | 高（受供需、天气、地缘政治影响） |
| 通胀对冲 | 通常与通胀正相关 |
| 与股债相关性 | 低（提供分散化好处） |

**大宗商品的细分类型**：

| 类别 | 例子 | 主要驱动因素 |
|------|------|--------------|
| 能源 | 原油、天然气 | OPEC政策、地缘政治、经济周期 |
| 贵金属 | 黄金、白银 | 避险情绪、通胀预期、美元走势 |
| 工业金属 | 铜、铝、锌 | 制造业需求、中国经济 |
| 农产品 | 大豆、玉米、小麦 | 天气、种植面积、需求 |

**黄金：特殊的大宗商品**

黄金在投资组合中有特殊地位：
- **避险资产**：市场恐慌时，黄金通常上涨
- **通胀对冲**：长期看，黄金保持购买力
- **零收益**：不产生利息或股息，持有成本实际为负（机会成本）

| 情景 | 黄金表现 |
|------|----------|
| 经济衰退 + 宽松货币 | 通常上涨 |
| 经济增长 + 紧缩货币 | 通常下跌或震荡 |
| 高通胀 | 通常上涨 |
| 地缘政治危机 | 短期上涨 |

> **雷·达里奥**："黄金是货币的替代品，是全球公认的财富储存形式。"
> — 桥水基金创始人

### 其他资产类别

#### 房地产（Real Estate）

- **直接投资**：购买物业
- **间接投资**：REITs（房地产投资信托）
- 特点：提供租金收入 + 资本增值，与通胀正相关

#### 现金及现金等价物

- 货币市场基金、短期国债、定期存款
- 特点：流动性最高，回报最低，几乎无波动

#### 另类投资

- 私募股权、对冲基金、加密货币、艺术品
- 特点：流动性低，门槛高，与传统资产相关性低

## 投资者视角

### 为什么投资者必须理解资产类别？

1. **资产配置决定回报**：研究表明，90%以上的投资组合回报来自资产配置，而非个股选择

2. **风险分散**：不同资产类别在不同经济环境下表现不同，组合持有可以降低整体风险

3. **匹配投资目标**：不同资产类别适合不同的投资目标和期限

> **大卫·斯文森**："资产配置是投资过程中最重要的决策，远比证券选择和市场时机重要。"
> — 耶鲁捐赠基金管理者

### 资产类别的历史表现对比

| 资产类别 | 长期年化回报（美国） | 年化波动率 | 最大回撤 |
|----------|----------------------|------------|----------|
| 美国股票 | ~10% | ~16% | -50%以上 |
| 美国长期国债 | ~5% | ~10% | -20%以上 |
| 黄金 | ~6% | ~15% | -40%以上 |
| 现金 | ~3% | ~1% | 几乎无 |

**关键洞察**：
- 股票长期回报最高，但波动也最大
- 债券提供稳定收入，但回报有限
- 黄金长期略跑赢通胀，但短期波动大
- 现金最稳定，但长期跑输通胀

### 不同经济环境下的资产表现

| 经济环境 | 股票 | 债券 | 大宗商品 | 现金 |
|----------|------|------|----------|------|
| 经济扩张 + 低通胀 | +++ | + | + | - |
| 经济扩张 + 高通胀 | + | - | +++ | - |
| 经济衰退 + 低通胀 | -- | +++ | - | ++ |
| 经济衰退 + 高通胀（滞胀）| -- | -- | ++ | - |

**全天候策略的启示**：没有单一资产在所有环境下都表现最好，因此需要组合配置。

## 关键要点

1. **资产类别是投资组合的基石**：股票提供增长、债券提供稳定、大宗商品对冲通胀

2. **资产配置决定90%的回报**：选择投资多少在股票/债券/其他，比选择具体股票更重要

3. **没有"最好"的资产类别**：不同资产在不同经济环境下表现不同，组合配置才是王道

4. **风险与回报成正比**：股票长期回报最高，但波动也最大；债券稳定但回报有限

5. **理解相关性是分散的关键**：选择相关性低的资产组合，才能真正降低风险

## 延伸思考

### 资产类别的边界正在模糊

> **雷·达里奥**："在高度全球化的今天，资产类别之间的相关性正在增加。"

传统上独立的资产类别，在危机时刻可能同涨同跌：
- 2008年金融危机：股票、高收益债、大宗商品同时暴跌
- 2020年3月：几乎所有资产都下跌（除了美国国债）
- 2022年：股票和债券同时下跌（40年来首次）

| 情况 | 传统预期 | 实际表现 | 教训 |
|------|----------|----------|------|
| 2008危机 | 债券对冲股票 | 高收益债与股票同跌 | 信用风险在危机时集中爆发 |
| 2020.3 | 分散化有效 | 流动性危机下全部下跌 | 现金才是终极避险 |
| 2022 | 60/40组合稳定 | 股债双杀 | 高通胀+加息是特殊情况 |

**投资者警示**：
- 分散化不是万能药，在极端情况下可能失效
- 现金储备永远重要
- 理解资产类别背后的驱动因素比简单分散更重要
