# IPS v2 综合整合

> **对应培养方案**：第7周交付成果——投资政策声明（IPS）v2
>
> 本笔记是第7周学习的总结，整合资产类别、多元化、行为金融的所有知识，形成一份完整的投资政策声明。

---

## 核心概念

### 什么是IPS v2？

**IPS v2** 是在 IPS v1（[[6. 投资政策声明 (IPS)]]）基础上的升级版本，新增了：

| 版本 | 核心内容 | 新增要素 |
|------|----------|----------|
| IPS v0 | 目标、期限、初步配置 | 基础框架 |
| IPS v1 | 风险限制、再平衡规则 | 风险意识 |
| **IPS v2** | 资产类别详解、分散化规则、行为控制 | 完整系统 |

> **大卫·斯文森**："投资政策声明是投资者与自己签订的最重要合同。它不是一次性文件，而是随着认知提升不断迭代的活文档。"
> — 耶鲁捐赠基金管理者

#### 生活中的例子

**例子1：健身计划的升级**

想象你的健身计划：
- **v0**：每周运动3次（只有目标）
- **v1**：每周跑步2次+力量1次，心率控制在120-150（有具体规则）
- **v2**：分部位训练计划、饮食配合、休息日安排、受伤预防机制（完整系统）

投资也是如此：IPS v2不仅告诉你"做什么"，还告诉你"为什么"以及"如何应对异常情况"。

**例子2：公司章程的完善**

一家成熟公司的章程会包含：
- 公司目标（投资目标）
- 组织架构（资产配置）
- 决策流程（交易规则）
- 风险控制（止损、仓位限制）
- 危机处理预案（行为控制规则）

IPS v2就是你个人投资的"公司章程"。

## 模块详解

### 模块1：投资目标与期限

> **沃伦·巴菲特**："如果你不知道自己要去哪里，任何道路都不会带你到达目的地。"

**目标设定SMART原则**：

| 原则 | 含义 | 示例 |
|------|------|------|
| **S**pecific | 具体 | "退休时有XXX万" vs "积累财富" |
| **M**easurable | 可衡量 | 年化收益率7-10% |
| **A**chievable | 可实现 | 基于历史数据合理 |
| **R**elevant | 相关 | 与人生目标挂钩 |
| **T**ime-bound | 有期限 | 20年后退休 |

**经典案例：耶鲁捐赠基金的目标设定**

大卫·斯文森管理的耶鲁捐赠基金：
- **目标**：永续支持大学运营
- **期限**：无限期（代际传承）
- **收益目标**：实际收益率5%+（扣除通胀和支出后）
- **风险限制**：任何单一年度亏损不超过25%

**启示**：越长的投资期限，越能承受短期波动，越应该配置高风险资产。

### 模块2：风险承受能力

**两个维度的综合评估**：

| 维度 | 评估因素 | 你的情况 |
|------|----------|----------|
| **能力（Ability）** | 收入稳定性、资产规模、负债、投资期限 | __________ |
| **意愿（Willingness）** | 情绪反应、睡眠质量、焦虑程度 | __________ |

> **查理·芒格**："如果你不能在投资亏损20%时保持冷静，你就不应该投资股票。"

**最大回撤承受度测试**：

想象你的投资组合价值100万，问自己：

| 亏损金额 | 你的反应 | 风险等级建议 |
|----------|----------|--------------|
| -10万（-10%） | 完全不在意 | 可承受高风险 |
| -20万（-20%） | 有些焦虑但能接受 | 中等风险 |
| -30万（-30%） | 开始失眠、频繁查看 | 低风险 |
| -40万（-40%） | 无法承受、考虑清仓 | 极低风险 |

**核心规则**：风险承受 = min(能力, 意愿)

### 模块3：资产类别与配置

参见：[[43. 资产类别概览]]

**三大核心资产类别**：

| 资产类别 | 作用 | 长期年化回报 | 波动率 |
|----------|------|--------------|--------|
| **股票** | 增长引擎 | ~10% | ~16% |
| **债券** | 稳定器 | ~5% | ~10% |
| **大宗商品** | 通胀对冲 | ~6% | ~15% |

**经典配置模板**：

| 配置类型 | 股票 | 债券 | 其他 | 适合人群 |
|----------|------|------|------|----------|
| 激进型 | 80% | 15% | 5% | 长期限、高风险承受 |
| 平衡型 | 60% | 30% | 10% | 中等期限、中等风险 |
| 保守型 | 40% | 50% | 10% | 短期限、低风险 |
| 全天候 | 30% | 55% | 15% | 追求稳定、任何周期 |

> **雷·达里奥**："风险平价不是平均分配资金，而是平均分配风险。"

### 模块4：分散化规则

参见：[[44. 多元化与相关性]]

> **哈里·马科维茨**："分散投资是唯一的免费午餐。"
> — 但前提是资产之间相关性足够低

**分散化三层架构**：

| 层次 | 规则 | 你的设定 |
|------|------|----------|
| **同类资产内** | 单只股票不超过X% | ____% |
| **跨资产类别** | 至少持有X种资产类别 | ____种 |
| **跨地域** | 单一国家不超过X% | ____% |

**核心限制规则**：

| 规则类型 | 说明 | 你的设定 |
|----------|------|----------|
| 单只股票最大权重 | 防止个股风险 | ≤ ____% |
| 单一行业最大权重 | 防止行业风险 | ≤ ____% |
| 单一国家最大权重 | 防止国别风险 | ≤ ____% |
| 相关性检查 | 核心持仓相关性 | < ____ |
| 现金最低比例 | 危机应对能力 | ≥ ____% |

**经典案例：长期资本管理公司（LTCM）的教训**

1998年LTCM崩溃：
- 持有大量"低相关"资产
- 危机时相关性飙升至接近1
- 杠杆放大了损失
- **教训**：危机时的相关性与正常时期完全不同

### 模块5：再平衡机制

**再平衡的本质**：强制执行"低买高卖"

```
再平衡逻辑：
资产A涨了 → 比例超标 → 卖出部分A（高卖）
资产B跌了 → 比例不足 → 买入更多B（低买）
```

**再平衡触发条件**：

| 触发类型 | 说明 | 你的设定 |
|----------|------|----------|
| **阈值触发** | 偏离目标比例X%时 | ±____% |
| **时间触发** | 每X个月检查一次 | ____个月 |
| **混合触发** | 以上两者结合 | __________ |

> **约翰·博格**："再平衡不是为了提高收益，而是为了控制风险。"

**再平衡执行规则**：

| 步骤 | 行动 | 检查项 |
|------|------|--------|
| 1 | 计算当前各资产比例 | 是否偏离目标？ |
| 2 | 确定需要调整的金额 | 考虑交易成本 |
| 3 | 优先在新增资金中调整 | 减少卖出操作 |
| 4 | 必要时卖出超配资产 | 考虑税务影响 |
| 5 | 记录再平衡操作 | 更新投资日志 |

**你的再平衡计划**：

### 模块6：行为控制规则

参见：[[45. 行为金融陷阱]]、[[46. 行为金融陷阱]]

> **查理·芒格**："如果我知道我会死在哪里，我就永远不去那个地方。"
> — 避免错误比追求正确更重要

**行为偏误防控清单**：

| 偏误 | 触发场景 | 对抗规则 |
|------|----------|----------|
| **过度自信** | 连续盈利后 | 单笔不超过____% |
| **损失厌恶** | 持仓亏损时 | 止损规则____% |
| **追涨杀跌** | 市场极端波动 | 冷静期____小时 |
| **锚定效应** | 盯着买入成本 | 只看当前价值 |
| **FOMO** | 看到别人赚钱 | 不追涨幅>____% |

**冷静期规则**：

以下情况必须等待24-48小时再决定：

- [ ] 单笔买入超过组合_____%
- [ ] 一周内亏损超过_____%后想加仓
- [ ] 买入近期涨幅超过_____%的资产
- [ ] 想要清仓某只持续亏损的资产
- [ ] 市场大涨或大跌当天

**"当我不交易"清单**：

| 场景 | 正确做法 |
|------|----------|
| 市场大跌时 | 检查IPS，按规则行动 |
| 听到"必涨"消息时 | 等待48小时，独立研究 |
| 情绪激动时 | 关闭交易软件，明天再看 |
| 喝酒/疲劳时 | 绝不交易 |
| 报复性交易冲动时 | 运动或散步冷静 |

### 模块7：审查与更新机制

**定期审查内容**：

| 审查频率 | 内容 | 行动 |
|----------|------|------|
| **每周** | 检查是否偏离配置 | 记录，不轻易行动 |
| **每月** | 更新宏观仪表盘 | 调整观点，不调仓 |
| **每季** | 绩效回顾+行为复盘 | 微调规则 |
| **每年** | 全面IPS审查 | 考虑目标/能力变化 |

**IPS更新触发条件**：

| 条件 | 可能的调整 |
|------|------------|
| 收入大幅变化 | 重新评估风险承受能力 |
| 重大人生事件（结婚/生子/买房） | 调整投资期限和目标 |
| 年龄增长 | 逐步降低风险资产比例 |
| 市场认知提升 | 优化配置和规则 |
| 发现自己违反规则 | 强化行为控制机制 |

## 投资者视角

### 为什么IPS v2比v1更重要？

1. **系统性思维**：不再是零散的规则，而是完整的投资系统

2. **行为保护**：明确的行为控制规则，防止最大的敌人——自己

3. **可执行性**：每个规则都有具体的数字和检查清单

> **彼得·伯恩斯坦**："投资成功的关键不是你知道多少，而是你能多大程度上控制自己的行为。"

## 关键要点

1. **IPS是活文档**：随着认知提升和人生变化不断迭代，v2不是终点

2. **规则 > 情绪**：提前制定规则是唯一可靠的行为保护措施

3. **分散需要深度**：不只是持有更多资产，而是持有低相关资产

4. **再平衡强制低买高卖**：这是普通投资者唯一可靠的"择时"方法

5. **最大的敌人是自己**：行为控制规则和IPS一样重要

## 延伸思考

### IPS的心理学本质

> **丹尼尔·卡尼曼**："人们在情绪激动时做出的决定，几乎总是错误的。"

IPS本质上是一种**预承诺机制（Pre-commitment Device）**：
- 在冷静时制定规则
- 在冲动时被规则约束
- 把"要不要遵守"的决策提前做好

### 为什么大多数人不写IPS？

| 原因 | 真相 |
|------|------|
| "太麻烦了" | 亏损时更麻烦 |
| "我能控制自己" | 研究证明不能 |
| "市场变化太快" | 规则应对变化 |
| "我还在学习" | 学习过程中更需要规则 |

### IPS的终极检验

问自己这个问题：

> **如果明天市场暴跌30%，你的IPS能否告诉你应该做什么？**

如果答案是"否"，你的IPS还不够完善。

**投资者警示**：IPS不是写完就放抽屉里的文件。它应该是你每次想要交易时首先查看的文档。如果你发现自己经常违反IPS，要么修改IPS，要么反思自己的行为——但绝不能假装IPS不存在。
