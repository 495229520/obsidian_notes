# 期权价格的硬约束：界限与单调性

## 核心概念

### 为什么要学"界限"？

**问题**：期权定价模型（如Black-Scholes）很复杂，普通投资者怎么判断一个期权报价是否"合理"？

**答案**：不需要精确计算，只需要知道**硬约束**——期权价格不能低于某个下限，也不能高于某个上限。如果市场报价违反了这些约束，要么是套利机会，要么是你看错了。

> **约翰·赫尔**："在学习复杂的定价模型之前，先理解期权价格的基本界限。这些界限来自无套利原理，是任何合理报价必须满足的条件。"
> — 《期权、期货及其他衍生品》

#### 生活中的例子

**例子1：二手车定价**

你要买一辆二手车：
- **下限**：不可能低于废铁价（假设5000元）——再破的车也值这个价
- **上限**：不可能高于同款新车价（假设20万）——否则直接买新车

如果有人报价3000元，你会怀疑：是骗局还是车有大问题？

**期权类似**：
- **下限**：期权不可能低于"立即行权能赚的钱"（内在价值）
- **上限**：看涨期权不可能贵过股票本身

**例子2：彩票定价**

一张彩票最高奖金100万：
- **下限**：不可能免费送你（成本>0）
- **上限**：不可能卖100万以上（否则买彩票不如直接拿100万）

期权也是一种"彩票"——买方付出权利金，博取未来的收益。

> **查理·芒格**："投资中最重要的是知道什么是不可能的。期权界限告诉你价格的'不可能区间'，这比知道'精确价格'更有用。"

---

### 期权价格的下界

**核心原则**：期权价格不能低于其**内在价值**（如果是美式期权），因为否则可以立即行权获利。

#### 看涨期权下界

**欧式看涨期权下界**：

$$C \geq \max(S_0 - Ke^{-rT}, 0)$$

**美式看涨期权下界**：

$$C \geq \max(S_0 - K, 0)$$

| 符号 | 含义 |
|------|------|
| $C$ | 看涨期权价格 |
| $S_0$ | 当前股价 |
| $K$ | 行权价 |
| $r$ | 无风险利率 |
| $T$ | 到期时间（年） |

**直觉解释**：

看涨期权给你"以K买入股票"的权利。如果现在股价是110，行权价是100：
- 期权至少值10元（立即行权赚10元）
- 如果期权只卖8元，套利者会：买期权(8元) → 立即行权(付100元买股票) → 卖股票(得110元) → 净赚2元

**例子：股票ETF看涨期权**

| 数据 | 数值 |
|------|------|
| 当前ETF价格 | 3.50元 |
| 行权价 | 3.20元 |
| 到期时间 | 3个月 |
| 无风险利率 | 3%/年 |

**计算下界**：
$$C \geq 3.50 - 3.20 \times e^{-0.03 \times 0.25} = 3.50 - 3.176 = 0.324$$

**结论**：这个看涨期权至少值0.324元。如果市场报价0.30元，存在套利机会。

#### 看跌期权下界

**欧式看跌期权下界**：

$$P \geq \max(Ke^{-rT} - S_0, 0)$$

**美式看跌期权下界**：

$$P \geq \max(K - S_0, 0)$$

**直觉解释**：

看跌期权给你"以K卖出股票"的权利。如果现在股价是90，行权价是100：
- 期权至少值10元（立即行权赚10元）
- 如果期权只卖8元，套利者会获利

---

### 期权价格的上界

**核心原则**：期权价格不能超过其"最好情况下的收益"。

#### 看涨期权上界

$$C \leq S_0$$

**直觉**：看涨期权给你买股票的权利，但这个权利不可能比股票本身更值钱。

**极端情况**：行权价K=0的看涨期权，意味着"免费获得股票的权利"，这个权利恰好等于股票价格。

**例子**：
- 股票价格：100元
- 看涨期权报价：105元

**问题**：这合理吗？

**答案**：不合理。花105元买期权，不如直接花100元买股票。期权最多值100元。

#### 看跌期权上界

**欧式看跌期权**：

$$P \leq Ke^{-rT}$$

**美式看跌期权**：

$$P \leq K$$

**直觉**：看跌期权最好的情况是股票跌到0，此时你能以K卖出一文不值的东西，赚K元。所以期权不可能比K（或其现值）更贵。

---

### 期权价格的单调性

**单调性**告诉我们：当某个因素变化时，期权价格如何变化。

#### 行权价的影响

**看涨期权**：行权价越高，期权越便宜

$$K_1 < K_2 \Rightarrow C(K_1) > C(K_2)$$

**直觉**：行权价100的看涨期权（以100元买股票的权利）比行权价120的（以120元买股票的权利）更值钱。

**看跌期权**：行权价越高，期权越贵

$$K_1 < K_2 \Rightarrow P(K_1) < P(K_2)$$

**直觉**：行权价120的看跌期权（以120元卖股票的权利）比行权价100的更值钱。

#### 到期时间的影响

**美式期权**：到期时间越长，期权越贵（或至少不会更便宜）

$$T_1 < T_2 \Rightarrow C(T_1) \leq C(T_2)$$

**直觉**：更长的时间意味着更多的可能性。3个月的期权包含了1个月期权的所有可能，还多了2个月的机会。

**欧式期权**：通常也是如此，但有例外（如高分红股票的看涨期权）。

#### 单调性速查表

| 因素增加 | 看涨期权价格 | 看跌期权价格 |
|----------|--------------|--------------|
| 股价 $S$ ↑ | ↑ | ↓ |
| 行权价 $K$ ↑ | ↓ | ↑ |
| 到期时间 $T$ ↑ | ↑（美式） | ↑（美式） |
| 波动率 $\sigma$ ↑ | ↑ | ↑ |
| 无风险利率 $r$ ↑ | ↑ | ↓ |
| 股息 $D$ ↑ | ↓ | ↑ |

> **记忆口诀**：
> - 看涨喜欢：高股价、低行权价、高波动、高利率、长时间、低股息
> - 看跌喜欢：低股价、高行权价、高波动、低利率、长时间、高股息

## 投资者视角

### 为什么投资者必须理解期权界限？

1. **快速筛选不合理报价**：不需要复杂计算，一眼看出报价是否离谱

2. **理解结构化产品**：银行产品中的期权成分是否定价合理？

3. **避免被"便宜"误导**：看似便宜的期权可能已经接近下界，没有安全边际

4. **套利思维训练**：理解"如果违反界限会发生什么"是金融思维的基础

### 界限检验实战流程

```
拿到一个期权报价 → 计算下界 → 计算上界 → 判断合理性

如果 报价 < 下界 → 要么是套利机会，要么你算错了
如果 报价 > 上界 → 要么是报价错误，要么你看错了
如果 下界 < 报价 < 上界 → 报价在合理范围内
```

### 常见陷阱

| 陷阱 | 说明 | 避免方法 |
|------|------|----------|
| **忽略利率** | 欧式期权下界需要折现 | 记住公式中的 $e^{-rT}$ |
| **混淆欧式/美式** | 美式下界更高（可以提前行权） | 先确认期权类型 |
| **忘记股息** | 高股息股票的看涨期权下界更低 | 检查标的是否分红 |
| **只看下界** | 接近下界的期权可能已经没有投资价值 | 同时考虑时间价值 |

## 关键要点

1. **期权下界 = 内在价值的折现**：期权不可能比"立即行权的收益"更便宜
2. **看涨期权上界 = 股票价格**：买股票的权利不可能比股票本身更贵
3. **看跌期权上界 = 行权价的现值**：卖出权利最多让你赚到行权价
4. **单调性提供直觉**：知道因素变化方向，就知道期权价格变化方向
5. **界限检验是第一道防线**：在复杂定价之前，先用界限排除明显错误

## 延伸思考

### 为什么界限比精确价格更重要？

> **霍华德·马克斯**："在投资中，知道什么是不可能的，往往比知道什么会发生更有价值。期权界限告诉你'不可能的区间'。"

**思考题**：

1. 如果一个期权报价正好等于下界，你应该买入吗？
2. 为什么美式期权的下界比欧式期权高？
3. 在什么情况下，提前行权美式期权是有利的？

### 界限失效的情况

| 情况 | 说明 | 投资者应对 |
|------|------|-----------|
| **市场极端波动** | 买卖价差极大，报价可能暂时违反界限 | 等待市场平静 |
| **流动性枯竭** | 没人愿意卖，报价虚高 | 不要追价 |
| **数据错误** | 行情软件显示错误 | 多方验证 |
| **特殊公司行动** | 拆股、合并等导致参数变化 | 确认调整后的参数 |

**投资者警示**：界限是"无套利"世界的规则。在现实中，交易成本、借券困难、资金限制等摩擦可能让套利无法执行。看到"违反界限"的报价，先检查自己是否算错了。

## 本笔记总结

期权价格的界限是**不依赖任何模型的硬约束**，来自无套利原理：

| 约束类型 | 核心逻辑 |
|----------|----------|
| **下界** | 期权不能比立即行权的收益更便宜 |
| **上界** | 期权不能比最好情况的收益更贵 |
| **单调性** | 因素变化方向决定价格变化方向 |

**一句话总结**：在学习复杂的期权定价模型之前，先掌握这些简单的界限检验——它们是你判断报价合理性的第一道防线。
