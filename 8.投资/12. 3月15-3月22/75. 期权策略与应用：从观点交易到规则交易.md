# 期权策略与应用：从观点交易到规则交易

> **对应培养方案：** 第12周Day 5——策略与应用：从"观点交易"转向"规则交易/对冲交易"
> **前置知识：** [[74. Greeks的风控化表达]]

---

## 核心概念

### 观点交易 vs 规则交易

**问题**：大多数期权亏损者的共同特征是什么？

**答案**：他们在做"观点交易"——"我觉得股票会涨，所以买看涨期权"。这种交易方式把期权当成了高杠杆赌博工具。

> **纳西姆·塔勒布**："业余交易者用期权表达观点，专业交易者用期权管理风险。前者在猜方向，后者在控制敞口。"
> — 《动态对冲》

#### 两种交易方式对比

| 维度 | 观点交易 | 规则交易 |
|------|----------|----------|
| **出发点** | "我觉得会涨/跌" | "我要保护/增强收益" |
| **决策依据** | 预测 | 规则 |
| **进场条件** | 模糊 | 明确的触发条件 |
| **出场条件** | "感觉差不多了" | 预设的止损/止盈 |
| **最大亏损** | "没想过" | 提前计算 |
| **成功率** | 依赖预测准确性 | 依赖纪律执行 |

#### 生活中的例子

**例子1：买保险 vs 买彩票**

| 行为 | 心态 | 类比 |
|------|------|------|
| 买彩票 | "我觉得会中奖" | 观点交易：买虚值期权赌大涨 |
| 买保险 | "我不想承受意外损失" | 规则交易：买保护性看跌 |

**例子2：安全带 vs 赛车**

- **观点交易**：把车当赛车开，期望一路绿灯
- **规则交易**：系好安全带，不管路况如何都有保护

> **霍华德·马克斯**："投资的目标不是'做对'，而是'在错的时候不会死'。期权策略应该是后者的工具。"

### 三大核心策略

#### 策略一：保护性看跌（Protective Put）

**定义**：持有股票的同时买入看跌期权，为下跌提供保护。

**结构**：
- 持有股票（多头）
- 买入看跌期权（支付权利金）

**收益图**：

```
收益
  |        /
  |      /
  |    /
  |--*-------------- 股价
  |  （权利金成本）
  |
```

**适用场景**：

| 场景 | 说明 |
|------|------|
| 持有盈利股票 | 想保护利润，但不想卖出 |
| 重大事件前 | 财报、政策发布前的保护 |
| 长期持有 | 看好长期，但担心短期波动 |

**成本与收益**：

| 项目 | 内容 |
|------|------|
| 最大亏损 | 权利金 +（股价 - 行权价）|
| 最大收益 | 无限（股票上涨） |
| 盈亏平衡点 | 股价 + 权利金 |

**经典案例：2020年疫情初期**

- **背景**：2020年2月，某投资者持有10万元ETF
- **操作**：买入3个月后到期的平价看跌期权，权利金3000元
- **结果**：3月大跌20%，看跌期权盈利17000元，抵消了大部分亏损
- **教训**：保护成本3%，换来了"安心持有"

> **约翰·赫尔**："保护性看跌是最直观的期权策略——它就是给你的股票买一份保险。"

---

#### 策略二：备兑看涨（Covered Call）

**定义**：持有股票的同时卖出看涨期权，收取权利金作为"增强收益"。

**结构**：
- 持有股票（多头）
- 卖出看涨期权（收取权利金）

**收益图**：

```
收益
  |    ____*（封顶）
  |   /
  |  /
  | /
  +-------------- 股价
  |（有权利金垫底）
```

**适用场景**：

| 场景 | 说明 |
|------|------|
| 震荡市 | 股票不大涨不大跌 |
| 高估时 | 股票已经涨了一些，愿意在更高价卖出 |
| 增加收入 | 长期持股，想增加现金流 |

**成本与收益**：

| 项目 | 内容 |
|------|------|
| 最大收益 | 权利金 +（行权价 - 买入价）|
| 最大亏损 | 买入价 - 权利金（股票跌到0） |
| 盈亏平衡点 | 买入价 - 权利金 |

**经典案例：长期持有ETF的增强策略**

- **背景**：某投资者长期持有50ETF，成本3.0元
- **操作**：每月卖出虚值5%的看涨期权，平均收取0.03元/份
- **结果**：年化增加约12%的收益（0.03×12/3.0）
- **风险**：如果ETF大涨超过3.15元，收益被封顶

> **沃伦·巴菲特**（伯克希尔曾使用类似策略）："如果你愿意以某个价格卖出股票，为什么不收取一笔费用来承诺这样做呢？"

**备兑看涨的隐藏风险**：

| 风险 | 说明 |
|------|------|
| **踏空风险** | 股票大涨，你只能以行权价卖出 |
| **下跌保护有限** | 权利金只能抵消小幅下跌 |
| **机会成本** | 放弃了大涨的可能性 |

---

#### 策略三：垂直价差（Vertical Spread）

**定义**：同时买入和卖出相同到期日、不同行权价的期权，限制最大收益和最大亏损。

**两种类型**：

| 类型 | 构成 | 观点 |
|------|------|------|
| **牛市价差** | 买低K看涨 + 卖高K看涨 | 温和看涨 |
| **熊市价差** | 买高K看跌 + 卖低K看跌 | 温和看跌 |

**牛市看涨价差收益图**：

```
收益
  |    *____（封顶）
  |   /
  |  /
  +-*-------------- 股价
  |（有限亏损）
```

**适用场景**：

| 场景 | 说明 |
|------|------|
| 温和看涨/看跌 | 预期有方向但幅度有限 |
| 降低成本 | 买期权太贵，用卖期权抵消部分成本 |
| 控制风险 | 想要有限亏损、有限收益 |

**牛市看涨价差示例**：

| 项目 | 内容 |
|------|------|
| 买入 | 行权价100的看涨，权利金5元 |
| 卖出 | 行权价110的看涨，权利金2元 |
| 净成本 | 5 - 2 = 3元 |
| 最大亏损 | 3元（股价≤100） |
| 最大收益 | 10 - 3 = 7元（股价≥110） |
| 盈亏平衡 | 103元 |

> **谢尔登·纳坦恩伯格**："价差策略是把'无限'变成'有限'——有限的成本，有限的收益，有限的风险。对于大多数投资者，这是更合理的选择。"

## 投资者视角

### 策略选择框架

**核心问题**：你想要什么？

| 目标 | 推荐策略 | 代价 |
|------|----------|------|
| 保护下跌 | 保护性看跌 | 支付权利金 |
| 增加收入 | 备兑看涨 | 放弃大涨收益 |
| 降低成本看涨 | 牛市价差 | 收益封顶 |
| 降低成本看跌 | 熊市价差 | 收益封顶 |
| 赌大涨大跌 | 买入跨式 | 高权利金成本 |
| 赚时间价值 | 卖出跨式 | 无限风险 |


### 从"我觉得"到"规则触发"

**错误示范**：

> "我觉得这只股票会涨，买个看涨期权。"

**正确示范**：

> "根据规则，当股票RSI低于30且我长期看好时，我会买入保护性看跌来建仓。最大亏损控制在总资金的2%，如果股价在1个月内不反弹，平仓止损。"

## 策略实战案例

### 案例1：保护性看跌的规则化

**背景**：你持有10万元的沪深300ETF（代码510300），担心短期下跌但长期看好。

**规则设定**：

| 规则项 | 设定 |
|--------|------|
| **触发条件** | 每次盈利超过10%时，买入保护 |
| **期权选择** | 平价或虚值5%的看跌期权 |
| **到期时间** | 1-3个月 |
| **权利金预算** | 不超过持仓市值的2% |
| **滚仓规则** | 到期前1周，如仍需保护则滚仓 |
| **退出条件** | 股价跌破行权价5%时，考虑平仓兑现保护 |

**执行记录**：

| 日期 | ETF价格 | 操作 | 权利金 | 结果 |
|------|---------|------|--------|------|
| 3月1日 | 4.20 | 买入4.20看跌 | 800元 | |
| 3月15日 | 3.80 | 期权盈利 | +3200元 | 抵消部分亏损 |

### 案例2：备兑看涨的规则化

**背景**：你长期持有某银行股，股价30元，愿意在35元卖出。

**规则设定**：

| 规则项 | 设定 |
|--------|------|
| **触发条件** | 股价接近前高或横盘超过1个月 |
| **期权选择** | 虚值10-15%的看涨期权 |
| **到期时间** | 1个月 |
| **滚仓规则** | 到期时未被行权，继续卖出下月期权 |
| **买回条件** | 如果股价大涨导致期权翻倍，买回止损 |
| **底线** | 如果被行权，接受在35元卖出 |

### 案例3：牛市价差的规则化

**背景**：你温和看好某科技股，但不想付太多权利金。

**规则设定**：

| 规则项 | 设定 |
|--------|------|
| **触发条件** | 技术面确认上涨趋势 |
| **头寸构建** | 买入平价看涨，卖出虚值10%看涨 |
| **到期时间** | 2个月 |
| **最大亏损** | 净权利金支出 |
| **止盈条件** | 价差价值达到最大收益的70%时平仓 |
| **止损条件** | 价差价值跌至50%时平仓 |

## 关键要点

1. **规则交易优于观点交易**：预设进出场条件，避免情绪决策
2. **保护性看跌 = 给股票买保险**：付出权利金，换取下跌保护
3. **备兑看涨 = 卖出上涨潜力**：收取权利金，但收益封顶
4. **价差策略 = 有限风险有限收益**：用卖出期权抵消买入成本
5. **每个策略都需要策略卡片**：明确进出场规则、最大亏损、时间管理

## 延伸思考

### 策略的"隐性成本"

> **查理·芒格**："世界上没有免费的午餐。每一个期权策略都有看得见和看不见的成本。"

| 策略 | 显性成本 | 隐性成本 |
|------|----------|----------|
| 保护性看跌 | 权利金 | 放弃了"不买保护反而没事"的可能 |
| 备兑看涨 | 无 | 放弃了大涨的收益 |
| 牛市价差 | 净权利金 | 收益封顶 |

### 策略失效的警示

| 失效情况 | 原因 | 教训 |
|----------|------|------|
| **保护性看跌到期作废** | 股票一直涨 | 保护是保险，不是投资 |
| **备兑被行权后股票继续大涨** | 低估了上涨空间 | 只在愿意卖出的价格卖 |
| **价差亏损最大金额** | 方向判断错误 | 最大亏损要能承受 |
| **流动性陷阱** | 期权买卖价差太大 | 只做流动性好的期权 |

**投资者警示**：期权策略是工具，不是圣杯。工具用对了是保护，用错了是自伤。核心原则：**在使用任何策略之前，确保你完全理解最坏情况。**

## 本笔记总结

期权策略的核心转变：

| 从 | 到 |
|----|----|
| "我觉得会涨" | "如果涨，我赚多少；如果跌，我亏多少" |
| 模糊的进出场 | 规则化的触发条件 |
| 追求暴利 | 控制风险 |
| 单一头寸 | 组合策略 |

**一句话总结**：期权策略不是用来"赌对方向"的，而是用来"在不确定性中保护自己"的。规则交易的核心是：**进场前就知道出场条件，开仓前就算好最大亏损。**
