# 看跌-看涨期权平价：把期权与远期连起来

> **对应培养方案：** 第12周Day 2——看跌-看涨期权平价：把期权与远期连起来
> **前置知识：** [[71. 期权价格的硬约束：界限与单调性]]

---

## 核心概念

### 什么是看跌-看涨期权平价？

**定义**：看跌-看涨期权平价（Put-Call Parity）是描述欧式看涨期权、看跌期权、标的资产和无风险债券之间关系的等式。它告诉我们：这四种资产可以相互复制。

> **约翰·赫尔**："看跌-看涨期权平价是期权定价最基础、最重要的关系。它不依赖任何定价模型，只依赖无套利原理。如果平价关系被打破，市场就存在'免费午餐'。"
> — 《期权、期货及其他衍生品》

#### 公式

$$C + Ke^{-rT} = P + S_0$$

或者写成：

$$C - P = S_0 - Ke^{-rT}$$

| 符号 | 含义 |
|------|------|
| $C$ | 欧式看涨期权价格 |
| $P$ | 欧式看跌期权价格 |
| $S_0$ | 当前股票价格 |
| $K$ | 行权价（看涨和看跌相同） |
| $r$ | 无风险利率 |
| $T$ | 到期时间 |

#### 生活中的例子

**例子1：租房 vs 买房的等价关系**

想象你要在一个城市住一年：

**方案A：直接买房**
- 花100万买房，一年后房子还是你的

**方案B：租房 + 买入"买房权"**
- 花5万租一年（相当于买看涨期权的权利金）
- 手里留95万存银行
- 一年后如果房价涨了，用存款买房

这两种方案在财务上是等价的——这就是平价关系的直觉。

**例子2：保险的等价视角**

你持有一辆价值20万的车：

**方案A：车 + 保险**
- 持有车（价值20万）
- 买一份全险（保费5000元）——相当于买看跌期权

**方案B：存钱 + 买车权**
- 把20万存银行
- 买一份"以20万买这辆车的权利"——相当于买看涨期权

无论车是否出事，两种方案的最终财富是一样的。

> **费希尔·布莱克**："平价关系告诉我们，看涨期权和看跌期权不是独立的产品，而是同一枚硬币的两面。"

---

### 平价关系的直觉理解

**核心思想**：用不同的资产组合，可以创造出相同的收益结构。

#### 组合A：看涨期权 + 现金

| 资产 | 到期时股价 $S_T > K$ | 到期时股价 $S_T < K$ |
|------|---------------------|---------------------|
| 看涨期权 | $S_T - K$ | 0 |
| 现金 $Ke^{-rT}$（到期变成K） | $K$ | $K$ |
| **组合A总价值** | $S_T$ | $K$ |

#### 组合B：看跌期权 + 股票

| 资产 | 到期时股价 $S_T > K$ | 到期时股价 $S_T < K$ |
|------|---------------------|---------------------|
| 看跌期权 | 0 | $K - S_T$ |
| 股票 | $S_T$ | $S_T$ |
| **组合B总价值** | $S_T$ | $K$ |

**结论**：组合A和组合B在任何情况下的价值都相同！

$$\text{组合A} = \text{组合B}$$
$$C + Ke^{-rT} = P + S_0$$

---

### 合成远期：平价的另一种理解

**从平价公式变形**：

$$C - P = S_0 - Ke^{-rT}$$

**右边**：$S_0 - Ke^{-rT}$ 是远期合约的价值（买入远期的价值）

**左边**：$C - P$ 是"买看涨 + 卖看跌"的组合

**含义**："买看涨 + 卖看跌"= 合成远期多头

#### 合成远期的收益图

```
买入看涨期权        卖出看跌期权        合成远期（=远期多头）
      /                   |___              /
     /                        \            /
____/                          \___   ____/
                                    =
```

**用途**：
- 如果远期市场流动性差，可以用期权合成
- 检验期权定价是否合理
- 理解期权与远期的内在联系

> **约翰·赫尔**："理解了平价关系，你就理解了期权定价的一半。它把看涨、看跌、远期三个看似不同的产品统一起来。"

---

### 平价关系的应用：套利检验

**如果平价关系被打破，就存在套利机会。**

#### 情况1：$C + Ke^{-rT} > P + S_0$（看涨期权太贵）

**套利策略**：
1. 卖出看涨期权（收入C）
2. 买入看跌期权（支出P）
3. 买入股票（支出$S_0$）
4. 借入现金$Ke^{-rT}$（到期还K）

**现金流分析**：

| 时点 | 现金流 |
|------|--------|
| 现在 | $+C - P - S_0 + Ke^{-rT} > 0$（净收入） |
| 到期（$S_T > K$） | 股票被call走得K，还债K，净值=0 |
| 到期（$S_T < K$） | 用put以K卖股票，还债K，净值=0 |

**结论**：无论股价涨跌，到期净值都是0，但开始时净赚了$(C + Ke^{-rT}) - (P + S_0) > 0$。

#### 情况2：$C + Ke^{-rT} < P + S_0$（看跌期权太贵）

**套利策略**：反向操作
1. 买入看涨期权
2. 卖出看跌期权
3. 卖空股票
4. 存入现金$Ke^{-rT}$

---

## 投资者视角

### 为什么投资者必须理解平价关系？

1. **检验报价合理性**：如果看涨和看跌价格不满足平价，要么是套利机会，要么是你算错了

2. **理解结构化产品**：很多保本产品的定价基于平价关系

3. **灵活构建头寸**：买不到某个期权？用其他资产合成

4. **风险管理**：理解"保护性看跌 = 持有股票 + 买看跌 = 买看涨 + 存钱"

### 平价关系的实际应用

#### 应用1：检验期权报价

**例子**：某股票当前价格50元

| 数据 | 数值 |
|------|------|
| 股价 $S_0$ | 50元 |
| 行权价 $K$ | 50元 |
| 到期时间 $T$ | 3个月 |
| 无风险利率 $r$ | 4%/年 |
| 看涨期权报价 $C$ | 3.5元 |
| 看跌期权报价 $P$ | 3.0元 |

**验证平价**：

$$C + Ke^{-rT} = 3.5 + 50 \times e^{-0.04 \times 0.25} = 3.5 + 49.50 = 53.0$$

$$P + S_0 = 3.0 + 50 = 53.0$$

**结论**：平价成立，报价合理。

#### 应用2：合成保护性看跌

**问题**：你持有股票，想保护下跌风险，但看跌期权太贵或流动性差。

**解决方案**：用平价关系合成

$$\text{股票} + \text{看跌} = \text{看涨} + \text{现金}$$

**操作**：
1. 卖出股票（得$S_0$）
2. 买入看涨期权（支出C）
3. 剩余资金存银行

**效果**：与"持有股票+买看跌"完全一样！

#### 应用3：理解"领口策略"

**领口策略（Collar）**：持有股票 + 买入看跌 + 卖出看涨

**用平价理解**：

$$\text{股票} + P - C = \text{股票} + P - C$$

由平价：$C - P = S_0 - Ke^{-rT}$

**如果用同一行权价**：$C = P$（当$K = S_0 e^{rT}$，即平价期权）

**零成本领口**：买看跌的钱正好被卖看涨赚回来！

## 关键要点

1. **平价公式**：$C + Ke^{-rT} = P + S_0$，这是期权定价的基石
2. **无模型依赖**：平价关系只依赖无套利原理，不需要Black-Scholes等模型
3. **合成头寸**：任何一种资产都可以用其他三种合成
4. **套利检验**：平价关系是检验期权报价合理性的第一工具
5. **看涨与看跌的关系**：它们是同一枚硬币的两面，通过平价相连

### 平价关系速查

| 公式形式 | 含义 |
|----------|------|
| $C + Ke^{-rT} = P + S_0$ | 看涨+现金 = 看跌+股票 |
| $C - P = S_0 - Ke^{-rT}$ | 买看涨卖看跌 = 合成远期 |
| $C = P + S_0 - Ke^{-rT}$ | 用看跌和股票合成看涨 |
| $P = C - S_0 + Ke^{-rT}$ | 用看涨和现金合成看跌 |

## 延伸思考

### 平价关系的哲学意义

> **费希尔·布莱克**："金融市场的奇妙之处在于，看似不同的产品，在本质上可能是一样的。平价关系揭示了这种'本质相同'。"

**思考题**：

1. 为什么美式期权的平价关系不严格成立？
2. 如果市场上只有看涨期权，你如何获得看跌期权的保护？
3. 平价关系在极端市场（如2008年金融危机）中还成立吗？

### 平价关系失效的情况

| 情况 | 原因 | 影响 |
|------|------|------|
| **高借券成本** | 卖空股票很贵 | 合成看跌变贵 |
| **高交易成本** | 套利利润被侵蚀 | 小幅偏离无法套利 |
| **股息不确定** | 影响远期价值 | 平价需要调整 |
| **美式期权** | 提前行权可能性 | 平价变成不等式 |
| **流动性差** | 买卖价差大 | 实际套利困难 |

**投资者警示**：平价关系是"理论上"的等式。在实际市场中，总会有小幅偏离。只有当偏离大到能覆盖所有交易成本时，才是真正的套利机会。

## 本笔记总结

看跌-看涨期权平价是期权定价的**基石**：

| 核心公式 | $C + Ke^{-rT} = P + S_0$ |
|----------|--------------------------|
| **本质** | 两种不同的资产组合，产生相同的收益 |
| **应用** | 检验报价、合成头寸、理解产品结构 |
| **依赖** | 只依赖无套利，不依赖任何定价模型 |

**一句话总结**：平价关系告诉我们，看涨期权、看跌期权、股票、现金——这四种资产可以相互转化。理解了这个关系，你就掌握了期权定价的一半。
