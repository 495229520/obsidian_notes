## 选项 A：对冲计划（推荐）

### 什么是对冲计划？

**定义**：一份系统化的文档，明确你将如何使用期权保护投资组合免受严重回撤，包括触发条件、成本预算、执行规则和失效模式。

> **霍华德·马克斯**："风险管理不是在危机发生后才做的事，而是在危机发生前就写好规则。"
> — 《周期》

#### 生活中的例子

**例子1：汽车保险**

你的对冲计划就像汽车保险：
- **被保护资产**：你的车（价值20万）
- **保护区间**：出事故时赔偿（免赔额1000元以上）
- **成本**：每年保费3000元（1.5%）
- **规则**：什么情况下报险，什么情况下自己修

投资组合对冲也一样——你需要提前决定：保护什么、保护多少、花多少钱、什么时候启动。

**例子2：降落伞**

跳伞运动员不会在跳出飞机后才检查降落伞。对冲计划就是你的"金融降落伞"——必须在市场平静时准备好，而不是恐慌时临时找。

## 选项 B：结构化产品分解

### 什么是结构化产品分解？

**定义**：把银行或券商销售的"结构化产品"拆解成基础组件（通常是债券+期权），理解其真实风险收益特征。

> **查理·芒格**："如果你不理解一个产品的结构，那就意味着风险被隐藏了——而隐藏的风险通常由你承担。"
> — 《穷查理宝典》

#### 生活中的例子

**例子：保本理财产品**

银行说："这个产品保本，还能参与股市上涨！"

听起来很美好？让我们拆解：
- **债券部分**：95元买入零息债券，到期还本100元
- **期权部分**：5元买入看涨期权
- **结果**：最差拿回100元（保本），最好拿回100元+期权收益

**真相**：你放弃了95元的利息收入（约3-4%/年），换取了5元的期权博弈机会。银行的"保本"是用你本该得到的利息买的。

### 结构化产品分解示例

**产品**：某银行"稳健增利"结构化存款
**期限**：1年
**保本比例**：100%
**挂钩标的**：沪深300指数
**收益规则**：
- 若到期时指数涨幅 ≤ 5%：收益 = 1.5%
- 若涨幅 > 5% 且 ≤ 20%：收益 = 1.5% + (涨幅-5%) × 50%
- 若涨幅 > 20%：收益 = 1.5% + 7.5% = 9%（封顶）

**拆解**：

| 组件 | 分析 |
|------|------|
| 债券 | 97.09万（3%贴现） |
| 看涨期权 | 执行价=现价×105%，参与率50%，敲出价=现价×120% |
| 银行费用 | 约1.5万（估计） |

**评价**：
- ✅ 本金安全
- ❌ 涨幅超20%被封顶（敲出）
- ❌ 参与率只有50%
- ❌ 费用不透明

**结论**：适合极度保守、不愿自己操作的投资者。理解结构后，我选择（自己复制/购买/放弃）。

## 选项 C：期权策略备忘录

### 什么是期权策略备忘录？

**定义**：一份规则化的文档，明确你将在什么条件下使用什么期权策略，以及每种策略的进入规则、退出规则和风险控制。

> **彼得·林奇**："投资成功的关键不是预测市场，而是有一套规则并严格执行。"

#### 生活中的例子

**例子：健身计划**

一个好的健身计划不是"我要多运动"，而是：
- 周一、三、五：力量训练（具体动作、组数、重量）
- 周二、四：有氧30分钟
- 周日：休息
- 如果生病：休息直到恢复
- 如果出差：用酒店健身房做简化版

期权策略备忘录也一样——不是"我觉得可以做"，而是明确的规则。

### 期权策略备忘录模板

---

#### 策略1：备兑看涨期权（Covered Call）

**策略描述**：
持有股票/ETF的同时，卖出对应的看涨期权，收取权利金。

**适用场景**：
| 条件 | 要求 |
|------|------|
| 市场观点 | 中性偏多，不预期大涨 |
| 波动率环境 | 隐含波动率 > 历史波动率 |
| 持仓状态 | 已持有标的，愿意在高位卖出 |

**进入规则**：

| 规则 | 你的参数 |
|------|----------|
| 标的条件 | 只对 ______ 做备兑 |
| 执行价选择 | 选择 Delta = ______ 的执行价（通常0.2-0.3） |
| 到期日选择 | 选择 ______ 天后到期的期权 |
| 权利金要求 | 至少收取股价的 ______% |
| 隐含波动率 | IV > ______ 时才卖 |

**退出规则**：

| 情况 | 行动 |
|------|------|
| 期权价值跌至初始的20% | 提前平仓，锁定利润 |
| 股价逼近执行价 | （滚动/接受行权/买回期权） |
| 突发重大利好 | 考虑买回期权保留上涨空间 |
| 到期 | 让期权自然到期或被行权 |

**风险控制**：

| 风险 | 控制措施 |
|------|----------|
| 股价大涨错过收益 | 只卖 ______% 仓位的备兑 |
| 股价大跌 | 备兑不提供下行保护，需配合止损 |
| 被提前行权 | 了解美式期权规则，接受被行权可能 |

**备兑收益计算**：

| 项目 | 数值 |
|------|------|
| 持股成本 | ¥______ |
| 卖出执行价 | ¥______ |
| 收取权利金 | ¥______ |
| 最大收益 | (执行价 - 成本 + 权利金) = ¥______ |
| 盈亏平衡点 | (成本 - 权利金) = ¥______ |

---

#### 策略2：保护性看跌期权（Protective Put）

**策略描述**：
持有股票/ETF的同时，买入对应的看跌期权作为保险。

**适用场景**：
| 条件 | 要求 |
|------|------|
| 市场观点 | 长期看多，但担心短期回调 |
| 波动率环境 | 隐含波动率相对较低（VIX < 20） |
| 持仓状态 | 重要持仓，不愿止损卖出 |

**进入规则**：

| 规则 | 你的参数 |
|------|----------|
| 保护比例 | 保护 ______% 的持仓 |
| 执行价选择 | 选择比现价低 ______% 的执行价 |
| 到期日选择 | 选择 ______ 个月后到期 |
| 成本上限 | 每次保护成本不超过持仓的 ______% |
| VIX 阈值 | 只在 VIX < ______ 时买入 |

**退出规则**：

| 情况 | 行动 |
|------|------|
| 股价大跌触及执行价 | （行权卖出/卖出期权+继续持股） |
| 股价上涨，期权变废纸 | 接受保险成本，不追悔 |
| 到期前股价回升 | 期权失去价值，考虑是否续保 |

**成本效益分析**：

| 项目 | 计算 |
|------|------|
| 年化保护成本 | ______ % |
| 保护的回撤范围 | ______ % 到 ______ % |
| 成本合理性 | 年化成本 < 预期回撤×概率×损失？ |

---

#### 策略3：垂直价差（Vertical Spread）

**策略描述**：
同时买入和卖出不同执行价的同类期权（看涨价差或看跌价差）。

**看涨价差（Bull Call Spread）**：
- 买入低执行价看涨期权
- 卖出高执行价看涨期权
- 适用：温和看涨

**看跌价差（Bear Put Spread）**：
- 买入高执行价看跌期权
- 卖出低执行价看跌期权
- 适用：温和看跌或对冲

**进入规则**：

| 规则 | 你的参数 |
|------|----------|
| 价差宽度 | 执行价相差 ______ 档 |
| 最大风险 | 不超过 ¥______ |
| 盈亏比要求 | 最大收益 / 最大亏损 > ______ |
| 到期选择 | ______ 天后到期 |

**退出规则**：

| 情况 | 行动 |
|------|------|
| 达到最大收益的50% | 考虑提前平仓 |
| 接近到期但未盈利 | （持有到期/止损平仓） |
| 标的大幅突破预期 | 调整策略或认亏 |

---

#### 策略选择决策树

```
你对标的的看法是什么？
│
├── 看涨
│   ├── 强烈看涨 → 买入看涨期权
│   └── 温和看涨 → 看涨价差 或 备兑看涨
│
├── 看跌
│   ├── 强烈看跌 → 买入看跌期权
│   └── 温和看跌 → 看跌价差
│
├── 中性
│   ├── 预期波动小 → 卖出跨式
│   └── 预期波动大 → 买入跨式
│
└── 不确定但想保护
    └── 保护性看跌 或 领口策略
```

| 禁止事项 | 原因 |
|----------|------|

## 关键要点

1. **毕业设计是规则化的体现**：把"我觉得"变成"规则规定"
2. **三个选项各有侧重**：对冲保护、产品理解、策略系统
3. **失效模式比成功路径更重要**：提前想好什么会出错
4. **可执行性是核心**：写出来后能照着做，不需要临时判断
5. **这是起点不是终点**：随着经验积累持续迭代

## 延伸思考

### 从学习者到实践者

> **彼得·德鲁克**："计划只是意图，执行才是结果。"

完成毕业设计后，你将从"学习投资知识"转变为"使用投资工具"。这个转变的关键不是知道更多，而是能够在压力下按规则行动。

### 持续迭代

| 迭代周期 | 内容 |
|----------|------|
| 每月 | 回顾执行情况，记录偏离 |
| 每季度 | 评估策略效果，调整参数 |
| 每年 | 大修框架，纳入新学到的知识 |

**投资者警示**：毕业设计不是考试，没有标准答案。最好的设计是你能坚持执行的设计，而不是理论上最优的设计。
