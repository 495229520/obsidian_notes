## 核心概念

### 12周学习的终点：从"学习"到"执行"

你已经走完了12周的金融学习之旅。但学习的目的不是知道更多，而是**做得更好**。

> **查理·芒格**："知道自己不知道什么比知道很多更有价值。但知道自己应该做什么，并且真正去做，才是最有价值的。"
> — 《穷查理宝典》

**本笔记的目标**：把12周学到的一切，压缩成**10条可执行的要点**——能直接放进你的IPS和风险报告里，成为你投资决策的"宪法"。

#### 生活中的例子

**例子1：驾照考试 vs 真正开车**

驾校学了很多：交规、灯光、倒库、坡起...
但真正开车时，你需要的是：
- 几条核心规则（红灯停、绿灯行、保持车距）
- 几个关键习惯（上车系安全带、变道看后视镜）
- 几个应急预案（刹车失灵怎么办）

投资也一样：12周学了很多概念，但最终需要**压缩成几条核心规则**，才能真正执行。

**例子2：菜谱 vs 厨艺**

学做菜可以看100道菜谱，但真正成为好厨师需要：
- 掌握几个核心技法（火候、调味、刀工）
- 形成肌肉记忆
- 知道什么时候该调整

投资知识需要从"菜谱"变成"厨艺"——内化成你的决策习惯。

> **沃伦·巴菲特**："投资很简单，但不容易。规则很简单——低买高卖、分散风险、长期持有。困难的是真正做到。"

## 10条可执行要点

### 这10条规则的设计原则

> **霍华德·马克斯**："风险管理不是避免风险，而是知道自己承担了多少风险，并确保它在可承受范围内。"

每条规则都必须满足：
1. **可检验**：能用数字衡量
2. **可执行**：有明确的触发条件和行动
3. **可坚持**：在情绪波动时仍能遵守

---

### 要点1：先建立防线，再追求收益

**规则**：
```
投资前必须完成：
□ 应急基金 = 6个月生活费（流动性资产）
□ 高息债务清零（年利率>5%的债务）
□ 基础保险到位（医疗、意外）
```

**检验标准**：
| 项目 | 达标标准 | 你的状态 |
|------|----------|----------|
| 应急基金 | ≥6个月支出 | □达标 □未达标 |
| 高息债务 | 0 | □达标 □未达标 |
| 保险覆盖 | 医疗+意外 | □达标 □未达标 |

**来源**：[[2. 应急基金与流动性计划]] | [[1.债务、利息和无风险回报]]

> **彼得·林奇**："在股市里亏掉的钱，大部分是那些本不该用来投资的钱。"

---

### 要点2：永远知道自己在赌什么

**规则**：
```
每笔投资前必须回答：
1. 我在赌什么？（方向/波动/时间/利率/汇率）
2. 如果我错了，最多亏多少？
3. 什么情况下我会止损？
```

**执行模板**：

| 投资 | 在赌什么 | 最大亏损 | 止损条件 |
|------|----------|----------|----------|
| [填写] | | | |

**来源**：[[40. 希腊字母]] | [[74. Greeks 的风控化表达]]

> **纳西姆·塔勒布**："如果你不知道自己在赌什么，你就是赌桌上的傻瓜。"

---

### 要点3：单一持仓永远不超过10%

**规则**：
```
单一股票/债券/基金 ≤ 组合总值的10%
单一行业 ≤ 组合总值的25%
例外：宽基指数ETF（如沪深300）可以超过10%
```

**检验方法**：

| 持仓 | 占比 | 是否超限 |
|------|------|----------|
| [最大持仓] | ___% | □是 □否 |
| [第二大持仓] | ___% | □是 □否 |
| [最大行业] | ___% | □是 □否 |

**触发行动**：超过限制 → 下个交易日减仓至限制内

**来源**：[[29. 仓位规模与集中风险]] | [[6. 投资政策声明 (IPS)]]

> **霍华德·马克斯**："分散投资是免费的午餐。集中投资赚大钱的故事背后，是无数亏光本金的沉默案例。"

---

### 要点4：再平衡用规则触发，不用感觉

**规则**：
```
触发条件（满足任一即再平衡）：
□ 任一资产类别偏离目标配置 ≥5个百分点
□ 距上次再平衡 ≥6个月

再平衡方法：
1. 优先用新增资金买入低配资产
2. 不足时卖出高配资产
```

**执行示例**：

| 资产类别 | 目标配置 | 当前配置 | 偏离 | 行动 |
|----------|----------|----------|------|------|
| 股票 | 60% | 68% | +8% | 减持 |
| 债券 | 30% | 24% | -6% | 增持 |
| 现金 | 10% | 8% | -2% | 不动 |

**来源**：[[48. 投资组合再平衡策略]] | [[26. 核心投资组合构建]]

> **约翰·博格**："再平衡不是预测市场，而是维持你预定的风险水平。"

### 要点6：看财报只看三个问题

**规则**：
```
每次读财报，只回答：
1. 赚的钱是真钱吗？（经营现金流 vs 净利润）
2. 能还得起债吗？（利息保障倍数 ≥3）
3. 增长靠什么？（收入增长 vs 利润率提升）
```

**速查表**：

| 问题 | 指标 | 健康标准 | 危险信号 |
|------|------|----------|----------|
| 真金白银 | 经营现金流/净利润 | ≥1 | <0.5 |
| 偿债能力 | 利息保障倍数 | ≥3 | <1.5 |
| 增长质量 | 收入增长率 | 稳定增长 | 收入降利润涨 |

**来源**：[[22. 损益表与现金流量表]] | [[23. 资产负债表与杠杆风险]]

> **沃伦·巴菲特**："会计是商业的语言。如果你不能读懂财报，你就不知道公司在做什么。"

---

### 要点7：估值用区间，不用精确数字

**规则**：
```
估值检验（不需要精确计算DCF）：
1. 当前市盈率 vs 历史区间（5年/10年）
2. 当前市盈率 vs 同行业对比
3. "如果利率+1%，估值会怎样？"
```

**执行模板**：

| 标的 | 当前PE | 5年PE区间 | 行业中位数PE | 利率敏感度 |
|------|--------|-----------|--------------|------------|
| | | - | | □高 □中 □低 |

**估值判断**：
- 低于历史25%分位 → 可能低估
- 高于历史75%分位 → 可能高估
- 处于中间区间 → 合理

**来源**：[[25. 估值基础知识]]

> **本杰明·格雷厄姆**："精确的错误不如模糊的正确。估值的目的是找到安全边际，不是算出精确价格。"

### 要点8：期权/衍生品只用于对冲，不用于投机

衍生品使用限制：
□ 只在有明确对冲需求时使用
□ 名义敞口 ≤ 被对冲资产价值
□ 入场前必须填写策略卡片（目标、最大亏损、止损规则）

|------|------|
| 对冲什么风险 | |
| 使用什么工具 | |
| 最大亏损 | |
| 止损规则 | |
| 到期/平仓时间 | |

**来源**：[[74. Greeks 的风控化表达]] | [[62. 对冲备忘录]]

> **约翰·赫尔**："衍生品是风险管理工具。把它当成投机工具，就像用手术刀削苹果——可以做到，但很危险。"

### 要点9：压力测试每季度做一次

**规则**：
```
每季度末，对投资组合做以下压力测试：
1. 股票下跌25%
2. 利率上升1.5%
3. 汇率变动10%

问自己：
□ 这种情况下我会亏多少？
□ 我能承受吗？
□ 我需要调整什么？
```

**压力测试模板**：

| 情景 | 组合影响 | 最大亏损 | 能否承受 | 应对措施 |
|------|----------|----------|----------|----------|
| 股票-25% | | | □是 □否 | |
| 利率+1.5% | | | □是 □否 | |
| 汇率±10% | | | □是 □否 | |

**来源**：[[53. 压力测试]] | [[54. 投资组合风险报告实战]]

> **纳西姆·塔勒布**："平时的波动不重要，极端事件才决定你的生死。压力测试就是提前演练极端情况。"

### 要点10：交易日志每笔都写

**规则**：
```
每笔交易（买入/卖出）后立即记录：
□ 日期、标的、方向、金额
□ 买入/卖出理由（用一句话写清楚）
□ 止损/止盈条件
□ 1个月后复盘：理由对了吗？
```

**交易日志模板**：

| 日期 | 标的 | 方向 | 金额 | 理由 | 止损条件 | 1个月复盘 |
|------|------|------|------|------|----------|-----------|
| | | 买/卖 | | | | |

**为什么必须写？**
1. 防止冲动交易（写下来要花时间，给自己冷静期）
2. 事后复盘（理由是否正确）
3. 发现自己的行为模式（什么情况下容易犯错）

**来源**：[[33. 行为风险控制]] | [[27. 绩效评估与行为复盘]]

> **查理·芒格**："如果你不能解释你为什么买一只股票，那你就不该买它。"

## 投资者视角

### 为什么是"10条"而不是"50条"？

> **查理·芒格**："人类的大脑不是为了记住复杂规则而进化的。简单的规则，严格执行，胜过复杂的规则，偶尔执行。"

**3个原因**：

1. **记得住**：10条规则可以背下来，50条规则只能查手册
2. **能执行**：规则太多会导致"分析瘫痪"
3. **可检验**：每条都有明确的数字标准

## 关键要点

1. **学习的目的是执行**：12周学了很多，但最终要压缩成可执行的规则

2. **规则要可检验**：每条规则都有数字标准，不是模糊的"建议"

3. **简单胜过复杂**：10条规则足够，50条规则反而执行不了

4. **防线最重要**：没有应急金和零高息债，其他规则都是空谈

5. **持续改进**：交易日志和定期审计让你不断进步

### 核心引用

> **查理·芒格**："如果我知道我会死在哪里，我就永远不会去那里。投资也一样——知道什么会让你亏钱，然后避免它。"

### 最后的提醒

> **塔勒布**："投资的目标不是赚大钱，而是不亏大钱。能在市场中生存下来的人，才有机会等到真正的好机会。"

| 阶段 | 目标 | 时间框架 |
|------|------|----------|
| 第一年 | 不亏大钱、建立系统 | 现在 |
| 第二年 | 稳定执行、持续改进 | +1年 |
| 第三年+ | 逐步优化、适度进取 | +2年 |

**投资者警示**：12周学习只是起点。真正的投资能力来自于：
1. 长期实践（至少3-5年）
2. 从错误中学习（这就是为什么要写交易日志）
3. 持续阅读和思考

**永远记住**：在市场中生存比赚大钱更重要。
