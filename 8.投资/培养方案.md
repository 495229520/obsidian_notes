## 1）结构和时间预算

**建议进度：**每周 8-12 小时，持续 12 周（速度快但切实可行）。  
**每周学习计划安排（避免“光看不练”）：**

- 40% 概念（阅读/讲座）
    
- 40% 问题（计算 + 简短证明 + 图表）
    
- 20% 申请材料（纸质投资组合 + 风险报告 + 说明文）
    

**真正重要的产出（你的“技能证明”）：**

1. 个人理财体系（预算、应急基金规则、IPS）
    
2. 多元化投资组合设计 + 再平衡规则
    
3. 衍生品/风险工具包：风险价值 (VaR) + 压力测试 + 期权希腊字母直觉
    
4. 总结备忘录：对冲计划或结构化产品分解

## 2）学习里程碑（你应该能够做到的事情）

### 里程碑 A — 经济学 + 金融素养（第四周结束）

- 用通俗易懂的语言解释供给/需求冲击、弹性以及市场结构。
    
- 阅读宏观经济数据（通货膨胀率、失业率、GDP），并将它们与利率和股票市场联系起来。
    
- 了解货币时间价值；债券收益率/价格走势；复利与贴现。
    

### 里程碑 B — 投资 + 投资组合管理（第 7 周末）

- 从宏观层面解读财务报表（现金流量与收益）
    
- 构建一个基础投资组合（ETF 或指数基金），并衡量其相对于基准的表现。
    
- 实施风险控制措施：仓位规模控制、最大回撤规则、再平衡触发机制
    

### 里程碑 C — 金融工程核心（第 12 周结束）

- 使用**无套利逻辑**对远期合约/期货合约/互换合约进行定价和合理性检验
    
- 理解期权收益、看跌-看涨期权平价、界限、布莱克-斯科尔斯直觉
    
- 从概念上运用希腊字母进行对冲；评估常见的期权策略
    
- 计算基本风险价值 (VaR) 并运行压力测试情景（利率、股市崩盘、外汇冲击）
    

这一里程碑映射与 FI408 的发展进程相吻合：风险管理/VaR → 无套利 → 外汇和利率衍生品 → 股票指数期货 → 互换 → 商品期货 → 期权定价（布莱克-斯科尔斯模型） → 希腊字母 → 策略 → 应用（结构化存款、外汇风险、上限/下限、复合期权） → 产品创新。

## 3）12周快速计划（按周划分）

### 第 0 周（2-3 天）：设置 + 个人财务基线

**目标：**在学习的同时，安全地开始投资。

- 制定预算和净资产表
- 定义：应急基金目标、保险清单、债务偿还规则
- 创建模拟交易观察列表（股票ETF、债券ETF、现金代理）

**交付成果：** 1 页"个人理财操作系统"（规则 + 数字）。

#### 每日学习安排

**第1天——构建你的"理财操作系统"**
- **学习：**现金流与净资产；为什么没有流程的投资会失败
- **查看：**您最近 2-3 个月的消费记录（银行/信用卡账单）
- **练习：**将费用分为固定费用/可变费用/可自由支配费用
- **应用：**制作**预算表 v1**（收入、支出、储蓄率）

**第2天——应急基金+债务+利息的计算**
- **学习：**复利、年利率、"无风险收益=偿还债务"概念
- **查看：**所有债务、利率、最低还款额
- **练习：**计算一年内支付的利息；收益情景
- **应用：**制定**债务规则**（例如，"投资前先偿还年利率高于 X% 的债务"）

**第3天——保险+巨灾风险**
- **了解：**保险的目的是什么（应对尾部风险），而不是"投资"
- **查看：**医疗保险、租房/住房保险、汽车保险；以及适用的雇主福利
- **练习：**列举 3 件灾难性事件，并说明你将如何应对经济困境
- **应用：**创建**保险清单**并找出需要弥补的缺口

### 第 1 周：个人理财体系 + 市场基础知识

**目标：**建立投资基础设施，避免日后遭受损失。

#### 每日学习安排

**第4天——投资工具：ETF、债券、股票**
- **了解：**什么是ETF、费用、多元化投资、跟踪误差
- **查看：** 2 只大型 ETF（例如，涵盖整体市场和债券市场的 ETF）概况介绍
- **练习：**比较费用比率和主要持仓
- **应用：**创建一个**核心观察名单**（3-6只你了解的ETF）

**第5天——你必须了解的市场机制**
- **学习：**买卖价差、流动性、滑点、市价单与限价单
- **查看：**实时报价屏幕：液态和非液态报价之间的差异
- **练习：**模拟糟糕的市价单与限价单的成本对比
- **应用：**编写您的**执行规则**（限价单，避免流动性差的股票）

**第6天——撰写您的投资政策声明 (IPS) v0**
- **学习：**为什么规则胜过情绪；现实困境
- **实践：**明确投资期限、目标、风险承受能力以及可承受的最大回撤
- **应用：**写入**IPS v0**：
    - 目标配置比例（例如，70/30），再平衡触发条件（例如，5%区间）
    - 单笔持仓最大限额（例如，5-10%）
    - "当我不进行交易时。"

**第7天——补习/轻松回顾**
- 修正预算表、IPS v0、观察名单

**交付成果：** IPS v0（目标、期限、配置、再平衡规则）

### 第 2 周：投资者微观经济学

Micro 是你的"价格形成 + 激励"引擎。

**核心概念：**
- 供给/需求、均衡、弹性
- 消费者/生产者剩余
- 税收、补贴、价格上限/下限（市场扭曲）
- 成本曲线，边际推理
- 市场结构：竞争、垄断与寡头垄断
- 定价权 + 行业结构（利润率为何能够维持或崩溃）

#### 每日学习安排

**第8天——供需冲击（价格变动因素）**
- **学习：**需求与供给的变化；弹性（通俗易懂的语言）
- **关注：**一条商品或行业新闻；将其归类为供给冲击或需求冲击
- **练习：**绘制 4 个图表：需求上升/下降、供给上升/下降
- **应用：**写5句话："如果X发生，价格可能会出现Y，因为……"

**第9天——定价权和竞争**
- **学习：**定价权、利润率、竞争为何会扼杀收益
- **比较**两家同一行业的公司，看它们的毛利率
- **练习：**用竞争案例解释利润差异
- **应用：**在自选股名称中添加"定价权"备注

**第10天——成本、经营杠杆和衰退脆弱性**
- **学习：**固定成本与变动成本；经营杠杆
- **查看：**公司的成本结构（任何简单的概括）
- **练习：**假设收入下降 10%，利润会发生什么变化？
- **应用：**将您关注列表中的行业标记为"高/低经营杠杆"

**第11天——市场结构：垄断/寡头垄断信号**
- **了解：**为什么寡头垄断既能带来丰厚利润，也可能带来政治风险
- **考察：**航空公司、软件公司和公用事业公司；对市场结构进行分类
- **练习：**写出三个表明某个行业属于寡头垄断的指标
- **应用：**为您追踪的每个行业添加"结构风险"注释

**第12天——监管与政策风险（投资者视角）**
- **了解：**监管如何改变成本曲线/需求
- **查看：**一个受监管行业的头条新闻
- **练习：**识别价值链中的赢家/输家
- **应用：**在您的观察列表中添加"政策敏感性"列

**第13天——每周微总结**
- **练习：**写一份一页纸的备忘录："影响X行业股票表现的三大微观因素"
- **应用：**如果您发现风险误解，请修订IPS v0

**第14天——补习/休息**

**练习（必须完成）：**
- 选择两个行业（例如，航空业与半导体行业）。请写出：
    - 需求驱动因素、成本结构、定价权、主要风险

### 第 3 周：宏观经济学核心

宏观经济就是"贴现率+增长率+通货膨胀"的引擎。

**核心概念：**
- 国内生产总值、通货膨胀、失业率、实际值与名义值
- 货币政策传导：政策利率→收益率曲线→估值

#### 每日学习安排

**第15天——唯一持续起作用的宏观变量**
- **学习：**通货膨胀、经济增长、失业率、各种比率（它们代表的含义）
- **查看：**最新的通货膨胀率和政策利率数据（任何你喜欢的公开来源）
- **练习：**用一句话定义每个概念，并解释市场为何关注它们
- **应用：**创建一个**宏仪表板表格**（仅需 6-8 个字段）

**第16天——利率：它们为何会影响资产价格**
- **学习：**贴现率的直观理解；实际利率与名义利率
- **查看：** 2年期和10年期国债收益率（当前水平）
- **练习：** "如果利率上升 1%，为什么成长型股票受到的冲击更大？"
- **应用：**制定规则："提高利率机制 → 我倾向于___ / 避免___"

**第17天——通货膨胀体制**
- **了解：**需求拉动型通货膨胀与成本推动型通货膨胀；谁赢谁输
- **查看：**一条与通胀相关的新闻标题；归类类型
- **练习：**绘制通货膨胀向上图 → 债券、股票、大宗商品、现金
- **应用：**在您的投资组合日志中添加"策略说明"

**第18天——经济衰退机制（哪些环节最先崩溃）**
- **学习：**杠杆 + 流动性 + 信心（简单的危机三要素）
- **观察：**信用利差代理指标或公司债券ETF的行为
- **练习：**列出你将监测的 3 个早期预警指标
- **应用：**写下"如果经济衰退信号增强，我将___"（行动计划）

**第19天——外汇基础知识（仅限投资者需要了解的内容）**
- **了解：**利率↔货币；为什么强势美元会损害部分收益
- **关注**美元指数或美元/人民币走势
- **练习：**外汇对公司影响的3个方面（收入转换、成本、需求）
- **应用：**在自选股名称上添加"外汇风险敞口：高/中/低"标签

**第20天——每周宏观备忘录**
- **练习：** 1页备忘录：当前体制（紧缩/宽松）、风险、超配/低配
- **应用：**更新宏观仪表盘 + 日志

**第21天——补习/休息**

**交付成果：** 单页"宏观经济仪表盘"（通货膨胀率、政策利率、2年期/10年期国债收益率、主要股指、美元指数、信用利差代理指标）

### 第 4 周：开放经济 + 周期

**核心概念：**
- 外汇基础知识以及汇率如何影响货币走势
- 商业周期模式（紧缩/宽松；扩张/衰退）
- 基本的经济衰退指标以及通常最先崩溃的因素（流动性/杠杆率）

#### 每日学习安排

**第22天——损益表与现金流量表（新手最常犯的错误）**
- **了解：**收益与现金；为什么利润可能撒谎
- **查看：**一家公司的现金流量概要
- **练习：**找出利润≠现金的三个原因
- **应用：**在公司备注中添加"现金质量"复选框

**第23天——资产负债表：杠杆风险**
- **学习：**债务、流动性、利息保障倍数
- **查看：**一家公司的债务和现金状况
- **练习：**计算简单比率：负债权益比率、利息保障倍数（粗略估算）
- **应用：**制定"危险信号"规则（例如，保险覆盖率下降+债务上升）

**第24天——债券：价格收益率和久期（必知）**
- **了解：**债券价格走势与收益率相反；久期=利率敏感性
- **查看：**债券ETF的久期（已公布的指标）
- **练习：**利用久期估算收益率变动±1%时的价格变化
- **应用：**在您的投资政策声明（IPS）中添加债券套利逻辑（说明您持有债券的原因）

**第25天——估值基础知识（不深入探讨DCF）**
- **学习：**市盈率的构成要素（增长、风险、利率）
- **查看：**市场指数和公司的市盈率
- **练习：**解释高市盈率的三个成因；并说明高市盈率何时危险
- **应用：**在您的关注列表中添加"估值合理性"部分

**第26天——构建你的"核心投资组合"草稿**
- **学习：**核心卫星策略（核心ETF + 小额主动投资）
- **实践：**草案分配、限制、再平衡规则
- **应用：** **投资组合草案 v1**（股票代码/权重 + 再平衡触发条件）

**第27天——每周回顾：绩效+行为**
- **查看：**纸面投资组合损益表及主要驱动因素
- **练习：**写下："我做对了/做错了什么；我违反了什么规则"
- **应用：**将IPS修订为**IPS v1**（更清晰的风险限制）

**第28天——补习/休息**

### 第 5 周：会计与财务报表（最低限度可行）

你不需要是会计师，但你不能是盲人。

**核心概念：**
- 损益表、资产负债表和现金流量表
- 权责发生制与现金制；营运资本；资本支出；自由现金流
- 危险信号：杠杆率、利息保障倍数、激进的收入确认

#### 每日学习安排

**第29天——仓位规模和集中风险**
- **了解：**为什么大多数零售损失都是尺码错误造成的，而不是"糟糕的想法"
- **练习：**选择最大仓位；模拟回撤影响
- **应用：**在 Portfolio Draft v1 中强制执行最大单仓位规则

**第30天——波动性、相关性和多元化**
- **学习：**危机期间相关性飙升；多元化投资的局限性
- **练习：**选择 3 项资产并分析它们与压力之间的相关性
- **应用：**建立一份"多元化投资清单"（不仅仅是增加股票代码）

**第31天——VaR 概念（简化版）+ 其不足之处**
- **学习：** VaR 的含义、置信水平、期限；尾部风险限制
- **练习：**使用历史每日走势数据（电子表格）估算简单的"95%概率下的1天亏损"
- **应用：**在您的投资组合表中添加每周"风险评估"栏目

**第32天——压力测试（比 VaR 更重要）**
- **学习：**情景 > 大多数人的公式
- **练习：**应用以下 3 个场景：股票下跌25%，利率上涨1%，汇率上涨5%
- **应用：**撰写一份"危机应对计划"：你会卖出/持有/买入什么以及原因

**第33天——行为风险控制**
- **学习：**常见陷阱：错失恐惧症 (FOMO)、报复性交易、锚定效应
- **练习：**写下3个个人触发因素和预防规则
- **应用：**添加"冷却规则"（例如，任何新交易前需等待 24 小时）

**第34天——每周风险报告**
- **交付成果：**一份单页报告：分配、最大风险、情景影响、计划行动

**第35天——补习/休息**

**交付成果：** 1 家公司概况（10 个要点 + 5 个比率）

这自然而然地融入了典型的经济/金融课程中使用的更广泛的金融学习顺序（原理→会计→公司金融）。

### 第 6 周：公司财务+估值基础（投资基础）

**核心概念：**
- 货币时间价值（净现值直觉）
- 风险与回报；折现率直觉（影响估值倍数的因素）
- 资本结构基础：债务与权益的权衡

#### 每日学习安排

**第36天——无套利思维（你唯一需要的"工程思维"）**
- **学习：** "相同回报→相同价格"逻辑
- **练习：** 5 个复制示例（即使是简单的现金流）
- **应用：**制定一条规则："如果我无法解释回报，我就不会购买。"

**第37天——远期合约/期货：它们的用途**
- **学习：**远期合约 = 锁定价格；期货 = 标准化远期合约
- **练习：**写一个对冲示例（外汇或商品风险敞口）
- **应用：**确定您个人是否存在任何自然外汇/汇率风险敞口

**第38天——期权收益（看涨期权/看跌期权）**
- **学习：**收益图、内在价值与时间价值
- **练习：**绘制以下收益图：买入看涨期权、买入看跌期权、备兑看涨期权、保护性看跌期权
- **应用：**决定：你需要保护（看跌期权）还是应该减少仓位规模？

**第39天——看跌-看涨期权平价 + 界限（以避免不合理的定价）**
- **学习：**平价直觉；为什么免费的钱很少见
- **练习：** 5 个快速检查：从概念上讲，这个选项是否"太便宜/太贵"？
- **应用：**建立一份"选项合理性检查清单"

**第40天——希腊字母（仅指实际含义）**
- **学习：** delta = 方向敞口；gamma = 凸性；theta = 时间衰减；vega = 波动率敞口
- **练习：**对于以下四种策略，分别写出哪种希腊字母风险占主导地位

## 5）一个简单的每周评估标准（让你保持诚实）

每个星期天，给自己每项得分0-2分：

1. 我可以不用笔记解释这周的概念吗？
    
2. 我能正确解答 5-10 道基础题吗？
    
3. 我是否完成了每周的工作目标？
    
4. 我更新了我的投资政策声明/投资组合/风险报告吗？
    

如果你的得分低于 6/8，请在继续学习之前重做本周的习题。

## 6）安全性和实用性注意事项（对“投资+快速学习”很重要）

- 衍生品威力强大但容错率极低；您的计划有意**先建立风险管理**（VaR/压力测试），然后再制定策略。
    
    
- 只有当你能产出成果（投资政策声明、风险报告、对冲备忘录）时，“快速学习”才真正有效。单靠阅读无法转化为投资技能。


## A. 你的每日模板（每个学习日都这样做）

**1) 学习（20-30分钟）：**阅读/观看特定章节  
**2) 练习（25-35分钟）：**进行简单的计算、绘制图表或回答简短问题  
**3) 应用（15-25分钟）：**更新您的电子表格/纸质投资组合/日志  
**4) 日志（5分钟）：** “我学到了什么 → 它如何影响我的资金/投资 → 下一步行动”

**你每天都会用到的工具**

- 一张电子表格（预算+投资组合+风险）
    
- 纸上投资组合（虚拟交易）或真实的微型投资组合（如果您已经投资）
    
- 一个简单的“宏观仪表盘”（利率、通货膨胀率、指数水平）

## B. 我们_没有_做的事情（明确省略的部分）

- 深层微观理论（效用最大化、拉格朗日量）
    
- 深度宏观模型（IS-LM 推导，DSGE）
    
- 随机微积分/测度论
    
- 奇异衍生物工程
    
- 学术因素模型证明
    

你仍然会学到足够的经济学/金融知识来解读市场和管理资金，但你不会浪费时间。

# C. 6周后：每周的维护计划（维持系统）

如果你想提升投资技巧，“每周循环”比额外的理论更重要。

每周（总计 60-90 分钟）：

1. 更新宏仪表板（10 分钟）
    
2. 更新投资组合和风险表（20分钟）
    
3. 撰写一份包含 10 条要点的每周备忘录（20 分钟）
    
4. 仅当规则触发时才进行重新平衡（10 分钟）
