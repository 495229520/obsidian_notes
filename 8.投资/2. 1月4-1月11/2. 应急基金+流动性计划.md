# 应急基金+流动性计划

## 核心概念

### 应急基金（Emergency Fund）
- **定义**：专门用于应对突发财务状况的现金储备
- **目的**：避免在紧急情况下被迫卖出投资资产（可能亏损）或借高息债务
- **本质**：用"低收益"换取"高确定性"——这是一种保险，不是投资

### 流动性（Liquidity）
- **定义**：资产转换为现金的速度和成本
- **流动性谱系**：
  - 高流动性：现金、货币基金、活期存款
  - 中流动性：股票、债券ETF（可快速卖出，但价格波动）
  - 低流动性：房产、私募基金、定期存款（变现慢或有罚金）

### 流动性风险 vs 市场风险

| 风险类型 | 定义 | 应对方式 |
|----------|------|----------|
| **市场风险** | 资产价格波动导致的损失 | 分散投资、长期持有 |
| **流动性风险** | 需要用钱时无法及时变现，或被迫低价卖出 | 保持应急基金 |

> **关键洞察**：应急基金解决的是流动性风险，不是市场风险。即使你的投资组合很分散，没有应急基金仍然危险。

---

## 投资者视角

### 为什么应急基金是投资的前提？

1. **避免被迫卖出**：市场下跌时正是买入良机，但如果此时需要用钱，你会被迫在最差时机卖出
2. **心理稳定**：有应急基金的人更能承受市场波动，不会恐慌性抛售
3. **机会成本可控**：应急基金的"低收益"是为投资组合买的保险

### 应急基金的机会成本计算

假设：
- 应急基金：6个月支出 = ¥60,000
- 货币基金年化：2%
- 股票长期年化：8%

```
年机会成本 = 60,000 × (8% - 2%) = ¥3,600
```

这 ¥3,600 买到的是：
- 失业时不用卖股票
- 突发医疗不用借高息贷款
- 市场暴跌时能保持冷静

## 应急基金规模设定

### 标准建议：3-6个月支出

| 情况 | 建议月数 | 原因 |
|------|----------|------|
| 收入稳定（公务员、大企业） | 3个月 | 失业风险低 |
| 收入一般稳定 | 6个月 | 标准配置 |
| 收入不稳定（自由职业、销售） | 9-12个月 | 收入波动大 |
| 单收入家庭 | 6-12个月 | 风险集中 |

### 计算你的目标金额

```
应急基金目标 = 月固定支出 × 目标月数

月固定支出 = 房租/房贷 + 水电 + 食物 + 交通 + 保险 + 最低还款
           （不含可削减的娱乐、购物）
```

**示例**：
- 月固定支出：¥8,000
- 目标月数：6个月
- 应急基金目标：¥48,000

## 应急基金存放位置

### 原则：安全 > 收益 > 便捷

| 工具 | 收益 | 流动性 | 适合比例 |
|------|------|--------|----------|
| 活期存款 | ~0.2% | 即时 | 1个月支出 |
| 货币基金 | ~2% | T+0/T+1 | 主力存放 |
| 短期理财 | ~2.5% | 7-30天 | 部分配置 |

### 分层策略

```
第一层（即时可用）：1个月支出 → 活期/余额宝
第二层（1-2天可用）：2-3个月支出 → 货币基金
第三层（7天可用）：2-3个月支出 → 短期理财
```

## 关键要点

1. **应急基金不是投资**，是保险——接受低收益换取确定性
2. **先建应急基金，再投资**——否则市场下跌时你会被迫卖出
3. **3-6个月支出**是标准配置，根据收入稳定性调整
4. **分层存放**：即时可用 + 1-2天可用 + 7天可用
5. **自动化**：设置自动转账，避免"忘记存"

## 应用记录

### 自动转账计划设置

```
每月工资日 → 自动转账 → 应急基金账户

转账金额 = (应急基金目标 - 当前余额) / 剩余月数
```

**示例**：
- 目标：¥48,000
- 当前：¥12,000
- 计划6个月完成
- 每月转账：(48,000 - 12,000) / 6 = ¥6,000
