# 美港股交易机制

> **对应培养方案：** 第1周第5天——你必须了解的市场机制

---

## 核心概念

### 1. 买卖价差 (Bid-Ask Spread)

| 术语 | 定义 | 投资者视角 |
|------|------|-----------|
| Bid（买入价） | 买方愿意支付的最高价 | 你卖出时得到的价格 |
| Ask（卖出价） | 卖方愿意接受的最低价 | 你买入时支付的价格 |
| Spread（价差） | Ask - Bid | **隐性交易成本** |

**价差的直觉理解：**
- 价差 = 做市商的"利润"+ 流动性风险补偿
- 价差越大 → 交易成本越高 → 需要更大涨幅才能盈利

**示例计算：**
```
股票A: Bid $99.90 / Ask $100.10 → Spread = $0.20 (0.2%)
股票B: Bid $9.80 / Ask $10.20 → Spread = $0.40 (4%)

买入$10,000股票A的隐性成本: $10
买入$10,000股票B的隐性成本: $200
```

### 2. 流动性 (Liquidity)

**定义：** 能以接近当前价格快速买卖资产的能力

| 流动性指标 | 高流动性特征 | 低流动性特征 |
|-----------|-------------|-------------|
| 价差 | < 0.1% | > 1% |
| 日均成交量 | 数百万股 | 数千股 |
| 深度 | 各价位有大量挂单 | 订单簿稀疏 |

**流动性风险场景：**
- 想卖出时找不到买家 → 被迫降价
- 大额订单"吃掉"多个价位 → 成交均价偏离报价

### 3. 滑点 (Slippage)

**定义：** 预期成交价与实际成交价的差异

**滑点产生原因：**
1. 市价单执行时价格已变动
2. 订单量超过当前最优价位的可用量
3. 高波动时段（开盘/收盘/重大新闻）

**滑点示例：**
```
预期买入价: $100.00
实际成交价: $100.15
滑点: $0.15 (0.15%)
```

### 4. 订单类型

| 订单类型 | 机制 | 优点 | 缺点 |
|---------|------|------|------|
| **市价单** | 以当前最优价立即成交 | 保证成交 | 价格不确定，可能有滑点 |
| **限价单** | 仅以指定价格或更优价格成交 | 控制成交价 | 可能不成交 |
| **止损单** | 价格触及后变为市价单 | 自动风控 | 可能在跳空时成交价偏离 |
| **止损限价单** | 价格触及后变为限价单 | 控制最差成交价 | 极端行情可能不成交 |

## 美股交易机制

### 交易时间（北京时间）

| 时段 | 夏令时 (3月-11月) | 冬令时 (11月-3月) |
|------|------------------|------------------|
| 盘前交易 | 16:00 - 21:30 | 17:00 - 22:30 |
| **正常交易** | **21:30 - 04:00** | **22:30 - 05:00** |
| 盘后交易 | 04:00 - 08:00 | 05:00 - 09:00 |

**盘前/盘后注意事项：**
- 流动性显著降低 → 价差扩大
- 只有限价单可用
- 重大财报/新闻常在此时段公布

### T+0 交易规则

| 账户类型 | 规则 |
|---------|------|
| 现金账户 | 资金T+2结算，买入后需等结算才能再用 |
| 融资账户 < $25,000 | 5个交易日内最多3次日内交易 (PDT规则) |
| 融资账户 ≥ $25,000 | 无限制日内交易 |

### 美股特有机制

| 机制 | 说明 |
|------|------|
| **无涨跌停** | 理论上单日涨跌无限制 |
| **熔断机制** | S&P 500跌7%/13%/20%触发暂停 |
| **个股熔断 (LULD)** | 5分钟内涨跌超阈值暂停交易 |
| **做空无限制** | 可借券做空（需支付借券费） |

## 港股交易机制

### 交易时间（北京时间）

| 时段 | 时间 |
|------|------|
| 开市前时段 | 09:00 - 09:30 |
| **早市** | **09:30 - 12:00** |
| 午休 | 12:00 - 13:00 |
| **午市** | **13:00 - 16:00** |
| 收市竞价 | 16:00 - 16:10 |

### T+0 与结算

| 规则 | 说明 |
|------|------|
| 交易 | **T+0**（当日可买卖同一股票） |
| 结算 | T+2（资金/股票两个工作日后交割） |

### 港股特有机制

| 机制 | 说明 |
|------|------|
| **最小交易单位** | 1手（手数因股票而异，常见100/500/1000股） |
| **碎股交易** | 需通过碎股市场，价差较大 |
| **印花税** | 买卖双向各0.1% |
| **涨跌幅** | 无限制（但有波动调节机制VCM） |

### 港股通规则（A股投资者）

| 限制 | 内容 |
|------|------|
| 交易时间 | 仅港交所+内地均开市时 |
| 订单类型 | 仅增强限价单 |
| 当日额度 | 每日额度限制（一般充足） |
| 结算 | T+0交收，但卖出资金T+1可用 |

## ADR vs 香港上市

### 什么是ADR？

**ADR (American Depositary Receipt)** = 美国存托凭证
- 代表外国公司股票的美国交易证券
- 由美国托管银行发行，1个ADR可代表多股原股

### 对比分析

| 维度 | ADR（美股） | 港股 |
|------|-----------|------|
| 交易时间 | 北京时间晚间 | 北京时间白天 |
| 货币 | 美元 | 港币 |
| 流动性 | 中概股ADR通常较高 | 因股票而异 |
| 费用 | 无印花税，有ADR托管费 | 印花税0.1%双向 |
| 最小单位 | 1股 | 1手（如100股） |
| 股息 | 可能有额外扣税 | 直接发放 |

### 同一公司的价差套利？

理论上ADR和港股应该价格一致（否则有套利空间），但实际存在价差：
- 汇率波动
- 交易时段不重叠
- 流动性差异
- 转换成本

**投资者启示：** 买入前比较两地价格，选择折价端

2. 小盘股价差可能 > 1%

### 市价单 vs 限价单成本对比

假设你要买入 $10,000 的股票：

| 场景 | 市价单 | 限价单 |
|------|--------|--------|
| 高流动性股票 | 滑点~$5 | 可能立即成交 |
| 低流动性股票 | 滑点可达$100-200 | 可能需要等待/部分成交 |
| 高波动时段 | 滑点不可预测 | 保护成交价格 |

## 关键要点

1. **价差是隐性成本** — 每次买卖都在付出，频繁交易成本累积惊人
2. **流动性决定执行质量** — 流动性差的股票，理论收益被滑点吃掉
3. **限价单是默认选择** — 市价单只在"必须立即成交"时使用
4. **美港股机制差异大** — 交易时间、结算、税费、最小单位都不同
5. **ADR vs 港股需比价** — 同一公司可能存在折溢价
