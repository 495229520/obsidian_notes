# 供需冲击与价格变动

## 核心概念

### 供给与需求的基本框架

```
价格 ↑
  │
  │    供给曲线 (S)
  │        ╱
  │       ╱
  │      ╱ ← 均衡点 (P*, Q*)
  │     ╱
  │    ╱
  │   ╱  需求曲线 (D)
  │  ╱    ╲
  │ ╱      ╲
  │╱        ╲
  └──────────────→ 数量
```

| 概念 | 定义（投资者语言） |
|------|-------------------|
| 需求 | 消费者愿意且能够购买的数量（在各价格水平） |
| 供给 | 生产者愿意且能够提供的数量（在各价格水平） |
| 均衡 | 供需相等时的价格和数量 |
| 冲击 | 导致曲线移动的外部事件 |

### 四种基本冲击

| 冲击类型 | 曲线变化 | 价格影响 | 数量影响 |
|----------|----------|----------|----------|
| 需求增加 | D右移 | ↑ | ↑ |
| 需求减少 | D左移 | ↓ | ↓ |
| 供给增加 | S右移 | ↓ | ↑ |
| 供给减少 | S左移 | ↑ | ↓ |

### 弹性（Elasticity）

**弹性 = 变化的敏感度**

$$E_d = \frac{\%\Delta Q_d}{\%\Delta P}$$

| 弹性类型 | 含义 | 投资含义 |
|----------|------|----------|
| 需求弹性高（>1） | 价格小涨，销量大跌 | 定价权弱，利润率脆弱 |
| 需求弹性低（<1） | 价格涨，销量影响小 | 定价权强，可转嫁成本 |
| 供给弹性高 | 价格涨，供给快速增加 | 难以维持高价（竞争进入） |
| 供给弹性低 | 供给调整缓慢 | 价格波动剧烈 |

**弹性决定因素：**
- 替代品数量（多→弹性高）
- 必需品 vs 奢侈品（必需→弹性低）
- 时间跨度（短期弹性低，长期弹性高）
- 支出占比（占比高→弹性高）

## 投资者视角

### 为什么投资者必须理解供需？

1. **预测价格走势**：识别冲击类型 → 判断价格方向
2. **评估企业利润**：成本上涨能否转嫁？取决于需求弹性
3. **行业选择**：低弹性需求 = 定价权 = 更稳定的利润率

### 投资决策框架

```
事件发生 → 识别冲击类型 → 判断弹性 → 预测价格/利润影响 → 投资决策
```

| 场景 | 冲击分析 | 投资含义 |
|------|----------|----------|
| 芯片短缺 | 供给减少 | 芯片价格↑，有库存的企业受益 |
| AI需求爆发 | 需求增加 | GPU价格↑，英伟达等受益 |
| 经济衰退 | 需求减少 | 非必需品价格↓，必需品相对稳定 |
| 新产能投产 | 供给增加 | 价格↓，高成本生产商承压 |

### 弹性与定价权的关系

| 企业特征 | 需求弹性 | 定价权 | 投资吸引力 |
|----------|----------|--------|------------|
| 独特产品/品牌 | 低 | 强 | 高 |
| 高转换成本 | 低 | 强 | 高 |
| 大宗商品 | 高 | 弱 | 周期性 |
| 激烈竞争 | 高 | 弱 | 低 |

## 关键要点

1. **供需框架是分析一切价格变动的基础**：股票、商品、房产都适用
2. **识别冲击类型是第一步**：先判断是供给还是需求变化
3. **弹性决定影响幅度**：弹性低的市场，价格波动更剧烈
4. **定价权 = 低需求弹性**：这是评估企业质量的核心指标之一
5. **时间维度很重要**：短期弹性往往低于长期弹性

## 延伸思考

> **投资者问题**：当供给和需求同时变化时，价格方向如何判断？

答案取决于**哪个变化幅度更大**：
- 需求增加 > 供给增加 → 价格上涨
- 供给增加 > 需求增加 → 价格下跌

这就是为什么分析师要量化冲击的**相对强度**。
