## 微观经济学要点速记卡

### 卡片1：定价权

```
定价权 = 涨价不丢客户的能力

判断标准：
✓ 毛利率稳定或上升
✓ 涨价后销量不降
✓ 竞争对手跟随涨价

来源：品牌、转换成本、网络效应、稀缺性
```

### 卡片2：经营杠杆

```
经营杠杆 = 利润对收入变化的放大倍数

DOL = 贡献利润 / 营业利润

高固定成本 → 高经营杠杆 → 利润波动大

危险组合：高经营杠杆 + 强周期需求
```

### 卡片3：市场结构

```
CR4 > 60% → 寡头垄断 → 利润可持续

四种结构：
完全竞争 → 零利润趋势
垄断竞争 → 薄利
寡头垄断 → 厚利（投资甜点区）
垄断 → 暴利但受监管
```

### 卡片4：政策风险

```
政策改变游戏规则的两种方式：
1. 成本曲线上移（环保、合规）
2. 需求曲线移动（补贴、禁令）

价值链分析：政策变化有赢家也有输家
```

| 第20天 | 每周总结 | 构建你的宏观仪表盘 |

### 微观 vs 宏观

| 维度 | 微观（第2周） | 宏观（第3周） |
|------|--------------|--------------|
| 分析对象 | 企业、行业 | 整体经济 |
| 核心问题 | 这家公司值得投资吗？ | 现在是投资的好时机吗？ |
| 关键变量 | 定价权、成本、竞争 | 利率、通胀、增长 |
| 决策作用 | 选什么 | 什么时候买/卖 |

> **投资智慧**：微观分析帮你选对公司，宏观分析帮你选对时机。两者结合，才是完整的投资框架。
