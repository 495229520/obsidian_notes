判断以下说法是否正确：

| 序号 | 说法 | 对/错 | 理由 |
|------|------|-------|------|
| 1 | VaR告诉你最坏情况下会亏多少 | | |

| 3 | 压力测试比VaR更能捕捉尾部风险 | | |
| 4 | 市场风险可以通过分散化完全消除 | | |
| 5 | 信用风险只存在于债券投资中 | | |
| 6 | 流动性风险在牛市中最高 | | |
| 7 | 操作风险与市场涨跌无关 | | |
| 8 | 危机时各资产相关性会上升 | | |
| 9 | 风险预算是把风险平均分配给各资产 | | |
| 10 | 逆向压力测试问的是"什么会让我破产" | | |

**参考答案**：

1. **错** — VaR只告诉你在给定置信度下的最大损失，不是真正的最坏情况
2. **对** — 这是VaR的定义
3. **对** — 压力测试专门针对极端情景
4. **错** — 系统性风险无法消除
5. **错** — 股票（公司破产）、衍生品（交易对手违约）也有信用风险
6. **错** — 流动性风险在危机/熊市中最高
7. **对** — 操作风险来自系统、人员、流程失败
8. **对** — "危机中没有分散"
9. **错** — 风险预算可以根据预期回报或风险承受能力不均匀分配
10. **对** — 逆向思维，找出致命情景

### 测试2：计算题

**题目1：VaR计算**

你的投资组合价值100万元，年化波动率15%。

计算95% VaR（1天）：

$$VaR_{1天} = 组合价值 \times 波动率_{日} \times Z_{95\%}$$

其中：$波动率_{日} = \frac{波动率_{年}}{\sqrt{252}}$，$Z_{95\%} = 1.65$

### 第9周预习要点

**无套利定价（No-Arbitrage Pricing）**是金融工程的核心思想：

> **核心原理**："相同的回报 → 相同的价格"

如果两个资产/组合有完全相同的未来现金流，它们必须有相同的价格。否则，就存在套利机会。

**预习问题**：

1. 如果一年后确定收到100元，今天应该值多少钱？（提示：需要知道无风险利率）

2. 如果股票现价100元，一年后确定涨到110元，无风险利率5%，有套利机会吗？

3. 什么是"复制"（Replication）？为什么它是定价的关键？

**推荐预习材料**：
- Hull《期权、期货及其他衍生品》第1-5章

## 关键要点

1. **风险管理是投资的基础**：先学会不亏钱，再考虑怎么赚钱

2. **VaR有用但不够**：VaR管"正常"，压力测试管"极端"，两者结合

3. **风险预算是纪律**：给风险设上限，而不是事后后悔

4. **定期执行是关键**：风险报告不是做一次就完，要定期更新

5. **里程碑C是进阶**：金融工程工具（衍生品）是锦上添花，不是必需品
