# 风险报告模板

## 核心概念

### 什么是风险报告？

**定义**：风险报告是一份定期更新的文档，系统性地记录投资组合面临的风险敞口、量化指标和应对策略。它是投资者的"风险仪表盘"，帮助你在市场波动时保持清醒。

> **霍华德·马克斯**："风险控制是投资中最重要但最不被理解的事情。你不能等到风险实现后才开始管理它。"
> — 《投资最重要的事》

#### 为什么需要风险报告？

**例子：飞行员的仪表盘**

想象你是一位飞行员：
- 没有仪表盘 → 凭感觉飞行 → 天气变化时容易坠机
- 有仪表盘 → 实时监控高度、速度、油量 → 提前发现问题

投资也是如此：
- 没有风险报告 → 凭感觉投资 → 市场暴跌时恐慌
- 有风险报告 → 清楚知道最坏情况 → 危机时能冷静执行

---

### 风险报告的核心要素

根据培养方案要求，你的风险报告必须包含：

| 要素 | 内容 | 目的 |
|------|------|------|
| **组合概览** | 资产配置、持仓明细 | 了解当前状态 |
| **历史VaR** | 95%/99%置信度下的最大损失 | 量化正常风险 |
| **Expected Shortfall** | 超过VaR时的平均损失 | 量化尾部风险 |
| **压力测试** | 3种极端情景下的损失 | 准备最坏情况 |
| **行动计划** | 触发条件和应对措施 | 危机时不慌乱 |

## 填写指南

### 如何计算历史VaR

**方法1：参数法（假设正态分布）**

```
日波动率 = 组合收益率的标准差（可用过去60-250天数据）
VaR(95%,1天) = 组合市值 × 1.65 × 日波动率
VaR(99%,1天) = 组合市值 × 2.33 × 日波动率
```

**方法2：历史模拟法（推荐）**

1. 收集过去250天的组合日收益率
2. 从小到大排序
3. 第13个最差收益率 ≈ VaR(95%)
4. 第3个最差收益率 ≈ VaR(99%)

**简化估算**（如果没有详细数据）：

| 组合类型 | 估计年化波动率 | VaR(95%,1天) |
|----------|----------------|--------------|
| 保守型（30%股票） | ~8% | ~0.8% |
| 平衡型（60%股票） | ~12% | ~1.2% |
| 激进型（80%股票） | ~16% | ~1.6% |
| 纯股票 | ~20% | ~2.0% |

### 如何计算Expected Shortfall

$$ES_{95\%} = \frac{VaR(95\%) + VaR(97.5\%) + VaR(99\%) + ...}{n}$$

**简化方法**：ES ≈ VaR × 1.3 至 1.5

### 如何选择压力测试情景

| 情景类型 | 参考历史事件 | 典型假设 |
|----------|--------------|----------|
| 股市暴跌 | 2008金融危机 | 股票-40%，债券+10% |
| 利率飙升 | 2022年 | 成长股-25%，长债-20% |
| 流动性危机 | 2020.3 | 所有资产同跌20-35% |
| 滞胀 | 1970s | 股票-20%，债券-15%，商品+30% |
| 地缘冲突 | 2022俄乌 | 能源+50%，股票-15% |

## 投资者视角

### 为什么要定期做风险报告？

1. **心理准备**：提前知道最坏情况，危机时不恐慌

2. **行动指南**：预设规则，避免临场决策

3. **纪律约束**：定期检查，防止风险累积

> **沃伦·巴菲特**："在别人恐惧时贪婪的前提是：你必须在别人恐惧之前就知道自己能承受多少恐惧。"

### 风险报告的更新频率

| 市场状态 | 建议频率 |
|----------|----------|
| 正常市场 | 每月1次 |
| 波动加剧（VIX>25） | 每周1次 |
| 危机时期（VIX>35） | 每天1次 |
| 重大事件后 | 立即更新 |

## 关键要点

1. **风险报告是投资的"体检报告"**：定期检查，早发现早处理

2. **VaR是门槛，ES是深渊**：两个都要看

3. **压力测试比VaR更重要**：为"不可能"做准备

4. **预设规则，危机时执行**：不要临场决策

5. **报告要简洁，一页纸足够**：复杂的报告没人看

## 延伸思考

### 风险报告的局限

| 局限 | 说明 | 应对 |
|------|------|------|
| 基于历史 | 无法预测黑天鹅 | 保持安全边际 |
| 模型假设 | 可能不符合现实 | 多种方法交叉验证 |
| 虚假安全感 | 有报告≠安全 | 保持敬畏之心 |
| 执行困难 | 知道≠做到 | 预设规则，自动执行 |

### 从报告到行动

**风险报告的价值不在于数字本身，而在于它帮你**：
1. 提前思考最坏情况
2. 预设应对规则
3. 危机时有据可依
4. 事后复盘学习

> **查理·芒格**："所有聪明的飞行员都使用检查清单，即使他们认为自己不需要。"
> — 《穷查理宝典》

**你的风险报告就是你的投资检查清单。**
