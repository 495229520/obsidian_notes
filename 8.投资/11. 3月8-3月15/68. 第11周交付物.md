### 示例：企业固定-浮动利率互换

#### 背景

某企业有一笔浮动利率贷款（LPR+1%），担心利率上升增加融资成本，决定通过互换锁定利率。

#### 完整现金流图说明

```markdown
# 利率互换现金流图说明

**日期**：2024-03-08
**互换类型**：固定-浮动利率互换（付固定、收浮动）

---

## 1. 合约基本信息

| 项目 | 内容 |
|------|------|
| **名义本金** | 1亿人民币 |
| **起始日** | 2024-03-15 |
| **到期日** | 2027-03-15（3年） |
| **支付频率** | 每季度 |
| **计息惯例** | ACT/365 |

---

## 2. 固定端（我支付）

| 项目 | 内容 |
|------|------|
| **固定利率** | 3.5%/年 |
| **付款方** | 我（企业） |
| **每期付款金额** | 1亿 × 3.5% × (90/365) ≈ 86.3万元 |

---

## 3. 浮动端（我收取）

| 项目 | 内容 |
|------|------|
| **参考利率** | 3个月LPR |
| **利差（Spread）** | +0bp |
| **重置规则** | 每季度初重置 |
| **付款方** | 对手方（银行） |

---

## 4. 现金流时间线

```
2024/3   2024/6   2024/9   2024/12  2025/3   ...   2027/3
  |        |        |        |        |              |
  |   我付固定  我付固定  我付固定  我付固定         我付固定
  |   86.3万   86.3万   86.3万   86.3万            86.3万
  |   ──────→  ──────→  ──────→  ──────→          ──────→
  |
  |   ←──────  ←──────  ←──────  ←──────          ←──────
  |   收浮动   收浮动   收浮动   收浮动            收浮动
  |   (LPR×季) (LPR×季) (LPR×季) (LPR×季)         (LPR×季)
```

---

## 5. 现金流方向说明

| 我的角色 | 我支付 | 我收取 |
|----------|--------|--------|
| 付固定方 | 固定利率3.5%/年 | 浮动利率LPR |

**净效果**：每季度净现金流 = 收到的LPR利息 - 支付的3.5%利息

---

## 6. 风险转移解释

**我把什么风险换成了什么风险？**

| 转换前 | 转换后 |
|--------|--------|
| 浮动利率风险（LPR可能上升） | 固定利率成本（锁定3.5%） |
| 不确定的融资成本 | 确定的融资成本 |

**原有贷款 + 互换 = 合成固定利率贷款**

| 原贷款 | 互换 | 净效果 |
|--------|------|--------|
| 付LPR+1% | 收LPR，付3.5% | 付4.5%固定 |

---

## 7. 情景分析

| 利率变动 | 固定端影响 | 浮动端影响 | 我的净效果 |
|----------|------------|------------|------------|
| LPR上升1% | 不变（仍付3.5%） | 多收1% | 盈利（互换赚钱） |
| LPR下降1% | 不变（仍付3.5%） | 少收1% | 亏损（互换亏钱） |
| LPR不变 | 不变 | 不变 | 中性 |

**关键洞察**：互换让我锁定了融资成本，但也放弃了利率下降时的好处。
```

### 示例：航空公司燃油套保

#### 背景

某航空公司预计3个月后需要采购10,000吨航油，担心油价上涨增加成本。

#### 完整套保示例

```markdown
# 商品套期保值示例：航空公司燃油套保

**日期**：2024-03-08
**套保类型**：买入套保（担心价格上涨）

---

## 1. 风险敞口描述

| 项目 | 内容 |
|------|------|
| **商品类型** | 航空煤油（航油） |
| **敞口规模** | 10,000吨 |
| **风险方向** | 担心价格上涨 |
| **风险期限** | 3个月（2024年6月采购） |
| **当前价格** | 6,500元/吨 |
| **采购预算** | 6,500万元 |

---

## 2. 对冲工具选择

| 项目 | 内容 |
|------|------|
| **期货品种** | 原油期货（INE） |
| **合约规格** | 1,000桶/手 |
| **选择理由** | 无航油期货，原油与航油高度相关 |
| **相关系数ρ** | 0.88（历史数据估算） |

**为什么用原油期货？**
- 国内没有航油期货
- 原油是航油的主要成本来源
- 两者价格高度相关

---

## 3. 套保比率计算

| 参数 | 数值 | 来源 |
|------|------|------|
| 航油价格波动率σS | 22%/年 | 历史数据 |
| 原油期货波动率σF | 25%/年 | 历史数据 |
| 相关系数ρ | 0.88 | 历史相关性分析 |
| **最优套保比率h*** | 0.88 × (22/25) = **0.77** | |

**解读**：每1吨航油敞口，用0.77吨当量的原油期货对冲

---

## 4. 合约数量计算

| 计算项 | 数值 |
|--------|------|
| 航油敞口 | 10,000吨 |
| 吨转换为桶 | 10,000吨 × 7.35桶/吨 = 73,500桶 |
| 需对冲原油桶数 | 73,500 × 0.77 = 56,595桶 |
| 原油期货合约规模 | 1,000桶/手 |
| 需要合约数 | 56,595 / 1,000 = 56.6手 |
| **取整后合约数** | **57手** |
| **对冲方向** | **做多**（买入期货） |

---

## 5. 情景分析

假设当前：航油6,500元/吨，原油期货540元/桶

| 期末航油价格 | 航油采购成本变动 | 期货盈亏 | 净效果 |
|--------------|------------------|----------|--------|
| 7,500元/吨 | +1,000万 | +约800万 | -200万 |
| 7,000元/吨 | +500万 | +约400万 | -100万 |
| 6,500元/吨 | 0 | 0 | 0 |
| 6,000元/吨 | -500万 | -约400万 | -100万 |
| 5,500元/吨 | -1,000万 | -约800万 | -200万 |

**对冲效率**：ρ² = 0.88² = 77.4%
**残留波动**：约23%的价格波动未被对冲

---

## 6. 风险与失效模式

| 风险类型 | 描述 | 应对措施 |
|----------|------|----------|
| **基差风险** | 航油与原油价差可能变化 | 监控裂解价差 |
| **交叉对冲风险** | 品种不同导致对冲不完美 | 接受77%对冲效率 |
| **滚仓风险** | 期货到期需换约 | 提前5个交易日滚仓 |
| **保证金风险** | 价格大跌需追加保证金 | 预留20%额外资金 |

---

## 7. 结论

| 指标 | 评估 |
|------|------|
| **对冲目标** | 锁定3个月后的航油采购成本 |
| **对冲工具** | 原油期货做多57手 |
| **对冲效率** | 77%（消除77%的价格波动） |
| **残留风险** | 基差风险、交叉对冲误差 |
| **保证金占用** | 约540 × 1000 × 57 × 10% = 308万 |
| **对冲成本** | 保证金机会成本 + 交易费用 ≈ 5万 |

**总结**：
- 如果航油涨到7,500元/吨，对冲节省约800万成本
- 如果航油跌到5,500元/吨，对冲损失约800万机会收益
- 对冲让采购成本波动从±1,000万降至±200万
```

## 关键要点

1. **互换是现金流的交换**：画出现金流图是理解互换的第一步

2. **明确风险转移**：互换让你"把一种风险换成另一种风险"，不是消除风险

3. **套保比率不是100%**：h* = ρ × σS/σF，考虑相关性和波动率

4. **情景分析很重要**：提前想清楚"如果价格涨/跌，我会怎样"

5. **识别失效模式**：基差风险、交叉对冲风险是主要问题

## 延伸思考

### 为什么要把交付物写下来？

> **雷·达利欧**："把你的原则写下来。在压力下，人会做出情绪化的决定。写下来的规则是你最好的朋友。"

| 不写下来 | 写下来 |
|----------|--------|
| 理解模糊 | 理解清晰 |
| 容易遗忘 | 随时查阅 |
| 无法复盘 | 可以改进 |
| 无法分享 | 可以讨论 |

**投资者警示**：交付物不是作业，而是你自己的工具。写得越清楚，将来用得越顺手。如果你自己都看不懂自己写的东西，说明你还没真正理解。
