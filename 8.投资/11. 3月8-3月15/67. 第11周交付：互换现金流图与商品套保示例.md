## 交付物一：利率互换现金流图

### 模板结构

根据培养方案的交付物模板，互换现金流图需要包含以下要素：

| 项目 | 内容 |
|------|------|
| **名义本金** | 互换基于的本金金额（不实际交换） |
| **支付频率** | 每季度/每半年/每年 |
| **固定端利率** | 固定支付的年化利率 |
| **浮动端参考利率** | SOFR/LIBOR/LPR等 |
| **重置规则** | 浮动利率何时重置、如何计算 |
| **现金流方向** | 你付固定收浮动，还是相反 |
| **风险转移解释** | 你把什么风险换成了什么风险 |

### 示例：普通利率互换（Plain Vanilla IRS）

**场景**：某企业借了一笔浮动利率贷款（SOFR + 1%），担心利率上升，决定用利率互换锁定成本。

#### 互换基本信息

| 项目 | 内容 |
|------|------|
| **名义本金** | 1000万美元 |
| **期限** | 3年 |
| **支付频率** | 每半年（共6期） |
| **固定端利率** | 4.5%/年 |
| **浮动端参考利率** | 6个月SOFR |
| **重置规则** | 每期初重置，期末支付 |
| **企业角色** | 付固定、收浮动 |

#### 现金流时间线

```
时间轴（年）
    0        0.5        1.0        1.5        2.0        2.5        3.0
    |----------|----------|----------|----------|----------|----------|

企业支付（固定）：
             ↓$22.5万   ↓$22.5万   ↓$22.5万   ↓$22.5万   ↓$22.5万   ↓$22.5万
            (4.5%×0.5   (固定      (固定      (固定      (固定      (固定
             ×1000万)    不变)      不变)      不变)      不变)      不变)

企业收取（浮动）：
             ↑变动      ↑变动      ↑变动      ↑变动      ↑变动      ↑变动
            (SOFR_0     (SOFR_1    (SOFR_2    (SOFR_3    (SOFR_4    (SOFR_5
             ×0.5       ×0.5       ×0.5       ×0.5       ×0.5       ×0.5
             ×1000万)   ×1000万)   ×1000万)   ×1000万)   ×1000万)   ×1000万)

净现金流 = 收到的浮动 - 支付的固定
```

**计算示例（第1期）**：

假设期初SOFR = 4.0%：
- 企业支付固定：$1000万 × 4.5% × 0.5 = $22.5万
- 企业收取浮动：$1000万 × 4.0% × 0.5 = $20万
- **净现金流：支付 $2.5万**

#### 现金流明细表

| 期数 | 时点 | SOFR（假设） | 浮动端收入 | 固定端支出 | 净现金流 |
|------|------|-------------|-----------|-----------|----------|
| 1 | 0.5年 | 4.0% | $20万 | $22.5万 | -$2.5万 |
| 2 | 1.0年 | 4.5% | $22.5万 | $22.5万 | $0 |
| 3 | 1.5年 | 5.0% | $25万 | $22.5万 | +$2.5万 |
| 4 | 2.0年 | 5.5% | $27.5万 | $22.5万 | +$5万 |
| 5 | 2.5年 | 5.0% | $25万 | $22.5万 | +$2.5万 |
| 6 | 3.0年 | 4.8% | $24万 | $22.5万 | +$1.5万 |

#### 风险转移解释

| 转移前 | 转移后 | 说明 |
|--------|--------|------|
| 浮动利率风险 | 固定利率成本 | 不再担心利率上升 |
| 成本不确定 | 成本锁定 | 现金流可预测 |
| 但也放弃了：如果利率下降，企业无法享受低成本 | | |

> **关键理解**："互换不是消除风险，而是把一种风险换成另一种风险。付固定收浮动的企业，把利率风险换成了信用风险（交易对手违约）和机会成本（利率下降时）。"

## 交付物二：商品套保示例

### 模板结构（10条要点）

一个完整的商品套保示例应该回答以下问题：

| # | 要点 | 说明 |
|---|------|------|
| 1 | **风险定义** | 我要对冲什么风险？ |
| 2 | **现货敞口** | 我的现货持仓/需求是多少？ |
| 3 | **期货选择** | 用什么期货品种对冲？ |
| 4 | **相关性分析** | 现货与期货的相关性如何？ |
| 5 | **波动率分析** | 现货与期货的波动率差异？ |
| 6 | **最优对冲比率** | h* 是多少？怎么算的？ |
| 7 | **期货头寸** | 需要多少期货合约？ |
| 8 | **成本估算** | 保证金、手续费、滚仓成本？ |
| 9 | **失效模式** | 什么情况下对冲会失效？ |
| 10 | **监控规则** | 什么时候调整/退出对冲？ |

### 示例：航空燃油套保

**背景**：某航空公司每月消耗价值1亿元人民币的航空燃油，担心油价上涨增加成本。

#### 10条要点

**1. 风险定义**

| 项目 | 内容 |
|------|------|
| 风险类型 | 航空燃油价格上涨 |
| 影响 | 油价每涨10%，每月成本增加1000万 |
| 风险期限 | 未来12个月 |

**2. 现货敞口**

| 项目 | 数量 |
|------|------|
| 月度燃油需求 | 约2万吨 |
| 月度成本 | 约1亿元（按当前价格） |
| 年度敞口 | 约12亿元 |

**3. 期货选择**

| 选项 | 优缺点 | 决策 |
|------|--------|------|
| 航空燃油期货 | 最相关，但流动性差 | ❌ |
| 原油期货（INE） | 流动性好，但相关性次之 | ✅ 选择 |
| 柴油期货 | 相关性中等 | 备选 |

**4. 相关性分析**

| 对比项 | 数据 | 来源 |
|--------|------|------|
| 航空燃油 vs 原油（近3年） | ρ = 0.88 | 历史数据回测 |
| 相关性稳定性 | 中等（危机时下降） | |

**5. 波动率分析**

| 标的 | 年化波动率 |
|------|-----------|
| 航空燃油价格（$\sigma_S$） | 28% |
| 原油期货（$\sigma_F$） | 32% |

**6. 最优对冲比率**

$$h^* = \rho \times \frac{\sigma_S}{\sigma_F} = 0.88 \times \frac{0.28}{0.32} = 0.77$$

**含义**：每100万元燃油敞口，用77万元原油期货对冲。

**7. 期货头寸**

| 计算项 | 结果 |
|--------|------|
| 月度敞口 | 1亿元 |
| 对冲比率 | 0.77 |
| 需要期货名义价值 | 7700万元 |
| 原油期货合约规模（假设） | 1000桶 ≈ 50万元 |
| 需要合约数 | 约154张 |

**8. 成本估算**

| 成本项 | 估算 | 说明 |
|--------|------|------|
| 初始保证金 | 约770万元（10%） | 占用资金成本 |
| 手续费 | 约1万元/月 | |
| 滚仓成本 | 约0.5-1%/月 | 升水市场下的隐性成本 |
| **年化总成本** | 约1-1.5%的敞口 | |

**9. 失效模式**

| 失效情景 | 原因 | 后果 |
|----------|------|------|
| **基差暴涨** | 航空燃油与原油价格脱钩 | 对冲无效甚至亏损 |
| **相关性崩溃** | 供应链中断、地缘政治 | 原油跌但航油涨 |
| **流动性枯竭** | 极端市场 | 无法平仓 |
| **滚仓成本暴增** | 深度升水结构 | 持仓成本超预期 |

**10. 监控规则**

| 监控指标 | 阈值 | 行动 |
|----------|------|------|
| 基差偏离 | >15% | 检查相关性，考虑减仓 |
| 滚仓损耗 | >2%/月 | 考虑减少对冲期限 |
| 保证金占用 | >初始的150% | 补充保证金或减仓 |
| 相关性滚动值 | <0.7 | 重新评估对冲有效性 |

#### 套保效果模拟

| 情景 | 燃油涨幅 | 原油涨幅 | 燃油成本增加 | 期货盈利 | 净影响 |
|------|----------|----------|--------------|----------|--------|
| 正常上涨 | +10% | +10% | +1000万 | +770万 | -230万 |
| 原油超涨 | +10% | +15% | +1000万 | +1155万 | +155万 |
| 基差扩大 | +10% | +5% | +1000万 | +385万 | -615万 |
| 价格下跌 | -10% | -10% | -1000万 | -770万 | -230万 |

> **关键理解**："对冲不是为了赚钱，而是为了减少波动。在'正常上涨'情景下，净影响-230万看似亏损，但实际上是把1000万的不确定性降到了230万。"

## 延伸思考

### 互换与套保的哲学

> **纳西姆·塔勒布**："对冲的本质是承认无知。你不知道价格会涨还是跌，所以你选择两边都下注。这是智慧，不是软弱。"

**思考题**：

1. 如果你"确定"利率会上升，还需要做利率互换吗？
2. 套保降低了波动，但也降低了潜在收益。这个交换什么时候值得？
3. 大多数企业不做套保——是因为不需要，还是因为不懂？

### 真实世界的教训

| 事件 | 发生了什么 | 教训 |
|------|-----------|------|
| **中航油事件（2004）** | 投机性卖出看涨期权，油价上涨巨亏 | 套保≠投机，方向要和风险匹配 |
| **西南航空燃油套保** | 长期锁定低油价，竞争优势明显 | 系统性套保可以成为战略优势 |
| **LTCM危机（1998）** | 对冲策略在极端市场失效 | 模型假设在危机时会崩溃 |

**投资者警示**：套保是工具，使用得当是保护，使用不当是自杀。关键在于：你是否真正理解你在对冲什么、付出什么、以及什么时候会失效。
