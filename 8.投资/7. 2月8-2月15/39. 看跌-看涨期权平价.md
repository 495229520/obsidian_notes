# 看跌-看涨期权平价

## 核心概念

### 什么是看跌-看涨期权平价（Put-Call Parity）？

**定义**：看跌-看涨期权平价是期权定价的核心关系式，它揭示了同一标的、同一行权价、同一到期日的看涨期权和看跌期权之间存在确定的数学关系。如果这个关系被打破，就存在无风险套利机会。

> **约翰·赫尔（John Hull）**："看跌-看涨期权平价是期权市场最基本的无套利关系。任何偏离这一平价关系的情况，都会被套利者迅速消除。"
> — 《期权、期货及其他衍生品》

#### 核心公式

$$C + Ke^{-rT} = P + S_0$$

或等价形式：

$$C - P = S_0 - Ke^{-rT}$$

| 符号 | 含义 | 说明 |
|------|------|------|
| $C$ | 欧式看涨期权价格 | Call Premium |
| $P$ | 欧式看跌期权价格 | Put Premium |
| $S_0$ | 标的资产当前价格 | Spot Price |
| $K$ | 行权价格 | Strike Price |
| $r$ | 无风险利率 | Risk-free Rate |
| $T$ | 到期时间（年） | Time to Expiration |
| $Ke^{-rT}$ | 行权价的现值 | Present Value of Strike |

#### 生活中的例子

**例子1：保险与资产的等价关系**

想象你拥有一幢价值100万的房子：

**方案A：持有房子 + 买火灾保险**
- 你拥有房子（价值100万）
- 你买了保险（如果房子烧了，保险公司赔你100万）
- 无论发生什么，你最终至少有100万

**方案B：把房子卖掉 + 持有现金 + 买"房子看涨期权"**
- 你把房子卖了，拿到100万现金存银行
- 你买了一个"看涨期权"（有权以100万买回房子）
- 如果房价涨了，你行权买回房子
- 如果房价跌了，你保留现金（100万+利息）

两个方案的**最终财富相同**！这就是平价关系的本质——不同的"包装"，相同的"结果"。

**例子2：股票+看跌 = 债券+看涨**

假设苹果股票现价100美元，你想持有苹果但担心下跌：

| 组合 | 构成 | 到期时股价=120 | 到期时股价=80 |
|------|------|----------------|---------------|
| **组合A** | 股票 + 看跌期权(K=100) | 120（持有股票） | 100（行权卖出） |
| **组合B** | 债券(面值100) + 看涨期权(K=100) | 100+20=120（行权买入） | 100（持有债券） |

两个组合在**任何情况下收益相同**！这就是为什么它们的价格必须相等。

#### 直觉理解

**平价关系的本质**：你可以用两种不同的方式"制造"相同的收益结构：

```
持有股票 + 买看跌期权 ≡ 持有现金 + 买看涨期权
（保护性看跌）           （合成多头）
```

如果两种方式成本不同，聪明人会：
- 买便宜的组合
- 卖贵的组合
- 锁定无风险利润

这种套利行为会**自动消除**价格差异。

### 期权价格的界限（Option Bounds）

**为什么需要界限？** 即使不知道精确的期权价格，我们也能确定它的合理范围。超出这个范围的价格意味着存在套利机会。

#### 看涨期权的界限

**上界**：$C \leq S_0$（看涨期权价格不能超过股价）

> **直觉**：看涨期权给你"买股票的权利"，这个权利的价值不可能比直接持有股票更贵。否则，为什么不直接买股票？

**下界**：$C \geq \max(S_0 - Ke^{-rT}, 0)$

> **直觉**：看涨期权至少值它的"内在价值"。如果期权价格低于内在价值，套利者会买入期权、卖空股票、存入现金，锁定利润。

```
看涨期权价格范围：
|-------------------------------------------|
0           内在价值                       股价
            $S_0 - Ke^{-rT}$                $S_0$
            ↑                               ↑
          下界                            上界
```

#### 看跌期权的界限

**上界**：$P \leq Ke^{-rT}$（看跌期权价格不能超过行权价现值）

> **直觉**：看跌期权最好的情况是股价跌到0，你以K卖出。这个权利最多值K的现值。

**下界**：$P \geq \max(Ke^{-rT} - S_0, 0)$

> **直觉**：看跌期权至少值它的"内在价值"。

#### 界限示例

**股票价格**：$S_0 = 100$
**行权价**：$K = 95$
**无风险利率**：$r = 5\%$
**到期时间**：$T = 1$ 年

计算：
- $Ke^{-rT} = 95 \times e^{-0.05} = 90.37$

| 期权类型 | 下界 | 上界 | 合理范围 |
|----------|------|------|----------|
| 看涨期权 | $\max(100 - 90.37, 0) = 9.63$ | 100 | 9.63 ~ 100 |
| 看跌期权 | $\max(90.37 - 100, 0) = 0$ | 90.37 | 0 ~ 90.37 |

如果看涨期权报价8美元（低于下界9.63），存在套利机会！

### 平价关系的套利应用

**场景：发现定价偏差**

假设市场数据：
- 股票价格：$S_0 = 50$
- 行权价：$K = 50$
- 无风险利率：$r = 4\%$
- 到期时间：$T = 0.5$ 年
- 看涨期权价格：$C = 4$
- 看跌期权价格：$P = 2$

**检验平价关系**：

左边：$C + Ke^{-rT} = 4 + 50 \times e^{-0.04 \times 0.5} = 4 + 49.01 = 53.01$

右边：$P + S_0 = 2 + 50 = 52$

**差异**：$53.01 - 52 = 1.01$（左边贵，右边便宜）

**套利策略**：
1. **卖出**组合A（卖看涨 + 借出现金）→ 收入53.01
2. **买入**组合B（买看跌 + 买股票）→ 支出52
3. **净利润**：1.01美元（无风险）

> **查理·芒格**："在投资中，真正的'免费午餐'极为罕见。当你看到明显的套利机会时，首先要问自己是否遗漏了什么。"
> — 《穷查理宝典》

## 投资者视角

### 为什么投资者必须理解平价关系？

1. **识别错误定价**：平价关系是检验期权价格合理性的第一道防线。如果某个期权价格"感觉太便宜"，用平价公式检验是否真的便宜。

2. **理解合成头寸**：你可以用不同的方式构建相同的收益结构。当直接买某种期权太贵时，可以用"合成"方式替代。

3. **避免被骗**：不良交易员可能推销"神奇的"期权策略。懂平价关系的投资者知道：相同收益结构的组合，成本应该相同。

### 合成头寸策略

| 目标头寸 | 合成方式 | 何时使用 |
|----------|----------|----------|
| **合成看涨** | 买股票 + 买看跌 | 看涨期权流动性差或太贵时 |
| **合成看跌** | 卖股票 + 买看涨 | 看跌期权流动性差时 |
| **合成股票** | 买看涨 + 卖看跌 | 股票难以借入做空时 |
| **合成债券** | 买股票 + 买看跌 - 卖看涨 | 构建无风险收益 |

#### 合成看涨期权示例

**场景**：你看好某股票，但看涨期权买卖价差很大（流动性差）。

**替代方案**：
1. 买入股票（$S_0 = 100$）
2. 买入看跌期权（$K = 100, P = 3$）
3. 总成本：103

**等价于**：持有看涨期权 + 持有100元面值债券

根据平价关系：$C = P + S_0 - Ke^{-rT} = 3 + 100 - 98 = 5$

如果直接买看涨期权报价为6美元，合成更便宜！

### 期权合理性快速检查

**检查清单**（在交易任何期权前使用）：

| 检查项 | 方法 | 不合理信号 |
|--------|------|------------|
| 平价检验 | $C + Ke^{-rT} \approx P + S_0$ | 差异超过交易成本 |
| 内在价值 | $C \geq S_0 - K$（实值看涨） | 期权价格低于内在价值 |
| 时间价值 | 期权价格 > 内在价值 | 时间价值为负 |
| 单调性 | 行权价越低，看涨越贵 | 违反单调关系 |
| 凸性 | 中间行权价期权不能太贵 | 蝶式价差为负 |

## 关键要点

1. **平价关系是无套利的直接结果**：相同收益结构的组合必须有相同的价格，否则存在"免费午餐"

2. **平价公式**：$C + Ke^{-rT} = P + S_0$，任何偏离都是套利机会

3. **期权有明确的价格界限**：超出界限的报价意味着存在套利，或者你漏算了什么

4. **合成头寸是平价的实际应用**：当某种期权太贵或流动性差时，可以用其他工具"合成"

5. **先检验再交易**：在买卖任何期权前，用平价关系和界限做"合理性检查"

## 延伸思考

### 为什么套利机会转瞬即逝？

> **尤金·法玛**："在有效市场中，价格反映所有可用信息。套利者就像市场的'清道夫'，他们消除了定价错误，使市场更有效。"

在现实中，平价关系可能出现微小偏离，原因包括：

| 因素 | 影响 | 结果 |
|------|------|------|
| 交易成本 | 买卖价差、佣金 | 套利利润被吞噬 |
| 借贷利差 | 借入利率 > 无风险利率 | 套利成本上升 |
| 股息 | 预期股息影响期权价格 | 公式需要调整 |
| 美式期权 | 提前行权可能性 | 平价变成不等式 |
| 执行风险 | 滑点、延迟 | 理论利润无法实现 |

**投资者警示**：当你看到"明显"的套利机会时，先问自己：
1. 我是否漏算了交易成本？
2. 我是否忽略了股息？
3. 这是美式还是欧式期权？
4. 为什么专业套利者没有消除这个机会？

如果找不到合理解释，那可能真的是机会——但这种情况极为罕见。
