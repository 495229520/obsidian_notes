# 期权收益

> **对应培养方案：** 第6周第38天——期权收益（看涨期权/看跌期权）

---

## 核心概念

### 期权：买"权利"而非"义务"

**定义**：期权是一种合约，赋予买方在**未来某一时间**以**约定价格**买入或卖出标的资产的**权利**（而非义务）。为了获得这个权利，买方需要支付一笔费用，叫做**期权费（Premium）**。

> **约翰·赫尔**："期权与远期最根本的区别是：远期是义务，期权是权利。这个区别决定了它们完全不同的风险收益特征。"
> — 《期权、期货及其他衍生品》

#### 期权 vs 远期的核心区别

| 特征 | 远期/期货 | 期权 |
|------|----------|------|
| 买方义务 | **必须**执行 | **可以选择**执行或放弃 |
| 初始成本 | 零 | 支付期权费 |
| 最大亏损 | 无限（可能很大） | 期权费（有限） |
| 最大收益 | 无限 | 无限（看涨）/有限（看跌） |

#### 生活中的例子

**例子1：买房的"购房意向金"**

你看中一套500万的房子，但需要1个月筹钱和考虑：
- 你付5万"意向金"，锁定这套房1个月
- 1个月后：
  - 如果你想买 → 付剩余495万，房子是你的
  - 如果你不想买 → 放弃，损失5万意向金
  - 如果房价涨到550万 → 你仍按500万买入，赚了50万（减去5万成本）
  - 如果房价跌到450万 → 你放弃意向金，去市场上买更便宜的

这就是**看涨期权**的逻辑：
- 标的资产：房子
- 行权价：500万
- 期权费：5万
- 到期日：1个月后

**例子2：汽车延保服务**

你买了一辆车，4S店推荐"延保服务"：
- 支付5000元，获得3年内任何故障免费维修的权利
- 如果车不出问题 → 你损失5000元保费
- 如果车大修花了3万 → 你只付了5000元，"赚"了2.5万

延保服务本质上是一个**看跌期权**——对冲车辆价值下降（故障）的风险。

**例子3：机票退改签**

| 机票类型 | 价格 | 退改规则 | 期权类比 |
|----------|------|----------|----------|
| 特价票 | 500元 | 不可退改 | 没有期权 |
| 标准票 | 800元 | 可退改，手续费50元 | 买了期权（多付300元） |

你多付300元买的是"如果计划有变，可以退票"的**权利**。

### 看涨期权（Call Option）

**定义**：看涨期权赋予买方在到期日（或之前）以**行权价买入**标的资产的权利。买方看涨，希望价格上涨后以低于市场价买入。

$$看涨期权到期收益 = \max(S_T - K, 0)$$

其中：
- $S_T$ = 到期时标的资产价格
- $K$ = 行权价（Strike Price）

**直觉解释**：
- 如果到期价格 > 行权价 → 执行期权，赚差价
- 如果到期价格 ≤ 行权价 → 放弃期权，损失期权费

#### 看涨期权收益图

```
收益
  ↑
  │                    ／
  │                  ／
  │                ／
  │--------------●
  │              K
  └──────────────────────→ 股价
              行权价
```

| 到期股价 | 是否行权 | 收益 | 净收益（减期权费） |
|----------|----------|------|-------------------|
| 低于K | 否 | 0 | -期权费 |
| 等于K | 无所谓 | 0 | -期权费 |
| 高于K | 是 | S-K | S-K-期权费 |

**经典案例：特斯拉看涨期权**

2020年初，特斯拉股价约100美元（拆股后）：
- 买入行权价150美元、1年到期的看涨期权，期权费20美元
- 2020年底，特斯拉涨到700美元
- 收益 = 700 - 150 - 20 = 530美元
- 收益率 = 530 / 20 = **2650%**

如果直接买股票：
- 收益 = 700 - 100 = 600美元
- 收益率 = 600 / 100 = **600%**

**期权的杠杆效应**：用更少的本金获得更大的收益率——但如果股价不涨，期权费全损。

### 看跌期权（Put Option）

**定义**：看跌期权赋予买方在到期日（或之前）以**行权价卖出**标的资产的权利。买方看跌，希望价格下跌后以高于市场价卖出。

$$看跌期权到期收益 = \max(K - S_T, 0)$$

**直觉解释**：
- 如果到期价格 < 行权价 → 执行期权，高价卖出
- 如果到期价格 ≥ 行权价 → 放弃期权，损失期权费

#### 看跌期权收益图

```
收益
  ↑
  │＼
  │  ＼
  │    ＼
  │      ●--------------
  │      K
  └──────────────────────→ 股价
        行权价
```

| 到期股价 | 是否行权 | 收益 | 净收益（减期权费） |
|----------|----------|------|-------------------|
| 低于K | 是 | K-S | K-S-期权费 |
| 等于K | 无所谓 | 0 | -期权费 |
| 高于K | 否 | 0 | -期权费 |

**经典案例：2008年金融危机的看跌期权**

电影《大空头》中的投资者：
- 2006年买入大量房贷相关证券的看跌期权（CDS本质类似）
- 支付了连续两年的"保费"（期权费）
- 2008年房市崩盘，证券价格暴跌
- 看跌期权价值飙升，获得数十倍收益

> **迈克尔·刘易斯**："他们付出的代价是每年的保费，换来的是在灾难发生时的巨额回报。这就是期权的本质。"
> — 《大空头》

### 内在价值与时间价值

**期权价格 = 内在价值 + 时间价值**

#### 内在价值（Intrinsic Value）

**定义**：如果**立即行权**能获得的价值。

$$看涨期权内在价值 = \max(S - K, 0)$$
$$看跌期权内在价值 = \max(K - S, 0)$$

| 状态 | 英文 | 定义 | 内在价值 |
|------|------|------|----------|
| 实值 | In-the-money (ITM) | 立即行权有利可图 | > 0 |
| 平值 | At-the-money (ATM) | 股价≈行权价 | ≈ 0 |
| 虚值 | Out-of-the-money (OTM) | 立即行权无利可图 | = 0 |

**例子**：股价100元

| 期权 | 行权价 | 状态 | 内在价值 |
|------|--------|------|----------|
| 看涨 | 90元 | 实值 | 10元 |
| 看涨 | 100元 | 平值 | 0元 |
| 看涨 | 110元 | 虚值 | 0元 |
| 看跌 | 110元 | 实值 | 10元 |
| 看跌 | 100元 | 平值 | 0元 |
| 看跌 | 90元 | 虚值 | 0元 |

#### 时间价值（Time Value）

**定义**：期权价格中超出内在价值的部分，反映"未来可能变得更有利"的可能性。

$$时间价值 = 期权价格 - 内在价值$$

**影响时间价值的因素**：

| 因素 | 对时间价值的影响 | 原因 |
|------|------------------|------|
| 到期时间↑ | ↑ | 更多时间=更多可能性 |
| 波动率↑ | ↑ | 波动大=大涨大跌可能性都增加 |
| 接近平值 | 最大 | 不确定性最高 |

> **纳西姆·塔勒布**："期权的时间价值本质上是对不确定性的定价。不确定性越大，期权越贵。"
> — 《动态对冲》

**时间衰减（Theta Decay）**：

```
时间价值
  ↑
  │＼
  │  ＼
  │    ＼
  │      ＼
  │        ＼___
  └──────────────→ 到期日
```

**关键洞见**：时间价值随到期日临近而加速衰减，尤其在最后30天。这就是为什么卖期权通常比买期权更容易赚钱——时间站在卖方这边。

## 四种基本策略的收益图

### 1. 买入看涨期权（Long Call）

**适用场景**：看涨标的资产

```
损益
  ↑
  │                    ／
  │                  ／
  │                ／
  ├─────────────●
  │             K ＼
  │                 期权费
  └──────────────────────→ 股价
```

| 要素 | 说明 |
|------|------|
| 最大亏损 | 期权费（有限） |
| 最大收益 | 无限 |
| 盈亏平衡点 | K + 期权费 |

---

### 2. 买入看跌期权（Long Put）

**适用场景**：看跌标的资产 / 保护持仓

```
损益
  ↑
  │＼
  │  ＼
  │    ＼
  │      ●─────────────
  │      K           期权费
  └──────────────────────→ 股价
```

| 要素 | 说明 |
|------|------|
| 最大亏损 | 期权费（有限） |
| 最大收益 | K - 期权费（股价跌到0） |
| 盈亏平衡点 | K - 期权费 |

---

### 3. 保护性看跌期权（Protective Put）

**定义**：持有股票 + 买入看跌期权 = 给股票买"保险"

**适用场景**：持有股票但担心下跌

```
损益
  ↑
  │                    ／
  │                  ／
  │                ／
  ├───────────●／
  │           K
  │           最大亏损 = 期权费 + (S₀-K)
  └──────────────────────→ 股价
```

| 要素 | 说明 |
|------|------|
| 最大亏损 | 期权费 + (买入价 - 行权价) |
| 最大收益 | 无限 |
| 本质 | 给下跌设定"地板" |

**经典案例：投资组合保险**

你持有价值100万的股票组合：
- 担心市场下跌，但不想卖出
- 买入行权价=当前价格的看跌期权，期权费3万
- 如果股市跌30% → 你的损失限制在3万（期权费），而非30万
- 如果股市涨20% → 你赚20万，减去3万期权费 = 净赚17万

> **沃伦·巴菲特**："衍生品是金融大规模杀伤性武器——但保护性看跌期权是例外，它本质上是保险。"

---

### 4. 备兑看涨期权（Covered Call）

**定义**：持有股票 + 卖出看涨期权 = 用上涨潜力换取额外收入

**适用场景**：持有股票，认为短期不会大涨

```
损益
  ↑
  │          期权费收入
  │    ●─────────────────
  │  ／      K
  │／
  │
  └──────────────────────→ 股价
```

| 要素 | 说明 |
|------|------|
| 最大亏损 | 买入价 - 期权费（股价跌到0） |
| 最大收益 | K - 买入价 + 期权费（有限） |
| 本质 | 用上涨空间换取现金流 |

**经典案例：持有蓝筹股收租**

你持有茅台股票，当前价格1800元：
- 你认为短期涨不到2000元
- 卖出行权价2000元、1个月到期的看涨期权，收取期权费30元
- 如果茅台没涨到2000 → 你白赚30元期权费
- 如果茅台涨到2200 → 你被迫以2000元卖出，错过200元上涨，但还是赚了200+30=230元

> **查理·芒格**："备兑看涨期权是一种保守的增强收益策略——放弃极端上涨的可能性，换取确定的现金流。"

## 投资者视角

### 为什么个人投资者需要理解期权？

1. **保护工具**：看跌期权是最直接的投资组合保险
2. **增强收益**：备兑看涨期权可以增加持股的现金流
3. **识别风险**：理解期权才能识别结构化产品中的隐藏风险
4. **理解市场**：期权价格反映市场对波动性的预期

### 期权 vs 减仓：保护投资组合的两种方式

| 维度 | 买入看跌期权 | 直接减仓 |
|------|--------------|----------|
| 成本 | 期权费（确定） | 交易成本（较小） |
| 下跌保护 | 完全保护 | 部分保护（减仓部分） |
| 上涨参与 | 完全参与 | 部分参与（减仓部分） |
| 适用场景 | 短期担忧，长期看好 | 真正想降低仓位 |
| 心理负担 | 可能"浪费"期权费 | 可能"错过"反弹 |

> **霍华德·马克斯**："保护的成本是确定的，保护的收益是不确定的。你必须问自己：你愿意为安心付出多少？"

### 个人投资者的期权使用建议

| 策略 | 适合度 | 原因 |
|------|--------|------|
| 买入看跌期权（保护） | ⭐⭐⭐⭐ | 风险有限，保护明确 |
| 备兑看涨期权 | ⭐⭐⭐ | 增强收益，风险可控 |
| 买入看涨期权（投机） | ⭐⭐ | 杠杆高，容易亏光 |
| 裸卖期权 | ⭐ | 风险无限，不建议 |

## 关键要点

1. **期权是权利，不是义务**：买方有权选择执行或放弃，最大亏损是期权费
2. **看涨期权获利于上涨**：股价涨过行权价+期权费，开始盈利
3. **看跌期权获利于下跌**：股价跌破行权价-期权费，开始盈利
4. **时间是期权买方的敌人**：时间价值每天在流失，卖方收取"时间租金"
5. **保护性看跌是投资保险**：确定的成本（期权费）换取不确定的保护

## 延伸思考

### 期权的常见误区

| 误区 | 真相 |
|------|------|
| "期权是赌博" | 期权是风险管理工具，赌博取决于使用方式 |
| "买期权稳赚" | 大多数期权到期时一文不值 |
| "卖期权稳赚" | 卖期权收小钱但可能亏大钱（尾部风险） |
| "期权太复杂" | 基本的保护性策略其实很简单 |

### 个人投资者的期权使用原则

> **查理·芒格**："如果你不完全理解一个金融工具，就不要使用它。"

| 原则 | 说明 |
|------|------|
| 1. 只买不卖 | 买期权亏损有限，卖期权风险无限 |
| 2. 只做保护 | 用期权对冲，不要用期权投机 |
| 3. 成本有限 | 期权费不超过组合价值的1-2% |
| 4. 理解再用 | 不理解的期权不要碰 |

**投资者警示**：期权是专业工具，威力强大但容错率低。对于大多数个人投资者，调整仓位规模可能比使用期权更简单有效。
