# 远期合约与期货

> **对应培养方案：** 第6周第37天——远期合约/期货：它们的用途

---

## 核心概念

### 远期合约（Forward Contract）

**定义**：远期合约是买卖双方约定在**未来某一特定日期**，以**今天确定的价格**买卖某种资产的协议。它是一种"锁定价格"的工具，让你现在就知道未来要付多少钱（或收多少钱）。

> **约翰·赫尔**："远期合约是最简单的衍生品。理解了远期，你就理解了衍生品定价的核心逻辑。"
> — 《期权、期货及其他衍生品》

#### 生活中的例子

**例子1：预订婚宴酒席**

你要在6个月后举办婚礼，今天去酒店预订：
- 酒店报价：每桌5000元
- 你签合同锁定这个价格
- 6个月后，无论酒店价格涨到6000还是跌到4000，你都按5000元付款

这就是一个**远期合约**：
- 标的资产：婚宴酒席
- 到期日：6个月后
- 远期价格：5000元/桌
- 双方义务：酒店必须提供服务，你必须付款

**例子2：农民卖粮食**

春天播种时，农民担心秋天粮价下跌：
- 现在小麦价格：3元/斤
- 农民与粮商签订远期合约：秋天以2.8元/斤卖出10吨
- 秋天到来：
  - 如果市场价跌到2.5元 → 农民赚了（锁定了更高价格）
  - 如果市场价涨到3.5元 → 农民亏了（错过了更高价格）

**关键洞见**：远期合约不是为了"赚钱"，而是为了**消除不确定性**。农民可能放弃上涨收益，但换来了确定的收入，可以安心规划生活。

**例子3：留学换汇**

你的孩子明年9月要去美国留学，需要5万美元学费：
- 今天汇率：1美元 = 7.2人民币
- 你担心人民币贬值（汇率变成7.5甚至8）
- 银行提供远期换汇：锁定1美元 = 7.3人民币

| 情景 | 没有远期 | 有远期（锁定7.3） |
|------|----------|------------------|
| 明年汇率7.0 | 支付35万 | 支付36.5万（多付1.5万） |
| 明年汇率7.5 | 支付37.5万 | 支付36.5万（省1万） |
| 明年汇率8.0 | 支付40万 | 支付36.5万（省3.5万） |

远期让你把"不确定的36.5万~40万"变成"确定的36.5万"。

### 期货合约（Futures Contract）

**定义**：期货合约是**标准化**的远期合约，在**交易所**交易，具有统一的合约规格、到期日和交割方式。期货通过**每日盯市**（Mark-to-Market）机制结算盈亏，大大降低了违约风险。

> **利奥·梅拉梅德**（芝加哥商业交易所前主席）："期货市场的天才之处在于标准化。它把定制西装变成了成衣，让每个人都能买得起风险管理工具。"

#### 远期 vs 期货：核心区别

| 特征 | 远期合约 | 期货合约 |
|------|----------|----------|
| **交易场所** | 场外（OTC），双方直接协商 | 交易所，公开交易 |
| **标准化** | 定制化（金额、日期可协商） | 标准化（固定规格） |
| **对手方** | 交易对手（有违约风险） | 清算所（几乎无违约风险） |
| **流动性** | 低（难以转让） | 高（可随时买卖平仓） |
| **保证金** | 通常无（或一次性） | 每日盯市，追加保证金 |
| **结算** | 到期一次性结算 | 每日结算盈亏 |
| **适用对象** | 机构、大客户 | 所有投资者 |

#### 生活中的类比

**远期 = 定制西装**
- 你找裁缝量身定制
- 可以完全按你的要求
- 但只能在这家店做，不能转让
- 如果裁缝跑路，你的定金就没了

**期货 = 成衣店的西装**
- 标准尺码（S/M/L/XL）
- 可能不是100%合身
- 但可以随时退换、转卖
- 店铺是大品牌，不用担心跑路

#### 期货的每日盯市机制

**例子：买入1手沪深300股指期货**

假设：
- 合约乘数：300元/点
- 买入价格：4000点
- 合约价值：4000 × 300 = 120万元
- 保证金比例：10% → 需缴纳12万元

| 日期 | 结算价 | 当日盈亏 | 累计盈亏 | 保证金账户 |
|------|--------|----------|----------|------------|
| 第1天 | 4000 | 0 | 0 | 12万 |
| 第2天 | 4050 | +1.5万 | +1.5万 | 13.5万 |
| 第3天 | 3950 | -3万 | -1.5万 | 10.5万 |
| 第4天 | 3900 | -1.5万 | -3万 | 9万（需追加保证金） |

**每日盯市的好处**：
- 盈亏每天结算，不会累积巨额亏损
- 如果账户余额不足，会被强制平仓
- 大大降低了违约风险

### 远期/期货的用途

#### 用途1：对冲（Hedging）

**定义**：对冲是使用远期/期货来锁定价格，消除未来的价格不确定性。

> **沃伦·巴菲特**："衍生品本身不是坏东西。它们就像刀子——外科医生用来救人，罪犯用来杀人。关键是谁在用，怎么用。"
> — 致股东的信，2002年

**经典案例：西南航空的燃油对冲**

| 时期 | 西南航空的策略 | 结果 |
|------|----------------|------|
| 2000-2008年 | 大量买入燃油期货对冲 | 油价从$25涨到$147，西南航空锁定低价，节省数十亿美元 |
| 2008年金融危机 | 油价暴跌至$35 | 对冲合约产生账面亏损，但主业稳定 |
| 长期来看 | 持续对冲策略 | 成本可预测，竞争优势明显 |

> **投资启示**：西南航空的对冲不是为了"在期货上赚钱"，而是为了让燃油成本可预测。这让他们能够提供稳定的票价，建立竞争优势。

**对冲的核心逻辑**：

| 你的风险敞口 | 对冲操作 | 效果 |
|--------------|----------|------|
| 未来要买入（怕涨价） | 买入远期/期货 | 锁定买入价格 |
| 未来要卖出（怕跌价） | 卖出远期/期货 | 锁定卖出价格 |
| 持有美元资产（怕美元跌） | 卖出美元远期 | 锁定汇率 |
| 有美元债务（怕美元涨） | 买入美元远期 | 锁定汇率 |

#### 用途2：投机（Speculation）

**定义**：投机是利用远期/期货的杠杆效应，押注价格方向以获取利润。

> **杰西·利弗莫尔**："投机不是赌博，它是有根据的冒险。但大多数人把投机变成了赌博。"
> — 《股票作手回忆录》

**投机的杠杆效应**：

| 投资方式 | 资金 | 价格涨10% | 收益率 |
|----------|------|-----------|--------|
| 直接买入现货 | 100万 | 赚10万 | +10% |
| 买入期货（10%保证金） | 10万（控制100万资产） | 赚10万 | +100% |

**警告**：杠杆是双刃剑——如果价格跌10%，现货亏10%，期货则**亏光全部本金**。

#### 用途3：套利（Arbitrage）

**定义**：套利是利用远期/期货与现货之间的定价偏差，无风险获利。

**例子：期现套利**

如果：
- 黄金现货：$2000/盎司
- 3个月期货：$2100/盎司
- 无风险利率：2%/年（3个月约0.5%）
- 理论期货价格：$2000 × 1.005 = $2010

期货价格$2100明显高于理论价格$2010，存在套利机会：
1. 借钱买入现货黄金（$2000）
2. 同时卖出期货（锁定$2100卖出价）
3. 3个月后交割：获得$2100，还债$2010
4. 无风险利润：$90/盎司

### 远期/期货定价原理

#### 持有成本模型（Cost of Carry Model）

**核心公式**：

$$F = S_0 \times e^{(r-q)T}$$

简化版（离散复利）：

$$F = S_0 \times (1 + r - q)^T$$

其中：
- F = 远期/期货价格
- S₀ = 现货价格
- r = 无风险利率
- q = 持有收益率（股息率、便利收益等）
- T = 到期时间

**直觉理解**：

| 因素 | 对远期价格的影响 | 原因 |
|------|------------------|------|
| 现货价格↑ | 远期价格↑ | 复制成本增加 |
| 利率↑ | 远期价格↑ | 持有现货的机会成本增加 |
| 股息/便利收益↑ | 远期价格↓ | 持有现货有额外收益 |
| 到期时间↑ | 远期价格↑ | 持有成本累积更多 |

#### 不同资产的定价特点

| 资产类型 | 定价公式 | 特点 |
|----------|----------|------|
| **无收益资产**（黄金） | F = S₀(1+r)ᵀ | 只有利息成本 |
| **有股息股票** | F = S₀(1+r)ᵀ - 股息终值 | 扣除股息 |
| **外汇** | F = S₀ × (1+r_d)/(1+r_f) | 两国利率差 |
| **商品** | F = S₀(1+r+u-y)ᵀ | u=存储成本，y=便利收益 |

## 投资者视角

### 为什么投资者必须理解远期/期货？

1. **管理生活中的风险**：汇率、利率、商品价格波动都可能影响你的财务
2. **理解企业风险管理**：分析公司时，了解其对冲策略很重要
3. **避免被"杠杆"诱惑**：期货的高杠杆既是机会也是陷阱
4. **识别投机 vs 对冲**：明白自己在做什么，是管理风险还是承担风险

### 普通人可能有的风险敞口

| 风险类型 | 具体情景 | 对冲工具 |
|----------|----------|----------|
| **汇率风险** | 子女留学、海外购物、持有美元资产 | 外汇远期 |
| **利率风险** | 浮动利率房贷 | 利率互换（需通过银行） |
| **商品风险** | 经营餐厅（食材成本）、开车（油价） | 商品期货 |
| **股市风险** | 持有大量股票 | 股指期货 |

### 对冲 vs 投机的本质区别

| 维度 | 对冲 | 投机 |
|------|------|------|
| **目的** | 减少不确定性 | 利用不确定性获利 |
| **风险敞口** | 已有现货敞口，用期货抵消 | 主动创造敞口 |
| **预期** | 期货盈亏与现货相抵 | 期货盈利 |
| **心态** | "锁定"心态 | "赌"心态 |
| **风险** | 放弃潜在收益 | 可能巨额亏损 |

> **查理·芒格**："如果你不知道自己在做什么，那你就是在投机。如果你知道自己在做什么，那也可能是在投机——但至少是有根据的投机。"

## 关键要点

1. **远期=锁定价格**：远期合约让你现在就确定未来的买卖价格，消除不确定性
2. **期货=标准化远期**：期货在交易所交易，更安全、更灵活，但规格固定
3. **对冲的目的是减少风险，不是赚钱**：用期货对冲时，期货的盈亏会与现货相抵
4. **杠杆是双刃剑**：期货的低保证金意味着高杠杆，放大收益也放大亏损
5. **先识别风险敞口，再考虑是否对冲**：没有风险敞口却买期货，那就是投机

## 延伸思考

### 为什么大多数人不应该交易期货？

| 原因 | 解释 |
|------|------|
| **杠杆风险** | 10%保证金意味着10倍杠杆，亏损被放大10倍 |
| **专业门槛** | 需要理解合约规格、交割规则、保证金机制 |
| **心理压力** | 每日盯市的盈亏波动，普通人难以承受 |
| **零和游戏** | 期货市场是零和的，你的盈利就是别人的亏损 |
| **机构优势** | 机构有信息、资金、技术优势 |

### 什么情况下可以考虑使用远期/期货？

| 情景 | 合适程度 | 原因 |
|------|----------|------|
| 有真实风险敞口需要对冲 | ✅ 合适 | 这是衍生品的本来用途 |
| 对某商品有深入研究，想投机 | ⚠️ 谨慎 | 需要专业知识和风险承受能力 |
