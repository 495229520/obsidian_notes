# 12.5 const 成员函数

## 简介

在 C++ 中，`const` 成员函数（也称为常量成员函数）是指在类的成员函数声明和定义的末尾（参数列表之后，函数体之前）加上 `const` 关键字的函数。

```cpp
ReturnType FunctionName(parameters) const;
```

---

## 内含

`const` 成员函数**不会修改对象的任何非静态或 mutable 成员变量**。这是一种对对象状态的承诺，只允许读取对象数据而不允许写入。

### 核心作用

- **允许 const 对象调用**：普通的非 const 成员函数不能被 const 对象调用，但 const 成员函数可以
- **接口承诺**：表明这个函数是一个纯粹的"观察者"，使用者无需担心调用它会改变对象的状态
- **const 正确性**：const 成员函数是 C++ 中"const 正确性"的重要组成部分，它帮助编写更安全、更易于理解的代码

---

## 对 this 指针的影响

- **普通成员函数**：`this` 指针类型为 `ClassName* const`（指针不可变，但指向的对象可修改）
- **const 成员函数**：`this` 指针类型为 `const ClassName* const`（指针和指向的对象都不可修改）

---

## mutable 关键字

有些成员变量需要在 `const` 成员函数中修改（如缓存、计数器等），可以使用 `mutable` 关键字：

```cpp
class Counter {
private:
    mutable int accessCount;  // mutable 成员可在 const 函数中修改
    int value;

public:
    Counter() : accessCount(0), value(0) {}

    int getValue() const {
        accessCount++;  // OK：mutable 成员可以修改
        return value;
    }

    int getAccessCount() const {
        return accessCount;
    }
};
```

---

## 示例

```cpp
#include <iostream>

class Rectangle {
private:
    int width;
    int height;

public:
    Rectangle(int w, int h) : width(w), height(h) {}

    // const 成员函数：获取宽度
    int getWidth() const {
        // width = 10;  // 错误！const 成员函数不能修改成员
        return width;
    }

    // const 成员函数：获取高度
    int getHeight() const {
        return height;
    }

    // const 成员函数：计算面积
    int getArea() const {
        return width * height;
    }

    // 非 const 成员函数：修改尺寸
    void resize(int w, int h) {
        width = w;
        height = h;
    }
};

int main() {
    // 普通对象：可以调用所有成员函数
    Rectangle rect1(10, 20);
    std::cout << "Area: " << rect1.getArea() << std::endl;
    rect1.resize(30, 40);  // OK

    // const 对象：只能调用 const 成员函数
    const Rectangle rect2(100, 200);
    std::cout << "Width: " << rect2.getWidth() << std::endl;  // OK
    // rect2.resize(1, 1);  // 错误！const 对象不能调用非 const 成员函数

    return 0;
}
```

## const 成员函数重载

可以根据 `const` 进行函数重载：

```cpp
class MyArray {
private:
    int* data;
    int size;

public:
    // 非 const 版本：返回引用，可以修改
    int& operator[](int index) {
        return data[index];
    }

    // const 版本：返回 const 引用，只能读取
    const int& operator[](int index) const {
        return data[index];
    }
};

int main() {
    MyArray arr;
    arr[0] = 10;  // 调用非 const 版本

    const MyArray constArr;
    int val = constArr[0];  // 调用 const 版本
    // constArr[0] = 10;    // 错误！const 版本返回 const 引用
}
```

---

## const 成员函数规则总结

| 情况 | 是否允许 |
|------|----------|
| const 函数读取成员变量 | ✓ |
| const 函数修改普通成员变量 | ✗ |
| const 函数修改 mutable 成员 | ✓ |
| const 函数调用其他 const 函数 | ✓ |
| const 函数调用非 const 函数 | ✗ |
| 非 const 对象调用 const 函数 | ✓ |
| const 对象调用 const 函数 | ✓ |
| const 对象调用非 const 函数 | ✗ |

---

## 为什么要使用 const 成员函数

1. **代码安全性**：编译器帮你检查不应该修改的地方
2. **接口清晰**：明确告诉使用者哪些方法不会修改对象
3. **允许 const 对象使用**：const 对象只能调用 const 成员函数
4. **支持 const 引用参数**：当函数参数是 `const ClassName&` 时，只能调用 const 成员函数

```cpp
void printRect(const Rectangle& rect) {
    // 只能调用 rect 的 const 成员函数
    std::cout << rect.getArea() << std::endl;  // OK
    // rect.resize(1, 1);  // 错误！
}
```

---

## 关键要点

1. **const 在末尾**：`returnType funcName() const;`
2. **不能修改成员**：const 成员函数不能修改非 mutable 成员变量
3. **mutable 例外**：mutable 成员可以在 const 函数中修改
4. **this 变化**：const 函数中 this 是 `const ClassName* const` 类型
5. **可以重载**：同名函数可以有 const 和非 const 两个版本
