# 12.4 类的成员函数

## 简介

成员函数是定义在类内部的函数，它可以访问类的所有成员（包括私有成员）。成员函数分为**类内定义**和**类外定义**两种方式。

---

## 类内定义 vs 类外定义

### 类内定义

在类的声明体内直接提供函数体。这种方式定义的成员函数**默认被视为 inline 函数**（尽管编译器可能不会真正内联）。

```cpp
class Role {
public:
    int hp;

    // 类内定义的成员函数（隐式 inline）
    int GetHp() {
        return hp;
    }
};
```

### 类外定义

在类声明中只提供函数原型（声明），在类外部使用**作用域解析运算符 `::`** 提供函数体（定义）。

```cpp
// 头文件 Role.h
class Role {
public:
    int hp;
    int GetHp();  // 声明
};

// 源文件 Role.cpp
int Role::GetHp() {  // 定义
    return hp;
}
```

## this 指针

`this` 是一个隐式传递给每个非静态成员函数的指针，指向调用该函数的**当前对象**。

### this 指针的特性

- `this` 的类型是 `ClassName* const`（指针本身是常量）
- 在 `const` 成员函数中，`this` 的类型是 `const ClassName* const`
- `this` 指针是**隐式传递**的，不需要显式声明

### 使用场景

```cpp
class Role {
private:
    int hp;

public:
    // 当参数名与成员名相同时，使用 this 区分
    void SetHp(int hp) {
        this->hp = hp;  // this->hp 是成员，hp 是参数
    }

    // 返回当前对象的引用（链式调用）
    Role& SetHpChain(int hp) {
        this->hp = hp;
        return *this;
    }
};

int main() {
    Role r;
    r.SetHpChain(100).SetHpChain(200);  // 链式调用
}
```

### this 指针示意图

```
┌─────────────────┐
│  Role 对象      │
│  ┌───────────┐  │
│  │ hp = 100  │  │ ← this 指向这个对象
│  └───────────┘  │
└─────────────────┘
```

当调用 `r.GetHp()` 时，编译器实际上将其转换为类似 `GetHp(&r)` 的形式，`this` 就是这个传入的指针。

## 成员函数的调用机制

成员函数的代码只有一份，存储在代码区，被所有对象共享。区分不同对象的方式是通过 `this` 指针。

```cpp
class Point {
private:
    int x;
    int y;

public:
    Point(int x_val, int y_val) : x(x_val), y(y_val) {}

    void move(int dx, int dy) {
        x += dx;
        y += dy;
        std::cout << "Point moved to (" << x << ", " << y << ")" << std::endl;
    }
};

int main() {
    Point p1(0, 0);
    Point p2(10, 20);

    p1.move(5, 5);   // this 指向 p1
    p2.move(3, 3);   // this 指向 p2
}
```

## 成员函数重载

成员函数可以重载，规则与普通函数相同：

```cpp
class Calculator {
public:
    int add(int a, int b) {
        return a + b;
    }

    double add(double a, double b) {
        return a + b;
    }

    int add(int a, int b, int c) {
        return a + b + c;
    }
};
```

## 内联成员函数

### 隐式内联

类内定义的成员函数**默认是内联的**：

```cpp
class MyClass {
public:
    void func() {  // 隐式 inline
        // 函数体
    }
};
```

### 显式内联

类外定义时可以使用 `inline` 关键字：

```cpp
class MyClass {
public:
    void func();
};

inline void MyClass::func() {  // 显式 inline
    // 函数体
}
```

> 关于 inline 的详细说明，参见 [[11.3 inline]]。

---

## 关键要点

1. **类内定义**：隐式 inline，适合简短函数
2. **类外定义**：使用 `::` 作用域解析符，适合复杂函数
3. **this 指针**：指向当前对象，用于区分成员和参数、实现链式调用
4. **代码共享**：所有对象共享同一份成员函数代码
5. **成员函数重载**：规则与普通函数相同
