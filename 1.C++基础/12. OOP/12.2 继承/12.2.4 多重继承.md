# 12.2.4 多重继承

## 简介

**多重继承**（Multiple Inheritance）是指一个派生类可以同时继承多个基类。这允许派生类组合多个基类的特性。

---

## 一般的多重继承

### 语法

```cpp
class DerivedClass : public Base1, public Base2 {
    // ...
};
```

### 示例：狼人类

```cpp
class Wolf {
public:
    void eat() {
        std::cout << "Wolf eat!\n";
    }

    void bite() {
        std::cout << "Wolf Bite!\n";
    }
};

class Man {
public:
    void eat() {
        std::cout << "human eat!\n";
    }
};

// WolfMan 同时继承 Wolf 和 Man
class WolfMan : public Wolf, public Man {
public:
    using Man::eat;  // 指定使用 Man 的 eat
};

int main() {
    WolfMan Jack;
    Jack.bite();       // 调用 Wolf::bite()
    Jack.Wolf::eat();  // 显式调用 Wolf::eat()
    Jack.eat();        // 调用 Man::eat()（因为 using 声明）
}
```

---

## 菱形继承问题（二义性）

当多重继承的两个基类有共同的祖先时，会产生**菱形继承**问题，也称为**钻石继承**。

### 问题示意

```
      MoveObject
       /      \
      /        \
   Wolf        Man
      \        /
       \      /
       WolfMan
```

### 问题代码

```cpp
class MoveObject {
public:
    int x;
    int y;
};

class Wolf : public MoveObject {
public:
    void eat() {
        std::cout << "Wolf eat!\n";
    }

    void bite() {
        std::cout << "Wolf Bite!\n";
    }
};

class Man : public MoveObject {
public:
    void eat() {
        std::cout << "human eat!\n";
    }
};

class WolfMan : public Wolf, public Man {
private:
    bool IsWolf = false;
public:
    void Change() {
        IsWolf = !IsWolf;
        std::cout << "变身!\n";
    }

    void eat() {
        if (IsWolf)
            Wolf::eat();
        else
            Man::eat();
    }
};

int main() {
    WolfMan Jack;
    Jack.x = 1;      // 错误！二义性：是 Wolf::x 还是 Man::x？
    Jack.y = 2;      // 错误！二义性

    Jack.Wolf::x = 250;   // 必须显式指定
    Jack.Man::x = 350;    // Wolf 和 Man 各有一份 x
}
```

**问题**：`WolfMan` 中存在**两份** `MoveObject` 的成员（一份来自 `Wolf`，一份来自 `Man`），导致：
- 访问 `x`、`y` 时产生二义性
- 浪费内存空间

---

## 使用虚继承解决菱形问题

**虚继承**（Virtual Inheritance）确保在菱形继承中，共同基类只存在一份副本。

### 语法

```cpp
class Wolf : public virtual MoveObject { ... };
class Man : public virtual MoveObject { ... };
```

### 解决方案代码

```cpp
class MoveObject {
public:
    int x;
    int y;
};

// 使用虚继承
class Wolf : public virtual MoveObject {
public:
    void eat() {
        std::cout << "Wolf eat!\n";
    }

    void bite() {
        std::cout << "Wolf Bite!\n";
    }
};

// 使用虚继承
class Man : public virtual MoveObject {
public:
    void eat() {
        std::cout << "human eat!\n";
    }
};

class WolfMan : public Wolf, public Man {
private:
    bool IsWolf = false;
public:
    void Change() {
        IsWolf = !IsWolf;
        std::cout << "变身!\n";
    }

    void eat() {
        if (IsWolf)
            Wolf::eat();
        else
            Man::eat();
    }
};

int main() {
    WolfMan Jack;
    Jack.x = 1;      // 正确！只有一份 x
    Jack.y = 2;      // 正确！只有一份 y

    Jack.Wolf::x = 250;   // 与 Jack.Man::x 是同一个变量
    Jack.Man::x = 350;    // 最终 x 的值为 350
    std::cout << Jack.Wolf::x;  // 输出 350
}
```

### 虚继承内存布局

**不使用虚继承**：
```
WolfMan 对象内存布局：
┌─────────────────┐
│  Wolf 部分       │
│  ├─ x (来自 MoveObject) │
│  ├─ y (来自 MoveObject) │
│  └─ Wolf 成员    │
├─────────────────┤
│  Man 部分        │
│  ├─ x (来自 MoveObject) │  ← 重复！
│  ├─ y (来自 MoveObject) │  ← 重复！
│  └─ Man 成员     │
├─────────────────┤
│  WolfMan 成员    │
└─────────────────┘
```

**使用虚继承**：
```
WolfMan 对象内存布局：
┌─────────────────┐
│  Wolf 部分       │
│  └─ vbptr (虚基类指针) │
├─────────────────┤
│  Man 部分        │
│  └─ vbptr (虚基类指针) │
├─────────────────┤
│  WolfMan 成员    │
├─────────────────┤
│  MoveObject 部分 │  ← 只有一份！
│  ├─ x           │
│  └─ y           │
└─────────────────┘
```

## 多重继承的注意事项

| 要点 | 说明 |
|------|------|
| **二义性** | 当多个基类有同名成员时，需要使用作用域解析运算符指定 |
| **菱形继承** | 使用 `virtual` 关键字进行虚继承，避免基类重复 |
| **构造顺序** | 虚基类最先构造，然后按声明顺序构造其他基类 |
| **性能开销** | 虚继承会引入额外的指针和间接访问开销 |
| **设计建议** | 优先考虑组合（composition）而非多重继承 |

---

## 关键要点

1. **多重继承语法**：`class D : public A, public B { };`
2. **二义性问题**：同名成员需要用 `Base::member` 显式指定
3. **菱形继承**：共同基类会被重复继承，造成二义性和内存浪费
4. **虚继承**：使用 `virtual` 关键字确保共同基类只存在一份
5. **using 声明**：可以使用 `using Base::member;` 选择使用哪个基类的成员
6. **更多细节**：参见 [[12.2.5 虚基类|虚基类]]
