# 判断运算符

## if 语句

### 基本语法

```cpp
if (条件) {
    变量声明;
}
else {
    变量声明;
}
```

### C++17 新语法：条件中声明变量

C++17 允许在 `if` 条件中同时声明变量：

```cpp
if (变量声明; 条件) {
    // ...
}
else {
    // ...
}
```

**示例**：

```cpp
if (int a, int b, int c; true) {
    std::cout << a << b << c;
}
else {
    std::cout << a + 1 << b + 1 << c + 1;
}
```

## switch 语句

### 基本语法

```cpp
switch (值) {
    case 选项1:
        break;
    case 选项2:
        break;
    case 选项3:
        break;
    case 选项4:
        break;
    default:
        break;
}
```

### 示范用法：学生抽奖

```cpp
int main()
{
    unsigned int sId{}, sPr{};
    std::cin >> sId;
    switch (sId)
    {
        case 15692:
            sPr = 500;
            break;
        case 65421:
            sPr = 450;
            break;
        case 98741:
            sPr = 250;
            break;
        case 2531:
            sPr = 150;
            break;
        case 666:
            [[fallthrough]];  // C++17：显式声明 fallthrough
        case 321:
            sPr = sId * 2;
            break;
        default:
            sPr = 0;
            std::cout << "对不起，同学，您本次活动没有中奖，下次继续！\n";
            break;
    }
    std::cout << "同学你的奖金总额为" << sPr << std::endl;
}
```

> **注意**：`[[fallthrough]]` 是 C++17 引入的属性，用于显式声明 case 穿透行为，避免编译器警告。

### C++17 新语法：switch 中声明变量

```cpp
switch (int a{}; 3) {
    case 1:
        a++;
    case 2:
        a++;
    case 3:
        a++;
        break;
}
```

## goto 语句

`goto` 语句可以无条件跳转到程序中指定的标签位置，通常与 `if` 配合使用实现条件跳转。

### 示例代码

```cpp
int main()
{
    char a;

linput:
    printf("请您输入一个大写字母，我们会将其转换为小写字母\n");
    std::cin >> a;

    if (a > 64 && a < 91) {  // ASCII: 65='A', 90='Z'
        a += 32;              // 大写转小写
        std::cout << a;
    } else {
        printf("您输入的不是字符\n");
        goto linput;
    }
}
```

> **说明**：通常 `goto` 是配合 `if` 一起使用的，用于实现简单的循环或错误重试逻辑。现代 C++ 中应尽量避免使用 `goto`。

## 汇编分析

### if-else 的汇编实现

**代码部分**：

```cpp
if (a > b) {
    // do something
} else {
    // do something else
}
```

**汇编实现**：

1. **比较操作**：编译器首先将变量 a 和 b 的值加载到寄存器中，并通过 `cmp` 指令进行比较
2. **条件跳转**：根据比较结果，会使用条件跳转指令（如 `jg`, `jl`, `je` 等），这些指令会检查条件是否成立，并跳转到对应的代码块

#### 常用条件跳转指令

| 指令 | 含义 | 使用场景 |
|------|------|----------|
| `je` | Jump if Equal | ZF=1 时跳转，用于比较两个值是否相等 |
| `jg` | Jump if Greater | 有符号数比较，第一个操作数大于第二个数时跳转 |
| `jl` | Jump if Less | 有符号数比较，第一个操作数小于第二个数时跳转 |
| `jz` | Jump if Zero | 等价于 `je`，因为相等时零标志位会被置 1 |

**汇编示例**：

```asm
mov eax, [a]      ; 将变量 a 的值加载到寄存器 eax 中
mov ebx, [b]      ; 将变量 b 的值加载到寄存器 ebx 中
cmp eax, ebx      ; 比较 a 和 b 的值
jg  L1            ; 如果 a > b，则跳转到 L1 代码块
; else 部分代码
jmp L2            ; 执行完 else 部分后跳转到 L2
L1:
; if 部分代码
L2:
```

### switch 的汇编实现

**代码部分**：

```cpp
switch (x) {
    case 1:
        // do something
        break;
    case 2:
        // do something else
        break;
    // ...
    default:
        // do default
        break;
}
```

**两种汇编实现方式**：

#### 1. 跳转表（Jump Table）

适用于 **case 值范围紧凑** 的情况，编译器可以生成一个跳转表，通过表中的索引值直接跳转到相应的代码块。

**实现逻辑**：
- 如果 case 值是紧凑的（如 1 到 5），编译器会使用跳转表
- 先通过 x 的值确定表中对应的偏移，然后直接跳转到对应的 case 代码块

```asm
mov eax, [x]                    ; 将变量 x 的值加载到寄存器 eax 中
cmp eax, 5                      ; 确保 x 的值在合理范围内
ja  default_case                ; 如果 x > 5，跳转到 default
jmp [jump_table + eax * 4]      ; 根据 x 的值跳转到相应的 case 代码块

jump_table:
    dd case_1        ; case 1 的地址
    dd case_2        ; case 2 的地址
    dd case_3        ; ...
    dd default_case  ; default 的地址
```

> - `jmp [jump_table + eax * 4]`：根据 x 的值在跳转表中查找对应的地址，并跳转到相应的代码块
> - `dd`：数据定义，用于生成跳转表

#### 2. 链式比较（Cascaded Comparisons）

适用于 **case 值范围稀疏** 的情况，编译器可能会使用链式比较，也就是类似于多个 if-else 的结构。

```asm
mov eax, [x]
cmp eax, 1
je  case_1         ; 如果 x == 1，跳转到 case_1
cmp eax, 2
je  case_2         ; 如果 x == 2，跳转到 case_2
jmp default_case   ; 如果都不满足，跳转到 default
```

### 总结

| 语句 | 汇编实现方式 |
|------|--------------|
| `if-else` | 主要通过条件判断和条件跳转指令（如 `jg`, `je`, `jl`）实现，在汇编中，通常会通过 `cmp` 进行比较，后面紧跟条件跳转指令 |
| `switch` | 有两种主要实现方式：**跳转表**（通过 `jmp` 和查表跳转）适用于 case 值紧凑的情况；**链式比较**（类似 if-else 的条件跳转）适用于 case 值稀疏的情况 |
