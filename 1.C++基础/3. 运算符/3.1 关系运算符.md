## 关系运算符

关系运算符用于比较两个值之间的关系，返回布尔值 `true`（1）或 `false`（0）。

### 运算符列表

| 运算符 | 含义 | 示例 | 结果 |
|--------|------|------|------|
| `==` | 等于 | `5 == 5` | `true` |
| `!=` | 不等于 | `5 != 3` | `true` |
| `<` | 小于 | `3 < 5` | `true` |
| `>` | 大于 | `5 > 3` | `true` |
| `<=` | 小于等于 | `5 <= 5` | `true` |
| `>=` | 大于等于 | `5 >= 3` | `true` |

---

### 基本用法

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 10, b = 20;

    cout << boolalpha;  // 让 bool 输出 true/false 而非 1/0

    cout << "a == b: " << (a == b) << endl;  // false
    cout << "a != b: " << (a != b) << endl;  // true
    cout << "a < b: "  << (a < b)  << endl;  // true
    cout << "a > b: "  << (a > b)  << endl;  // false
    cout << "a <= b: " << (a <= b) << endl;  // true
    cout << "a >= b: " << (a >= b) << endl;  // false

    return 0;
}
```

### 常见错误：`=` 与 `==` 混淆

```cpp
int x = 5;

// 错误写法：赋值而非比较
if (x = 10) {  // 将 10 赋给 x，表达式值为 10（非零即真）
    cout << "永远会执行" << endl;
}

// 正确写法
if (x == 10) {
    cout << "x 等于 10" << endl;
}
```

> **技巧**：可以把常量写在左边 `if (10 == x)`，这样误写成 `=` 时编译器会报错。

### 浮点数比较的陷阱

由于浮点数的精度问题，直接使用 `==` 比较可能得到错误结果：

```cpp
#include <iostream>
#include <cmath>
using namespace std;

int main() {
    double a = 0.1 + 0.2;
    double b = 0.3;

    // 直接比较可能失败
    cout << boolalpha;
    cout << "a == b: " << (a == b) << endl;  // 可能输出 false

    // 正确做法：使用误差范围（epsilon）比较
    const double epsilon = 1e-9;
    if (abs(a - b) < epsilon) {
        cout << "a 和 b 近似相等" << endl;
    }

    return 0;
}
```

**原因**：`0.1 + 0.2` 在二进制浮点表示中并不精确等于 `0.3`。

### 字符与字符串比较

#### 字符比较（基于 ASCII 码）

```cpp
char c1 = 'A';  // ASCII: 65
char c2 = 'a';  // ASCII: 97

cout << (c1 < c2) << endl;   // true，因为 65 < 97
cout << (c1 == 'A') << endl; // true
```

#### C 风格字符串比较

```cpp
#include <cstring>

char str1[] = "hello";
char str2[] = "hello";

// 错误：比较的是地址，不是内容
if (str1 == str2) { }  // false，地址不同

// 正确：使用 strcmp
if (strcmp(str1, str2) == 0) {
    cout << "字符串相等" << endl;
}
```

#### C++ string 比较

```cpp
#include <string>

string s1 = "hello";
string s2 = "hello";

// 可以直接使用关系运算符
if (s1 == s2) {
    cout << "字符串相等" << endl;  // 正确输出
}
```

### 运算符优先级

更加的详细的运算符优先级排序可以参考[[2.2 格式化输出流、运算符优先级#运算符优先级表|运算符优先级表]]
关系运算符的优先级：
- `<`、`>`、`<=`、`>=` 优先级相同（较高）
- `==`、`!=` 优先级相同（较低）

```cpp
int a = 5, b = 10, c = 5;

// a < b == c 等价于 (a < b) == c
// (5 < 10) == 5 → true == 5 → 1 == 5 → false
cout << (a < b == c) << endl;  // false

// 建议：使用括号明确优先级
cout << ((a < b) && (b > c)) << endl;  // true
```

---

### 关键要点

1. 关系运算符返回 `bool` 类型（`true` 或 `false`）
2. 注意 `=`（赋值）和 `==`（比较）的区别
3. 浮点数比较要使用误差范围
4. C 风格字符串比较要用 `strcmp`，C++ `string` 可直接比较
5. 关系表达式可用于 `if`、`while`、`for` 等控制语句
