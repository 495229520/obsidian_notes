## 条件运算符（三元运算符）

条件运算符 `? :` 是 C++ 中唯一的三元运算符，可以根据条件选择两个值中的一个。

### 基本语法

```
条件表达式 ? 表达式1 : 表达式2
```

- 如果条件为 `true`，返回表达式1的值
- 如果条件为 `false`，返回表达式2的值

---

### 基本用法

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 10, b = 20;

    // 使用条件运算符找出较大值
    int max = (a > b) ? a : b;
    cout << "较大值: " << max << endl;  // 输出: 20

    // 等价的 if-else 写法
    int max2;
    if (a > b) {
        max2 = a;
    } else {
        max2 = b;
    }

    return 0;
}
```

---

### 条件运算符 vs if-else

| 特性 | 条件运算符 | if-else |
|------|-----------|---------|
| 类型 | 表达式（有返回值） | 语句（无返回值） |
| 用途 | 简单的二选一赋值 | 复杂的条件逻辑 |
| 可读性 | 简洁但复杂时难读 | 清晰直观 |
| 嵌套 | 可以但不推荐 | 更适合多层嵌套 |

---

### 实际应用场景

#### 1. 初始化变量

```cpp
int score = 85;
string grade = (score >= 60) ? "及格" : "不及格";
```

#### 2. 函数返回值

```cpp
int abs(int x) {
    return (x >= 0) ? x : -x;
}
```

#### 3. 输出控制

```cpp
int count = 1;
cout << count << " item" << ((count > 1) ? "s" : "") << endl;
// 输出: 1 item

count = 5;
cout << count << " item" << ((count > 1) ? "s" : "") << endl;
// 输出: 5 items
```

#### 4. 求最值

```cpp
int a = 5, b = 3, c = 8;

// 两数最大值
int max2 = (a > b) ? a : b;

// 三数最大值（嵌套使用）
int max3 = (a > b) ? ((a > c) ? a : c) : ((b > c) ? b : c);
```

---

### 嵌套条件运算符

可以嵌套使用，但要注意可读性：

```cpp
int score = 75;

// 嵌套写法（不推荐过度嵌套）
string level = (score >= 90) ? "优秀" :
               (score >= 80) ? "良好" :
               (score >= 60) ? "及格" : "不及格";

cout << level << endl;  // 输出: 及格
```

> **注意**：超过两层嵌套时，建议改用 `if-else` 或 `switch` 以提高可读性。

---

### 条件运算符的返回类型

条件运算符的返回类型由表达式1和表达式2共同决定：

```cpp
int a = 10;
double b = 20.5;

// 结果类型为 double（int 会被提升为 double）
auto result = (a > 5) ? a : b;
cout << result << endl;  // 输出: 10（但类型是 double）

// 明确类型转换
int intResult = (a > 5) ? a : static_cast<int>(b);
```

---

### 作为左值使用（C++ 特性）

在 C++ 中，如果表达式1和表达式2都是左值，条件运算符的结果也可以是左值：

```cpp
int a = 10, b = 20;

// 条件运算符作为左值
((a > b) ? a : b) = 100;  // 将较大的那个变量赋值为 100

cout << "a = " << a << ", b = " << b << endl;
// 输出: a = 10, b = 100
```

---

### 运算符优先级

条件运算符的优先级很低，仅高于赋值运算符和逗号运算符：

```cpp
int x = 5;

// 可能出错的写法
cout << x > 3 ? "大" : "小";  // 编译错误！

// 正确写法：加括号
cout << (x > 3 ? "大" : "小") << endl;
```

**运算符优先级（从高到低）**：
1. 算术运算符（`+`、`-`、`*`、`/`）
2. 关系运算符（`<`、`>`、`==`、`!=`）
3. 逻辑运算符（`&&`、`||`）
4. **条件运算符（`?:`）**
5. 赋值运算符（`=`、`+=`）

---

### 常见错误

#### 1. 忘记括号

```cpp
// 错误
int max = a > b ? a : b + 10;
// 实际解析为: a > b ? a : (b + 10)

// 正确
int max = (a > b ? a : b) + 10;
```

#### 2. 类型不匹配

```cpp
int a = 5;
// 错误：两个分支类型差异太大
// auto x = (a > 0) ? "正数" : 123;  // 编译可能出问题

// 正确：保持类型一致
const char* x = (a > 0) ? "正数" : "非正数";
```

---

### 关键要点

1. 条件运算符是表达式，有返回值，适合简单的二选一
2. 语法：`条件 ? 值1 : 值2`
3. 避免过度嵌套，超过两层考虑使用 `if-else`
4. 注意优先级，必要时使用括号
5. 两个分支的类型应该兼容
