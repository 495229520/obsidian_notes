# 6.9 总结

> 本节是第六章 [[6.1基本指针|指针]] 系列的总结，汇总所有关键概念和最佳实践。

---

## 函数声明时避免直接在指针内存空间

### 指针作为函数返回值

```cpp
// type* functionName();

int main() {
    int a;
    // 声明指针
    int* ptr = &a;
}
```

### 注意事项

- **不能返回局部变量的地址**：函数返回时局部变量被销毁，返回的指针成为[[6.3 动态内存分配的风险#2. 悬空指针 (Dangling Pointer)|悬空指针]]
- 如果需要返回指针，应返回：
  - 动态分配的内存（调用者负责释放）
  - 静态变量的地址
  - 传入参数的地址

## 指针运算

### 指针与整数运算

```cpp
int arr[5] = {1, 2, 3, 4, 5};
int* ptr = arr;

// ptr + i 指向 arr[i]
std::cout << *(ptr + 1) << std::endl;  // 2
std::cout << *(ptr + 2) << std::endl;  // 3
```

### 指针与指针运算

```cpp
int arr[5] = {1, 2, 3, 4, 5};
int* ptr = arr;
int* ptr2 = &arr[3];

// 指针相减得到元素个数差
std::cout << ptr2 - ptr << std::endl;  // 3
```

## 指针访问数组

### 一维数组

```cpp
int arr[5] = {1, 2, 3, 4, 5};
int* ptr = arr;

// 声明一个 5 个 int 元素的数组
std::cout << "当前访问数据的存储地址：" << ptr << std::endl;

// 遍历数组
for (int i = 0; i < 5; i++) {
    std::cout << "arr[" << i << "] = " << *(ptr + i) << std::endl;
}
```

### 二维数组

```cpp
int size = 5;
int* arr = (int*)malloc(size * sizeof(int));

// 正确释放
free(arr);
```

### 指针遍历二维数组

```cpp
#include <iostream>
#include <cstdlib>

int main() {
    int size = 5;
    int* arr = (int*)malloc(size * sizeof(int));

    for (int i = 0; i < size; i++) {
        arr[i] = i;
    }

    // 使用指针遍历
    for (int i = 0; i < size; i++) {
        std::cout << "arr[" << i << "] = " << *(arr + i) << std::endl;
    }

    free(arr);
    return 0;
}
```

## 常见错误和最佳实践

### 多维数组的内存分配

```cpp
#include <iostream>
#include <cstdlib>

int main() {
    int rows = 3;
    int cols = 5;

    // 分配行指针数组
    int** arr = (int**)malloc(rows * sizeof(int*));

    // 为每行分配内存
    for (int i = 0; i < rows; i++) {
        arr[i] = (int*)malloc(cols * sizeof(int));
    }

    // 初始化
    for (int i = 0; i < rows; i++) {
        for (int j = 0; j < cols; j++) {
            arr[i][j] = i * cols + j;
        }
    }

    // 遍历输出
    for (int i = 0; i < rows; i++) {
        for (int j = 0; j < cols; j++) {
            std::cout << arr[i][j] << " ";
        }
        std::cout << std::endl;
    }

    // 释放内存（先释放每行，再释放行指针数组）
    for (int i = 0; i < rows; i++) {
        free(arr[i]);
    }
    free(arr);

    return 0;
}
```

## 共识整合

### 使用指针时避免问题

```cpp
int main() {
    int a;
    int* ptr = &a;
    // ptr 指向一个合法内存
    // 作用域结束后 a 被销毁，ptr 依然有效但不应继续使用
}
```

### 正确的内存管理

```cpp
#include <iostream>

int main() {
    int size = 5;
    int* arr = new int[size];

    for (int i = 0; i < size; i++) {
        arr[i] = i + 1;
        std::cout << "arr[" << i << "] = " << arr[i] << std::endl;
    }

    delete[] arr;  // 正确释放
    arr = nullptr; // 置空，避免悬空指针
    return 0;
}
```

## 核心要点总结

| 概念 | 要点 |
|------|------|
| **指针** | 存储内存地址的变量，可动态指向不同位置 |
| **引用** | 变量的别名，必须初始化且不能改变绑定 |
| **动态内存** | `new`/`delete` 配对使用，注意数组用 `delete[]` |
| **内存风险** | 泄漏、悬空指针、重复释放、错误释放 |
| **数组 vs 指针** | 数组名是常量指针，sizeof 行为不同 |
| **堆 vs 栈** | 堆手动管理空间大，栈自动管理速度快 |
| **指针与结构体** | 使用 `->` 访问成员，注意内存对齐 |
| **unique_ptr** | 独占所有权，不可复制只能移动 |
| **shared_ptr** | 共享所有权，引用计数自动释放 |
| **weak_ptr** | 弱引用，用于打破循环引用 |

---

## 最佳实践清单

- [ ] 指针使用前检查是否为 `nullptr`
- [ ] `new`/`delete` 和 `new[]`/`delete[]` 配对使用
- [ ] 释放内存后将指针置为 `nullptr`
- [ ] 不要返回局部变量的地址
- [ ] 使用引用传参避免不必要的拷贝
- [ ] 大对象或需要长期存在的对象使用堆分配
- [ ] 小对象或临时对象使用栈分配
- [ ] **优先使用智能指针**替代原始指针
- [ ] 独占资源使用 `unique_ptr`
- [ ] 共享资源使用 `shared_ptr`
- [ ] 避免循环引用，必要时使用 `weak_ptr`
