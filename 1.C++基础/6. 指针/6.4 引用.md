# 6.4 引用

> 引用是 C++ 对 [[6.1基本指针|指针]] 的一种更安全的替代方案，提供了更简洁的语法。

## 简介

**引用**为已存在的一个变量提供一个别名，而非存储一个新的值。引用**一定需要在声明的时候初始化**，而且引用**不能为空**。

```
数据类型 &引用名 = 已存在的变量;
```

---

## 示例

### 基本用法

```cpp
#include <iostream>

int main() {
    int num = 10;
    // 声明一个引用 ref 并初始化为 num 的别名
    int& ref = num;

    std::cout << "num 的值：" << num << std::endl;
    std::cout << "ref 的值：" << ref << std::endl;

    return 0;
}
```

**输出**：
```
num 的值：10
ref 的值：10
```

### 引用与原变量共享地址

```cpp
#include <iostream>

int main() {
    int num = 10;
    int& ref = num;

    // 输出变量 num 的地址
    std::cout << "变量 num 的地址：" << &num << std::endl;
    // 输出引用 ref 的地址
    std::cout << "引用 ref 的地址：" << &ref << std::endl;

    return 0;
}
```

**输出**：
```
变量 num 的地址：000000D062CFF5F4
引用 ref 的地址：000000D062CFF5F4
```

> **关键点**：引用和原变量指向**同一块内存地址**。

---

## 优点：函数参数传递

**通过引用传递参数**：可以避免值传递带来的额外拷贝开销，若函数内对参数进行改变也会改变外界参数的值，否则不会改变外界的值。

### 示例：通过引用传递参数

```cpp
#include <iostream>

// 通过引用传递参数
void swap(int& a, int& b) {
    int temp = a;
    a = b;
    b = temp;
}

int main() {
    int x = 5, y = 10;
    swap(x, y);
    std::cout << "x = " << x << ", y = " << y << std::endl;
    return 0;
}
```

**输出**：
```
x = 10, y = 5
```

---

## 函数可以返回一个引用

这样可以直接操作函数返回的对象，此时 `arr[2] = 10`。

```cpp
#include <iostream>

int arr[5] = {1, 2, 3, 4, 5};

// 返回数组元素的引用
int& getElement(int index) {
    return arr[index];
}

int main() {
    getElement(2) = 10;
    std::cout << "arr[2] = " << arr[2] << std::endl;
    return 0;
}
```

**输出**：
```
arr[2] = 10
```

## 本质

从上面的汇编代码可以看到引用实际存储地址的指针和存放对象的地址是一样的。

### 汇编层面分析

```cpp
int a = 5;
```
```asm
00044202 mov    dword ptr [ebp-0Ch], 5
```

```cpp
int& b = a;
```
```asm
00044210 lea    eax, [ebp-0Ch]
00044213 mov    dword ptr [ebp-18h], eax
```

```cpp
int& c = a;
```
```asm
00044216 lea    eax, [ebp-0Ch]
00044219 mov    dword ptr [ebp-24h], eax
```

```cpp
int* d = &a;
```
```asm
0004421C lea    eax, [ebp-0Ch]
0004421F mov    dword ptr [ebp-30h], eax
```

```cpp
b = 6;
```
```asm
00044222 mov    eax, dword ptr [ebp-18h]
00044225 mov    dword ptr [eax], 6
```

```cpp
c = 6;
```
```asm
00044228 mov    eax, dword ptr [ebp-24h]
0004422E mov    dword ptr [eax], 6
```

```cpp
*d = 6;
```
```asm
00044234 mov    eax, dword ptr [ebp-30h]
00044237 mov    dword ptr [eax], 6
```

```cpp
a = (int)&a;
```
```asm
0004423D lea    eax, [ebp-0Ch]
00044240 mov    dword ptr [ebp-0Ch], eax
```

### 汇编分析结论

应用的底层可以视为是一个**解引之后的指针**：常量指针。引用在底层通常被实现为一个**自动解引用**的指针（隐藏的常量指针）。

## 引用 vs 指针 对比

| 特性 | 指针 (Pointer) | 引用 (Reference) | 底层实现 |
|------|----------------|------------------|----------|
| **本质** | 存储地址的变量 | 已确定地址的名，是既定内存地址的别名 | |
| **占内存** | 占用（4或8字节） | 占用（4或8字节） | |
| **可为空** | 可以是 `nullptr`（空指针，不能为 `NULL`） | 必须绑定一个对象，不能为 `NULL` | |
| **可修改** | 可以指向不同的对象 | 初始化后不能指向其他对象 | |
| **间接寻址** | 需要手动使用 `*` | 编译器自动处理 | |

## 关键要点

1. **引用是别名**：引用不是新变量，而是已存在变量的另一个名字
2. **必须初始化**：声明时必须绑定到一个变量，之后不能改变绑定
3. **不能为空**：引用必须指向有效对象，不存在"空引用"
4. **函数参数**：使用引用传参可以避免拷贝，提高效率
5. **底层实现**：引用本质是一个自动解引用的常量指针
