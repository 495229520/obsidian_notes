## 格式化输出流操作

> 红色标记的操作符需要包含头文件 `<iomanip>`

| 操作符                      | 功能说明                                      | 示例                           |
| ------------------------ | ----------------------------------------- | ---------------------------- |
| `std::fixed`             | 以小数点模式输出浮点数                               | `3.1415926`                  |
| `std::scientific`        | 以科学计数法输出小数                                | `3E100`                      |
| `std::defaultfloat`      | 恢复默认的小数输出                                 | -                            |
| `std::setprecision(int)` | 设置小数精度（需要 `<iomanip>`）                    | `setprecision(2)` 后输出 `3.14` |
| `std::dec`               | 以后输出的数字采用十进制输出                            | `999`                        |
| `std::hex`               | 以后输出的数字采用十六进制输出                           | `FFF`                        |
| `std::oct`               | 以后输出的数字采用八进制输出                            | `777`                        |
| `std::showbase`          | 十六进制和八进制显示前缀                              | `0xFF`                       |
| `std::noshowbase`        | 关闭十六进制和八进制显示前缀                            | -                            |
| `std::setw(int)`         | 设置输出内容宽度（需要 `<iomanip>`）                  | `setw(10)` 输出 `       999`   |
| `std::setfill(char)`     | 当显示宽度超过字符宽度时，用指定字符串填充剩余内容（需要 `<iomanip>`） | `********999`                |
| `std::left`              | 设置字符对齐模式为左对齐                              | -                            |
| `std::right`             | 设置字符对齐模式为右对齐                              | -                            |

### 使用示例

```cpp
#include <iostream>
#include <iomanip>

int main() {
    double pi = 3.141592653589793;

    // 固定小数点格式
    std::cout << std::fixed << std::setprecision(2) << pi << std::endl;  // 输出: 3.14

    // 科学计数法
    std::cout << std::scientific << pi << std::endl;  // 输出: 3.14e+00

    // 进制转换
    int num = 255;
    std::cout << std::dec << num << std::endl;  // 输出: 255
    std::cout << std::hex << std::showbase << num << std::endl;  // 输出: 0xff
    std::cout << std::oct << num << std::endl;  // 输出: 0377

    // 宽度和填充
    std::cout << std::dec << std::setw(10) << std::setfill('*') << 999 << std::endl;  // 输出: *******999

    return 0;
}
```

## 运算符优先级表

> 优先级从高到低排列，同一行的运算符优先级相同

| 运算符 | 结合性 |
|--------|--------|
| `::` | 左 |
| `()` `[]` `->` `.` 后缀`++` 后缀`-` `typeid` `const_cast` `dynamic_cast` `static_cast` `reinterpret_cast` | 左 |
| `!` `~` 一元`+` 一元`-` 前缀`++` 前缀`--` `&` `*`（解引用） `sizeof` `new` `new[]` `delete` `delete[]` | 右 |
| `.*` `->*` | 左 |
| `*` `/` `%` | 左 |
| `+` `-` | 左 |
| `<<` `>>` | 左 |
| `<` `<=` `>` `>=` | 左 |
| `==` `!=` | 左 |
| `&`（按位与） | 左 |
| `^`（按位异或） | 左 |
| `\|`（按位或） | 左 |
| `&&` | 左 |
| `\|\|` | 左 |
| `?:` `op=`（复合赋值） | 右 |
| `throw` | 右 |
| `,` | 左 |

## 自增运算符 `a++` 和 `++a` 的反汇编分析

### 单独使用时的反汇编代码

```asm
; a++;
002717DF mov     eax, dword ptr [a]    ; 将a的值加载到eax
002717E2 add     eax, 1                 ; eax加1
002717E5 mov     dword ptr [a], eax    ; 将结果存回a

; ++a;
002717E8 mov     eax, dword ptr [a]    ; 将a的值加载到eax
002717EB add     eax, 1                 ; eax加1
002717EE mov     dword ptr [a], eax    ; 将结果存回a
```

> **结论**：单独使用时，`a++` 和 `++a` 生成的汇编代码完全相同，没有性能差异。

### 在表达式中使用时的反汇编分析

#### `c = b + a++;` 的反汇编

```asm
009B3CDF mov     eax, dword ptr [b]    ; eax = b
009B3CE2 add     eax, dword ptr [a]    ; eax = eax + a（先使用a的原值）
009B3CE5 mov     dword ptr [c], eax    ; c = eax
009B3CE8 mov     ecx, dword ptr [a]    ; ecx = a
009B3CEB add     ecx, 1                 ; ecx = ecx + 1
009B3CEE mov     dword ptr [a], ecx    ; a = ecx（后自增）
```

#### `c = b + ++a;` 的反汇编

```asm
009B3CF1 mov     eax, dword ptr [a]    ; eax = a
009B3CF4 add     eax, 1                ; eax = eax + 1（先自增）
009B3CF7 mov     dword ptr [a], eax    ; a = eax
009B3CFA mov     ecx, dword ptr [b]    ; ecx = b
009B3CFD add     ecx, dword ptr [a]    ; ecx = ecx + a（使用自增后的值）
009B3D00 mov     dword ptr [c], ecx    ; c = ecx
```

### 复杂表达式求值顺序

| 表达式            | 求值过程                                                       |
| -------------- | ---------------------------------------------------------- |
| `c = b+++a`    | 后缀优先，解析为 `c = (b++) + a`，先计算 `b++`，再计算 `b+a`               |
| `c = b---a`    | 后缀优先，解析为 `c = (b--) - a`，先计算 `b--`，再计算 `b-a`               |
| `c = 3*-b+++a` | 1. 计算 `3*(-b)` <br> 2. `b = b + 1` <br> 3. 计算 `3*(-b) + a` |
| `c = 2*-b---a` | 1. 计算 `2*(-b)` <br> 2. `b = b - 1` <br> 3. 计算 `2*(-b) - a` |
