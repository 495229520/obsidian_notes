# vector 数组

## 简介

`std::vector` 是 C++ 标准模板库（STL）中的**动态数组**容器，与 [[4.1 基本数组]] 中的静态数组不同，它可以在运行时动态改变大小。

```cpp
#include <vector>
```

## 与普通数组的比较

| 特性 | 普通数组（静态数组） | vector（动态数组） |
|------|---------------------|-------------------|
| 大小 | 编译时确定，不能改变 | 运行时可动态改变 |
| 内存分配 | 栈上分配（局部数组） | 堆上分配 |
| 边界检查 | 无 | `at()` 提供边界检查 |
| 内存管理 | 手动或自动（栈） | 自动管理 |
| 获取大小 | `sizeof(arr)/sizeof(arr[0])` | `size()` 成员函数 |

## 声明和初始化

```cpp
// 声明空 vector
std::vector<int> vec1;

// 声明指定大小的 vector，元素默认初始化为 0
std::vector<int> vec2(5);

// 声明指定大小并初始化为特定值
std::vector<int> vec3(5, 10);  // 5 个元素，全部为 10

// 使用初始化列表
std::vector<int> vec4 = {1, 2, 3, 4, 5};
std::vector<int> vec5{1, 2, 3, 4, 5};

// 从另一个 vector 复制
std::vector<int> vec6(vec4);

// 从数组初始化
int arr[] = {1, 2, 3};
std::vector<int> vec7(arr, arr + 3);
```

## 内存管理：size 与 capacity

`vector` 内部维护两个重要的概念：

- **size**：当前实际存储的元素数量
- **capacity**：当前分配的内存能容纳的元素数量

```
内存布局示意：
┌───┬───┬───┬───┬───┬───┬───┬───┐
│ 1 │ 2 │ 3 │ 4 │ 5 │   │   │   │
└───┴───┴───┴───┴───┴───┴───┴───┘
 ↑                   ↑           ↑
 begin              size      capacity
                   (5个)       (8个)
```

> 当 `size` 超过 `capacity` 时，`vector` 会自动扩容（通常是翻倍），这涉及重新分配内存并复制所有元素。

## 常用成员函数

### 元素访问

| 函数 | 说明 |
|------|------|
| `at(index)` | 访问指定位置的元素，**有边界检查**，越界抛出 `std::out_of_range` 异常 |
| `operator[index]` | 访问指定位置的元素，**无边界检查** |
| `front()` | 返回第一个元素的引用 |
| `back()` | 返回最后一个元素的引用 |
| `data()` | 返回指向底层数组的指针 |

```cpp
std::vector<int> vec = {10, 20, 30, 40, 50};

vec[0];       // 10，无边界检查
vec.at(1);    // 20，有边界检查
vec.front();  // 10
vec.back();   // 50
```

### 容量相关

| 函数 | 说明 |
|------|------|
| `size()` | 返回当前元素数量 |
| `capacity()` | 返回当前容量 |
| `empty()` | 检查是否为空 |
| `reserve(n)` | 预留至少能容纳 n 个元素的空间（不改变 size） |
| `resize(n)` | 改变 size 为 n（可能添加或删除元素） |
| `shrink_to_fit()` | 释放多余内存，使 capacity 等于 size |

```cpp
std::vector<int> vec;
vec.reserve(100);    // 预留空间，避免频繁扩容
// vec.size() == 0, vec.capacity() >= 100

vec.resize(10);      // 现在有 10 个元素（默认初始化为 0）
// vec.size() == 10

vec.shrink_to_fit(); // 释放多余内存
```

### 修改操作

| 函数 | 说明 | 时间复杂度 |
|------|------|-----------|
| `push_back(val)` | 在末尾添加元素 | 均摊 O(1) |
| `pop_back()` | 删除末尾元素 | O(1) |
| `insert(pos, val)` | 在指定位置插入元素 | O(n) |
| `erase(pos)` | 删除指定位置的元素 | O(n) |
| `clear()` | 清空所有元素（不释放内存） | O(n) |

```cpp
std::vector<int> vec = {1, 2, 3};

vec.push_back(4);              // {1, 2, 3, 4}
vec.pop_back();                // {1, 2, 3}

vec.insert(vec.begin() + 1, 10);  // {1, 10, 2, 3}
vec.erase(vec.begin());           // {10, 2, 3}

vec.clear();                   // {}，size=0，capacity 不变
```

## 迭代器

```cpp
std::vector<int> vec = {1, 2, 3, 4, 5};

// 正向迭代
for (auto it = vec.begin(); it != vec.end(); ++it) {
    std::cout << *it << " ";
}

// 反向迭代
for (auto it = vec.rbegin(); it != vec.rend(); ++it) {
    std::cout << *it << " ";
}

// 范围 for 循环（推荐）
for (int x : vec) {
    std::cout << x << " ";
}

// 使用引用避免拷贝
for (const auto& x : vec) {
    std::cout << x << " ";
}
```

## 扩容机制

当 `push_back()` 导致 `size` 超过 `capacity` 时，`vector` 会：

1. 分配一块更大的内存（通常是原来的 **2 倍**，具体取决于实现）
2. 将所有元素复制/移动到新内存
3. 释放旧内存

```cpp
std::vector<int> vec;
std::cout << "size\tcapacity\n";
for (int i = 0; i < 10; ++i) {
    vec.push_back(i);
    std::cout << vec.size() << "\t" << vec.capacity() << "\n";
}
```

**可能的输出**（MSVC）：
```
size    capacity
1       1
2       2
3       3
4       4
5       6
6       6
7       9
8       9
9       9
10      13
```

> **性能提示**：如果预先知道元素数量，使用 `reserve()` 预留空间可以避免多次扩容带来的性能开销。

## emplace_back vs push_back

`emplace_back()` 直接在容器内**原地构造**对象，避免了临时对象的创建和拷贝：

```cpp
struct Student {
    std::string name;
    int age;
    Student(const std::string& n, int a) : name(n), age(a) {
        std::cout << "构造: " << name << std::endl;
    }
    Student(const Student& other) : name(other.name), age(other.age) {
        std::cout << "拷贝: " << name << std::endl;
    }
};

std::vector<Student> vec;
vec.reserve(2);

// push_back: 先构造临时对象，再拷贝到容器
vec.push_back(Student("Alice", 20));  // 构造 + 拷贝

// emplace_back: 直接在容器内构造
vec.emplace_back("Bob", 21);          // 仅构造
```

## 二维 vector

```cpp
// 声明 3x4 的二维 vector，初始值为 0
std::vector<std::vector<int>> matrix(3, std::vector<int>(4, 0));

// 访问元素
matrix[0][1] = 5;

// 动态添加行
matrix.push_back({1, 2, 3, 4});

// 遍历
for (const auto& row : matrix) {
    for (int val : row) {
        std::cout << val << " ";
    }
    std::cout << "\n";
}
```

**内存布局**：

```
matrix (vector<vector<int>>)
   │
   ├──→ [0]: vector<int> ──→ [0][0] [0][1] [0][2] [0][3]
   │
   ├──→ [1]: vector<int> ──→ [1][0] [1][1] [1][2] [1][3]
   │
   └──→ [2]: vector<int> ──→ [2][0] [2][1] [2][2] [2][3]
```

> **注意**：二维 `vector` 的每一行可以有不同的长度（锯齿数组），这与 C 风格的二维数组不同。

## 性能总结

| 操作 | 时间复杂度 |
|------|-----------|
| 随机访问 `operator[]` / `at()` | O(1) |
| 尾部插入 `push_back()` | 均摊 O(1) |
| 尾部删除 `pop_back()` | O(1) |
| 头部/中间插入 `insert()` | O(n) |
| 头部/中间删除 `erase()` | O(n) |
| 查找（无序） | O(n) |

## 常见用法示例

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

int main() {
    std::vector<int> scores = {85, 92, 78, 96, 88};

    // 添加新成绩
    scores.push_back(91);

    // 排序
    std::sort(scores.begin(), scores.end());

    // 计算平均分
    int sum = 0;
    for (int s : scores) {
        sum += s;
    }
    double avg = static_cast<double>(sum) / scores.size();

    std::cout << "平均分: " << avg << std::endl;
    std::cout << "最高分: " << scores.back() << std::endl;
    std::cout << "最低分: " << scores.front() << std::endl;

    return 0;
}
```
