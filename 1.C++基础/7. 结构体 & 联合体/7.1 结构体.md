# 7.1 结构体

C++ 中的结构体（struct）是一种用户自定义的数据类型，它允许你将多个不同类型的数据组合成一个单一的单元。可以将结构体看作是一个自定义的"容器"，它可以容纳多个相关的变量，这些变量被称为结构体的**成员**。

> 结构体的基础概念可参考 [[2.4 枚举与结构体#结构体（struct）]]

---

## 定义

使用 `struct` 关键字定义一个结构体，其基本语法如下：

```cpp
struct 结构体名 {
    成员类型  成员名1;
    成员类型  成员名2;
    // ... 更多成员
};  // 注意分号结尾
```

**示例**：

```cpp
struct Student {
    int id;
    char name[50];
    float gpa;
};
```

---

## 内存分配

### 栈上分配

#### 1. 局部变量

```cpp
void someFunction() {
    Role myRole;  // myRole 在栈上分配
    myRole.hp = 100;
    myRole.mp = 50;
    // ...
}
// myRole 及其成员变量 hp 和 mp 都将在栈上分配。
// 当 someFunction 函数退出时，myRole 所占用的内存将被自动释放。
```

#### 2. 函数参数

当结构体作为函数参数按值传递的时候也会存放在栈中。

> 关于栈和堆的区别，参考 [[6.6 堆与栈的区别]]

### 堆上分配

如果使用 `new` 运算符动态创建 `Role` 结构体的实例，那么就是在堆上分配：

```cpp
Role* myRolePtr = new Role;  // myRole 在堆上分配
myRolePtr->hp = 100;
myRolePtr->mp = 50;
// ...
delete myRolePtr;  // 记得释放内存
// 需要手动使用 delete 运算符释放内存。
```

> 关于动态内存分配，参考 [[6.2 动态内存分配与释放]]

## 成员访问

### 点运算符（.）

可以使用点运算符来访问结构体成员：

```cpp
Student stu1;
stu1.id = 1001;
strcpy(stu1.name, "Alice");
stu1.gpa = 3.8;

std::cout << stu1.name << "'s GPA is " << stu1.gpa << std::endl;
```

### 箭头运算符（->）

如果有一个指向结构体的[[6.1基本指针|指针]]，可以使用箭头运算符来访问成员：

```cpp
Student* stuPtr = new Student;
stuPtr->id = 1002;
strcpy(stuPtr->name, "Bob");
stuPtr->gpa = 3.5;

std::cout << stuPtr->name << "'s GPA is " << stuPtr->gpa << std::endl;

delete stuPtr;  // 释放堆上分配的内存
```

> 关于指针访问结构体的详细内容，包括汇编分析和内存对齐，参考 [[6.7 指针结构体]]

## 结构体初始化

### 1. 花括号初始化

可以使用花括号 `{}` 初始化结构体：

```cpp
Student stu2 = {1003, "Charlie", 3.9};
```

### 2. 成员名称初始化（C++20）

可以使用成员名称初始化：

```cpp
Student stu3 = {.name = "David", .id = 1004, .gpa = 3.7};
```

---

## 结构体数组

可以创建结构体数组，存储多个结构体实例：

```cpp
Student students[3];
students[0] = {1001, "Alice", 3.8};
students[1] = {1002, "Bob", 3.5};
students[2] = {1003, "Charlie", 3.9};
```

---

## 结构体作为函数参数和返回值

结构体可以作为函数的参数和返回值：

```cpp
void printStudent(Student stu) {
    std::cout << "'s ID: " << stu.name << stu.id << ", GPA: " << stu.gpa;
}

Student createStudent(int id, const char* name, float gpa) {
    Student stu = {id, "", gpa};
    strcpy(stu.name, name);
    return stu;
}
```

---

## 结构体和类的区别

在 C++ 中，`struct` 和 `class` 的主要区别在于**默认访问权限**：
- `struct` 的成员默认是 **public**
- `class` 的成员默认是 **private**

除此之外，它们几乎是相同的。

---

## 结构体中的函数

结构体中可以定义成员函数：

```cpp
struct Rectangle {
    int width;
    int height;

    int area() {
        return width * height;
    }
};
```

---

## 关键要点

1. **结构体定义**：使用 `struct` 关键字将多个不同类型的数据组合在一起
2. **内存分配**：局部变量在栈上，`new` 创建的在堆上
3. **成员访问**：对象用 `.`，指针用 `->`
4. **初始化方式**：支持花括号初始化和指定成员名初始化
5. **与类的区别**：主要是默认访问权限不同
