# 字符处理

## 字符字符串

比较变量：每个字符代表1个字节 (Byte)，每个字符对应的是其实的地址编号。它以 `\0` 做结尾，这代表结束符。

| 字符 | H | e | l | l | o | \0 |
|------|---|---|---|---|---|----|
| 索引 | 0 | 1 | 2 | 3 | 4 | 5  |

> **注意**：C++ 和 C 语言中的字符串以 `\0` 结尾

### char 定义

`char` 定义**一个字符**，一个字符占用 1 个字节。使用 char 直接定义并初始化一个字符的方式：`'c'`（单引号），表示**字面量**，已经实际存在于内存。

**sizeof** - `char` 占用的字节长度是固定的：

```cpp
#include <iostream>

int main() {
    // 字符数组
    char str[6] = {'H', 'e', 'l', 'l', 'o', '\0'};

    // 遍历输出
    std::cout << str << std::endl;

    return 0;
}
```

### 遍历输出每个字符

```cpp
#include <iostream>
#include <cstring>

int main() {
    char str[6] = {'H', 'e', 'l', 'l', 'o', '\0'};

    // 遍历
    std::cout << str << std::endl;
    std::cout << strlen(str) << std::endl;  // 5（不包含 '\0'）

    return 0;
}
```

## 数字字符串 char

```cpp
#include <iostream>
#include <cstring>
#include <string>

char str[12] = { 'H', 'e', 'l', 'l', 'o', '\0' };
// 注意："Hello" 隐含 \0，占用 char
std::string str3 = "Hello";

// 遍历
std::cout << str << std::endl;
std::cout << strlen(str) << std::endl;  // strlen 返回字符串长度（不含 '\0'）
```

## 新字符字符串 char

数字字符串的声明和赋值直接使用双引号 `""` 来编写即可，内存会自动末尾添加结尾符。

| 字符 | H | e | l | l | o | \0 |
|------|---|---|---|---|---|----|

```cpp
char str[256] = "Hello世界";
const char* str2 = "Hello";
std::string str3 = "Hello";
```

## 宽字符字符串 wchar_t

数字字符串的声明和赋值需要使用 `wchar_t` 来声明或 `wchar_t*` 来赋值。

```cpp
wchar_t str[256] = L"Hello";
const wchar_t* str2 = L"Hello";
```

| 字符 | H | e | l | l | o | \0 |
|------|---|---|---|---|---|----|

> **注意**：宽字符字符串的赋值需要在双引号前添加 `L` 前缀

### 宽字符注意事项

- 使用宽字符需要设置 `locale` 输出宽字符：`setlocale(LC_ALL, "");`
- 对于中文环境，"chs" 表示简体中文 "Simplified Chinese"（简称中文）

```cpp
#include <iostream>
#include <cstring>

int main() {
    setlocale(LC_ALL, "");  // 设置本地化
    wchar_t str[256] = L"Hello世界";
    std::wcout << str << std::endl;

    return 0;
}
```

### wcslen() 获取宽字符字符串长度

```cpp
size_t len = wcslen(str);  // 获取宽字符字符串长度
```

- `strlen()` 用于 char 字符串，返回字符数（不含 `\0`）
- `wcslen()` 用于 wchar_t 宽字符字符串

## 字符输入输出

### 字符输出

```cpp
char Name[16];
printf("请输入姓名：");
scanf_s("%s", Name, 16);

printf("你好 %s！\n", Name);
```

> **注意**：Name 数组的大小是 `16`，包含末尾的结束符（最多存储 15 个有效字符）。

### 解决方案：设置输入安全限制

```cpp
#include <iostream>
#include <cstring>

int main() {
    char Name[16];
    printf("请输入姓名：");
    scanf_s("%15s", Name, 16);  // %15s 限制最多读取15个字符

    printf("你好 %s！\n", Name);
    return 0;
}
```

### 字符输入

```cpp
#include <iostream>
#include <cstring>

int main() {
    std::cout << "Your Name = ";
    std::cin >> Name;  // 读取直到空白字符
    std::cout << std::endl;
    return 0;
}
```

### 字符ASCII

ASCII 值的使用：

```cpp
char char1 = 'A';  // ASCII 值为 65
char char2 = 66;   // 'B'
int charSum = char1 + char2;  // 131

std::cout << charSum << std::endl;
std::cout << (char)charSum << std::endl;  // 输出对应字符
```

## 字符复制

将一个字符数组的内容复制到另一个数组：

```cpp
char source[] = "Hello";
char dest[20];
strcpy(dest, source);  // 复制 source 到 dest
```

### strcpy 和 strncpy

| 函数 | 说明 |
|------|------|
| `strcpy(dest, src)` | 将 src 复制到 dest |
| `strncpy(dest, src, n)` | 将 src 的前 n 个字符复制到 dest |

```cpp
char bound = strchr(s1, 'W');  // 查找字符 'W' 首次出现的位置
if (bound != nullptr) {
    std::cout << "Found 'W' at: " << bound << std::endl;
}
```

## 字符比较

比较两个字符串是否相等：

| 函数 | 说明 |
|------|------|
| `strcmp(s1, s2)` | 比较两个字符串，返回 0 表示相等 |
| `strncmp(s1, s2, n)` | 比较前 n 个字符 |
| `stricmp(s1, s2)` | 忽略大小写比较（非标准） |
| `strnicmp(s1, s2, n)` | 忽略大小写比较前 n 个字符（非标准） |

```cpp
char str1[] = "Hello";
char str2[] = "Hello";

if (strcmp(str1, str2) == 0) {
    std::cout << "Strings are equal" << std::endl;
}
```

## 字符拼接

将两个字符串连接：

| 函数 | 说明 |
|------|------|
| `strcat(dest, src)` | 将 src 追加到 dest 末尾 |
| `strncat(dest, src, n)` | 将 src 的前 n 个字符追加到 dest |

```cpp
char dest[50] = "Hello ";
char src[] = "World";
strcat(dest, src);  // dest 变为 "Hello World"
```

## 字符查找

| 函数 | 说明 |
|------|------|
| `strchr(s, c)` | 查找字符 c 在字符串 s 中首次出现的位置 |
| `strrchr(s, c)` | 查找字符 c 在字符串 s 中最后出现的位置 |
| `strstr(s1, s2)` | 查找子串 s2 在 s1 中首次出现的位置 |

## 常用字符处理函数（`<cctype>`）

这些函数来自 C 语言的 `<ctype.h>` 头文件，在 C++ 中可以使用 `<cctype>` 头文件。更详细的函数说明和示例请参考 [[3.1 字符串函数|Linux 字符分类函数]]。

| 函数 | 说明 |
|------|------|
| `isalpha(c)` | 判断是否为字母 |
| `isdigit(c)` | 判断是否为数字 |
| `isalnum(c)` | 判断是否为字母或数字 |
| `isspace(c)` | 判断是否为空白字符 |
| `isupper(c)` | 判断是否为大写字母 |
| `islower(c)` | 判断是否为小写字母 |
| `toupper(c)` | 转换为大写 |
| `tolower(c)` | 转换为小写 |

## 关键要点

1. C 风格字符串以 `\0` 结尾，需要手动管理缓冲区大小
2. `char` 用于 ASCII 字符，`wchar_t` 用于宽字符（支持中文等）
3. 使用 `strlen()` 获取 char 字符串长度，`wcslen()` 获取宽字符串长度
4. 字符串操作函数（`strcpy`、`strcat` 等）存在缓冲区溢出风险，推荐使用带 `n` 的安全版本
5. 现代 C++ 推荐使用 [[8.1 string|std::string]] 代替 C 风格字符串
