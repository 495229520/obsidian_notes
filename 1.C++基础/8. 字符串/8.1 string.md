# std::string 字符串

std::string 是 C++ 标准库中的一个强大类型，用于处理文本数据。它提供了许多方便的功能，已被许多开发者作为首选字符串处理工具。

> **注意**：C++ 程序中 C 语言中的字符串以 `\0` 结尾

## 概述

使用 `std::string` 需要包含头文件 `<string>`：

| 头文件 | 作用 |
|--------|------|
| `<string>` | 本文件中，依赖此函数来创建文本文件 |
| `<cstring>` | C 风格的字符串操作函数 |

> **提示**：`std::string` 内部使用 [[6.6 堆与栈的区别|堆内存]] 存储数据，实际保存的是指向字符串数据的指针，当超出了作用域以后会自动释放堆内存数据。

## ASCII 和 ANSI 字符

- ASCII（128 字符）
- ANSI 拓展字符集（256 字符）：兼容 ASCII，拓展了其他语言字符
- Unicode 大统一字符集（包含几乎所有语言）

## 创建 string

```cpp
#include <string>
#include <iostream>
using namespace std;

int main() {
    // 创建 string：三种方式创建

    // 方式1：创建 string 内存，将变量指向它得到 std::string
    std::string str_1;

    // 方式2：创建 string 后初始化
    std::string str_2("hello");

    // 方式3：使用赋值操作
    std::string str_3 = "Hello C++";

    // 接收输入
    std::string greeting = "Hello!";

    // 字符串拼接
    std::string message = greeting + " Welcome to C++";

    // 重复字符构造
    std::string str_repeat(5, '*');  // "*****"

    // 字符串成员访问
    char first_char = message[0];            // 'H'
    std::cout << "Message length: " << message.length() << std::endl;  // 22
    std::cout << "First character: " << first_char << std::endl;       // H

    // 遍历字符
    for (int i = 0; i < message.length(); i++) {
        std::cout << message[i];
    }
    std::cout << std::endl;

    // 修改字符串中的字符
    message[0] = 'h';
    std::cout << "Modified: " << message << std::endl;  // hello! Welcome to C++

    // 使用 at() 方法访问（带边界检查）
    char c = message.at(2);  // 'l'

    // 读取输入
    int std_num = 0;
    std::cout << "Enter number: ";
    std::cin >> std_num;
    std::cout << "Input: " << std_num << std::endl;

    return 0;
}
```

## 相关方法

### length() 和 size()

两个方法功能完全相同，都返回字符串长度：

```cpp
std::string str = "hello";
int len = str.length();  // 5
// 或者
int len = str.size();    // 5
```

- **empty()**：检测字符串是否为空，如果 `empty()` 返回 true，则表示该字符串为空
- **判空**：`if (str.empty()) { ... }`
- **获取第一个字符**：`char firstChar = str[0];`

### substr() - 子串截取

截取子字符串：

```cpp
std::string str = "HelloWorld";
std::string sub1 = str.substr(5);      // "World"（从索引5到末尾）
std::string sub2 = str.substr(0, 5);   // "Hello"（从索引0截取5个字符）
```

### append() - 追加

追加字符串到末尾：

```cpp
#include <iostream>
#include <string>

int main() {
    std::string str1 = "Hello";
    std::string str2 = " wonderful world!";
    str1.append(str2);  // str1 变为 "Hello wonderful world!"
    std::cout << str1 << std::endl;

    return 0;
}
```

### erase() - 删除

- **单参数**：`str.erase(5)` 删除从索引5开始到末尾的所有字符
- **双参数**：`str.erase(0, 5)` 删除从索引0开始的5个字符

```cpp
std::string str = "Hello World";
str.erase(5);       // "Hello"
// 或者
str.erase(0, 6);    // "World"
```

### insert() - 插入

```cpp
str.insert(5, ",");    // 在索引5处插入逗号
std::string sub = str.substr(5);  // 获取从索引5开始的子串
```

### replace() - 替换

```cpp
str.replace(0, 5, "Hi");  // 从索引0开始替换5个字符为"Hi"
```

### compare() - 比较

```cpp
std::string str1 = "Hello";
int result = str1.compare("Another string");  // 比较字符串
// 返回值：0 表示相等，负数表示 str1 < 参数，正数表示 str1 > 参数
```

### find() 和 rfind() - 查找

- **find()**：从前往后查找，返回首次出现的位置，未找到返回 `std::string::npos`
- **rfind()**：从后往前查找，返回最后一次出现的位置

```cpp
std::string str = "Hello World";
size_t pos = str.find("World");   // 返回 6
if (pos != std::string::npos) {
    std::cout << "Found at: " << pos << std::endl;
}
```

## 数值转换

### 字符串转数值

| 函数 | 作用 |
|------|------|
| `std::stoi()` | string 转 int |
| `std::stol()` | string 转 long |
| `std::stod()` | string 转 double |
| `std::stof()` | string 转 float |

```cpp
std::string str("3.14");
double val = std::stod(str);  // 3.14
```

### 数值转字符串

使用 `std::to_string()`：

```cpp
int num = 42;
std::string str = std::to_string(num);  // "42"
```

> **注意**：`std::to_string()` 将数值转为字符串；字符串的 `c_str()` 方法则返回 [[8.2 字符处理|C 风格字符串]]（`const char*`）。

## 字符串比较

- 使用 `==` 比较内容是否相等：`str1 == str2`
- 使用 `compare()` 方法：`str1.compare(str2)`

```cpp
std::string str1 = "Hello";
std::string str2 = "Hello";

if (str1 == str2) {
    std::cout << "相等" << std::endl;
}

// 或使用 compare
if (str1.compare(str2) == 0) {
    std::cout << "相等" << std::endl;
}
```

## 读取输入

### getline() 读取整行

`std::cin >> str` 遇到空格会停止读取，使用 `getline()` 可以读取整行：

```cpp
std::string line;
std::getline(std::cin, line);  // 读取整行包括空格
```

### cin.ignore()

`cin.ignore()` 用于忽略输入缓冲区中的一个字符（默认忽略一个换行符），常用于解决 `cin >>` 和 `getline()` 混用时的问题：

```cpp
#include <iostream>
#include <string>

int main() {
    int num;
    std::string line;

    std::cout << "Enter a number: ";
    std::cin >> num;

    std::cin.ignore();  // 忽略缓冲区中的换行符

    std::cout << "Enter a line: ";
    std::getline(std::cin, line);

    std::cout << "Number: " << num << ", Line: " << line << std::endl;
    return 0;
}
```

## string_view（C++17）

`std::string_view` 提供对字符串的只读视图，避免不必要的复制：

```cpp
#include <string_view>

void print(std::string_view sv) {
    std::cout << sv << std::endl;
}

int main() {
    std::string str = "Hello";
    print(str);           // 不会复制字符串
    print("World");       // 直接使用字面量
    return 0;
}
```

## 字符串格式化

### 使用宏定义拼接

```cpp
#define PATH1 "hello,"
#define PATH2 "World!"

int main() {
    const char* str = PATH1 PATH2;  // "hello,World!"
    std::cout << str << std::endl;
    return 0;
}
```

## 关键要点

1. `std::string` 自动管理内存，比 C 风格字符串更安全
2. 使用 `+` 或 `append()` 拼接字符串
3. `length()` 和 `size()` 功能相同
4. 混用 `cin >>` 和 `getline()` 时要注意使用 `cin.ignore()`
5. C++17 的 `string_view` 可避免不必要的字符串复制
