# 13.4 左右值参数

> 理解函数参数中的左值和右值传递机制，是掌握 C++ [[13.2 右值引用|移动语义]]和高效编程的关键。本节探讨不同参数类型如何匹配左值和右值实参。

---

## 函数参数类型概览

C++ 函数参数有四种主要的传递方式：

| 参数类型 | 语法 | 接受左值 | 接受右值 | 特点 |
|----------|------|---------|---------|------|
| **值传递** | `T` | ✓ | ✓ | 拷贝/移动，有开销 |
| **左值引用** | `T&` | ✓ | ✗ | 可修改原对象 |
| **const 左值引用** | `const T&` | ✓ | ✓ | 只读，最通用 |
| **右值引用** | `T&&` | ✗ | ✓ | 移动语义专用 |

---

## 重载解析规则

当存在多个[[9.2.1 函数重载|重载函数]]时，编译器根据参数的值类别选择最佳匹配：

### 基本示例

```cpp
#include <iostream>

void process(int& x) {
    std::cout << "左值引用版本: " << x << std::endl;
}

void process(int&& x) {
    std::cout << "右值引用版本: " << x << std::endl;
}

int main() {
    int a = 10;

    process(a);           // 调用 process(int&)，a 是左值
    process(20);          // 调用 process(int&&)，20 是右值
    process(a + 5);       // 调用 process(int&&)，a + 5 是右值
    process(std::move(a)); // 调用 process(int&&)，std::move(a) 是右值
}
```

**输出**：
```
左值引用版本: 10
右值引用版本: 20
右值引用版本: 15
右值引用版本: 10
```

---

## 重载优先级规则

当多个重载都可以匹配时，编译器按以下优先级选择：

### 完整优先级表

对于**左值**实参：
1. `T&` （最优先）
2. `const T&`
3. `T`（值传递，需要拷贝）

对于**右值**实参：
1. `T&&` （最优先）
2. `const T&&`（少用）
3. `const T&`
4. `T`（值传递，需要拷贝/移动）

### 示例：三种重载共存

```cpp
#include <iostream>
#include <string>

void func(std::string& s) {
    std::cout << "T&: " << s << std::endl;
}

void func(const std::string& s) {
    std::cout << "const T&: " << s << std::endl;
}

void func(std::string&& s) {
    std::cout << "T&&: " << s << std::endl;
}

int main() {
    std::string str = "hello";
    const std::string cstr = "const";

    func(str);           // T&       - 非const左值优先匹配 T&
    func(cstr);          // const T& - const左值匹配 const T&
    func("world");       // T&&      - 右值优先匹配 T&&
    func(std::move(str)); // T&&      - move产生右值
}
```

## 典型应用：移动与拷贝重载

实现类的构造函数和赋值运算符时，通常需要同时提供拷贝和移动版本：

```cpp
class MyString {
private:
    char* data;
    size_t size;

public:
    // 拷贝构造函数（接受左值）
    MyString(const MyString& other) : size(other.size) {
        std::cout << "拷贝构造" << std::endl;
        data = new char[size + 1];
        std::strcpy(data, other.data);
    }

    // 移动构造函数（接受右值）
    MyString(MyString&& other) noexcept
        : data(other.data), size(other.size) {
        std::cout << "移动构造" << std::endl;
        other.data = nullptr;
        other.size = 0;
    }

    // 拷贝赋值运算符
    MyString& operator=(const MyString& other) {
        std::cout << "拷贝赋值" << std::endl;
        if (this != &other) {
            delete[] data;
            size = other.size;
            data = new char[size + 1];
            std::strcpy(data, other.data);
        }
        return *this;
    }

    // 移动赋值运算符
    MyString& operator=(MyString&& other) noexcept {
        std::cout << "移动赋值" << std::endl;
        if (this != &other) {
            delete[] data;
            data = other.data;
            size = other.size;
            other.data = nullptr;
            other.size = 0;
        }
        return *this;
    }
};
```

### 使用示例

```cpp
MyString createString() {
    return MyString("temp");
}

int main() {
    MyString s1("hello");

    MyString s2 = s1;           // 拷贝构造（s1 是左值）
    MyString s3 = std::move(s1); // 移动构造（std::move(s1) 是右值）
    MyString s4 = createString(); // 移动构造（返回值是右值）

    s2 = s3;           // 拷贝赋值
    s2 = std::move(s3); // 移动赋值
}
```

## 转发引用（万能引用）

在模板中，`T&&` 不是普通的右值引用，而是**转发引用**（也叫万能引用），可以绑定左值和右值：

```cpp
template<typename T>
void wrapper(T&& arg) {
    // T&& 是转发引用
    // 传入左值时：T = int&，arg 类型为 int&
    // 传入右值时：T = int，arg 类型为 int&&
}

int main() {
    int x = 10;
    wrapper(x);    // T = int&，arg 是 int&
    wrapper(10);   // T = int，arg 是 int&&
}
```

### 配合 std::forward 使用

要保持参数原本的值类别，需要使用 `std::forward`：

```cpp
template<typename T>
void wrapper(T&& arg) {
    // 不使用 forward：arg 总是作为左值传递（因为 arg 有名字）
    // func(arg);

    // 使用 forward：保持原本的值类别
    func(std::forward<T>(arg));
}
```

**为什么需要 forward？**

- 右值引用变量本身是一个**左值**（它有名字、有地址）
- 即使 `arg` 的类型是 `int&&`，表达式 `arg` 仍然是左值
- `std::forward<T>(arg)` 根据 `T` 的类型恢复原本的值类别

## 参数传递最佳实践

### 1. 只读参数：使用 const T&

```cpp
// 推荐：大对象只读访问
void process(const std::vector<int>& data);

// 小对象（如 int）可以直接值传递
void calculate(int value);
```

### 2. 需要修改：使用 T&

```cpp
void increment(int& value) {
    ++value;
}
```

### 3. 需要存储/转移所有权：值传递 + move

```cpp
class Container {
    std::vector<int> data;
public:
    // 值传递 + move：既接受左值（拷贝）也接受右值（移动）
    void setData(std::vector<int> newData) {
        data = std::move(newData);
    }
};
```

### 4. 性能关键场景：重载 const T& 和 T&&

```cpp
class Container {
    std::string name;
public:
    void setName(const std::string& n) { name = n; }        // 左值：拷贝
    void setName(std::string&& n) { name = std::move(n); }  // 右值：移动
};
```

---

## 关键要点

1. **左值引用 (T&)**：只接受左值，用于需要修改的参数
2. **const 左值引用 (const T&)**：接受左值和右值，用于只读参数
3. **右值引用 (T&&)**：只接受右值，用于移动语义
4. **转发引用 (T&& 在模板中)**：接受任意值类别，配合 std::forward 使用
5. **重载解析**：编译器优先选择最精确匹配的重载版本
