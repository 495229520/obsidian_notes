# 9.2.6 函数模板参数

## 1. 指定默认参数

在 C++ 中，你可以为[[9.2.2 函数模版|函数模板]]的参数设置默认值。这使得在调用模板函数时，如果没有显式地提供某些模板参数，编译器会使用这些默认值。

### 两种用法

```cpp
// 用法1：在函数模板中指定默认参数
template <typename TR = int, typename T1, typename T2>
TR ave(T1 a, T2 b) {
    return (a + b) / 2;
}

// 用法2：使用相同的类型参数作为默认值
template <typename T1, typename T2, typename TR = T1>
TR ave(T1 a, T2 b) {
    return (a + b) / 2;
}
```

### 使用示例

```cpp
int main() {
    int a = 100;
    int b = 200;
    auto c = ave(a, b);  // 使用默认返回类型
    return 0;
}
```

## 2. 非类型模板参数 (Non-type Template Parameters)

函数模板有非类型的模板参数，指的是函数模板的**非类型参数 (Non-type template parameters)** 允许你在模板定义中引入**编译时常量值**。这些值可以在模板实例化时指定，并直接影响模板的行为和结构。

### 语法

```cpp
template <int max, int min, typename T>
bool AddHp(T& hp, T damage) {
    hp -= damage;
    if (hp > max) {
        hp = max;
    }
    return hp < min;
}
```

### 使用示例

```cpp
int main() {
    int hp = 2500;
    AddHp<2000, 1000>(hp, 100);  // max=2000, min=1000
    // 结果是2000
    std::cout << hp << std::endl;

    return 0;
}
```

### 非类型参数是常量

非类型参数数是一个**常量**，所以不可以在函数内部对其进行修改：

```cpp
template <int max, int min, typename T>
bool AddHp(T& hp, T damage) {
    // 这里的 max 是一个本质上一个编译值常量不可以被修改！
    // max = 2500;  // 错误！

    hp -= damage;
    if (hp > max) {
        hp = max;
    }
    return hp < min;
}

int main() {
    int hp = 2500;
    const int max = 2000;
    const int min = 1000;
    AddHp<max, min>(hp, 100);
    // 结果是2000
    std::cout << hp << std::endl;

    return 0;
}
```

## 3. 非类型参数自动推断数组长度

在显示指定 `typename` 的非类型参数时，可以传入常量参数。

### 求不同长度数组的平均值

```cpp
// 求不同长度数组的平均值
template <typename T, short count>
T ave(T(&arr)[count]) {
    T all = 0;
    for (int i = 0; i < count; i++) {
        all += arr[i];
    }
    return all / count;
}

int main() {
    int a[5] = {1, 2, 3, 4, 5};
    // 函数模版自动推断 count 的值，无需显示指定非类型参数
    std::cout << ave(a);
    return 0;
}
```

### 不使用非类型参数的写法

如果不使用非类型参数，在使用数组作为参数时需要指定数组的大小和返回的类型：

```cpp
// 求不同长度数组的平均值（需要显式指定大小）
int ave(int(&arr)[10]) {
    int all = 0;
    for (int i = 0; i < 10; i++) {
        all += arr[i];
    }
    return all / 10;
}

int main() {
    int a[5] = {1, 2, 3, 4, 5};
    std::cout << ave(a);  // 错误！数组大小不匹配
    return 0;
}
```

## 4. 问题：模板参数数量与函数参数数量

如果函数模板的参数数量要大于调用函数的参数数量，那么需要显示式指定：

```cpp
// 显示指定
template <typename TR, typename T1, typename T2>
TR ave(T1 a, T2 b) {
    return (a + b) / 2;
}

int main() {
    int a = 100;
    int b = 200;
    auto c = ave<int>(a, b);  // 必须显式指定 TR
    return 0;
}
```

或者设置函数模板参数的默认值：

```cpp
// 设置默认值
template <typename TR = int, typename T1, typename T2>
TR ave(T1 a, T2 b) {
    return (a + b) / 2;
}

int main() {
    int a = 100;
    int b = 200;
    auto c = ave(a, b);  // 使用默认的 TR = int
    return 0;
}
```

## 5. 非类型参数的限制

非类型模板参数有以下限制：

| 允许的类型 | 说明 |
|-----------|------|
| 整数类型 | `int`, `short`, `long`, `char`, `bool` 等 |
| 枚举类型 | `enum` |
| 指针类型 | 指向对象或函数的指针 |
| 引用类型 | 对对象或函数的引用 |
| `std::nullptr_t` | C++11 起 |

**不允许**的类型：

- 浮点类型（`float`, `double`）— C++20 前
- 类类型（除非满足特定条件）— C++20 前

---

## 6. 关键要点

1. **默认模板参数**：可以为模板参数指定默认类型，调用时可省略
2. **非类型参数**：允许传入编译时常量值（如整数、枚举）
3. **常量性**：非类型参数是编译期常量，不可修改
4. **自动推导**：数组长度等非类型参数可以自动推导
5. **显式指定**：当模板参数多于可推导参数时，需要显式指定
