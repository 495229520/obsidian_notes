# 9.2.4 特化函数模板 (Function Template Specialization)

## 1. 简介

实现了[[9.2.2 函数模版|函数模板]]之后，有时我们可能需要为**特定类型**提供**特殊的实现**。这就是**函数模板特化 (Function Template Specialization)** 的作用——为通用模板提供针对特定类型的自定义版本。

---

## 2. 为什么需要特化

考虑一个比较函数：

```cpp
template <typename T>
T bigger(T a, T b) {
    return a > b ? a : b;
}
```

对于 `int*` 类型，使用模板的 `>` 运算符会比较**指针地址**而非**指向的值**，这通常不是我们想要的行为。

```cpp
int main() {
    int x = 10;
    int y = 20;
    int* px = &x;
    int* py = &y;

    // bigger(px, py) 比较的是指针地址，而非 *px 和 *py 的值
    int* result = bigger(px, py);  // 可能返回地址较大的指针，而非值较大的
}
```

通过特化，我们可以为 `int*` 类型提供正确的比较逻辑。

---

## 3. 全特化 (Full Specialization)

### 3.1 语法

```cpp
template <>
返回类型 函数名<特定类型>(参数列表) {
    // 特化实现
}
```

- `template <>`：表示这是一个全特化，没有模板参数
- `<特定类型>`：指定要特化的具体类型

### 3.2 示例：指针类型特化

```cpp
// 通用模板
template <typename T>
T bigger(T a, T b) {
    return a > b ? a : b;
}

// 针对 int* 类型的全特化
template <>
int* bigger<int*>(int* a, int* b) {
    return *a > *b ? a : b;  // 比较指向的值
}

int main() {
    int x = 10;
    int y = 20;

    // 调用特化版本，比较 *(&x) 和 *(&y)
    int* result = bigger(&x, &y);
    std::cout << *result << std::endl;  // 输出: 20
}
```

### 3.3 示例：`char*` 字符串特化

```cpp
// 通用模板
template <typename T>
void swap(T& a, T& b) {
    T temp = a;
    a = b;
    b = temp;
}

// 针对 char* 类型的特化（交换字符串内容）
template <>
void swap<char*>(char*& a, char*& b) {
    // 保证缓冲区足够大，实际应用中需要更安全的处理
    char* temp = new char[strlen(a) + 1];
    strcpy(temp, a);
    strcpy(a, b);
    strcpy(b, temp);
    delete[] temp;
}
```

## 4. 特化 vs 重载

函数模板既可以**特化**也可以**[[9.2.3 重载函数模版|重载]]**。两者的区别：

| 特性 | 特化 (Specialization) | 重载 (Overloading) |
|------|----------------------|-------------------|
| 语法 | `template <>` | 普通函数声明 |
| 本质 | 模板的特殊版本 | 独立的函数 |
| 优先级 | 低于重载 | 高于模板 |

### 示例对比

```cpp
// 通用模板
template <typename T>
void process(T x) {
    std::cout << "Generic template" << std::endl;
}

// 特化版本
template <>
void process<int>(int x) {
    std::cout << "Specialized for int" << std::endl;
}

// 重载版本
void process(int x) {
    std::cout << "Overloaded for int" << std::endl;
}

int main() {
    process(10);  // 输出: "Overloaded for int"（重载优先）
}
```

---

## 5. 选择顺序

编译器在选择最匹配的版本时，遵循以下优先级：

| 优先级 | 匹配类型 |
|--------|---------|
| 1 | 完全匹配的**普通函数** |
| 2 | 完全匹配的**函数模板特化版本** |
| 3 | 可以通过模板参数推导实例化的**函数模板** |
| 4 | 可以通过类型转换匹配的**普通函数** |

> 编译器会选择最匹配的版本。如果找到一个完全匹配的特化版本，它会优先使用该版本，而不是通过通用模板实例化出来的函数。

---

## 6. 实际应用：`std::vector` 特化

```cpp
#include <iostream>
#include <vector>

// 通用模板
template <typename T>
void printInstances(T obj) {
    std::cout << "Generic template called" << std::endl;
}

// 针对 std::vector<bool> 的特化
template <>
void printInstances<std::vector<bool>>(std::vector<bool> obj) {
    std::cout << "Specialized for std::vector<bool>" << std::endl;
    for (bool b : obj) {
        std::cout << (b ? "true" : "false") << " ";
    }
    std::cout << std::endl;
}

int main() {
    std::vector<int> intVec = {1, 2, 3};
    std::vector<bool> boolVec = {true, false, true};

    printInstances(intVec);   // 调用通用模板
    printInstances(boolVec);  // 调用特化版本

    return 0;
}
```

---

## 7. 注意事项

1. **`template <>`**：全特化必须以 `template <>` 开头，尖括号内为空
2. **`<char*>`**：显式指定特化的类型，如果编译器能自动推导则可省略
3. **声明顺序**：特化必须在通用模板之后声明
4. **避免歧义**：当特化和重载同时存在时，重载优先

---

## 8. 关键要点

1. **特化目的**：为特定类型提供定制化实现
2. **语法**：使用 `template <>` 声明全特化
3. **优先级**：普通函数 > 特化模板 > 通用模板
4. **与重载的区别**：特化是模板的变体，重载是独立函数
