# 9.3.7 重载类型转换运算符 (Type Conversion Operator)

## 1. 什么是类型转换运算符？

类型转换运算符是一种特殊的成员函数，它定义了如何将一个类类型的对象转换为另一个类型的对象。这使得你的自定义类型可以与内置类型或其他类型进行无缝交互。

---

## 2. 语法

### 隐式转换

```cpp
class MyClass {
public:
    // 将 MyClass 对象转换为 type_name 类型
    operator type_name() const;
};
```

### 显式转换

默认情况下，类型转换运算符允许**隐式转换**。这虽然方便，但有时会导致意外的转换，降低代码的可读性和直观性。为了更加安全，从 C++11 开始，你可以使用 `explicit` 关键字来**禁止隐式转换**，只允许显式转换。

```cpp
class MyClass {
public:
    // 将 MyClass 对象显式转换为 type_name 类型
    explicit operator type_name() const;
};
```

---

## 3. 示例：Money 类

```cpp
class Money {
private:
    long long total_cents;  // 总分数

public:
    // 构造函数：接收元和分
    Money(long long dollars, int cents) {
        total_cents = dollars * 100 + cents;
    }

    void display() const {
        std::cout << "产品价格: " << total_cents / 100 << "元" << total_cents % 100 << "分" << std::endl;
    }

    // 1. 重载到 double 的类型转换运算符（显式）
    // 允许 Money对象 -> double 的自动转换
    explicit operator double() const {
        return static_cast<double>(total_cents) / 100.0;
    }

    // 2. 重载到 bool 的类型转换运算符（显式）
    // 用于在 if 语句中判断 Money 对象是否为零
    explicit operator bool() const {
        return total_cents > 0;
    }

    // 3. 重载到 std::string 的类型转换运算符（显式）
    // 允许 Money对象 -> std::string 的自动转换
    operator std::string() const {
        std::ostringstream oss;
        oss << std::fixed << std::setprecision(2);  // 设置固定小数，精度为2
        oss << "$" << static_cast<double>(total_cents) / 100.0;
        return oss.str();
    }
};
```

## 4. 使用示例

```cpp
int main() {
    Money wallet(123, 45);  // 123元45分
    wallet.display();
    std::cout << "-------------------------------" << std::endl;

    // --- 测试 operator double() ---
    std::cout << "测试 double 的转换:" << std::endl;
    double amount_in_double = static_cast<double>(wallet);  // wallet提供的转换为double
    std::cout << "转换后 double 的钱: " << amount_in_double << std::endl;

    // --- 测试 operator bool() ---
    // 使用 explicit operator bool 让你可以在 if 语句中使用对象
    Money empty_wallet(0, 0);
    std::cout << "-------------------------------" << std::endl;
    std::cout << "测试 bool 的转换:" << std::endl;
    if (wallet) {
        std::cout << "钱包不是空的。" << std::endl;
    } else {
        std::cout << "钱包是空的。" << std::endl;
    }

    if (empty_wallet) {
        std::cout << "空钱包有钱。" << std::endl;
    } else {
        std::cout << "空钱包是空的。" << std::endl;
    }

    // --- 测试 operator std::string() ---
    std::cout << "-------------------------------" << std::endl;
    std::cout << "测试 std::string 的转换:" << std::endl;
    std::string money_str = wallet;  // 隐式转换调用
    std::cout << "钱包金额 (string): " << money_str << std::endl;

    return 0;
}
```

### 输出

```
产品价格: 123元45分
-------------------------------
测试 double 的转换:
转换后 double 的钱: 123.45
-------------------------------
测试 bool 的转换:
钱包不是空的。
空钱包是空的。
-------------------------------
测试 std::string 的转换:
钱包金额 (string): $123.45
```

## 5. `explicit` 的重要性

### 没有 `explicit` 的问题

```cpp
class Distance {
public:
    operator int() const { return meters; }  // 隐式转换
private:
    int meters;
};

Distance d;
int x = d + 5;  // 隐式转换，可能不是预期行为
```

### 使用 `explicit` 更安全

```cpp
class Distance {
public:
    explicit operator int() const { return meters; }
private:
    int meters;
};

Distance d;
int x = d + 5;              // 错误！不允许隐式转换
int y = static_cast<int>(d) + 5;  // 正确，显式转换
```

## 6. `operator bool()` 的特殊性

`operator bool()` 通常应该是 `explicit` 的，但在以下上下文中会进行**隐式转换**：

- `if` 语句条件
- `while` 循环条件
- `for` 循环条件
- `!`、`&&`、`||` 运算符

```cpp
class Ptr {
public:
    explicit operator bool() const { return ptr != nullptr; }
private:
    void* ptr;
};

Ptr p;
if (p) { }       // OK：条件上下文允许
bool b = p;      // 错误：不允许隐式转换
bool c = !!p;    // OK：! 上下文允许
```

---

## 7. 关键要点

1. **语法**：`operator type_name() const`，无返回类型声明
2. **`explicit`**：防止意外的隐式转换，更安全
3. **`operator bool()`**：在条件上下文中特殊处理
4. **必须是成员函数**：类型转换运算符不能是友元函数
5. **慎用隐式转换**：可能导致难以调试的问题
