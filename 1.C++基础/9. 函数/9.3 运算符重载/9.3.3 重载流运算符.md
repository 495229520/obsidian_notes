# 9.3.3 重载流运算符 (Stream Operators)

## 1. 简介

流运算符 `<<`（输出）和 `>>`（输入）允许自定义类型与 `std::cout` 和 `std::cin` 配合使用，实现直观的输入输出操作。

---

## 2. 重载要求

### 必须使用友元函数

流运算符**必须**通过友元函数实现，因为：

- 左操作数是 `ostream`/`istream` 对象，不是我们的类
- 无法修改标准库的 `ostream`/`istream` 类

### 语法

```cpp
// 声明（在类中）
friend std::ostream& operator<<(std::ostream& _cout, const MyClass& obj);
friend std::istream& operator>>(std::istream& _cin, MyClass& obj);

// 定义（在类外）
std::ostream& operator<<(std::ostream& _cout, const MyClass& obj) {
    _cout << obj.member;
    return _cout;
}
```

---

## 3. 重载输出运算符 `<<`

### 头文件声明

```cpp
// .h
std::ostream& operator<<(std::ostream& _cout, const hstring& _str);
```

### 实现

```cpp
// .cpp
std::ostream& operator<<(std::ostream& _cout, const hstring& _str) {
    _cout << _str.c_str();
    return _cout;
}
```

### 使用

```cpp
hstring str = "Hello";
std::cout << str << std::endl;  // 输出: Hello
```

---

## 4. 重载输入运算符 `>>`

### 声明

```cpp
// .h
std::istream& operator>>(std::istream& _cin, hstring& _str);
```

### 实现

```cpp
// .cpp
std::istream& operator>>(std::istream& _cin, hstring& _str) {
    char buffer[1024];
    _cin >> buffer;
    _str = buffer;  // 使用赋值运算符
    return _cin;
}
```

### 使用

```cpp
hstring str;
std::cin >> str;  // 从键盘输入
```

---

## 5. 为什么返回引用？

返回 `ostream&` 或 `istream&` 引用是为了支持**链式操作**：

```cpp
std::cout << "Name: " << name << ", Age: " << age << std::endl;
// 等价于
((((std::cout << "Name: ") << name) << ", Age: ") << age) << std::endl;
```

每次 `<<` 操作返回同一个流对象，使下一次操作可以继续。

---

## 6. 注意事项

### 6.1 参数为 `const` 引用（输出）

对于 `operator<<`，第二个参数应为 `const` 引用，因为输出操作不应修改对象：

```cpp
std::ostream& operator<<(std::ostream& _cout, const hstring& _str);
                                              ^^^^^
```

### 6.2 参数为非 `const` 引用（输入）

对于 `operator>>`，第二个参数必须是非 `const` 引用，因为需要修改对象：

```cpp
std::istream& operator>>(std::istream& _cin, hstring& _str);
                                             ^（无 const）
```

### 6.3 临时对象问题

C++ 不允许将一个临时对象（由 `str(1, 2)` 返回的 `hstring` 对象就是一个临时对象）绑定到一个非 `const` 的左值引用上。

```cpp
// 错误：临时对象不能绑定到非 const 引用
std::ostream& operator<<(std::ostream& _cout, hstring& _str);  // 不能用于临时对象

// 正确：使用 const 引用
std::ostream& operator<<(std::ostream& _cout, const hstring& _str);

// 或者使用右值引用
std::ostream& operator<<(std::ostream& _cout, hstring&& _str);
```

## 7. 完整示例

```cpp
class Point {
    friend std::ostream& operator<<(std::ostream& os, const Point& p);
    friend std::istream& operator>>(std::istream& is, Point& p);
private:
    int x, y;
public:
    Point(int _x = 0, int _y = 0) : x(_x), y(_y) {}
};

std::ostream& operator<<(std::ostream& os, const Point& p) {
    os << "Point(" << p.x << ", " << p.y << ")";
    return os;
}

std::istream& operator>>(std::istream& is, Point& p) {
    std::cout << "Enter x y coordinates: ";
    is >> p.x >> p.y;
    return is;
}

int main() {
    Point p1(3, 4);
    std::cout << p1 << std::endl;  // 输出: Point(3, 4)

    Point p2;
    std::cin >> p2;  // 输入坐标
    std::cout << p2 << std::endl;
}
```

---

## 8. 关键要点

1. **友元函数**：流运算符必须通过友元函数实现
2. **返回引用**：返回流引用以支持链式操作
3. **const 正确性**：输出用 `const&`，输入用 `&`
4. **临时对象**：`const&` 可以绑定临时对象，`&` 不行
