# 9.3.10 实现加密 hint

## 1. 简介

本节通过一个实际的例子——实现一个简单的加密字符串类 `hint`，来综合运用前面学习的[[9.3.1 运算符重载|运算符重载]]知识。

---

## 2. hint 类设计

```cpp
#ifndef HINT_H
#define HINT_H

#include <iostream>
#include <cstring>

class hint {
private:
    int m_encrypt_value;   // 存储加密密钥
    char* data;            // 字符串数据
    int m_len;             // 字符串长度

public:
    // 构造函数
    hint();
    hint(const char* str, int key = 42);

    // 析构函数
    ~hint();

    // 拷贝构造函数
    hint(const hint& other);

    // 赋值运算符
    hint& operator=(const hint& other);

    // 加密/解密方法
    void encrypt();
    void decrypt();

    // 其他运算符重载声明...
    char& operator[](int index);
    const char& operator[](int index) const;
    operator int() const;
    operator bool() const;

    // 友元函数
    friend std::ostream& operator<<(std::ostream& os, const hint& h);
    friend std::istream& operator>>(std::istream& is, hint& h);
};

#endif
```

## 4. 核心功能实现

### 4.0 构造函数与析构函数

```cpp
// 默认构造函数
hint::hint() : m_encrypt_value(42), data(nullptr), m_len(0) {}

// 带参数的构造函数
hint::hint(const char* str, int key) : m_encrypt_value(key) {
    if (str) {
        m_len = strlen(str);
        data = new char[m_len + 1];
        strcpy(data, str);
    } else {
        m_len = 0;
        data = nullptr;
    }
}

// 拷贝构造函数（深拷贝）
hint::hint(const hint& other)
    : m_encrypt_value(other.m_encrypt_value), m_len(other.m_len) {
    if (other.data) {
        data = new char[m_len + 1];
        memcpy(data, other.data, m_len + 1);
    } else {
        data = nullptr;
    }
}

// 析构函数
hint::~hint() {
    delete[] data;
}
```

### 4.1 加密方法

使用简单的异或（XOR）加密：

```cpp
void hint::encrypt() {
    for (int i = 0; i < m_len; i++) {
        data[i] ^= m_encrypt_value;  // 异或加密
    }
}

void hint::decrypt() {
    for (int i = 0; i < m_len; i++) {
        data[i] ^= m_encrypt_value;  // 异或解密（同样操作）
    }
}
```

### 4.2 赋值运算符

```cpp
hint& hint::operator=(const hint& other) {
    if (this == &other) {
        return *this;  // 自赋值检查
    }

    delete[] data;  // 释放原有内存

    m_len = other.m_len;
    m_encrypt_value = other.m_encrypt_value;
    data = new char[m_len + 1];
    memcpy(data, other.data, m_len + 1);

    return *this;
}
```

### 4.3 类型转换运算符

```cpp
// 转换为 int（返回加密值）
hint::operator int() const {
    return m_encrypt_value;
}

// 转换为 bool（检查是否有数据）
hint::operator bool() const {
    return data != nullptr && m_len > 0;
}
```

### 4.4 下标运算符

```cpp
char& hint::operator[](int index) {
    if (index < 0 || index >= m_len) {
        throw std::out_of_range("Index out of bounds");
    }
    return data[index];
}

const char& hint::operator[](int index) const {
    if (index < 0 || index >= m_len) {
        throw std::out_of_range("Index out of bounds");
    }
    return data[index];
}
```

### 4.5 流运算符

```cpp
std::ostream& operator<<(std::ostream& os, const hint& h) {
    os << h.data;
    return os;
}

std::istream& operator>>(std::istream& is, hint& h) {
    char buffer[1024];
    is >> buffer;
    h = hint(buffer);
    return is;
}
```

## 5. 使用示例

```cpp
int main() {
    // 创建加密字符串
    hint secret("Hello, World!");

    // 输出原始数据
    std::cout << "原始: " << secret << std::endl;

    // 加密
    secret.encrypt();
    std::cout << "加密后: " << secret << std::endl;

    // 解密
    secret.decrypt();
    std::cout << "解密后: " << secret << std::endl;

    // 使用下标访问
    std::cout << "第一个字符: " << secret[0] << std::endl;

    // 类型转换
    if (secret) {
        std::cout << "hint 对象有效" << std::endl;
    }

    return 0;
}
```

## 6. 设计要点

### 内存管理

- 构造函数分配内存
- 析构函数释放内存
- 赋值运算符先释放后分配
- 拷贝构造函数深拷贝

### 安全性

- 自赋值检查
- 边界检查
- 空指针检查

### 易用性

- 支持流操作
- 支持下标访问
- 支持类型转换

---

## 7. 关键要点

1. **综合运用**：实际项目中通常需要重载多个运算符
2. **一致性**：相关运算符的行为应该一致
3. **安全性**：始终考虑边界情况和异常处理
4. **内存管理**：包含指针成员时，需要仔细处理深拷贝
5. **const 正确性**：适当使用 `const` 修饰符
