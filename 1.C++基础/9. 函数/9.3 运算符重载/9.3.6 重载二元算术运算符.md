# 9.3.6 重载二元算术运算符 (Binary Arithmetic Operators)

## 1. 简介

在 C++ 中，重载二元运算符（例如 `+`、`-`、`*`、`/`）允许你为自定义类型定义这些运算符的行为。这使得你的自定义类型可以像内置类型一样进行算术运算，从而使代码更加直观和易读。

---

## 2. 重载方式

二元运算符需要两个操作数（一个在左边，一个在右边）来进行运算。

| 实现方式 | 说明 | 推荐场景 |
|---------|------|---------|
| **成员函数** | 左操作数是 `*this`，右操作数是参数 | 左操作数必须是类对象 |
| **友元函数** | 两个操作数都是参数 | 支持左操作数不是类对象的情况 |

---

## 3. 成员函数 vs 友元函数

对于 `+`、`-`、`*`、`/` 等对称的二元运算符，**推荐使用友元函数**，因为：

1. 两个操作数地位相等
2. 支持 `2.0 * vector` 这样左操作数不是类对象的情况

```cpp
// 成员函数方式
Vector2D operator+(const Vector2D& other) const {
    return Vector2D(x + other.x, y + other.y);
}

// 友元函数方式（推荐）
friend Vector2D operator+(const Vector2D& a, const Vector2D& b) {
    return Vector2D(a.x + b.x, a.y + b.y);
}
```

---

## 4. 设计一个二维向量类 Vector2D

```cpp
class Vector2D {
private:
    double x;
    double y;

public:
    // 默认构造函数
    Vector2D() : x(0.0), y(0.0) {}

    // 带参数的构造函数
    Vector2D(double x, double y) : x(x), y(y) {}

    // 打印向量信息的函数
    void display() const {
        std::cout << "(" << x << ", " << y << ")" << std::endl;
    }

    // 方式一：使用【成员函数】重载二元运算符 +
    // v = v1 + v2 被解释为 v1.operator+(v2)
    Vector2D operator+(const Vector2D& other) const {
        return Vector2D(x + other.x, y + other.y);
    }

    // 方式二：使用【友元函数】重载二元运算符 *
    // v = v * 2.0 被解释为 operator*(v, 2.0)
    friend Vector2D operator*(const Vector2D& vec, double scalar);
    friend Vector2D operator*(double scalar, const Vector2D& vec);
};

// 向量乘以标量（友元函数定义）
Vector2D operator*(const Vector2D& vec, double scalar) {
    return Vector2D(vec.x * scalar, vec.y * scalar);
}

// 标量乘以向量（为了对称性）
// 2.0 * v 被解释为 operator*(2.0, v)
Vector2D operator*(double scalar, const Vector2D& vec) {
    return vec * scalar;
}
```

## 5. 使用示例

```cpp
int main() {
    // 创建两个 Vector2D 对象
    Vector2D v1(2.0, 3.0);
    Vector2D v2(5.0, 1.0);

    // 1. 测试向量加法 (v1 + v2)
    // 这里会调用 v1.operator+(v2)
    Vector2D v3 = v1 + v2;
    std::cout << "v1 + v2 的结果 v3: ";
    v3.display();

    // 2. 测试向量与标量乘法 (v1 * 2.5)
    // 这里会调用 operator*(v1, 2.5)
    Vector2D v4 = v1 * 2.5;
    std::cout << "v1 * 2.5 的结果 v4: ";
    v4.display();

    // 3. 测试对称性 (0.5 * v2)
    // 这里会调用 operator*(0.5, v2)
    Vector2D v5 = 0.5 * v2;
    std::cout << "0.5 * v2 的结果 v5: ";
    v5.display();

    return 0;
}
```

### 代码剖析

1. **`operator+` (成员函数)**：我们定义了两个版本的 `operator*`。
   - `v1 + v2` 这行代码内，`v1.operator+(v2)` 函数被调用，函数体内的 `this` 指向 `v1`，`other` 是 `v2` 的引用。它返回一个全新的 `Vector2D` 对象。

2. **`operator*` (非成员函数)**：
   - 第一个版本 `operator*(const Vector2D& vec, double scalar)` 处理 `vector * scalar` 的情况。
   - 第二个版本 `operator*(double scalar, const Vector2D& vec)` 处理 `scalar * vector` 的情况，这保证了乘法的对称性。

## 6. 复合赋值运算符

通常还需要实现 `+=`、`-=`、`*=`、`/=` 等复合赋值运算符：

```cpp
class Vector2D {
public:
    // 复合赋值运算符（必须是成员函数）
    Vector2D& operator+=(const Vector2D& other) {
        x += other.x;
        y += other.y;
        return *this;
    }

    Vector2D& operator*=(double scalar) {
        x *= scalar;
        y *= scalar;
        return *this;
    }
};

// 基于复合赋值实现算术运算符
Vector2D operator+(Vector2D lhs, const Vector2D& rhs) {
    lhs += rhs;  // 使用 +=
    return lhs;
}
```

---

## 7. 关键要点

1. **对称运算符**：`+`、`-`、`*`、`/` 推荐使用友元函数
2. **复合赋值**：`+=`、`-=` 必须是成员函数，返回 `*this` 引用
3. **对称性**：如果支持 `v * 2.0`，也应支持 `2.0 * v`
4. **返回值**：算术运算符返回新对象，复合赋值返回引用
5. **const 正确性**：参数和成员函数应适当使用 `const`
