# 9.3.5 重载括号运算符 (Function Call Operator)

## 1. 简介

在 C++ 中重载 `()` 运算符被称为**函数调用运算符重载**或**仿函数**。它允许你将类的对象像函数一样调用。当你对一个类的对象使用 `()` 运算符时，实际上是在调用该类中重载的 `operator()` 成员函数。

---

## 2. 语法

```cpp
class Adder {
public:
    // 重载函数调用运算符
    int operator()(int a, int b) const {
        return a + b;
    }
};
```

---

## 3. 基本示例

```cpp
class Adder {
public:
    // 重载函数调用运算符
    int operator()(int a, int b) const {
        return a + b;
    }
};

int main() {
    Adder add_obj;  // 创建 Adder 类的对象

    // 像调用函数一样调用对象
    int result = add_obj(10, 20);
    std::cout << "Result of 10 + 20: " << result << std::endl;

    // 你也可以直接在表达式中使用
    std::cout << "Result of 5 + 7: " << Adder()(5, 7) << std::endl;

    return 0;
}
```

---

## 4. 仿函数 (Functor)

重载了 `operator()` 的类对象被称为**仿函数**或**函数对象**。它们可以：

- 像普通函数一样被调用
- 拥有状态（成员变量）
- 作为参数传递给 STL 算法

### 有状态的仿函数

```cpp
class Counter {
private:
    int count = 0;
public:
    int operator()() {
        return ++count;
    }

    int getCount() const {
        return count;
    }
};

int main() {
    Counter counter;
    std::cout << counter() << std::endl;  // 1
    std::cout << counter() << std::endl;  // 2
    std::cout << counter() << std::endl;  // 3
}
```

---

## 5. 与 STL 算法配合

仿函数常用于 STL 算法，作为自定义操作：

```cpp
#include <algorithm>
#include <vector>

class IsGreaterThan {
private:
    int threshold;
public:
    IsGreaterThan(int t) : threshold(t) {}

    bool operator()(int value) const {
        return value > threshold;
    }
};

int main() {
    std::vector<int> nums = {1, 5, 3, 8, 2, 9};

    // 使用仿函数作为谓词
    int count = std::count_if(nums.begin(), nums.end(), IsGreaterThan(4));
    std::cout << "大于4的元素个数: " << count << std::endl;  // 3
}
```

## 6. 截取字符串示例

### 头文件声明

```cpp
// 截取字符串
hstring operator()(const unsigned short start, const unsigned short length) const;
```

### 实现

```cpp
hstring hstring::operator()(const unsigned short start, const unsigned short length) const {
    // 字符串的最后一个字符是 '\0'，要把这个排除
    // 不允许截取最后一个字符，"123456"\0
    if (start > uslen - 2) {
        return hstring("");
    }

    // 如果开始截取的位置+截取的长度比字符串长度要长，就只截取开始截取的位置到字符串末尾
    unsigned slen = start + length > uslen - 1 ? uslen - start - 1 : length;

    char* newstr = new char[slen + 1];
    memcpy(newstr, cstr + start, slen + 1);
    newstr[slen] = 0;
    return hstring(newstr);
}
```

### 使用

```cpp
hstring str = "a123";
hstring str1 = str(1, 2);
std::cout << str(1, 2);  // 输出: 12
```

## 7. 仿函数 vs Lambda

C++11 引入的 Lambda 表达式在很多场景下可以替代仿函数：

```cpp
// 仿函数
class Add {
public:
    int operator()(int a, int b) const { return a + b; }
};

// Lambda
auto add = [](int a, int b) { return a + b; };

// 使用
Add functor;
std::cout << functor(1, 2) << std::endl;  // 3
std::cout << add(1, 2) << std::endl;      // 3
```

| 特性 | 仿函数 | Lambda |
|------|--------|--------|
| 状态 | 可以有成员变量 | 通过捕获实现 |
| 复用 | 可以多次实例化 | 通常一次性使用 |
| 代码量 | 需要定义类 | 更简洁 |
| 可读性 | 适合复杂逻辑 | 适合简单操作 |

---

## 8. 关键要点

1. **仿函数**：重载 `operator()` 使对象可以像函数一样调用
2. **有状态**：仿函数可以保存状态（与普通函数不同）
3. **STL 兼容**：仿函数可以作为 STL 算法的谓词
4. **参数灵活**：`operator()` 可以接受任意数量和类型的参数
5. **Lambda 替代**：简单场景下可以使用 Lambda 代替
