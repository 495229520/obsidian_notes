# 9.3.8 重载递增递减运算符 (Increment/Decrement Operators)

## 1. 简介

在 C++ 中，递增 (`++`) 和递减 (`--`) 运算符可以被重载，以用于用户自定义的类型（如迭代器）。这使得你的类型可以像基本数据类型一样使用这些运算符。

重载递增或递减运算符时，需要区分**前置 (prefix)** 和**后置 (postfix)** 形式。

---

## 2. 语法

| 运算符 | 作为成员函数的原型 | 返回类型 | 行为 |
|--------|------------------|---------|------|
| **前缀递增** | `T& operator++()` | `T&` | 修改对象并返回自身的引用 |
| **后缀递增** | `T operator++(int)` | `T` | 复制对象，修改原对象，返回复制的旧值 |
| **前缀递减** | `T& operator--()` | `T&` | 修改对象并返回自身的引用 |
| **后缀递减** | `T operator--(int)` | `T` | 复制对象，修改原对象，返回复制的旧值 |

> **注意**：后缀版本的 `int` 参数是一个**哑参数 (dummy parameter)**，仅用于区分前缀和后缀版本，调用时不需要传递任何值。

---

## 3. 示例1：Counter 类

```cpp
#include <iostream>

class Counter {
private:
    int value;

public:
    Counter(int v = 0) : value(v) {}

    // 前缀递增 (++c)
    Counter& operator++() {
        ++value;
        return *this;  // 返回修改后的对象的引用
    }

    // 后缀递增 (c++)
    Counter operator++(int) {
        Counter temp = *this;  // 创建一个当前对象的副本
        ++(*this);             // 调用前缀递增来增加对象的值
        return temp;           // 返回修改前的旧值
    }

    // 前缀递减 (--c)
    Counter& operator--() {
        --value;
        return *this;
    }

    // 后缀递减 (c--)
    Counter operator--(int) {
        Counter temp = *this;
        --(*this);
        return temp;
    }

    void display() const {
        std::cout << "Value: " << value << std::endl;
    }
};

int main() {
    Counter c1(3);
    std::cout << "--- 前缀递增 ---" << std::endl;
    c1.display();
    (++c1).display();
    c1.display();

    std::cout << "\n--- 后缀递增 ---" << std::endl;
    Counter c2(10);
    c2.display();
    (c2++).display();  // 返回的是修改前的值
    c2.display();      // c2本身的值已经增加

    std::cout << "\n--- 前缀递减 ---" << std::endl;
    Counter c3(3);
    c3.display();
    (--c3).display();
    c3.display();

    std::cout << "\n--- 后缀递减 ---" << std::endl;
    Counter c4(8);
    c4.display();
    (c4--).display();
    c4.display();

    return 0;
}
```

## 4. 示例2：Point 类

```cpp
class Point {
private:
    int x, y;

public:
    Point(int x_val = 0, int y_val = 0) : x(x_val), y(y_val) {}

    // 前缀递增
    Point& operator++() {
        ++x;
        ++y;
        return *this;
    }

    // 后缀递增
    Point operator++(int) {
        Point temp = *this;
        ++(*this);
        return temp;
    }

    // 前缀递减
    Point& operator--() {
        --x;
        --y;
        return *this;
    }

    // 后缀递减
    Point operator--(int) {
        Point temp = *this;
        --(*this);
        return temp;
    }

    void print() const {
        std::cout << "Point(" << x << ", " << y << ")" << std::endl;
    }
};

int main() {
    Point p1(1, 2);
    std::cout << "初始点 p1: ";
    p1.print();

    Point p2 = ++p1;
    std::cout << "p2 = ++p1 后:" << std::endl;
    std::cout << "p1: ";
    p1.print();
    std::cout << "p2: ";
    p2.print();

    Point p3 = p1++;
    std::cout << "\np3 = p1++ 后:" << std::endl;
    std::cout << "p1: ";
    p1.print();
    std::cout << "p3: ";
    p3.print();

    return 0;
}
```

## 5. 前缀 vs 后缀对比

| 特性 | 前缀 (`++x`) | 后缀 (`x++`) |
|------|-------------|-------------|
| **返回类型** | 引用 (`T&`) | 值 (`T`) |
| **返回内容** | 修改后的对象 | 修改前的副本 |
| **效率** | 更高（无需复制） | 较低（需要复制） |
| **参数** | 无参数 | `int` 哑参数 |

---

## 6. 总结表格

| 运算符 | 作为成员函数的原型 | 返回类型建议 | 行为 |
|--------|------------------|-------------|------|
| 前缀递增 | `T& operator++()` | `T&` | 修改对象并返回自身的引用 |
| 后缀递增 | `T operator++(int)` | `T` | 复制对象，修改原对象，返回复制的旧值 |
| 前缀递减 | `T& operator--()` | `T&` | 修改对象并返回自身的引用 |
| 后缀递减 | `T operator--(int)` | `T` | 复制对象，修改原对象，返回复制的旧值 |

---

## 7. 关键要点

1. **区分前缀和后缀**：通过 `int` 哑参数区分
2. **前缀返回引用**：`T&`，直接返回 `*this`
3. **后缀返回值**：`T`，返回修改前的副本
4. **前缀更高效**：无需复制，优先使用前缀版本
5. **后缀调用前缀**：后缀版本通常调用前缀版本来实现递增/递减逻辑
