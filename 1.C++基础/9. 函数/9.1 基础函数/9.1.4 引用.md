> 本节内容是对 [[6.4 引用]] 的补充，重点讲解 C++11 引入的右值引用和移动语义。

在 C++ 中，左值 (lvalue) 和右值 (rvalue) 是表达式的分类，用于区分表达式所代表的对象的性质和使用方式。理解这两个概念对于掌握 C++ 中的资源管理（尤其是移动语义）至关重要。

---

## 1. 左值 (lvalue)

**定义**：左值是指一个持久的对象，它拥有内存地址，你可以通过取地址运算符 `&` 获取它的地址。换句话说，左值是可以放在赋值运算符 `=` 左边的表达式。

### 特征

- **具有身份 (identity)**：你可以确定两个左值是否引用同一个对象
- **可以被修改**（除非它是 `const` 的）
- **通常可以多次使用**

### 例子

- 变量名：`int x = 10;`（`x` 是一个左值）
- 数组元素：`int arr[5]; arr[0] = 1;`（`arr[0]` 是一个左值）
- 解引用指针：`int* ptr = &x; *ptr = 20;`（`*ptr` 是一个左值）
- 返回左值引用的函数：
  ```cpp
  int& get_x(int& value) { return value; }
  int y = 5;
  get_x(y) = 10;  // get_x(y) 是一个左值
  ```

### 左值表达式

- 赋值表达式（在大多数情况下）：`x = 5`（`x` 和整个赋值表达式 `x = 5` 都是左值）
- 前置递增/递减运算符：`++x`，`--x`
- 字符串字面量：`"hello"`（虽然它们是 `const` 的）

---

## 2. 右值 (rvalue)

**定义**：右值是指一个临时的、不具名的对象，它没有持久的内存地址。通常，右值出现在赋值运算符 `=` 的右边，并且在表达式结束后就会被销毁。C++11 引入了右值引用，使得我们可以捕获和利用右值。

### 特征

- **没有身份**（无法确定两个右值是否相同，因为它们是临时的）
- **通常不能被修改**（虽然可以通过右值引用修改某些类型的右值）
- **通常只使用一次**（因为它们是临时的）

### 例子

- 字面量（除了字符串字面量）：`10`，`3.14`，`true`
- 算术表达式的结果：`x + y`，`a * b`
- 返回非引用类型的函数：
  ```cpp
  int get_value() { return 5; }
  int z = get_value();  // get_value() 是一个右值
  ```

### 右值表达式

- 后置递增/递减运算符：`x++`，`y--`
- 类型转换表达式（大多数情况下）：`static_cast<int>(3.14)`
- 将亡值 (xvalue)：通过 `std::move()` 将左值转换为右值：
  ```cpp
  std::string s = "hello";
  std::string s2 = std::move(s);  // std::move(s) 是一个将亡值
  ```

---

## 右值引用

### 语法

右值引用的声明使用 `&&` 符号：

```cpp
int&& rref = 20;  // rref 是一个右值引用
```

### 移动语义 (Move Semantics)

右值引用允许我们将资源（如动态分配的内存）从一个即将销毁的右值对象"移动"到另一个对象，而不是进行深拷贝。这样可以避免不必要的拷贝操作，提高性能。

```cpp
// std::move 函数可以将左值转换为右值引用，以便应用移动语义
std::string str1 = "Hello";
std::string str2 = std::move(str1);  // str1 的资源被移动到 str2
// 此时 str1 处于有效但未定义状态
```

### 完美转发 (Perfect Forwarding)

右值引用与模板结合使用，可以实现完美转发，即将函数参数以其原始的值类别（左值或右值）传递给另一个函数。这在编写泛型库时非常有用。

### 提高程序性能

通过移动语义和完美转发，右值引用可以显著提高程序的性能，尤其是在处理大型对象和临时对象时。

## 引用数组

```cpp
int arr[5] = {1, 2, 3, 4, 5};
int (&ref)[5] = arr;

#include <iostream>

void printArray(int (&arr)[5]) {
    for (int i = 0; i < 5; ++i) {
        std::cout << arr[i] << " ";
    }
    std::cout << std::endl;
}

int main() {
    int myArray[5] = {10, 20, 30, 40, 50};
    printArray(myArray);
    return 0;
}
```

## 右值引用汇编分析

```cpp
int&& e = 320 + 230;
```

```asm
00C346CF mov    dword ptr [ebp-30h], 226h
00C346D6 lea    eax, [ebp-30h]
00C346D9 mov    dword ptr [ebp-24h], eax
```

分析：
- `int a[10];` / `// *(a + 1) = 100;` / `e = 1500;`
- `00C346DC mov eax, dword ptr [ebp-24h]`
- `00C346DF mov dword ptr [eax], 5DCh`
- `int x = c + 320 + 250;`
- `00C346E5 mov eax, dword ptr [ebp-0Ch]`
- `00C346E8 add eax, 23Ah`
- `00C346ED mov dword ptr [ebp-3Ch], eax`

右值 320+230 的计算结果存储在 `[ebp-30h]`，然后其地址被存储在 `[ebp-24h]`（即右值引用 `e`）。

---

## 关键要点

1. **左值**：有持久内存地址，可以取地址，可以放在赋值左边
2. **右值**：临时对象，无持久地址，表达式结束后销毁
3. **右值引用 (&&)**：C++11 特性，用于捕获右值，实现移动语义
4. **移动语义**：避免不必要的深拷贝，提高性能
5. **std::move()**：将左值转换为右值引用
