# 9.1.12 C++ 输入函数详解

> C++ 提供了多种输入方式：`cin >>`、`cin.get()`、`cin.getline()`、`getline()` 和 C 风格的 `getchar()`。理解它们对**空白字符**和**换行符**的处理差异，是避免输入问题的关键。

---

## 1. 概述

### 1.1 什么是 cin？

`std::cin` 是 C++ 标准库 `<iostream>` 中预定义的**标准输入流对象**，类型为 `std::istream`。它与标准输入设备（通常是键盘）绑定，用于从用户获取输入数据。

```cpp
#include <iostream>  // cin 定义在此头文件中

// cin 是 istream 类的全局对象
extern std::istream cin;
```

### 1.2 为什么需要多种输入函数？

不同的输入场景有不同的需求：

| 场景 | 推荐函数 | 原因 |
|------|----------|------|
| 读取单个数值/单词 | `cin >>` | 自动跳过空白，类型安全 |
| 读取整行文本（含空格） | `std::getline()` | 保留空格，丢弃换行符 |
| 逐字符处理输入 | `cin.get()` | 不跳过任何字符 |
| 读取到 C 风格字符数组 | `cin.getline()` | 固定缓冲区，丢弃换行符 |
| 密码输入等特殊场景 | `cin.get()` | 可逐字符控制回显 |

### 1.3 输入缓冲区机制

理解输入函数的关键是理解**输入缓冲区**：

```
用户输入: "hello world\n"
                ↓
┌─────────────────────────────┐
│  h e l l o   w o r l d \n  │  ← 输入缓冲区
└─────────────────────────────┘
                ↓
         cin >> / cin.get() / getline() 从缓冲区读取
```

- 用户按下回车后，整行数据进入缓冲区
- 输入函数从缓冲区**逐步读取**，而非直接从键盘
- 不同函数对**空白字符**和**换行符**的处理方式不同，这是产生问题的根源

## 2. cin >> (流提取运算符)

### 2.1 基本原理

`cin >>` 等价于 `cin.operator>>()`，调用成员函数进行读取。

```cpp
int n;
cin >> n;  // 等价于 cin.operator>>(n);
```

### 2.2 行为特点

| 特性 | 描述 |
|------|------|
| 跳过前导空白 | 自动忽略缓冲区开头的空格、Tab、换行符 |
| 遇空白停止 | 读取到空白字符时停止，**空白符留在缓冲区** |
| 类型安全 | 根据变量类型自动解析 |

### 2.3 示例

```cpp
#include <iostream>
#include <string>

int main() {
    std::string word;
    std::cin >> word;  // 输入 "hello world"
    // word = "hello"，" world\n" 残留在缓冲区

    std::cout << "读取到: " << word << std::endl;
    return 0;
}
```

### 2.4 连续读取

```cpp
int a, b;
std::cin >> a >> b;  // 输入 "10 20" 或 "10\n20" 都可以
// 自动跳过中间的空白字符
```

## 3. cin.get()

`cin.get()` 有多个重载版本，用于逐字符或逐行读取。

### 3.1 单字符版本

```cpp
// 方式一：返回字符的 ASCII 值
int ch = std::cin.get();

// 方式二：通过引用参数获取字符
char c;
std::cin.get(c);
```

**特点**：
- **不跳过空白字符**，包括空格、Tab、换行符
- 读取单个字符，无论它是什么

### 3.2 字符数组版本

```cpp
char buffer[100];
std::cin.get(buffer, 100);       // 最多读取 99 个字符
std::cin.get(buffer, 100, '#');  // 遇到 '#' 或读满停止
```

**特点**：
- 遇到换行符停止，**但不读取换行符**（换行符留在缓冲区）
- 自动添加 `'\0'` 结尾
- 可指定自定义分隔符

### 3.3 示例：处理残留换行符

```cpp
#include <iostream>

int main() {
    int num;
    char ch;

    std::cout << "输入数字: ";
    std::cin >> num;        // 输入 "42\n"，'\n' 残留

    std::cout << "输入字符: ";
    std::cin.get(ch);       // 直接读取到残留的 '\n'！

    std::cout << "ch 的 ASCII: " << static_cast<int>(ch) << std::endl;  // 输出 10
    return 0;
}
```

## 4. cin.getline()

### 4.1 语法

```cpp
std::cin.getline(char* buffer, std::streamsize count);
std::cin.getline(char* buffer, std::streamsize count, char delim);
```

### 4.2 行为特点

| 特性 | 描述 |
|------|------|
| 读取整行 | 读取直到遇到换行符（或指定分隔符） |
| **丢弃换行符** | 换行符被读取但不存入 buffer |
| 自动添加 `'\0'` | 在末尾添加空字符 |
| 仅用于 C 风格字符数组 | 不能用于 `std::string` |

### 4.3 示例

```cpp
#include <iostream>

int main() {
    char name[50];

    std::cout << "输入姓名: ";
    std::cin.getline(name, 50);  // 输入 "张三 李四\n"

    std::cout << "姓名: " << name << std::endl;  // 输出 "张三 李四"
    return 0;
}
```

---

## 5. std::getline() (推荐)

### 5.1 语法

```cpp
#include <string>

std::getline(std::istream& is, std::string& str);
std::getline(std::istream& is, std::string& str, char delim);
```

### 5.2 行为特点

| 特性 | 描述 |
|------|------|
| 支持 `std::string` | Modern C++ 推荐方式 |
| **丢弃换行符** | 换行符被读取但不存入 string |
| 自动扩容 | 无需预设缓冲区大小 |
| 可指定分隔符 | 默认为 `'\n'` |

### 5.3 示例

```cpp
#include <iostream>
#include <string>

int main() {
    std::string line;

    std::cout << "输入一行文字: ";
    std::getline(std::cin, line);  // 可以包含空格

    std::cout << "你输入了: " << line << std::endl;
    return 0;
}
```

## 6. getchar() (C 风格)

### 6.1 语法

```cpp
#include <cstdio>  // 或 <stdio.h>

int getchar();  // 返回字符的 ASCII 值，EOF 表示结束
```

### 6.2 特点

- C 标准库函数，返回 `int` 类型
- **不跳过任何字符**，包括空白字符
- 到达输入末尾或出错时返回 `EOF`（通常为 -1）

### 6.3 示例

```cpp
#include <cstdio>

int main() {
    int ch;
    while ((ch = getchar()) != '\n' && ch != EOF) {
        putchar(ch);  // 逐字符输出
    }
    return 0;
}
```

> **注意**：在 Modern C++ 中，优先使用 `std::cin.get()` 而非 `getchar()`。

## 7. 对比总结

| 函数 | 跳过前导空白 | 遇空白停止 | 处理换行符 | 适用类型 |
|------|:------------:|:----------:|:----------:|----------|
| `cin >>` | **是** | **是** | 残留 | 任意类型 |
| `cin.get(ch)` | 否 | 否 | 读取 | `char` |
| `cin.get(buf, n)` | 否 | 否 | **残留** | `char[]` |
| `cin.getline(buf, n)` | 否 | 否 | **丢弃** | `char[]` |
| `std::getline(cin, str)` | 否 | 否 | **丢弃** | `std::string` |
| `getchar()` | 否 | 否 | 读取 | `int` |

## 8. 常见陷阱与解决方案

### 8.1 陷阱：cin >> 后接 getline()

```cpp
int age;
std::string name;

std::cin >> age;              // 输入 "25\n"，'\n' 残留
std::getline(std::cin, name); // 立即读取到空行！
```

### 8.2 解决方案

**方案一：使用 `cin.ignore()`**

```cpp
std::cin >> age;
std::cin.ignore();  // 忽略一个字符（换行符）
std::getline(std::cin, name);
```

**方案二：忽略到换行符为止**

```cpp
std::cin >> age;
std::cin.ignore(std::numeric_limits<std::streamsize>::max(), '\n');
std::getline(std::cin, name);
```

**方案三：使用 `cin.get()` 消耗换行符**

```cpp
std::cin >> age;
std::cin.get();  // 读取并丢弃换行符
std::getline(std::cin, name);
```

### 8.3 陷阱：cin.get(buf, n) 后继续读取

```cpp
char buf1[50], buf2[50];
std::cin.get(buf1, 50);  // 换行符残留
std::cin.get(buf2, 50);  // 立即遇到换行符，读取空字符串！
```

**解决方案**：

```cpp
std::cin.get(buf1, 50);
std::cin.get();          // 消耗换行符
std::cin.get(buf2, 50);
```

## 9. 输入状态检查

### 9.1 流状态标志

```cpp
if (std::cin.fail()) {
    std::cerr << "输入失败" << std::endl;
    std::cin.clear();  // 清除错误状态
    std::cin.ignore(std::numeric_limits<std::streamsize>::max(), '\n');
}
```

### 9.2 常用状态函数

| 函数 | 描述 |
|------|------|
| `cin.good()` | 流状态正常 |
| `cin.fail()` | 上次操作失败 |
| `cin.eof()` | 到达输入末尾 |
| `cin.bad()` | 发生严重错误 |
| `cin.clear()` | 重置流状态 |

## 10. 最佳实践

### 10.1 统一使用 std::getline()

```cpp
#include <iostream>
#include <string>
#include <sstream>

int main() {
    std::string line;
    int age;
    std::string name;

    // 全部用 getline 读取，再解析
    std::cout << "输入年龄: ";
    std::getline(std::cin, line);
    std::istringstream(line) >> age;

    std::cout << "输入姓名: ";
    std::getline(std::cin, name);

    std::cout << name << " 今年 " << age << " 岁" << std::endl;
    return 0;
}
```

### 10.2 封装安全的输入函数

```cpp
#include <iostream>
#include <string>
#include <limits>

template<typename T>
T safeInput(const std::string& prompt) {
    T value;
    while (true) {
        std::cout << prompt;
        if (std::cin >> value) {
            std::cin.ignore(std::numeric_limits<std::streamsize>::max(), '\n');
            return value;
        }
        std::cerr << "输入无效，请重试" << std::endl;
        std::cin.clear();
        std::cin.ignore(std::numeric_limits<std::streamsize>::max(), '\n');
    }
}
```

---

## 11. 关键要点

1. **`cin >>`**：跳过空白，遇空白停止，**换行符残留**
2. **`cin.get()`**：不跳过空白，可逐字符读取
3. **`cin.getline()`**：读整行到 `char[]`，**丢弃换行符**
4. **`std::getline()`**：读整行到 `std::string`，**丢弃换行符**（推荐）
5. **混合使用时**：必须处理残留的换行符，使用 `cin.ignore()` 或 `cin.get()`
