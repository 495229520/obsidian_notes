## 返回值概述

在 C++ 中，函数可以返回各种类型的值，包括基本类型、指针、引用和对象。

---

## 返回引用 (Returning a Reference)

- **本质**：返回一个变量的别名，允许函数结果被直接赋值
- **优点**：避免了返回值的拷贝，提高了效率
- **风险**：返回局部变量的引用会造成**悬空引用**（函数结束后局部变量内存被释放）

### 危险示例：返回局部变量的引用

```cpp
int& foo(int val) {
    int res = val * 2;
    return res;  // 危险！返回局部变量的引用
}
```

> **原因**：局部变量 `res` 在函数返回后被销毁，返回的引用指向已释放的内存。

### 安全用法

不要返回函数内部新建的局部变量的引用。**函数体内部创建的局部变量的生命周期仅限于本函数内。** 当函数返回时，这些变量的值会被销毁。如果返回了局部变量的引用，调用者接收到的将是一个悬空引用。

可以返回：
- 全局变量的引用
- 静态变量的引用
- 通过参数传入的引用
- 在堆上分配的对象的引用（需注意内存管理）

```cpp
// 返回静态变量的引用 - 安全
int& getStaticValue() {
    static int value = 0;
    return value;
}

// 返回参数引用 - 安全
int& getMax(int& a, int& b) {
    return (a > b) ? a : b;
}
```

### 返回引用的 C++ 灵活用法

C++ 支持通过返回引用来实现如下效果：如果我们返回对变量的引用,这可实现类似通过函数调用修改变量的效果。

```cpp
struct Point {
    double x, y;
};

Point& getPointReferenceByRef(Point& ref) {
    return ref;
}

int main() {
    Point myPoint = { 1, 2 };
    std::cout << "原始坐标为: x = " << myPoint.x << ", y = " << myPoint.y << std::endl;

    // getPointReferenceByRef 返回一个 Point 引用，也返回了类型提示的权限
    // 被赋值的 myPoint 的引用，给引用赋值 = 给引用的对象赋值
    getPointReferenceByRef(myPoint) = { 3, 4 };
    std::cout << "修改后坐标为: x = " << myPoint.x << ", y = " << myPoint.y << std::endl;
}
```

## 返回指针 (Returning a Pointer)

- **本质**：返回一个内存地址
- **优点**：可以返回动态分配的对象，最终又可以返回 `nullptr` 表示失败
- **风险**：需要关心内存管理
  - 需要在外部释放动态分配的 ptr，否则内存泄漏
  - 不要返回指向栈内存的指针

### 安全用法

```cpp
int* getPointerToNewInt(int val) {
    int* ptr = new int(val);
    return ptr;  // 返回堆上分配的内存
}

int main() {
    int* myPtr = getPointerToNewInt(10, 20);  // 需要外部释放
    std::cout << "结果坐标为: x = " << myPtr->x << ", y = " << myPtr->y << std::endl;
    delete myPtr;  // 必须释放，否则内存泄漏
}
```

### 危险示例：返回局部变量的指针

```cpp
int* dangerousFunction() {
    int localVar = 42;
    return &localVar;  // 危险！返回局部变量的地址
}
```

> **原因**：`localVar` 在函数返回后被销毁，指针指向无效内存。

## 指针返回值与引用返回值的区别

在处理返回值时，它们对于返回值的处理方式有二三个区别：

- **指针返回值**：返回了内存地址的副本
- **引用返回值**：返回了原始对象本身

### 示例对比

```cpp
#include <iostream>

struct Point {
    int x, y;
};

Point createPointOnStack(int x, int y) {
    Point p;
    std::cout << "栈里创建的坐标地址: " << &p << " (p.x = " << p.x << ")";
    std::cout << std::endl;
    return p;
}

int main() {
    Point myPoint = createPointOnStack(10, 20);  // 返回p的副本
    std::cout << "返回后的坐标地址: " << &myPoint << ", x = " << myPoint.x;
    delete myPoint;  // 不可删除栈上的变量
}
```

```cpp
#include <iostream>

struct Point {
    int x, y;
};

Point* createPointOnHeap(int x, int y) {
    Point* p = new Point{x, 20};  // 在堆里创建，在主函数里消失
    return p;
}

int main() {
    Point* myPoint = createPointOnHeap(10, 20);
    std::cout << "结果坐标为: x = " << myPoint->x << ", y = " << myPoint->y << std::endl;
    delete myPoint;  // 必须在堆上释放
}
```

---

## 关键要点

1. **返回引用**：避免拷贝开销，但不能返回局部变量的引用
2. **返回指针**：可返回堆上对象或 `nullptr`，需注意内存管理
3. **悬空引用/指针**：返回局部变量的地址或引用会导致未定义行为
4. **安全返回**：可返回静态变量、全局变量、参数引用或堆上分配的对象
