## 数组参数

### 数组参数退化

将数组作为函数的参数时需要传递数组的长度，因为在 C 和 C++ 中，当**将数组作为函数的参数传递时，数组会退化为指向其第一个元素的指针**。这意味着函数实际上接收到的是数组的起始地址，而不是整个数组的副本。

```cpp
// 这两种声明方式是等价的
void Sort(int ary[], unsigned count)    void Sort(int* ary, unsigned count)
{                                       {
    for(int i=0; i<count; i++)              for(int i=0; i<count; i++)
        std::cout << ary[i];                    std::cout << ary[i];
}                                       }

int main()
{
    int a[4]{1,2,3,4};
    Sort(a, 4);
}
```

### 二维数组参数

将数组作为函数参数时必须指定第二个维度的大小，例如 `int add(int a[][2])`。这是因为 C++ 编译器需要知道数组的列数，以便正确计算数组元素的内存地址。

```cpp
// 二维数组参数：必须指定列数
void Sort(int ary[][2], unsigned count)
{
    // ...
}

int main()
{
    int a[3][2]{{1,2},{3,4},{5,6}};
    Sort(a, 3);
}
```

在 C++ 中，当二维数组作为函数参数传递时，必须指定第二个维度的大小。例如 `int add(int a[][2])`。这是因为 C++ 编译器需要知道数组的列数，以便正确计算数组元素的[[6.1基本指针#多维数组的内存布局|内存地址]]。

## 一维数组

```cpp
#include <iostream>

void printArray(int arr[], int size) {
    for (int i = 0; i < size; ++i) {
        std::cout << arr[i] << " ";
    }
    std::cout << std::endl;
}

int main() {
    int myArray[] = {10, 20, 30, 40, 50};
    int arraySize = sizeof(myArray) / sizeof(myArray[0]);
    printArray(myArray, arraySize);  // 传递数组和大小
    return 0;
}
```

## 二维数组

```cpp
#include <iostream>

void print2DArray(int arr[][3], int rows) {
    for (int i = 0; i < rows; ++i) {
        for (int j = 0; j < 3; ++j) {
            std::cout << arr[i][j] << " ";
        }
        std::cout << std::endl;
    }
}

int main() {
    int myArray[2][3] = {{1, 2, 3}, {4, 5, 6}};
    print2DArray(myArray, 2);  // 传递数组和行数
    return 0;
}
```

> **注意**：必须指定列数（这里是 3），而行数可以作为单独的参数传递。

## 使用模板和数组引用

**将数组作为函数的参数时需要传递数组的长度**，但是如果将数组引用作为参数传入函数数就可以不传递数组的长度。

### 数组指针

- 当将普通数组传递给函数时，数组会"退化"为指向其首元素的指针
- 这意味着函数仅接收数组的起始地址，而丢失了数组的长度信息
- 因此，通常需要显式地传递数组长度

### 数组引用

- 数组引用保留了数组的完整类型信息，包括其长度
- 因此，在函数内部，可以通过引用直接访问向数组，并且编译器在编译时已知数组的大小
- 这允许编译器执行更严格的类型检查，并防止越界访问

```cpp
#include <iostream>

template <typename T, size_t N>
void printArrayTemplate(T (&arr)[N]) {
    for (size_t i = 0; i < N; ++i) {
        std::cout << arr[i] << " ";
    }
    std::cout << std::endl;
}

int main() {
    int myArray[] = {1, 2, 3, 4, 5};
    double myDoubleArray[] = {1.1, 2.2, 3.3};
    printArrayTemplate(myArray);       // 无需传递大小
    printArrayTemplate(myDoubleArray);
    return 0;
}
```

`template <typename T, size_t N>` 表示一个模板，它接受两个参数：
- 一个类型参数 T，表示数组的类型
- 一个非类型参数 N，表示一个常量大小

这种模板常用于处理固定大小的数组。例如，你可以编写一个函数，该函数可以处理任何类型的固定大小小数组，而无需知道通数组的具体类型或大小。

## 默认参数

### 注意点

1. 给默认值的时候，从右向左给
2. 默认值的调用效率会更高，因为不用 mov 汇编将常量保存在 ptr 中，然后再压栈。而是直接压入默认值
3. 定义可以给出形参默认值，声明也可以给形参默认值
4. 形参给默认值的时候，不管是定义处给还是声明处给，形参默认值只能出现一次

### 案例

```cpp
#include <iostream>
using namespace std;

// 压栈是从右向左的，因此默认值也是从右向左
// 如果左边有默认值，右边没有就会报错
int sum(int a = 10, int b = 20) {
    return a + b;
}

int main()
{
    int a = 10;
    int b = 20;

    /*
    mov eax, dword ptr[ebp-8]
    push eax
    mov ecx, dword ptr[ebp-4]
    push ecx
    call sum
    */
    int ret = sum(a, b);
    cout << "ret:" << ret << endl;

    /*
    没有 b，因此直接调用默认值: 20
    push 14H
    mov ecx, dword ptr[ebp-4]
    push ecx
    call sum
    */
    ret = sum(a);

    /*
    push 14H
    push 0AH
    call sum
    */
    ret = sum();
}
```

## 关键要点

1. **数组退化**：数组作为参数传递时会退化为指针，丢失长度信息
2. **二维数组**：必须指定列数，行数可作为参数传递
3. **数组引用模板**：使用模板和数组引用可以保留数组大小信息
4. **默认参数**：从右向左设置，只能出现一次（声明或定义处）
