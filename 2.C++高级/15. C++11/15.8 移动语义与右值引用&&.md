# 15.8 移动语义与右值引用

---

## 简介

在 C++ 中，当一个对象被赋值给另一个对象时，通常会发生**拷贝（Copy）**，这意味着会创建一个新的副本，然后再把原对象的所有数据复制到新副本中去。如果这是一个大对象，比如一个包含几百万元素的数组（比如一个大 vector），那么这个拷贝操作的成本就会非常大。

**[[13.2 右值引用|移动语义]]就是为了解决这个问题的**。它的核心思想是：当源对象是一个即将被销毁的临时对象（右值）时，我们可以直接"窃取"它的资源，而不是复制它的资源。

### 类比理解

- **拷贝语义**：你要读一本书去复印一份，然后还给图书馆。你手里是一本全新的复印件
- **移动语义**：你直接把书从图书馆拿走，你手里是那本书本身，不需要复印过程，图书馆也不再有这本书了

---

## 作用

在以下场景中，移动语义可以带来巨大的性能提升：

### 临时对象

当一个函数返回一个大对象（比如返回一个大的 vector）时，如果没有移动语义，返回值会被复制。有了移动语义，我们可以直接将返回值这个临时对象内部的资源移动到接收变量，避免了不必要的拷贝。

### std::vector 扩容

当你在 `std::vector` 中 `push_back` 一个大对象时，如果这个元素需要容器内部重新分配内存，所有已有的元素都必须移动或复制到新的内存位置。使用移动语义可以让这个过程更加高效，因为不需要复制每个元素的内容。

---

## std::move

`std::move` 本身并不执行任何移动操作，它的作用非常简洁：它是一个类型转换操作符。

**它会将任何类型的左值转换为[[13.2 右值引用|右值引用]]**。它的作用在于表达意图："这个对象我不再需要了，可以被另一个对象'窃取'了。"编译器看到这个右值引用后，就会优先选择调用移动语义版本的函数（如移动构造函数）。从而实现真正的移动。

> **注意**：`std::move` 的本质是类型转换！"这个对象我不再需要它，你们安全地把它'搬走'吧。"

### 语法

```cpp
// 将一个左值对象 object 转换为右值引用
std::move(object);
```

## 案例

```cpp
class MyString {
private:
    char* data;
    size_t size;

public:
    // 普通构造函数
    MyString(const char* s) {
        std::cout << "调用普通构造函数" << std::endl;
        size = strlen(s);
        data = new char[size + 1];
        memcpy(data, s, size + 1);
    }

    // 移动构造函数（核心）
    MyString(MyString&& other) noexcept {
        std::cout << "调用移动构造函数" << std::endl;
        // 1. 偷取资源
        data = other.data;
        size = other.size;
        // 2. 将 other 的资源指针设为空
        other.data = nullptr;
        other.size = 0;
    }

    // 移动赋值运算符
    MyString& operator=(MyString&& other) noexcept {
        std::cout << "调用移动赋值运算符" << std::endl;
        if (this != &other) {
            // 1. 偷取 other 的资源
            delete[] data;
            data = other.data;
            size = other.size;
            // 2. 将 other 的资源指针设为空，防止它在析构时释放
            other.data = nullptr;
            other.size = 0;
        }
        return *this;
    }

    // 拷贝构造函数（为了对比）
    MyString(const MyString& other) {
        std::cout << "调用拷贝构造函数" << std::endl;
        size = other.size;
        data = new char[size + 1];
        memcpy(data, other.data, size + 1);
    }

    // 析构函数
    ~MyString() {
        if (data) {
            delete[] data;
        }
        data = nullptr;
    }
};

int main() {
    // 临时对象，会调用移动构造函数
    MyString s1("Hello");  // 普通构造
    MyString s2(std::move(s1));  // 强制移动，调用移动构造函数

    std::cout << "---" << std::endl;

    MyString s3("World");  // 普通构造
    MyString s4("Test");   // 普通构造
    s3 = std::move(s4);    // 调用移动赋值运算符

    return 0;
}
```

---

## 关键要点

| 概念 | 说明 |
|------|------|
| 移动语义 | 通过"窃取"资源避免深拷贝，提升性能 |
| 右值引用 (`&&`) | 用于绑定到临时对象（右值） |
| `std::move` | 将左值转换为右值引用，表达"可以被移动"的意图 |
| 移动构造函数 | 从另一个对象"窃取"资源来初始化新对象 |
| 移动赋值运算符 | 从另一个对象"窃取"资源来赋值 |
| `noexcept` | 移动操作应标记为 noexcept 以获得最佳 STL 兼容性 |
