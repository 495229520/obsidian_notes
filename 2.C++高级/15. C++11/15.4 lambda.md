# 15.4 Lambda 表达式

---

## 简介

简单来说，Lambda 表达式就是一种在 C++11 及更高版本中创建**匿名函数**的便捷方式，你可以把它想象成一个"一次性"的函数，它没有名字，可以在需要的地方快速创建，并且可以直接创建它也可以用。

Lambda 表达式非常适合用作[[15.5 for_each|std::for_each]]只需要用一次的小型函数对象，仿似作为 STL 算法的参数（如 std::sort、std::for_each）。

---

## 语法

```cpp
[捕获列表] (参数列表) -> 返回类型 {
    函数体;
};
```

- **[捕获列表] (Capture Clause)**：这是 Lambda 表达式最独特的部分。它用于指定 Lambda 表达式内部可以访问外部作用域的哪些变量
  - **[]**：不捕获任何外部变量
  - **[var]**：按值捕获变量 var，这会创建一个 var 的副本。在 Lambda 表达式内部无法修改变量和原始变量
  - **[&var]**：按引用捕获变量 var。在 Lambda 表达式内部可以直接修改原始变量，但它可能存在悬空引用问题
  - **[=]**：按值捕获所有外部变量
  - **[&]**：按引用捕获所有外部变量

- **(参数列表) (Parameter List)**：和普通函数一样，定义 Lambda 表达式接受的输入参数
- **-> 返回类型 (Return Type)**：指定 Lambda 表达式的返回类型，如果函数体只有一条 return 语句，或者不需要返回值（void），那么编译器通常可以自动推断返回类型，这部分可以省略
- **(函数体) (Function Body)**：包含 Lambda 表达式执行的代码

## 示例

### 示例 1：基本用法（不需要捕获）

一个例子使用 Lambda 表达式作为 std::for_each 的参数，打印容器中的每一个元素。

```cpp
int main() {
    std::vector<int> numbers = {1, 2, 3, 4, 5};

    std::cout << "使用 Lambda 表达式打印元素：" << std::endl;
    // [] 不捕获任何外部变量
    // (int n) 定义一个整型参数 n
    // { ... } 函数体，打印 n
    std::for_each(numbers.begin(), numbers.end(), [](int n) {
        std::cout << n << " ";
    });
    std::cout << std::endl;

    return 0;
}
```

### 示例 2：按值捕获

这个例子中，Lambda 表达式需要访问外部变量 limit，因为是按值捕获，Lambda 内部无法修改 limit 的值。

```cpp
int main() {
    std::vector<int> numbers = {10, 5, 20, 8, 15};
    int limit = 12;

    std::cout << "打印所有小于 " << limit << " 的元素：" << std::endl;
    // [limit] 按值捕获 limit 变量
    // auto 可以自动推断出返回类型
    std::for_each(numbers.begin(), numbers.end(), [limit](int n) {
        if (n < limit) {
            std::cout << n << " ";
        }
    });
    std::cout << std::endl;

    return 0;
}
```

### 示例 3：按引用捕获（最常用）

这个例子展示了如何使用引用捕获来修改外部变量。我们统计容器中有多少个偶数。

```cpp
int main() {
    std::vector<int> numbers = {1, 2, 3, 4, 5, 6};
    int even_count = 0;

    std::cout << "统计偶数个数：" << std::endl;
    // [&even_count] 按引用捕获 even_count 变量
    // 直接修改外部 even_count 变量
    std::for_each(numbers.begin(), numbers.end(), [&even_count](int n) {
        if (n % 2 == 0) {
            even_count++;  // 直接修改外部 even_count 变量
        }
    });

    std::cout << "偶数的总数是：" << even_count << std::endl;

    return 0;
}
```

### 示例 4：完整的语法形式

这个例子展示了包含参数列表和返回类型的完整语法。

```cpp
int main() {
    // 捕获列表为空 []
    // 参数列表 (int x, int y)
    // 返回类型 -> int
    // 函数体 { return ... }
    auto add = [](int x, int y) -> int {
        return x + y;
    };

    int result = add(5, 3);
    std::cout << "5 + 3 = " << result << std::endl;

    return 0;
}
```

---

## 关键要点

| 语法 | 说明 |
|------|------|
| `[]` | 不捕获任何外部变量 |
| `[var]` | 按值捕获 var（只读副本） |
| `[&var]` | 按引用捕获 var（可修改原值） |
| `[=]` | 按值捕获所有外部变量 |
| `[&]` | 按引用捕获所有外部变量 |
| `[=, &var]` | 默认按值捕获，但 var 按引用 |
| `[&, var]` | 默认按引用捕获，但 var 按值 |
