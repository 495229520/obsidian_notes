# 15.11 default、delete 与初始化列表

---

## = default 默认函数

### 语法

```cpp
ClassName() = default;
ClassName(const ClassName&) = default;
```

### 作用

`= default` 的作用是**显式地要求编译器生成该特殊成员函数的默认版本**。

在某些情况下，编译器会停止为你自动生成默认构造函数。例如，如果你的类中包含一个带有非默认[[12.1.6 构造函数|构造函数]]的成员变量，或者你自己定义了其他构造函数，那么编译器就不会为你生成默认构造函数了。

```cpp
class MyClass {
public:
    // 自己定义了带参数的构造函数
    MyClass(int x) {}

    // 编译器不会自动生成默认构造函数了！
    // MyClass obj;  // 这会导致编译错误
};

class AnotherClass {
public:
    // 显式地告诉编译器，我需要你生成默认构造函数
    MyClass() = default;
};
```

在这种情况下，`= default` 让你能强制编译器生成默认版本，而无需你自己编写一个空的函数体，这既省事又高效。

`= default` 还能用于明确指出你想要使用编译器生成的默认版本，尤其是在模板编程中，这能提高代码的清晰度。

### default 默认构造函数不会做的事情

理解 `= default` 生成的默认构造函数**不做什么**同样重要：

- **它不会初始化基本类型（如 int、char 等）的成员变量**。这些变量的值是未定义的，这是一个常见的陷阱，需要你手动初始化
- **它不会做任何复杂的初始化**。如果成员变量的默认构造函数需要特殊的逻辑，你需要自己编写构造函数

### 总结

`= default` 生成的默认构造函数是一个"懒惰"的构造函数。它不会为你做任何多余的事情，只是确保你的对象能够被创建，并让其成员变量各自进行默认初始化。它让你能够明确地告诉编译器"我不需要自定义的初始化逻辑，请使用标准的、最高效的空操作版本。"

## = delete 删除函数

### 语法

```cpp
ClassName() = delete;
ClassName(const ClassName&) = delete;
```

### 作用

`= delete` 的作用是**显式地禁用某个函数**。它主要用于以下几个目的：

### 1. 阻止拷贝和赋值

这是 `= delete` 最常见的用途。在 C++11 之前，为了防止对象被拷贝，开发者需要将拷贝构造函数和拷贝赋值运算符声明为 `private`，有了 `= delete`，你可以更清晰、更简单地实现这一点。

```cpp
class NonCopyable {
public:
    NonCopyable() = default;

    // 显式禁用拷贝构造函数
    NonCopyable(const NonCopyable&) = delete;

    // 显式禁用拷贝赋值运算符
    NonCopyable& operator=(const NonCopyable&) = delete;
};

NonCopyable a;
// NonCopyable b = a;  // 编译错误！拷贝构造函数被删除了
// NonCopyable c;
// c = a;  // 编译错误！拷贝赋值运算符被删除了
```

### 2. 禁用特定类型的隐式转换

你可以禁用某个特定参数类型的构造函数，从而防止不必要的隐式类型转换。

### 3. 禁用某个成员函数

`= delete` 不仅限于特殊成员函数，你还可以用它来禁用任何你不想被调用的成员函数，例如某个重载函数。

---

## 总结

| 关键字 | 作用 | 典型用法 |
|--------|------|----------|
| `= default;` | 显式地要求编译器生成默认版本 | 强制生成默认构造函数，或明确指出使用默认版本 |
| `= delete;` | 显式地禁用某个函数 | 防止对象被拷贝和赋值，或者禁用某个重载函数 |

## 统一初始化（列表初始化）

C++11 引入了**统一初始化**语法，使用花括号 `{}` 来初始化各种类型的对象。

### 初始化数组

```cpp
int arr[] = { 1, 2, 3, 4, 5 };
```

### 初始化容器

```cpp
int arr[] = { 1, 2, 3, 4, 5 };

// 初始化 std::vector
#include <iostream>
#include <vector>

int main() {
    std::vector<int> vec = { 1, 2, 3, 4, 5 };
    for (const auto& elem : vec) {
        std::cout << elem << " ";
    }
    std::cout << std::endl;
    return 0;
}
```

### 初始化自定义类

```cpp
class MyClass {
public:
    int x;
    double y;

    MyClass(int x, double y) : x(x), y(y) {}
};

int main() {
    MyClass obj{ 10, 3.14 };  // 列表初始化
    std::cout << "x = " << obj.x << ", y = " << obj.y << std::endl;
    return 0;
}
```

### 初始化结构体

```cpp
struct Point {
    int x;
    int y;
};

int main() {
    Point p = { 10, 20 };  // 列表初始化
    std::cout << "x = " << p.x << ", y = " << p.y << std::endl;
    return 0;
}
```

## 关键要点

| 特性 | 说明 |
|------|------|
| `= default` | 显式要求编译器生成默认版本 |
| `= delete` | 显式禁用某个函数 |
| 列表初始化 `{}` | C++11 统一初始化语法，适用于各种类型 |
| 防止拷贝 | 使用 `= delete` 禁用拷贝构造和赋值运算符 |
