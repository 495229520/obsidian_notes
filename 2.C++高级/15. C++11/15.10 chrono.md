# 15.10 std::chrono 时间库

---

## 简介

`std::chrono` 是 C++ 标准库中一个功能强大的时间库，它提供了用于处理时间、日期、时间间隔和计时的工具。

`std::chrono` 的核心思想是将时间的概念分解为三个基本组成部分：

1. **时钟 (Clocks)**：表示一个时间的源头，比如系统时钟（system_clock）、单调时钟（steady_clock）等
2. **时间点 (Time Points)**：表示一个特定的时间点，比如"今天下午 3:00"。它由一个时钟和一个时间间隔组成
3. **时间间隔 (Durations)**：表示一段时间，比如"30 分钟"或"2 小时"

这种设计让使用时间的代码作变得类型安全，不易出错，例如，你不能将分钟和小时直接混用而不进行换算和显式转换。

---

## 语法

`std::chrono` 的语法主要围绕这三个概念展开。

### 时间间隔 (std::chrono::duration)

- **语法**：`std::chrono::duration<Rep, Period>`
- **Rep**：存储时间间隔数值的类型（如 long long）
- **Period**：表示时间单位，一个 std::ratio 类型，比如：
  - `std::chrono::nanoseconds`
  - `std::chrono::microseconds`
  - `std::chrono::milliseconds`
  - `std::chrono::seconds`
  - `std::chrono::minutes`
  - `std::chrono::hours`

**创建时间间隔**：

```cpp
// 使用字面量（C++14 及更高版本）：
using namespace std::chrono;  // 引入字面量后缀
auto thirty_minutes = 30min;
auto two_seconds = 2s;

// 使用构造函数：
std::chrono::minutes thirty_minutes(30);
std::chrono::seconds two_seconds(2);
```

### 时钟 (std::chrono::clock)

`std::chrono` 提供了三种主要的时钟：

- **std::chrono::system_clock**
  - 作用：表示系统范围的实时时钟。它与墙上时钟（wall clock）同步，因此它可以被用户或系统时间同步（NTP）修改
  - 用途：用于获取当前的时间，比如文件修改时间、日志时间戳

- **std::chrono::steady_clock**
  - 作用：一个单调递增的时钟，它的时间点只会向前移动，不会倒退
  - 用途：用于测量时间间隔，比如代码执行时间、性能测试等

- **std::chrono::high_resolution_clock**
  - 作用：提供最高精度的时钟，它通常是 system_clock 或 steady_clock 的别名
  - 用途：用于需要非常精确的时间测量的场景

**获取当前时间点**：

```cpp
auto now = std::chrono::steady_clock::now();
```

### 时间点 (std::chrono::time_point)

- **语法**：`std::chrono::time_point<Clock, Duration>`
- **Clock**：时间点所属的时钟
- **Duration**：从时钟纪元（epoch）开始的时间间隔
- 通常通过 now() 成员函数或其他方式创建
- 也可以通过时间间隔相加相减获得新的时间点

## 案例

### 示例 1：测量函数执行时间

```cpp
void do_some_work() {
    // 模拟在这里做一些事情
    std::this_thread::sleep_for(std::chrono::milliseconds(200));
}

int main() {
    // 记录开始时间点
    auto start = std::chrono::steady_clock::now();

    do_some_work();

    // 记录结束时间点
    auto end = std::chrono::steady_clock::now();

    // 计算时间间隔
    auto duration = end - start;

    // 将时间间隔转换为毫秒
    auto ms = std::chrono::duration_cast<std::chrono::milliseconds>(duration);

    std::cout << "函数执行耗时：" << ms.count() << " 毫秒" << std::endl;

    return 0;
}
```

### 示例 2：时间点和时间间隔的加法

`std::chrono` 的类型安全设计使得不同单位的时间间隔不能直接相加，但你可以将它们转换为相同单位。

```cpp
int main() {
    using namespace std::chrono;

    // 创建一个时间点，表示现在
    auto now = system_clock::now();
    // 创建一个时间间隔，3小时50分钟
    auto three_hours = 3h;
    auto fifty_minutes = 50min;

    // 不同单位不能直接相加，需要先转换为相同单位
    auto total_minutes = duration_cast<minutes>(three_hours) + fifty_minutes;
    std::cout << "3 小时 50 分钟等于 " << total_minutes.count() << " 分钟" << std::endl;

    // 将时间间隔加到时间点上
    auto future_time = now + total_minutes;

    std::cout << "未来的时间点..." << std::endl;

    return 0;
}
```

## 关键要点

| 概念 | 说明 |
|------|------|
| `duration` | 时间间隔，如秒、毫秒、分钟 |
| `time_point` | 时间点，由时钟和持续时间组成 |
| `system_clock` | 系统时钟，可被调整 |
| `steady_clock` | 单调时钟，适合测量时间间隔 |
| `duration_cast` | 时间单位转换 |
| `now()` | 获取当前时间点 |
