# 15.5 std::for_each 算法

---

## 简介

`std::for_each` 是 C++ 标准库中的一个算法，定义在 `<algorithm>` 头文件中。它的主要作用是**对指定范围内的每一个元素应用一个函数或可调用对象**。

简单来说，它就是一个通用的、方便的循环，但它隐藏了循环的实现细节（比如迭代器管理、循环条件等）。你只需要告诉它要处理的元素范围，以及对每个元素要做什么操作，它就会帮你完成。

---

## 语法

```cpp
template< class InputIt, class UnaryFunction >
constexpr UnaryFunction for_each( InputIt first, InputIt last, UnaryFunction f );
```

- **InputIt first**：一个输入迭代器，指向要处理的**范围的起始位置**
- **InputIt last**：一个输入迭代器，指向要处理的**范围的结束位置**（不包含这个位置的元素）
- **UnaryFunction f**：一个一元函数或可调用对象（比如 [[15.4 lambda|Lambda 表达式]]），它接受一个参数，这个参数就是范围内的每个元素

---

## 作用特点

`std::for_each` 的核心作用就是**遍历**，但它比传统的 for 循环更具通用性，因为它能够与任何支持迭代器（begin() 和 end()）的容器一起工作。

**主要特点**：

1. **代码简洁**：它将遍历的逻辑（迭代器管理、循环条件等）封装起来，让你只需关心每个元素的操作，这让代码更易读
2. **通用性强**：可以用于 std::vector、std::list、std::deque、std::array 等所有标准容器
3. **支持任何可调用对象**：你可以传入普通函数、函数对象、或者最常用的 Lambda 表达式

---

## 案例

```cpp
// 1. 定义一个普通函数作为操作
void print_element(int n) {
    std::cout << n << " ";
}

// 2. 定义一个函数对象（仿函数）作为操作
struct Doubler {
    void operator()(int& n) const {
        n *= 2;
    }
};

int main() {
    std::vector<int> numbers = {1, 2, 3, 4, 5};

    // 示例 1：使用普通函数
    std::cout << "使用普通函数打印元素：";
    std::for_each(numbers.begin(), numbers.end(), print_element);
    std::cout << std::endl;

    // 示例 2：使用 Lambda 表达式
    std::cout << "使用 Lambda 表达式打印元素：";
    // lambda 表达式是 for_each 最常用的搭档
    std::for_each(numbers.begin(), numbers.end(), [](int n) {
        std::cout << n * 10 << " ";
    });
    std::cout << std::endl;

    // 示例 3：使用函数对象，并修改原始容器
    std::cout << "使用函数对象将元素翻倍：";
    // 注意这里传入的是 numbers.begin() 和 end() 的引用
    std::for_each(numbers.begin(), numbers.end(), Doubler());

    std::cout << "翻倍后的元素：";
    // 再次遍历，检查修改结果
    std::for_each(numbers.begin(), numbers.end(), print_element);
    std::cout << std::endl;

    return 0;
}
```

---

## 关键要点

| 特性 | 说明 |
|------|------|
| 头文件 | `<algorithm>` |
| 作用 | 对范围内每个元素执行操作 |
| 参数 | 起始迭代器、结束迭代器、可调用对象 |
| 常用搭档 | Lambda 表达式、仿函数 |
| 优势 | 代码简洁、通用性强 |
