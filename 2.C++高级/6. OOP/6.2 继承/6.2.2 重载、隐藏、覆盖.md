# 6.2.2 重载、隐藏、覆盖

## 概念区分

| 概念 | 作用域 | 条件 | 绑定方式 |
|------|--------|------|----------|
| **重载（Overload）** | 同一作用域 | 函数名相同，参数列表不同 | 静态绑定（编译时） |
| **隐藏（Hide）** | 不同作用域（基类/派生类） | 函数名相同 | 静态绑定（编译时） |
| **覆盖（Override）** | 不同作用域（基类/派生类） | 函数名、参数列表、返回值都相同，且基类是虚函数 | 动态绑定（运行时） |

---

## 重载（Overload）

**定义**：在**同一作用域**内，函数名相同但参数列表不同的函数

```cpp
class Base
{
public:
    Base(int data = 10) : ma(data) {}
    void show() { cout << "Base::show()" << endl; }
    void show(int i) { cout << "Base::show(int)" << endl; }  // 重载
};
```

**特点**：
- 必须在**同一作用域**
- 函数名相同，参数列表不同（类型、数量、顺序）
- 返回值类型可以不同
- 在**编译阶段**就确定好调用的函数

---

## 隐藏（Hide）

**定义**：在继承结构中，派生类定义了与基类**同名**的函数，导致基类函数被隐藏

```cpp
class Base
{
public:
    Base(int data = 10) : ma(data) {}
    void show() { cout << "Base::show()" << endl; }
    void show(int i) { cout << "Base::show(int)" << endl; }
protected:
    int ma;
};

class Derive : public Base
{
public:
    Derive(int data = 20) : Base(data), mb(data) {}
    void show() { cout << "Derive::show()" << endl; }  // 隐藏了基类所有的 show
private:
    int mb;
};

int main()
{
    Derive d;
    d.show();       // 调用 Derive::show()
    // d.show(10);  // 错误！Base::show(int) 被隐藏了
    d.Base::show(); // 正确，显式调用基类的 show()
}
```

**特点**：
- 派生类的函数会**隐藏基类所有同名函数**（无论参数是否相同）
- 要调用被隐藏的基类函数，需要使用**作用域限定符**：`d.Base::show()`
- 静态绑定（编译时决定）

---

## 覆盖（Override）

**定义**：在继承结构中，派生类重写基类的**虚函数**，实现[[6.3.1 深入理解多态|多态]]

```cpp
class Base
{
public:
    Base(int data = 10) : ma(data) {}
    virtual void show() { cout << "Base::show()" << endl; }
    virtual void show(int i) { cout << "Base::show(int)" << endl; }
protected:
    int ma;
};

class Derive : public Base
{
public:
    Derive(int data = 20) : Base(data), mb(data) {}
    // 覆盖基类的虚函数
    void show() { cout << "Derive::show()" << endl; }
private:
    int mb;
};

int main()
{
    Derive d;
    Base* pb = &d;
    pb->show();      // 动态绑定，调用 Derive::show()
    pb->show(10);    // 调用 Base::show(int)，因为派生类没有覆盖它
}
```

**覆盖的条件**：
1. 基类函数必须是**虚函数**（`virtual`）
2. 派生类函数的**函数名、参数列表、返回值**必须与基类完全相同
3. 派生类函数的访问权限可以不同

## 一个完整的例子

```cpp
class Base
{
public:
    Base(int data = 10) : ma(data) {}
    virtual void show() { cout << "Base::show()" << endl; }
    void show(int i) { cout << "Base::show(int)" << endl; }  // 非虚函数
protected:
    int ma;
};

class Derive : public Base
{
public:
    Derive(int data = 20) : Base(data), mb(data) {}
    void show() { cout << "Derive::show()" << endl; }  // 覆盖
private:
    int mb;
};

int main()
{
    Derive d;
    Base* pb = &d;

    pb->show();      // 输出：Derive::show()  （动态绑定，覆盖）
    pb->show(10);    // 输出：Base::show(int)  （静态绑定）

    d.show();        // 输出：Derive::show()
    // d.show(10);   // 错误！Base::show(int) 被隐藏
    d.Base::show(10); // 输出：Base::show(int)（显式调用）
}
```

## 关键要点

| 特性 | 重载 | 隐藏 | 覆盖 |
|------|------|------|------|
| 作用域 | 同一作用域 | 基类与派生类 | 基类与派生类 |
| 函数名 | 相同 | 相同 | 相同 |
| 参数列表 | 不同 | 任意 | 相同 |
| 返回值 | 任意 | 任意 | 相同 |
| virtual | 不要求 | 不要求 | **必须** |
| 绑定时机 | 编译时 | 编译时 | **运行时** |
