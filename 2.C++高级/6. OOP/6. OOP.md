## 知识点

1. **类**（属性：成员变量，行为：成员方法）
2. C 各种各样的结构定义是 `struct`；C++ 类 => 实体的抽象类型
3. 一个对象的生成需要经历两个步骤：
   - 1. 分配内存 2. 调用构造函数
4. **结构体一般是私有的**，因此需要给外部提供共有的成员方法来访问私有的属性
5. **给成员变量提供一组 `getXXX` 或 `setXXX` 的方法**，类体内实现的 `getXXX` 或 `setXXX` 方法会自动处理为 [[2. inline内联|inline]] 内联函数
6. 在类的外面实现成员方法，不会自动处理为 inline，需要手动添加
7. `getXXX` 方法返回私有成员变量，不能让其被外部指针修改，因此需要使用 const char*
8. 定义可以给出形参默认值，声明也可以给形参默认值
9. **对象的内存大小只和成员变量有关**
   - 计算方式：找到成员变量中最大的那个类型，然后就是它的整数倍
   - 这里最长的是 `double` 8 个字节，但有 20 个字节，不是 8 的整数倍
   - 所以内存就是 20 + 4 = 24，刚好是 8 的三倍
   - 实例化一个对象

## 案例

```cpp
#include <iostream>
using namespace std;

const int NAME_LEN = 20;
// 商品的抽象数据类型
class CGoods
{
// 属性一般都是私有
private:
    char _name[NAME_LEN];
    double _price;
    int _amount;

// 给外部提供共有的成员方法来访问私有的属性
public:
    // 为商品数据提供初始化方法
    // 私有，需要使用指向常量的指针，不能让其被外部指针修改
    void init(const char* name, double price, int amount);
    // 打印商品信息
    void show();

    // 给成员变量提供一组getXXX或setXXX的方法
    // 类体内实现的方法，不会自动处理为 inline，需要手动添加
    // 需要加上类的作用域：void CGoods::init(xxx)
    void setName(char* name) { strcpy(_name, name); };
    void setPrice(double price) { _price = price; };
    void setAmount(int amount) { _amount = amount; };

    const char* getName() { return _name; };
    double getPrice() { return _price; };
    int getAmount() { return _amount; };
};

// 在类的外面实现成员方法，不会自动处理为 inline，需要手动添加
void CGoods::init(const char* name, double price, int amount)
{
    strcpy(_name, name);
    _price = price;
    _amount = amount;
}

void CGoods::show()
{
    cout << "name:" << _name << endl;
    cout << "price:" << _price << endl;
    cout << "amount:" << _amount << endl;
}

int main()
{
    /*
    CGoods 可以定义无数个对象，每一个对象都有自己的成员变量
    但是他们共享一套成员方法

    show() 是怎么知道处理哪个对象的信息?
    在调用成员方法的时候也把对象的地址传进了函数中
    因此类在编译之后，给所有的成员方法都加入了一个this指针，接收调用该方法对象的地址
    void init(CGood *this, const char* name, double price, int amount);
    void show(CGood *this);
    */
    // 对象的内存大小只和成员变量有关
    // 计算方式：找到成员变量中最大的那个类型，然后就是它的整数倍
    // 这里最长的是 double 8 个字节，但有20个字节，不是8的整数倍
    // 所以内存就是 20 + 4 = 24，刚好是8的三倍
    // 实例化一个对象
    CGoods good;
    good.init("面包", 10.0, 200);
    good.show();
    good.setPrice(20.5);
    good.setAmount(100);
    good.show();

    // 实例化一个对象
    CGoods good2;
    good2.init("空调", 10000.0, 50);
    good2.show();

    return 0;
}
```

## 类的各种成员方法

### 普通成员方法

1. **属于类的作用域** → 编译器不会生成 `this` 形参变量
2. **调用依赖一个对象**，普通对象或者常对象都可以调用
3. 可以任意访问对象的私有成员变量：`protected`、`extend`、`public`、`private`

### 静态成员

1. **属于类的作用域**
2. 调用**静态成员方法**需要使用类名作用域：`CGoods::showCGoodsCount();`
3. 可以访问对象的私有成员，**但只限不依赖对象**的私有成员变量
4. **static 静态成员变量一定要在类外进行定义并且初始化**
5. **静态方法有一个问句的限制和一个对象调用的限制**
   - 静态成员方法，打印所有商品共享的信息
   ```cpp
   static void showCGoodsCount()
   {
       cout << "所有商品种类的数量是：" << _count << endl;
   }
   ```
6. 静态成员变量需要再类外进行定义
   ```cpp
   class Test
   {
   public:
       static int mb;
   };
   int Test::mb;
   ```

### const 常成员方法

1. **属于类的作用域**
2. 调用依赖一个对象，**普通对象或者常对象**都可以调用
3. 可以任意访问对象的私有成员变量，但是**只读操作**的成员方法，一律实现成 const 常成员方法
4. 只要是只读操作的成员方法，一律实现成 const 常成员方法
   - 这样普通成员可以调用，常成员也可以调用
   - 因为还是会导致 `const this*` 与 `this*` 的转换问题
   - 需要对象调用的方法需要常方法，因为常方法生成后 this 才是 const 类型
   ```cpp
   // 常成员方法，传入的是 const CGoods *this
   // 只要是只读操作的成员方法，一律实现成 const 常成员方法
   void show() const
   {
       cout << "name:" << _name << endl;
       cout << "amount:" << _amount << endl;
       cout << "price:" << _price << endl;
       _date.show();
   }
   ```

## 案例：CDate 和 CGoods

```cpp
class CDate
{
private:
    int _year;
    int _month;
    int _day;
public:
    CDate(int y, int m, int d)
    {
        _year = y;
        _month = m;
        _day = d;
    }
    // 只要是只读操作的成员方法，一律实现成const常成员方法
    void show() const
    {
        cout << _year << "/" << _month << "/" << _day << endl;
    }
};

class CGoods
{
private:
    char _name[20];
    int _amount;
    double _price;
    CDate _date;    // 成员对象
public:
    // 报错：类CDate不存在默认构造函数
    CGoods(const char* n, int a, double p, int y, int m, int d)
        : _date(y, m, d)  // CDate _date(y, m, d)
        , _amount(a)
        , _price(p)
    {
        strcpy(_name, n);
    }

    void show() const
    {
        cout << "name:" << _name << endl;
        cout << "amount:" << _amount << endl;
        cout << "price:" << _price << endl;
        _date.show();
    }
};

int main()
{
    CGoods good1("商品1", 100, 35.0, 2019, 5, 12);
    good1.show();

    CGoods good2("商品2", 100, 35.0, 2019, 5, 12);
    good2.show();

    CGoods good3("商品3", 100, 33.0, 2019, 5, 12);
    good3.show();

    CGoods good4("商品4", 100, 35.0, 2019, 5, 12);
    good4.show();

    // 统计所有商品的总数量
    CGoods::showCGoodsCount();

    // 为什么常量对象无法调用普通的方法?
    // 以为普通方法会隐式传入this指针
    // (const CGoods* => ) CGoods* this, 这个无法转换
    const CGoods good5("非卖品", 100, 35.0, 2019, 5, 12);
    good5.show();

    return 0;
}
```

---

## 关键要点

1. **封装**：属性私有，通过公有方法访问
2. **this 指针**：成员方法隐式接收调用对象的地址
3. **内存对齐**：对象大小按最大成员类型对齐
4. **const 成员方法**：只读操作应声明为 const
5. **静态成员**：属于类而非对象，需要类外定义
