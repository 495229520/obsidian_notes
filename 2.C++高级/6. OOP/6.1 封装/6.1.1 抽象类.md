# 6.1.1 抽象类

## 抽象类和普通类的区别

| 特性 | 普通类 | 抽象类 |
|------|--------|--------|
| **定义** | 不含纯虚函数的类 | 拥有纯虚函数的类 |
| **实例化** | 可以直接实例化对象 | **不能直接实例化对象**，但可以定义指针和引用变量 |
| **用途** | 创建具体对象 | 作为基类，给所有派生类保留统一的覆盖/重写接口 |

> **基类**：将类抽象成一个实体的类型，给所有的派生类保留统一的覆盖/重写接口

---

## 什么是纯虚函数

**纯虚函数（Pure Virtual Function）**：将虚函数设置为 0 的函数

```cpp
virtual 返回类型 函数名(参数列表) = 0;
```

**特点**：
- 纯虚函数没有函数体实现
- 派生类**必须**重写（override）纯虚函数，否则派生类也会成为抽象类
- 一升油能跑多少公里，由于不同的车改变量是不同的，所以设计为纯虚函数

---

## 案例：汽车抽象类

```cpp
class Car
{
public:
    Car(string name, double oil) : _name(name), _oil(oil) {}
    virtual ~Car() {}

    // 获取汽车剩余油量还能跑的公里数
    double getLeftMiles()
    {
        // getMilesPerGallon 一升油能跑多少公里，不同的车改变量是不同的
        // 在调用 getMilesPerGallon() 时是动态绑定的
        return _oil * getMilesPerGallon();
    }

    string getName() const { return _name; }

protected:
    string _name;
    double _oil;

    // 纯虚函数
    // 一升油能跑多少公里，由于不同的车改变量是不同的，所以设计为纯虚函数
    virtual double getMilesPerGallon() = 0;
};

class Benz : public Car
{
public:
    Benz(string name, double oil) : Car(name, oil) {}
    ~Benz() {}
    double getMilesPerGallon() { return 20.0; }
};

class Audi : public Car
{
public:
    Audi(string name, double oil) : Car(name, oil) {}
    ~Audi() {}
    double getMilesPerGallon() { return 12.0; }
};

class BMW : public Car
{
public:
    BMW(string name, double oil) : Car(name, oil) {}
    ~BMW() {}
    double getMilesPerGallon() { return 18.0; }
};

// 给外部提供一个统一的获取汽车剩余路程数的API
void showCarLeftMiles(Car& car)
{
    // 静态绑定，因为已经明确了对象是car
    cout << car.getLeftMiles() << endl;
}

int main()
{
    Benz b1("奔驰", 20.0);
    Audi a("奥迪", 20.0);
    BMW b2("宝马", 20.0);

    showCarLeftMiles(b1);
    showCarLeftMiles(a);
    showCarLeftMiles(b2);

    return 0;
}
```

---

## 关键要点

1. **抽象类不能实例化**：只能作为基类被继承
2. **纯虚函数必须被重写**：派生类必须实现所有纯虚函数才能实例化
3. **抽象类可以有普通成员**：抽象类可以包含普通成员变量和普通成员函数
4. **统一接口设计**：抽象类用于定义接口规范，实现"开闭原则"——对修改关闭，对扩展开放
