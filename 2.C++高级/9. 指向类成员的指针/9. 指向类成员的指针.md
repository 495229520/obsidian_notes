# 9. 指向类成员的指针

## 指向成员变量的指针

### 知识点

1. **指向快成员的指针不直接指向内存**，&类名::成员变量
2. **指向成员变量的指针依赖于对象，只有通过对象才能够访问成员变量的内存地址**
3. 定义静态成员变量没有对象，因此可以使用普通指针
4. 对静态成员变量是不依赖对象的

### 初始化

| 类型 | 初始化 |
|------|--------|
| **指向快成员的指针** | `int Test::*p = &Test::ma;` |
| **&Test::ma** | 这类得了一个名为 `p` 的指针，它可以存放 Test 类型中 int 类型成员变量的 "相对位置" |

这是一个代码的核心部分，声明得到了 `p`：

- `int Test::*p`：这声明了一个指向 Test 类的 int 成员的指针。
- 简而言之，`p` 不是一个指向某个成员在 Test 对象内部的 "相对位置"。
- `&Test::ma`：这取的是成员 ma 在 Test 类中的 "偏移量" 或 "相对位置"，不是某个具体对象中 ma 的实际内存地址。

这段代码只记录了 ma 在 Test 对象内部的一个偏移量。

### 使用

```cpp
int Test::*p = &Test::ma;
t1.*p = 20;
cout << t1.*p << endl;

// &Test::ma 类表示 ma 这个成员变量在 Test 对象内部的 "相对位置"
// 而是表示 ma 这个成员变量在 Test 对象内部的 "相对位置"
t2->*p = 30;
cout << t2->*p << endl;
```

### 静态成员变量

```cpp
// 静态成员变量没有对象，因此可以使用普通指针
int* p1 = &Test::mb;
*p1 = 40;
cout << *p1 << endl;
```

## 指向成员方法的指针

### 知识点

1. **指向普通成员方法的指针**：`void (Test::*pfunc)() = &Test::func;`
2. **只要是普通的成员方法，无论是成员还是返回值，都需要依赖对象的方法来调用**
3. 定义普通方法指针和静态成员方法指针

### 普通成员方法

```cpp
// 指向普通成员方法的指针
void (Test::*pfunc)() = &Test::func;
(t1.*pfunc)();
(t2->*pfunc)();
```

### 静态成员方法

```cpp
// 定义函数指针指向类的静态成员方法
void (*pfunc2)() = &Test::static_func;
(*pfunc2)();
```

---

## 案例

```cpp
class Test
{
public:
    void func()
    {
        cout << "call Test::func" << endl;
    }

    static void static_func()
    {
        cout << "Test::static_func" << endl;
    }

    int ma;
    static int mb;
};

int Test::mb;

int main()
{
    Test t1;
    Test* t2 = new Test();

    // 指向普通成员变量的指针
    int Test::*p = &Test::ma;
    t1.*p = 20;
    cout << t1.*p << endl;

    t2->*p = 30;
    cout << t2->*p << endl;

    // 静态成员变量没有对象，因此可以使用普通指针
    int* p1 = &Test::mb;
    *p1 = 40;
    cout << *p1 << endl;

    // 指向普通成员方法的指针
    void (Test::*pfunc)() = &Test::func;
    (t1.*pfunc)();
    (t2->*pfunc)();

    // 定义函数指针指向类的静态成员方法
    void (*pfunc2)() = &Test::static_func;
    (*pfunc2)();

    delete t2;
    return 0;
}
```

## 指向成员变量/方法的指针

### 指向成员变量

```cpp
// 指向成员变量的指针
int Test::*p = &Test::ma;

// 对象.* 或 对象指针->*
t1.*p = 20;
t2->*p = 30;
```

### 指向成员方法

```cpp
// 指向成员方法的指针
void (Test::*pfunc)() = &Test::func;

// 对象.* 或 对象指针->*
(t1.*pfunc)();
(t2->*pfunc)();
```

### 静态成员

```cpp
// 静态成员变量：普通指针
int* p1 = &Test::mb;

// 静态成员方法：普通函数指针
void (*pfunc2)() = &Test::static_func;
```

---

## 关键要点

1. **成员指针不是真正的内存地址**，而是类内的"相对位置"或"偏移量"
2. **必须通过对象使用**：`对象.*指针` 或 `对象指针->*指针`
3. **静态成员**：不依赖对象，使用普通指针
4. **语法**：`类型 类名::*指针名`（成员变量），`返回类型 (类名::*指针名)(参数)` （成员方法）
