# 12.2 vector 容器的迭代器实现

## vector 的内存模型

vector 是 STL 中最常用的序列容器，其底层使用动态数组实现。vector 使用三个指针来管理内存：

```
内存布局：
┌─────────────────────────────────────────────────────┐
│  有效元素区域           │  预留空间（未使用）          │
└─────────────────────────────────────────────────────┘
↑                        ↑                            ↑
_first                   _last                        _end
```

| 指针 | 含义 |
|------|------|
| `_first` | 指向数组起始位置 |
| `_last` | 指向最后一个有效元素的下一个位置 |
| `_end` | 指向整个分配空间的末尾 |

### 关键计算

- **有效元素个数 (size)**：`_last - _first`
- **容量 (capacity)**：`_end - _first`
- **是否为空 (empty)**：`_first == _last`
- **是否已满 (full)**：`_last == _end`

## vector 迭代器实现

由于 vector 底层是连续内存，其迭代器可以直接使用指针，属于**随机访问迭代器**。

### 基本实现

```cpp
template<typename T>
class vector {
public:
    // 迭代器类型直接使用指针
    using iterator = T*;
    using const_iterator = const T*;

private:
    T* _first;    // 指向数组起始位置
    T* _last;     // 指向有效元素的后继位置
    T* _end;      // 指向数组空间的后继位置

public:
    vector(int size = 10) {
        _first = new T[size];
        _last = _first;
        _end = _first + size;
    }

    ~vector() {
        delete[] _first;
        _first = _last = _end = nullptr;
    }

    // 迭代器接口
    iterator begin() { return _first; }
    iterator end() { return _last; }
    const_iterator begin() const { return _first; }
    const_iterator end() const { return _last; }

    // 容量相关
    int size() const { return _last - _first; }
    int capacity() const { return _end - _first; }
    bool empty() const { return _first == _last; }
    bool full() const { return _last == _end; }
};
```

## 完整的 vector 实现

下面是一个更完整的 vector 实现，包含常用操作（基于[[10.2 类模版|类模板]]）：

```cpp
template<typename T>
class vector {
public:
    using iterator = T*;
    using const_iterator = const T*;

private:
    T* _first;
    T* _last;
    T* _end;

    // 扩容函数
    void expand() {
        int size = _end - _first;
        T* ptmp = new T[size * 2];
        for (int i = 0; i < size; ++i) {
            ptmp[i] = _first[i];
        }
        delete[] _first;
        _first = ptmp;
        _last = _first + size;
        _end = _first + size * 2;
    }

public:
    vector(int size = 10) {
        _first = new T[size];
        _last = _first;
        _end = _first + size;
    }

    ~vector() {
        delete[] _first;
        _first = _last = _end = nullptr;
    }

    // 拷贝构造（深拷贝）
    vector(const vector<T>& rhs) {
        int size = rhs._end - rhs._first;
        _first = new T[size];
        int len = rhs._last - rhs._first;
        for (int i = 0; i < len; ++i) {
            _first[i] = rhs._first[i];
        }
        _last = _first + len;
        _end = _first + size;
    }

    // 赋值运算符（深拷贝）
    vector<T>& operator=(const vector<T>& rhs) {
        if (this == &rhs)
            return *this;

        delete[] _first;
        int size = rhs._end - rhs._first;
        _first = new T[size];
        int len = rhs._last - rhs._first;
        for (int i = 0; i < len; ++i) {
            _first[i] = rhs._first[i];
        }
        _last = _first + len;
        _end = _first + size;
        return *this;
    }

    // 尾部插入
    void push_back(const T& val) {
        if (full())
            expand();
        *_last++ = val;
    }

    // 尾部删除
    void pop_back() {
        if (empty())
            return;
        --_last;
    }

    // 访问尾部元素
    T back() const {
        return *(_last - 1);
    }

    // 下标访问
    T& operator[](int index) {
        return _first[index];
    }

    const T& operator[](int index) const {
        return _first[index];
    }

    // 迭代器
    iterator begin() { return _first; }
    iterator end() { return _last; }
    const_iterator begin() const { return _first; }
    const_iterator end() const { return _last; }

    // 容量
    int size() const { return _last - _first; }
    int capacity() const { return _end - _first; }
    bool empty() const { return _first == _last; }
    bool full() const { return _last == _end; }

    // 在指定位置插入元素
    iterator insert(iterator pos, const T& val) {
        // 检查是否需要扩容
        if (full()) {
            // 记录位置偏移，因为扩容后指针会失效
            int offset = pos - _first;
            expand();
            pos = _first + offset;  // 重新计算位置
        }

        // 将 pos 及之后的元素后移
        for (iterator it = _last; it > pos; --it) {
            *it = *(it - 1);
        }

        *pos = val;
        ++_last;

        return pos;
    }

    // 删除指定位置的元素
    iterator erase(iterator pos) {
        // 将 pos 之后的元素前移
        for (iterator it = pos; it < _last - 1; ++it) {
            *it = *(it + 1);
        }
        --_last;
        return pos;
    }
};
```

## 使用示例

```cpp
int main() {
    vector<int> vec;

    // 添加元素
    for (int i = 0; i < 20; ++i) {
        vec.push_back(rand() % 100 + 1);
    }

    // 使用迭代器遍历
    for (vector<int>::iterator it = vec.begin(); it != vec.end(); ++it) {
        std::cout << *it << " ";
    }
    std::cout << std::endl;

    // 使用范围 for
    for (int val : vec) {
        std::cout << val << " ";
    }
    std::cout << std::endl;

    // 在位置 2 插入元素
    vec.insert(vec.begin() + 2, 999);

    // 删除第一个元素
    vec.erase(vec.begin());

    return 0;
}
```

## 迭代器失效问题预览

在 vector 中，以下操作会导致[[12.3 迭代器的失效问题|迭代器失效]]：

| 操作 | 失效范围 |
|------|----------|
| `push_back` | 若触发扩容，所有迭代器失效 |
| `insert` | 插入点及之后的迭代器失效；若扩容则全部失效 |
| `erase` | 删除点及之后的迭代器失效 |
| `clear` | 所有迭代器失效 |

> 详细的迭代器失效处理方法请参见 [[12.3 迭代器的失效问题]]

---

## 关键要点

1. **vector 迭代器本质是指针**：因为底层是连续内存
2. **三个指针管理内存**：`_first`、`_last`、`_end`
3. **随机访问迭代器**：支持 `+n`、`-n`、`[]` 等操作
4. **扩容机制**：空间不足时按 2 倍扩容
5. **迭代器失效**：扩容、插入、删除都可能导致迭代器失效
