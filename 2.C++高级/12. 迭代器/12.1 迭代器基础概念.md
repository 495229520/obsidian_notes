# 12.1 迭代器基础概念

## 什么是迭代器

**迭代器（Iterator）** 是一种泛型指针，提供了一种统一的方式来访问容器中的元素，而无需暴露容器的底层实现。迭代器是 STL 的核心组件之一，它将容器和算法解耦。

### 迭代器的作用

| 特性 | 说明 |
|------|------|
| **统一接口** | 为不同容器提供统一的遍历方式 |
| **解耦** | 算法不需要知道容器的具体实现 |
| **泛型编程** | 支持模板算法作用于不同容器 |

---

## 迭代器的分类

STL 定义了五种迭代器类型，按功能从弱到强排列：

| 迭代器类型 | 支持的操作 | 典型容器 |
|------------|------------|----------|
| **输入迭代器** (Input) | `++`, `*`(读), `==`, `!=` | `istream_iterator` |
| **输出迭代器** (Output) | `++`, `*`(写) | `ostream_iterator` |
| **前向迭代器** (Forward) | 输入迭代器 + 多次遍历 | `forward_list`, `unordered_set` |
| **双向迭代器** (Bidirectional) | 前向迭代器 + `--` | `list`, `set`, `map` |
| **随机访问迭代器** (Random Access) | 双向迭代器 + `+n`, `-n`, `[]`, `<`, `>` | `vector`, `deque`, `array` |

### 迭代器类型层次

```
随机访问迭代器 (Random Access)
      ↑
双向迭代器 (Bidirectional)
      ↑
前向迭代器 (Forward)
      ↑
输入迭代器 (Input)    输出迭代器 (Output)
```

## iterator_traits 萃取机制

`iterator_traits` 是一种类型萃取技术，用于统一获取迭代器的相关类型信息。

### 为什么需要 traits

不同迭代器可能以不同方式定义其关联类型。`iterator_traits` 提供统一接口来获取这些类型：

```cpp
template<typename Iterator>
struct iterator_traits {
    using difference_type   = typename Iterator::difference_type;
    using value_type        = typename Iterator::value_type;
    using pointer           = typename Iterator::pointer;
    using reference         = typename Iterator::reference;
    using iterator_category = typename Iterator::iterator_category;
};
```

### 指针的特化版本

原生指针没有内嵌类型，需要特化：

```cpp
// 针对原生指针的偏特化
template<typename T>
struct iterator_traits<T*> {
    using difference_type   = ptrdiff_t;
    using value_type        = T;
    using pointer           = T*;
    using reference         = T&;
    using iterator_category = random_access_iterator_tag;
};

// 针对 const 指针的偏特化
template<typename T>
struct iterator_traits<const T*> {
    using difference_type   = ptrdiff_t;
    using value_type        = T;
    using pointer           = const T*;
    using reference         = const T&;
    using iterator_category = random_access_iterator_tag;
};
```

## 自定义容器与迭代器实现

下面是一个简单容器类及其迭代器的实现示例：

```cpp
#include <iterator>  // for std::iterator_traits and category tags

template<typename T>
class SimpleContainer {
public:
    // 嵌套迭代器类
    class iterator {
    public:
        // 迭代器必须定义的五种类型
        using iterator_category = std::random_access_iterator_tag;
        using value_type        = T;
        using difference_type   = std::ptrdiff_t;
        using pointer           = T*;
        using reference         = T&;

    private:
        T* _ptr;

    public:
        iterator(T* ptr = nullptr) : _ptr(ptr) {}

        // 1. 解引用操作
        reference operator*() const { return *_ptr; }
        pointer operator->() const { return _ptr; }

        // 2. 前置递增/递减
        iterator& operator++() { ++_ptr; return *this; }
        iterator& operator--() { --_ptr; return *this; }

        // 3. 后置递增/递减
        iterator operator++(int) {
            iterator tmp = *this;
            ++_ptr;
            return tmp;
        }
        iterator operator--(int) {
            iterator tmp = *this;
            --_ptr;
            return tmp;
        }

        // 4. 随机访问 (random_access)
        iterator operator+(difference_type n) const { return iterator(_ptr + n); }
        iterator operator-(difference_type n) const { return iterator(_ptr - n); }
        difference_type operator-(const iterator& other) const { return _ptr - other._ptr; }

        iterator& operator+=(difference_type n) { _ptr += n; return *this; }
        iterator& operator-=(difference_type n) { _ptr -= n; return *this; }

        reference operator[](difference_type n) const { return *(_ptr + n); }

        // 5. 算术比较 (random_access)
        bool operator==(const iterator& other) const { return _ptr == other._ptr; }
        bool operator!=(const iterator& other) const { return _ptr != other._ptr; }
        bool operator<(const iterator& other) const { return _ptr < other._ptr; }
        bool operator>(const iterator& other) const { return _ptr > other._ptr; }
        bool operator<=(const iterator& other) const { return _ptr <= other._ptr; }
        bool operator>=(const iterator& other) const { return _ptr >= other._ptr; }
    };

private:
    T* _data = nullptr;
    size_t _size = 0;
    size_t _capacity = 0;

public:
    SimpleContainer() = default;

    ~SimpleContainer() { delete[] _data; }

    void push_back(const T& val) {
        if (_size == _capacity) {
            size_t new_capacity = _capacity == 0 ? 1 : _capacity * 2;
            T* new_data = new T[new_capacity];
            for (size_t i = 0; i < _size; ++i) {
                new_data[i] = _data[i];
            }
            delete[] _data;
            _data = new_data;
            _capacity = new_capacity;
        }
        _data[_size++] = val;
    }

    size_t size() const { return _size; }

    // 返回迭代器
    iterator begin() { return iterator(_data); }
    iterator end() { return iterator(_data + _size); }
};
```

### 使用示例

```cpp
int main() {
    SimpleContainer<int> container;
    container.push_back(10);
    container.push_back(20);
    container.push_back(30);

    // 使用迭代器遍历（传统for循环）
    for (SimpleContainer<int>::iterator it = container.begin();
         it != container.end(); ++it) {
        std::cout << *it << " ";
    }
    std::cout << std::endl;

    // 使用范围for循环（C++11）
    for (int val : container) {
        std::cout << val << " ";
    }
    std::cout << std::endl;

    return 0;
}
```

## String 类的迭代器实现

下面展示一个完整的 String 类，包含[[11. 运算符重载|运算符重载]]和迭代器支持：

```cpp
class String {
public:
    String(const char* p = nullptr) {
        if (p != nullptr) {
            _pstr = new char[strlen(p) + 1];
            strcpy(_pstr, p);
        } else {
            _pstr = new char[1];
            *_pstr = '\0';
        }
    }

    ~String() {
        delete[] _pstr;
        _pstr = nullptr;
    }

    // 拷贝构造函数（深拷贝）
    String(const String& str) {
        _pstr = new char[strlen(str._pstr) + 1];
        strcpy(_pstr, str._pstr);
    }

    // 赋值运算符（深拷贝）
    String& operator=(const String& src) {
        if (this == &src)
            return *this;

        delete[] _pstr;
        _pstr = new char[strlen(src._pstr) + 1];
        strcpy(_pstr, src._pstr);
        return *this;
    }

    // 方括号运算符
    char& operator[](int index) { return _pstr[index]; }
    const char& operator[](int index) const { return _pstr[index]; }

    // 比较运算符
    bool operator>(const String& str) const {
        return strcmp(_pstr, str._pstr) > 0;
    }
    bool operator<(const String& str) const {
        return strcmp(_pstr, str._pstr) < 0;
    }
    bool operator==(const String& str) const {
        return strcmp(_pstr, str._pstr) == 0;
    }

    int length() const { return strlen(_pstr); }
    const char* c_str() const { return _pstr; }

    // ----- 迭代器实现 -----
    class iterator {
    public:
        iterator(char* p = nullptr) : _p(p) {}

        bool operator!=(const iterator& it) {
            return _p != it._p;
        }

        void operator++() { ++_p; }

        char& operator*() { return *_p; }

    private:
        char* _p;
    };

    // begin: 返回指向第一个元素的迭代器
    iterator begin() { return iterator(_pstr); }

    // end: 返回指向末尾（'\0'）的迭代器
    iterator end() { return iterator(_pstr + length()); }

    // 字符串拼接
    friend String operator+(const String& lhs, const String& rhs) {
        char* ptmp = new char[strlen(lhs._pstr) + strlen(rhs._pstr) + 1];
        strcpy(ptmp, lhs._pstr);
        strcat(ptmp, rhs._pstr);
        String tmp(ptmp);
        delete[] ptmp;
        return tmp;
    }

    // 流输出运算符
    friend std::ostream& operator<<(std::ostream& out, const String& str) {
        out << str._pstr;
        return out;
    }

private:
    char* _pstr;
};
```

### 迭代器使用示例

```cpp
int main() {
    String str = "Hello world!";

    // 使用迭代器遍历
    String::iterator it = str.begin();
    for (; it != str.end(); ++it) {
        std::cout << *it << " ";
    }
    std::cout << std::endl;

    // 使用范围for（C++11）
    // 底层调用 begin() 和 end()
    for (char ch : str) {
        std::cout << ch << " ";
    }
    std::cout << std::endl;

    return 0;
}
```

## 关键要点

1. **迭代器是泛型指针**：提供统一的容器遍历接口
2. **五种迭代器类型**：输入、输出、前向、双向、随机访问
3. **iterator_traits**：类型萃取，统一获取迭代器关联类型
4. **自定义迭代器**：需要定义五种类型别名和相应的运算符
5. **begin() 和 end()**：支持范围 for 循环的关键
