# 4. const、指针和引用

## 概述

本节深入讨论 `const` 关键字与[[6.1基本指针|指针]]、引用的组合使用，这是 C++ 中容易混淆但非常重要的概念。

---

## 一级指针与 const

### 指向常量的指针（常量指针）

```cpp
const int* ptr;
int const* ptr;  // 等价写法
```

**特点**：`ptr` 是一个指针，指向一个 `int` 类型的常量。可以修改指针指向，但不能通过指针修改值。

```cpp
const int a[ 1000 ];
const int b[ 2000 ];
const int c[ 3000 ];
const int* ptr( &a );

ptr = &c;      // 正确：可以改变指向
// *ptr = 1000;  // 错误：不能通过指针修改值
```

### 常量指针

```cpp
int* const ptr;
```

**特点**：`ptr` 是一个常量，这个常量是指向 `int` 的指针。可以修改值，但不能修改指向。

```cpp
int a[ 1000 ];
int* const ptr( &a );

*ptr = 1000;   // 正确：可以修改值
// ptr = &b;     // 错误：不能改变指向
```

### 指向常量的常量指针

```cpp
const int* const ptr;
```

**特点**：既不能改变指向，也不能通过指针修改值。

```cpp
const int a[ 1000 ];
const int* const ptr( &a );

// *ptr = 1000;  // 错误
// ptr = &b;     // 错误
```

### 总结表

| 声明方式 | `const` 修饰 `*ptr` | `const` 修饰指向 |
|---------|---------------------|-----------------|
| `int* ptr;` | 否 | 否 |
| `const int* ptr;` | 是 | 否 |
| `int* const ptr;` | 否 | 是 |
| `const int* const ptr;` | 是 | 是 |

---

## const 修饰与右边规则

**重要规则**：`const` 如果右边没有指针 `*` 的话，它就不参与类型。

### const 和二级指针的类型转换公式

| 左值 | 右值 | 结果 |
|------|------|------|
| `int*` | `int*` | 正确 |
| `int*` | `const int*` | 错误 |
| `const int*` | `int*` | 正确 |
| `int**` | `int**` | 正确 |
| `const int**` | `int**` | 错误 |
| `int* const*` | `int**` | 正确 |
| `int** const` | `int**` | 正确 |
| `int* const*` | `const int**` | 错误 |

**C++ 语言规范禁止任何可能导致 const 变量被意外修改的隐式类型转换。**

## 二级指针

### 指向常量的指针

```cpp
const int** q;
```

**特点**：`const` 修饰 `**q`，因此 `**q` 无法被赋值，但是 `*q` 和 `q` 不受影响。

```cpp
const int** q = ...;
*q = ...;     // 正确
**q = ...;    // 错误
q = ...;      // 正确
```

### 指向常量指针的指针

```cpp
int* const* q;
```

**特点**：`const` 修饰 `*q`，因此 `*q` 无法被赋值。

```cpp
int* const* q = ...;
*q = ...;     // 错误
q = ...;      // 正确
```

### 常量的二级指针

```cpp
int** const q;
```

**特点**：`const` 修饰 `q`，因此 `q` 无法被赋值。

```cpp
int** const q = ...;
q = ...;      // 错误
*q = ...;     // 正确
```

## 引用与指针

### 引用的本质

引用和指针的本质是一样的，在汇编层面都是将一个变量的地址存储在 `dword ptr` 指针的内存中。

```cpp
// 因此 int* ptr = 10; 或者 int& b = 20; 都是错误的，因为常量没有地址！

int a = 10;
// lea   eax, [a]              ; 取出 a 的地址
// mov   dword ptr [p], eax    ; 将 a 的地址放入 p 的内存中
int* p = &a;

// lea   eax, [a]              ; 取出 a 的地址
// mov   dword ptr [b], eax    ; 将 a 的地址放入 b 的内存中
int& b = a;
```

### 解引用对比

```cpp
// lea   eax, dword ptr [p]    ; 解引用，从 p 的内存里拿出 a 的地址
// mov   dword ptr [eax], 14h  ; 将 20 放入 a 地址的内存里
*p = 20;
// 20 20 20 a *p b
cout << a << " " << *p << " " << b << endl;

// lea   eax, dword ptr [b]    ; 解引用，从 b 的内存里拿出 a 的地址
// mov   dword ptr [eax], 1Eh  ; 将 30 放入 a 地址的内存里
b = 30;
// 30 30 30 a *p b
cout << a << " " << *p << " " << b << endl;
```

### 指针与引用的对比

| 特性 | 指针 | 引用 |
|------|------|------|
| **语法** | `int* p = &q;` | `int& p = q;` |
| **可以替换** | `int*  p = &q;` 可以替换为相对应的指针形式 | `int& p = q;` 因此当遇见引用的时候可以替换为相对应的指针形式 |

```cpp
int array[5] = {};
int* p = array;
// 20
cout << sizeof(array) << endl;

int (&q)[5] = array;
// 20
cout << sizeof(q) << endl;
```

## 右值引用

### 左值与右值

| 类型 | 特点 |
|------|------|
| 左值 | 有内存，有名字，值可以修改 |
| 右值 | 没内存，没名字，值不可以修改 |

前面说过引用的本质就是将地址放到编译器的 `ptr` 指针的内存中，因此右值是不可以被引用的。但是 C++11 提供了右值引用。

### 右值引用

1. 右值引用本质上就是在栈上生成一块临时的空间，然后将右值存储在这块内存上
2. 再将这块内存的地址放在引用 C 的内存中

```cpp
// mov    dword ptr [ebp-30h], 14h    ; 临时空间存放 20
// lea    eax, [ebp-30h]              ; 取出临时空间地址
// mov    dword ptr[c], eax           ; 将地址放入 c 的内存
int&& c = 20;
```

### 可修改与不可修改

| 类型 | 示例 | 说明 |
|------|------|------|
| 可修改 | `int&& c = 20;` `c = 30;` | 右值引用变量，本身就是一个左值，它有内存有名字 |
| 不可修改 | `const int& d = 20;` `d = 30;` // 错误 | 不能用一个右值引用变量来引用一个左值 |

## 引用案例

### 案例 1

```cpp
int a = 10;
int* const p = &a;      // 正确, int* <= int*
// 等于 int** q = &p;
int*& q = p;            // 错误, int** <= int* const*
```

### 案例 2

```cpp
int a = 10;
int* p = &a;            // 正确, int* <= int*
// 等于 int** q = &p;
int*& q = p;            // 正确, int** <= int**
```

### 案例 3

```cpp
int a = 10;
const int* p = &a;      // 正确, const int* <= int*
// int** q = &p;
int*& q = p;            // 错误, int** <= const int**
```

### 案例 4

```cpp
int a = 10;
int* p = &a;
const int*& q = &p;
// const int** <= const int**
const int*& q = p;      // 错误, const int** <= int**
```

## 关键要点

1. **const 右边规则**：`const` 修饰其右边紧邻的内容
2. **引用本质**：引用在底层就是指针
3. **右值引用**：C++11 特性，在栈上创建临时空间
4. **类型转换**：C++ 禁止任何可能导致 const 变量被意外修改的隐式转换
