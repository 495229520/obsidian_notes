# 11. 运算符重载

## 知识点

1. **编译器特殊生成的运算符重载函数**：赋值运算符(`operator=`)、取地址运算符(`operator&`)、`const` 取地址运算符
2. 对于编译器默认生成的函数，调用一次才会生成一次
3. **[[8. 浅拷贝与深拷贝|内存问题]]是 `data` 数据类型的对象进行拷贝内存的问题**
4. 运算符重载需要返回指向 `Complex` 对象的迭代器或引用，所以返回类型需要加 `const Complex&`
5. 左边作为参数时返回值相同，C++需要用 `reinterpret_cast` 或 `static_cast` 指出它的类型

---

## 编译器默认生成的运算符

编译器会为类自动生成以下运算符重载函数（如果未自定义）：

| 运算符 | 函数签名 | 说明 |
|--------|----------|------|
| `=` | `CComplex& operator=(const CComplex&)` | 赋值运算符 |
| `&` | `CComplex* operator&()` | 取地址运算符 |
| `&` (const) | `const CComplex* operator&() const` | const 取地址运算符 |

---

## 赋值运算符重载

### 基本形式

```cpp
CComplex& operator=(const CComplex& src)
{
    // 防止自赋值
    if (this == &src)
        return *this;

    // 释放当前对象的资源（如果有）
    // ...

    // 复制数据
    _real = src._real;
    _image = src._image;

    return *this;
}
```

### 为什么返回引用？

1. **支持链式赋值**：`a = b = c;`
2. **避免不必要的拷贝**：返回引用而非值

---

## 算术运算符重载

### 加法运算符

```cpp
// 成员函数实现
CComplex operator+(const CComplex& src)
{
    return CComplex(_real + src._real, _image + src._image);
}

// 全局函数实现（支持 int + CComplex）
CComplex operator+(int lhs, const CComplex& rhs)
{
    return CComplex(lhs + rhs._real, rhs._image);
}
```

### 复合赋值运算符

```cpp
// += 运算符
CComplex& operator+=(const CComplex& src)
{
    _real += src._real;
    _image += src._image;
    return *this;
}
```

---

## 前置/后置自增运算符

### 区分前置和后置

| 类型 | 语法 | 说明 |
|------|------|------|
| **前置++** | `CComplex& operator++()` | 先自增，返回引用 |
| **后置++** | `CComplex operator++(int)` | 先返回，再自增，`int` 是占位符 |

### 实现

```cpp
// 前置++
CComplex& operator++()
{
    ++_real;
    ++_image;
    return *this;
}

// 后置++
CComplex operator++(int)
{
    // return CComplex(_real++, _image++);
    CComplex tmp = *this;
    ++_real;
    ++_image;
    return tmp;  // 返回自增前的值
}
```

## 输入输出运算符

### 为什么用友元函数？

`<<` 和 `>>` 运算符的左操作数是流对象（`ostream`/`istream`），不是我们定义的类对象，所以不能作为成员函数实现。

### 实现

```cpp
class CComplex
{
    friend ostream& operator<<(ostream& out, const CComplex& src);
    friend istream& operator>>(istream& in, CComplex& src);
    // ...
};

// 输出运算符
ostream& operator<<(ostream& out, const CComplex& src)
{
    out << "real:" << src._real << " image:" << src._image << endl;
    return out;
}

// 输入运算符
istream& operator>>(istream& in, CComplex& src)
{
    in >> src._real >> src._image;
    return in;
}
```

## 完整案例：复数类

```cpp
class CComplex
{
private:
    int _real;
    int _image;

public:
    CComplex(int real = 0, int image = 0)
        : _real(real), _image(image) {}

    // 加法运算符
    CComplex operator+(const CComplex& src)
    {
        return CComplex(_real + src._real, _image + src._image);
    }

    // 复合赋值
    CComplex& operator+=(const CComplex& src)
    {
        _real += src._real;
        _image += src._image;
        return *this;
    }

    // 前置++
    CComplex& operator++()
    {
        ++_real;
        ++_image;
        return *this;
    }

    // 后置++
    CComplex operator++(int)
    {
        CComplex tmp = *this;
        ++_real;
        ++_image;
        return tmp;
    }

    // 友元声明
    friend ostream& operator<<(ostream& out, const CComplex& src);
    friend istream& operator>>(istream& in, CComplex& src);
    friend CComplex operator+(int lhs, const CComplex& rhs);
};

// 输出运算符
ostream& operator<<(ostream& out, const CComplex& src)
{
    out << "real:" << src._real << " image:" << src._image;
    return out;
}

// 输入运算符
istream& operator>>(istream& in, CComplex& src)
{
    in >> src._real >> src._image;
    return in;
}

// 支持 int + CComplex
CComplex operator+(int lhs, const CComplex& rhs)
{
    return CComplex(lhs + rhs._real, rhs._image);
}

int main()
{
    CComplex comp1(10, 20);
    CComplex comp2(30, 30);
    CComplex comp3 = comp1 + comp2;  // operator+(comp1, comp2)

    cout << comp3 << endl;           // 输出: real:40 image:50

    comp3 = comp1 + 10;              // CComplex + int
    cout << comp3 << endl;           // 输出: real:20 image:20

    comp3 = 10 + comp1;              // int + CComplex (需要全局函数)
    cout << comp3 << endl;           // 输出: real:20 image:20

    ++comp1;                         // 前置++
    cout << comp1 << endl;           // 输出: real:11 image:21

    comp1++;                         // 后置++
    cout << comp1 << endl;           // 输出: real:12 image:22

    CComplex comp4;
    cin >> comp4;                    // 输入
    cout << comp4 << endl;           // 输出

    return 0;
}
```

---

## 关键要点

1. **赋值运算符**：检查自赋值，返回 `*this` 引用
2. **前置 vs 后置**：后置版本有 `int` 占位符参数
3. **友元函数**：用于左操作数不是类对象的情况
4. **返回类型**：
   - 返回新对象：`CComplex`
   - 返回自身：`CComplex&`
5. **const 引用参数**：避免拷贝，防止修改
