# 7. 构造函数与析构函数

## 知识点

1. **函数的初始化列表**
2. **自定义构造函数之后，类就不会再创建默认构造函数了**
3. **成员变量的初始化和它们定义的顺序有关，和构造函数初始化列表中出现的先后顺序无关**

---

## 构造函数

构造函数用于在创建对象时初始化成员变量。

```cpp
class Test
{
private:
    int ma;
    int mb;
public:
    Test(int data = 10) : mb(data), ma(mb) { }
    // 构造函数列表初始化是按照成员变量的顺序来的
    // 因此这里先初始化 ma，再初始化 mb，因此 ma 的值并不是 10
    void show() { cout << "ma:" << ma << " " << "mb:" << mb << endl; }
};
```

**注意**：构造函数列表初始化是按照成员变量的**定义顺序**来的，而不是初始化列表的顺序。

## 错误案例

### 问题代码

```cpp
class CDate
{
private:
    int _year;
    int _month;
    int _day;
public:
    // 自定义构造函数之后，类就不会再创建默认构造函数了
    CDate(int y, int m, int d)
    {
        _year = y;
        _month = m;
        _day = d;
    }
    void show()
    {
        cout << _year << "/" << _month << "/" << _day << endl;
    }
};

class CGoods
{
private:
    char _name[20];
    int _amount;
    double _price;
    CDate _date;    // 成员对象
public:
    // 报错：类 CDate 不存在默认构造函数
    CGoods(char* n, int a, double p)
    {
        strcpy(_name, n);
        _amount = a;
        _price = p;
    }
    void show()
    {
        cout << "name:" << _name << endl;
        cout << "amount:" << _amount << endl;
        cout << "price:" << _price << endl;
        _date.show();
    }
};
```

### 错误原因

1. 一旦你为某个类定义了任何一个构造函数，编译器就不会再自动为你生成默认构造函数
2. 当你创建 `CGoods` 对象时，`_date` 成员对象试图初始化，但 `CDate` 类没有默认构造函数
3. 对于 `_date` 成员，没有提供任何形式的初始化，编译器会尝试调用 `CDate` 类的默认构造函数来初始化它，从而导致 "class CDate has no default constructor" 的描述

### 正确案例

```cpp
class CDate
{
private:
    int _year;
    int _month;
    int _day;
public:
    CDate(int y, int m, int d)
    {
        _year = y;
        _month = m;
        _day = d;
    }
    void show() const
    {
        cout << _year << "/" << _month << "/" << _day << endl;
    }
};

class CGoods
{
private:
    char _name[20];
    int _amount;
    double _price;
    CDate _date;    // 成员对象
public:
    CGoods(const char* n, int a, double p, int y, int m, int d)
        : _date(y, m, d)  // CDate _date(y, m, d) 初始化成员对象
        , _amount(a)
        , _price(p)
    {
        strcpy(_name, n);
    }

    void show() const
    {
        cout << "name:" << _name << endl;
        cout << "amount:" << _amount << endl;
        cout << "price:" << _price << endl;
        _date.show();
    }
};
```

## 简介

1. **构造函数和析构函数都折构造和析构函数数的名字和类名要相同，你不需要参数 `return this;` 或者 `return *this;`**
2. **构造函数和析构函数也可以声明为参数较少的，不需要参数 `return this;`**
3. 构造函数也可以有**默认参数**
4. **析构函数是不需要参数的**，所以**析构函数也不可以调用**

### 引用

```cpp
using namespace std;

// 实现一个顺序栈
class SeqStack
{
private:
    int* _pstack;   // 动态开辟数据，存储顺序栈的元素
    int _top;       // 指向栈顶元素的位置
    int _size;      // 数组扩容的总大小

public:
    // 构造函数，代替了初始化操作
    SeqStack(int size = 10)
    {
        cout << this << " is SeqStack()" << endl;
        _pstack = new int[size];
        _top = -1;
        _size = size;
    }

    // 析构函数，代替了内存释放操作
    ~SeqStack()
    {
        cout << this << " is ~SeqStack()" << endl;
        delete[] _pstack;
        _pstack = nullptr;
    }

    // 入栈操作
    void push(int val)
    {
        if (full())
            // 扩容操作
            resize();
        _pstack[++_top] = val;
    }

    // 出栈操作
    void pop()
    {
        if (empty())
            return;
        --_top;
    }

    // 返回栈顶元素
    int top()
    {
        return _pstack[_top];
    }

    bool empty() { return _top == -1; }
    bool full() { return _top == _size - 1; }

private:
    // 顺序栈底层数组按2倍的方式进行扩容
    void resize()
    {
        int* ptmp = new int[_size * 2];
        for (int i = 0; i < _top; i++)
        {
            ptmp[i] = _pstack[i];
        }
        delete[] _pstack;
        _pstack = ptmp;
        _size *= 2;
    }
};

int main()
{
    // 默认是开辟10的内存空间
    // 也可以自己分配内存空间，SeqStack s(50);
    SeqStack s;

    for (int i = 0; i < 15; i++) {
        s.push(rand() % 100);
    }

    while (!s.empty())
    {
        cout << s.top() << " ";
        s.pop();
    }

    // 退出程序之后，自动调用析构函数
    return 0;
}
```

---

## 关键要点

1. **初始化列表顺序**：按成员变量定义顺序，不是列表顺序
2. **默认构造函数**：自定义构造函数后不再生成
3. **成员对象初始化**：必须在初始化列表中完成
4. **析构函数**：无参数，无返回值，自动调用
