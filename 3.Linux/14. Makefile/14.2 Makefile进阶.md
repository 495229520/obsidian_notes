# Makefile 进阶

## 1. 自动变量

自动变量是 Makefile 中预定义的特殊变量，在规则执行时自动设置，用于简化规则的编写。

### 1.1 常用自动变量

| 变量 | 含义 | 示例 |
|------|------|------|
| `$@` | 目标文件名 | `app` |
| `$<` | 第一个依赖文件 | `main.c` |
| `$^` | 所有依赖文件（去重） | `main.o utils.o` |
| `$+` | 所有依赖文件（不去重） | 保留重复项 |
| `$?` | 比目标更新的依赖文件 | 修改过的文件 |
| `$*` | 模式规则中 `%` 匹配的部分 | `main`（从 `main.o`） |
| `$(@D)` | 目标文件的目录部分 | `src` |
| `$(@F)` | 目标文件的文件名部分 | `app` |

### 1.2 使用示例

```makefile
# 不使用自动变量
app: main.o utils.o
	gcc main.o utils.o -o app

main.o: main.c
	gcc -c main.c -o main.o

# 使用自动变量（更简洁）
app: main.o utils.o
	gcc $^ -o $@

main.o: main.c
	gcc -c $< -o $@
```

### 1.3 自动变量详解图

```
规则: app: main.o utils.o helper.o
           ↑       ↑        ↑
           │       │        │
      $<  ─┘       └────────┴─── $^
   (第一个依赖)        (所有依赖)

      $@ ─── app (目标)
```

## 2. 模式规则（Pattern Rules）

模式规则使用 `%` 通配符，可以用一条规则匹配多个目标。

### 2.1 基本语法

```makefile
%.o: %.c
	$(CC) $(CFLAGS) -c $< -o $@
```

这条规则表示：任何 `.o` 文件依赖于同名的 `.c` 文件。

### 2.2 模式规则示例

```makefile
CC = gcc
CFLAGS = -Wall -g

SRCS = main.c utils.c helper.c
OBJS = $(SRCS:.c=.o)
TARGET = app

$(TARGET): $(OBJS)
	$(CC) $^ -o $@

# 模式规则：所有 .o 文件的生成规则
%.o: %.c
	$(CC) $(CFLAGS) -c $< -o $@

clean:
	rm -f $(OBJS) $(TARGET)
```

### 2.3 静态模式规则

静态模式规则限定了模式规则应用的目标范围：

```makefile
# 语法：targets: target-pattern: prereq-patterns
#      命令

OBJS = main.o utils.o helper.o

# 只对 OBJS 中的文件应用此规则
$(OBJS): %.o: %.c
	$(CC) $(CFLAGS) -c $< -o $@
```

## 3. Makefile 函数

Makefile 提供了许多内置函数，用于字符串处理、文件操作等。

### 3.1 字符串处理函数

#### subst - 字符串替换

```makefile
$(subst from,to,text)

# 示例
FILES = a.c b.c c.c
OBJS = $(subst .c,.o,$(FILES))
# OBJS = a.o b.o c.o
```

#### patsubst - 模式替换

```makefile
$(patsubst pattern,replacement,text)

# 示例
SRCS = main.c utils.c
OBJS = $(patsubst %.c,%.o,$(SRCS))
# OBJS = main.o utils.o

# 简写形式
OBJS = $(SRCS:.c=.o)
```

#### strip - 去除空格

```makefile
$(strip string)

# 示例
STR =    hello   world
RESULT = $(strip $(STR))
# RESULT = "hello world"
```

#### findstring - 查找字符串

```makefile
$(findstring find,text)

# 示例
$(findstring a,a b c)  # 返回 "a"
$(findstring x,a b c)  # 返回 ""
```

#### filter / filter-out - 过滤

```makefile
$(filter pattern...,text)
$(filter-out pattern...,text)

# 示例
FILES = main.c utils.c config.h readme.txt
SRCS = $(filter %.c,$(FILES))
# SRCS = main.c utils.c

OTHERS = $(filter-out %.c,$(FILES))
# OTHERS = config.h readme.txt
```

#### sort - 排序去重

```makefile
$(sort list)

# 示例
FILES = b.c a.c c.c a.c
SORTED = $(sort $(FILES))
# SORTED = a.c b.c c.c （去重并排序）
```

#### word / words - 取单词

```makefile
$(word n,text)      # 取第 n 个单词（从 1 开始）
$(words text)       # 单词数量
$(firstword text)   # 第一个单词
$(lastword text)    # 最后一个单词

# 示例
LIST = one two three
$(word 2,$(LIST))      # two
$(words $(LIST))       # 3
$(firstword $(LIST))   # one
```

### 3.2 文件名函数

#### dir / notdir - 目录和文件名

```makefile
$(dir names...)       # 取目录部分
$(notdir names...)    # 取文件名部分

# 示例
PATH = src/main.c include/utils.h
$(dir $(PATH))      # src/ include/
$(notdir $(PATH))   # main.c utils.h
```

#### suffix / basename - 后缀和基本名

```makefile
$(suffix names...)    # 取后缀
$(basename names...)  # 去除后缀

# 示例
FILES = main.c utils.o readme.txt
$(suffix $(FILES))    # .c .o .txt
$(basename $(FILES))  # main utils readme
```

#### addsuffix / addprefix - 添加后缀/前缀

```makefile
$(addsuffix suffix,names...)
$(addprefix prefix,names...)

# 示例
FILES = main utils helper
SRCS = $(addsuffix .c,$(FILES))
# SRCS = main.c utils.c helper.c

OBJS = $(addprefix build/,$(addsuffix .o,$(FILES)))
# OBJS = build/main.o build/utils.o build/helper.o
```

#### wildcard - 通配符展开

```makefile
$(wildcard pattern)

# 示例
SRCS = $(wildcard *.c)           # 当前目录所有 .c 文件
SRCS = $(wildcard src/*.c)       # src 目录所有 .c 文件
SRCS = $(wildcard src/**/*.c)    # 递归查找（需要 GNU make 4.0+）
```

### 3.3 其他常用函数

#### shell - 执行 Shell 命令

```makefile
$(shell command)

# 示例
DATE = $(shell date +%Y%m%d)
FILES = $(shell find src -name "*.c")
GIT_HASH = $(shell git rev-parse --short HEAD)
```

#### foreach - 循环

```makefile
$(foreach var,list,text)

# 示例
DIRS = src lib include
CLEAN_DIRS = $(foreach dir,$(DIRS),$(dir)/clean)
# CLEAN_DIRS = src/clean lib/clean include/clean
```

#### if - 条件判断

```makefile
$(if condition,then-part,else-part)

# 示例
DEBUG = yes
CFLAGS = $(if $(DEBUG),-g -O0,-O2)
```

#### call - 调用自定义函数

```makefile
# 定义函数
define compile
$(CC) $(CFLAGS) -c $(1) -o $(2)
endef

# 调用函数
main.o: main.c
	$(call compile,$<,$@)
```

## 4. 条件判断

### 4.1 条件语法

```makefile
ifeq (arg1, arg2)
    # arg1 等于 arg2 时执行
endif

ifneq (arg1, arg2)
    # arg1 不等于 arg2 时执行
endif

ifdef variable
    # 变量已定义时执行
endif

ifndef variable
    # 变量未定义时执行
endif
```

### 4.2 条件判断示例

```makefile
# 根据 DEBUG 变量设置编译选项
DEBUG ?= 0

ifeq ($(DEBUG), 1)
    CFLAGS = -g -O0 -DDEBUG
else
    CFLAGS = -O2 -DNDEBUG
endif

# 根据操作系统选择编译器
ifeq ($(OS),Windows_NT)
    CC = cl
    RM = del /Q
else
    CC = gcc
    RM = rm -f
endif

# 检查变量是否定义
ifdef VERBOSE
    Q =
else
    Q = @
endif

%.o: %.c
	$(Q)$(CC) $(CFLAGS) -c $< -o $@
```

## 5. 自动依赖生成

### 5.1 问题：头文件依赖

```makefile
# 手动维护头文件依赖很容易出错
main.o: main.c utils.h config.h logger.h
```

当头文件修改后，如果 Makefile 中没有正确声明依赖，可能导致不重新编译。

### 5.2 解决方案：自动生成依赖

```makefile
CC = gcc
CFLAGS = -Wall -g
SRCS = $(wildcard *.c)
OBJS = $(SRCS:.c=.o)
DEPS = $(SRCS:.c=.d)

TARGET = app

all: $(TARGET)

$(TARGET): $(OBJS)
	$(CC) $^ -o $@

# 生成 .d 依赖文件
%.d: %.c
	$(CC) -MM $< > $@

# 包含依赖文件
-include $(DEPS)

%.o: %.c
	$(CC) $(CFLAGS) -c $< -o $@

clean:
	rm -f $(OBJS) $(DEPS) $(TARGET)
```

### 5.3 更优雅的方式

```makefile
CC = gcc
CFLAGS = -Wall -g -MMD -MP

SRCS = $(wildcard *.c)
OBJS = $(SRCS:.c=.o)
DEPS = $(OBJS:.o=.d)

TARGET = app

all: $(TARGET)

$(TARGET): $(OBJS)
	$(CC) $^ -o $@

%.o: %.c
	$(CC) $(CFLAGS) -c $< -o $@

-include $(DEPS)

clean:
	rm -f $(OBJS) $(DEPS) $(TARGET)
```

**说明**：
- `-MMD`：生成依赖文件（不包含系统头文件）
- `-MP`：为每个依赖生成伪目标，防止删除头文件时报错

## 6. 多目录项目

### 6.1 项目结构

```
project/
├── Makefile
├── src/
│   ├── main.c
│   └── utils.c
├── include/
│   └── utils.h
└── build/
    └── (编译产物)
```

### 6.2 Makefile 示例

```makefile
# 目录定义
SRC_DIR = src
INC_DIR = include
BUILD_DIR = build

# 编译器设置
CC = gcc
CFLAGS = -Wall -g -I$(INC_DIR)

# 源文件和目标文件
SRCS = $(wildcard $(SRC_DIR)/*.c)
OBJS = $(patsubst $(SRC_DIR)/%.c,$(BUILD_DIR)/%.o,$(SRCS))
DEPS = $(OBJS:.o=.d)

TARGET = $(BUILD_DIR)/app

.PHONY: all clean

all: $(TARGET)

# 创建 build 目录
$(BUILD_DIR):
	mkdir -p $(BUILD_DIR)

# 链接
$(TARGET): $(OBJS)
	$(CC) $^ -o $@

# 编译（依赖 build 目录存在）
$(BUILD_DIR)/%.o: $(SRC_DIR)/%.c | $(BUILD_DIR)
	$(CC) $(CFLAGS) -MMD -MP -c $< -o $@

-include $(DEPS)

clean:
	rm -rf $(BUILD_DIR)
```

### 6.3 Order-only 依赖

`| $(BUILD_DIR)` 是 **order-only 依赖**：
- 只检查目标是否存在，不检查时间戳
- 确保目录在编译前创建
- 目录时间戳变化不会触发重新编译

## 7. 递归 Make

### 7.1 子目录 Makefile

```
project/
├── Makefile
├── src/
│   ├── Makefile
│   └── main.c
└── lib/
    ├── Makefile
    └── utils.c
```

**顶层 Makefile**：
```makefile
SUBDIRS = src lib

.PHONY: all clean $(SUBDIRS)

all: $(SUBDIRS)

$(SUBDIRS):
	$(MAKE) -C $@

clean:
	for dir in $(SUBDIRS); do \
		$(MAKE) -C $$dir clean; \
	done
```

**src/Makefile**：
```makefile
CC = gcc
CFLAGS = -Wall -g

main.o: main.c
	$(CC) $(CFLAGS) -c $< -o $@

clean:
	rm -f *.o
```

### 7.2 传递变量

```makefile
# 导出变量到子 Makefile
export CC = gcc
export CFLAGS = -Wall -g

# 或者
$(MAKE) -C subdir CC=$(CC) CFLAGS="$(CFLAGS)"
```

## 8. 实用技巧

### 8.1 静默输出

```makefile
# 方法 1：在命令前加 @
clean:
	@echo "Cleaning..."
	@rm -f *.o

# 方法 2：使用 .SILENT
.SILENT: clean
clean:
	echo "Cleaning..."
	rm -f *.o

# 方法 3：全局静默
MAKEFLAGS += --silent
```

### 8.2 彩色输出

```makefile
# 定义颜色
RED    = \033[31m
GREEN  = \033[32m
YELLOW = \033[33m
RESET  = \033[0m

# 使用
compile:
	@echo "$(GREEN)Compiling...$(RESET)"
	$(CC) $(CFLAGS) -c $< -o $@
	@echo "$(GREEN)Done!$(RESET)"
```

### 8.3 进度显示

```makefile
SRCS = $(wildcard *.c)
TOTAL = $(words $(SRCS))
CURRENT = 0

define compile
$(eval CURRENT := $(shell echo $$(($(CURRENT)+1))))
@echo "[$(CURRENT)/$(TOTAL)] Compiling $<"
@$(CC) $(CFLAGS) -c $< -o $@
endef

%.o: %.c
	$(call compile)
```

### 8.4 调试 Makefile

```bash
# 打印变量值
make print-VARIABLE

# 在 Makefile 中添加
print-%:
	@echo '$* = $($*)'

# 查看 make 执行过程
make -n        # 只打印命令，不执行
make -d        # 调试模式，显示详细信息
make -p        # 打印所有规则和变量
```

## 9. 完整项目示例

```makefile
# ============================================
#              项目配置
# ============================================
PROJECT = myproject
VERSION = 1.0.0

# 目录
SRC_DIR   = src
INC_DIR   = include
BUILD_DIR = build
BIN_DIR   = bin

# 编译器
CC       = gcc
CFLAGS   = -Wall -Wextra -I$(INC_DIR)
LDFLAGS  =
LDLIBS   = -lm -lpthread

# Debug/Release 配置
DEBUG ?= 0
ifeq ($(DEBUG), 1)
    CFLAGS  += -g -O0 -DDEBUG
    BUILD_DIR := $(BUILD_DIR)/debug
else
    CFLAGS  += -O2 -DNDEBUG
    BUILD_DIR := $(BUILD_DIR)/release
endif

# 文件
SRCS   = $(wildcard $(SRC_DIR)/*.c)
OBJS   = $(patsubst $(SRC_DIR)/%.c,$(BUILD_DIR)/%.o,$(SRCS))
DEPS   = $(OBJS:.o=.d)
TARGET = $(BIN_DIR)/$(PROJECT)

# ============================================
#              伪目标
# ============================================
.PHONY: all clean rebuild install help

# ============================================
#              默认目标
# ============================================
all: $(TARGET)
	@echo "Build complete: $(TARGET)"

# ============================================
#              创建目录
# ============================================
$(BUILD_DIR) $(BIN_DIR):
	@mkdir -p $@

# ============================================
#              链接
# ============================================
$(TARGET): $(OBJS) | $(BIN_DIR)
	@echo "Linking $@"
	@$(CC) $(LDFLAGS) $^ $(LDLIBS) -o $@

# ============================================
#              编译
# ============================================
$(BUILD_DIR)/%.o: $(SRC_DIR)/%.c | $(BUILD_DIR)
	@echo "Compiling $<"
	@$(CC) $(CFLAGS) -MMD -MP -c $< -o $@

# ============================================
#              依赖
# ============================================
-include $(DEPS)

# ============================================
#              清理
# ============================================
clean:
	@echo "Cleaning..."
	@rm -rf build bin

rebuild: clean all

# ============================================
#              安装
# ============================================
install: $(TARGET)
	@echo "Installing to /usr/local/bin/"
	@install -m 755 $(TARGET) /usr/local/bin/

# ============================================
#              帮助
# ============================================
help:
	@echo "Usage: make [target] [DEBUG=1]"
	@echo ""
	@echo "Targets:"
	@echo "  all      - Build the project (default)"
	@echo "  clean    - Remove build files"
	@echo "  rebuild  - Clean and rebuild"
	@echo "  install  - Install to /usr/local/bin"
	@echo "  help     - Show this help"
	@echo ""
	@echo "Options:"
	@echo "  DEBUG=1  - Build with debug symbols"

# ============================================
#              调试
# ============================================
print-%:
	@echo '$* = $($*)'
```

## 10. 关键要点

1. **自动变量**：`$@`（目标）、`$<`（第一个依赖）、`$^`（所有依赖）
2. **模式规则**：`%.o: %.c` 简化多文件编译
3. **函数**：`wildcard`、`patsubst`、`shell` 等提高灵活性
4. **自动依赖**：使用 `-MMD -MP` 自动生成头文件依赖
5. **多目录**：使用 order-only 依赖和递归 Make
6. **调试**：`make -n`、`make -p`、`print-%` 目标
