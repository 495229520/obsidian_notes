# 字符分类函数

## 简介

头文件：`<ctype.h>`

字符分类函数用于测试单个字符的属性，返回非零值表示真（true），返回零表示假（false）。

| 函数 | 说明 |
|------|------|
| `int isalnum(int c)` | 测试字符是否为英文字母或数字 |
| `int isalpha(int c)` | 测试字符是否为英文字母 |
| `int isascii(int c)` | 测试字符是否为 ASCII 码字符 |
| `int isblank(int c)` | 测试字符是否为空格字符 |
| `int iscntrl(int c)` | 测试字符是否为 ASCII 码的控制字符 |
| `int isdigit(int c)` | 测试字符是否为阿拉伯数字 |
| `int isgraph(int c)` | 测试字符是否为可打印字符 |
| `int islower(int c)` | 测试字符是否为小写英文字母 |
| `int isprint(int c)` | 测试字符是否为可打印字符 |
| `int isspace(int c)` | 测试字符是否为空格字符 |
| `int ispunct(int c)` | 测试字符是否为标点符号或特殊符号 |
| `int isupper(int c)` | 测试字符是否为大写英文字母 |
| `int isxdigit(int c)` | 测试字符是否为十六进制数字 |

---

## 函数详解

### isalnum - 字母或数字
检测字符是否为字母（a-z, A-Z）或数字（0-9）。

### isalpha - 字母
仅检测字符是否为字母（a-z, A-Z）。

### isascii - ASCII 字符
检测字符是否在 ASCII 码范围内（0-127）。

### isblank - 空白字符
检测字符是否为空格（space）或制表符（tab）。

### iscntrl - 控制字符
检测字符是否为控制字符（ASCII 0-31 和 127）。

### isdigit - 数字
检测字符是否为十进制数字（0-9）。

### isgraph - 可显示字符
检测字符是否为可显示的图形字符（排除空格）。

### islower / isupper - 大小写字母
分别检测字符是否为小写字母或大写字母。

### isprint - 可打印字符
检测字符是否为可打印字符（包括空格）。

### isspace - 空白字符
检测字符是否为空白字符（空格、换行、制表符等）。

### ispunct - 标点符号
检测字符是否为标点符号。

### isxdigit - 十六进制数字
检测字符是否为十六进制数字（0-9, a-f, A-F）。

## 示例代码

```cpp
#include <ctype.h>
#include <iostream>

int main()
{
    const char str[] = "hello world!1.99\r\n\t2002-03-29\r\n\tby HWei\r\n";
    size_t len = sizeof(str);

    for (size_t i = 0; i < len; i++)
    {
        std::cout << str[i] << ":";

        // 是否为英文字母或数字
        if (isalnum(str[i])) std::cout << "A";

        // 是否为英文字母
        if (isalpha(str[i])) std::cout << "a";

        // 测试字符是否是ASCII码字符
        if (isascii(str[i])) std::cout << "B";

        // 测试字符是否为空格字符
        if (isblank(str[i])) std::cout << "b";

        // 测试字符是否为ASCII码的控制字符
        if (iscntrl(str[i])) std::cout << "C";

        // 测试字符是否为阿拉伯数字
        if (isdigit(str[i])) std::cout << "D";

        // 测试字符是否为可打印字符
        if (isgraph(str[i])) std::cout << "G";

        // 测试字符是否为小写英文字符
        if (islower(str[i])) std::cout << "L";

        // 测试字符是否为可打印字符
        if (isprint(str[i])) std::cout << "g";

        // 测试字符是否为空格字符
        if (isspace(str[i])) std::cout << "S";

        // 测试字符是否为标点符号或特殊符号
        if (ispunct(str[i])) std::cout << "P";

        // 测试字符是否为大写英文字符
        if (isupper(str[i])) std::cout << "U";

        // 测试字符是否为十六进制数字
        if (isxdigit(str[i])) std::cout << "H";

        std::cout << std::endl;
    }
    return 0;
}
```

---

## 关键要点

- 这些函数都接受 `int` 类型参数，但通常传入 `char` 类型
- 返回值为非零表示条件成立，零表示不成立
- 使用前需包含 `<ctype.h>`（C）或 `<cctype>`（C++）
- 这些函数是 locale 敏感的，在不同区域设置下可能有不同行为
