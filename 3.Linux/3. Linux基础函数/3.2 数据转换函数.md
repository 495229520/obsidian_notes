# 数据转换函数

## 简介

头文件：`<stdlib.h>`

数据转换函数用于在不同数据类型之间进行转换，主要包括字符串与数值类型的相互转换。

---

## 字符串转数值函数

### atof - 字符串转 double

```c
double atof(const char* str);
```

将字符串转换为双精度浮点数。

### atoi - 字符串转 int

```c
int atoi(const char* str);
```

将字符串转换为整数。

### atol - 字符串转 long int

```c
long int atol(const char* str);
```

将字符串转换为长整型。

### atoll - 字符串转 long long int（C++11）

```c
long long int atoll(const char* str);
```

将字符串转换为长长整型。

---

## 带错误检测的转换函数（strtox 系列）

这些函数比 atox 系列更安全，提供错误检测和进制转换功能。

### strtol - 字符串转 long int

```c
long int strtol(const char* str, char** endptr, int base);
```

### strtoll - 字符串转 long long int

```c
long long int strtoll(const char* str, char** endptr, int base);
```

### strtoul - 字符串转 unsigned long int

```c
unsigned long int strtoul(const char* str, char** endptr, int base);
```

### strtoull - 字符串转 unsigned long long int

```c
unsigned long long int strtoull(const char* str, char** endptr, int base);
```

### strtof - 字符串转 float

```c
float strtof(const char* str, char** endptr);
```

### strtod - 字符串转 double

```c
double strtod(const char* str, char** endptr);
```

### strtold - 字符串转 long double

```c
long double strtold(const char* str, char** endptr);
```

**参数说明：**
- `str`：要转换的字符串
- `endptr`：指向转换后第一个无效字符的指针（可以为 NULL）
- `base`：进制（2-36），0 表示自动检测

## 示例代码

### 基本类型转换

```cpp
#include <stdlib.h>
#include <iostream>

int main()
{
    // sizeof 各类型
    std::cout << "long:" << sizeof(long) << " int:" << sizeof(int) << std::endl;
    std::cout << "sizeunsigned int:" << sizeof(unsigned int) << std::endl;
    std::cout << "long long int:" << sizeof(long long int) << std::endl;

    // 字符串转数值示例
    std::cout << strtol("10#$#@#$@#@$RCV", NULL, 10) << std::endl;     // 10
    std::cout << strtol("1.CYHNUXT", NULL, 10) << std::endl;           // 1
    std::cout << strtoll("9223372036854775807", NULL, 10) << std::endl;// LLONG_MAX

    // printf 格式化输出
    std::cout << printf("%ld%s%lld%s", 100L, "&Aqipt", &a1qd) << std::endl;

    return 0;
}
```

### 进制转换示例

```cpp
#include <stdlib.h>
#include <iostream>

int main()
{
    // 不同进制的字符串转换
    std::cout << strtol("0.0123456799", 10, &despt, &a1qd) << std::endl;
    std::cout << strtol("1234", 10, &despt, &a1qd) << std::endl;
    std::cout << strtoul("0.123456789", 10, &despt, &a1qd) << std::endl;
    std::cout << strtoul("-123.456789", 10, &despt, &a1qd) << std::endl;
    std::cout << strtoul("-123.456789", 10, &despt, &a1qd) << std::endl;

    return 0;
}
```

## 格式化数值输出函数

### 整数格式化输出

| 格式符 | 类型 | 说明 | 示例 |
|--------|------|------|------|
| `%d` | int | 有符号十进制整数 | `printf("%d", 100)` |
| `%u` | unsigned int | 无符号十进制整数 | `printf("%u", 100)` |
| `%ld` | long | 长整型 | `printf("%ld", 100L)` |
| `%lld` | long long | 长长整型 | `printf("%lld", 100LL)` |
| `%x` | int | 十六进制（小写） | `printf("%x", 255)` → `ff` |
| `%X` | int | 十六进制（大写） | `printf("%X", 255)` → `FF` |
| `%o` | int | 八进制 | `printf("%o", 8)` → `10` |

### 浮点数格式化输出

| 格式符 | 类型 | 说明 | 示例 |
|--------|------|------|------|
| `%f` | double | 浮点数（默认6位小数） | `printf("%f", 3.14)` |
| `%e` | double | 科学计数法（小写e） | `printf("%e", 3.14)` |
| `%E` | double | 科学计数法（大写E） | `printf("%E", 3.14)` |
| `%g` | double | 自动选择 %f 或 %e | `printf("%g", 3.14)` |

### 宽度与精度控制

```c
// 宽度控制
printf("%10d", 100);      // 右对齐，宽度10
printf("%-10d", 100);     // 左对齐，宽度10
printf("%010d", 100);     // 用0填充，宽度10

// 精度控制
printf("%.2f", 3.14159);  // 保留2位小数：3.14
printf("%10.2f", 3.14);   // 宽度10，2位小数
```

## 关键要点

- `atox` 系列函数简单但不安全，无法检测转换错误
- `strtox` 系列函数更安全，可以检测转换错误和处理不同进制
- `endptr` 参数可用于检测是否完整转换了字符串
- `base` 参数为 0 时，会根据字符串前缀自动检测进制（0x=16进制，0=8进制，其他=10进制）
