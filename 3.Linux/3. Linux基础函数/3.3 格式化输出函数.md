# 格式化输出函数

## 简介

格式化输出函数是 C 语言中最常用的输出函数族，用于将数据按照指定格式输出到不同目标（标准输出、文件、字符串等）。

---

## printf 函数族

### printf - 输出到标准输出

```c
int printf(const char* format, ...);
```

将格式化的数据输出到标准输出（stdout）。

### fprintf - 输出到文件

```c
int fprintf(FILE* stream, const char* format, ...);
```

将格式化的数据输出到指定的文件流。

### sprintf - 输出到字符串

```c
int sprintf(char* str, const char* format, ...);
```

将格式化的数据输出到字符串缓冲区。**注意：不检查缓冲区溢出，存在安全风险。**

### snprintf - 安全输出到字符串

```c
int snprintf(char* str, size_t size, const char* format, ...);
```

将格式化的数据输出到字符串缓冲区，最多写入 `size-1` 个字符。**推荐使用此函数替代 sprintf。**

### vprintf / vfprintf / vsprintf / vsnprintf

带 `v` 前缀的版本接受 `va_list` 参数，用于可变参数函数的实现。

```c
int vprintf(const char* format, va_list arg);
int vfprintf(FILE* stream, const char* format, va_list arg);
int vsprintf(char* str, const char* format, va_list arg);
int vsnprintf(char* str, size_t size, const char* format, va_list arg);
```

## 格式说明符

格式说明符的完整形式：`%[flags][width][.precision][length]specifier`

### 转换说明符（specifier）

| 说明符 | 输出类型 | 示例 |
|--------|----------|------|
| `%d` / `%i` | 有符号十进制整数 | `printf("%d", -10)` → `-10` |
| `%u` | 无符号十进制整数 | `printf("%u", 10)` → `10` |
| `%o` | 无符号八进制整数 | `printf("%o", 8)` → `10` |
| `%x` | 无符号十六进制整数（小写） | `printf("%x", 255)` → `ff` |
| `%X` | 无符号十六进制整数（大写） | `printf("%X", 255)` → `FF` |
| `%f` | 浮点数（小数形式） | `printf("%f", 3.14)` → `3.140000` |
| `%e` | 浮点数（科学计数法，小写e） | `printf("%e", 3.14)` → `3.140000e+00` |
| `%E` | 浮点数（科学计数法，大写E） | `printf("%E", 3.14)` → `3.140000E+00` |
| `%g` | 浮点数（自动选择 %f 或 %e） | 根据值的大小自动选择更简洁的格式 |
| `%c` | 单个字符 | `printf("%c", 'A')` → `A` |
| `%s` | 字符串 | `printf("%s", "hello")` → `hello` |
| `%p` | 指针地址 | `printf("%p", ptr)` → `0x7fff5fbff8c8` |
| `%n` | 不输出，将已输出字符数存入参数 | 安全风险，谨慎使用 |
| `%%` | 输出百分号 | `printf("%%")` → `%` |

### 长度修饰符（length）

| 修饰符 | 适用于 | 说明 |
|--------|--------|------|
| `hh` | d, i, o, u, x, X | signed/unsigned char |
| `h` | d, i, o, u, x, X | short int |
| `l` | d, i, o, u, x, X | long int |
| `ll` | d, i, o, u, x, X | long long int |
| `L` | f, e, E, g, G | long double |
| `z` | d, i, o, u, x, X | size_t |

### 标志（flags）

| 标志 | 说明 | 示例 |
|------|------|------|
| `-` | 左对齐（默认右对齐） | `printf("%-10d", 42)` → `42        ` |
| `+` | 强制显示正负号 | `printf("%+d", 42)` → `+42` |
| ` ` | 正数前加空格 | `printf("% d", 42)` → ` 42` |
| `#` | 替代形式（八进制加0，十六进制加0x） | `printf("%#x", 255)` → `0xff` |
| `0` | 用零填充 | `printf("%010d", 42)` → `0000000042` |

### 宽度与精度

- **宽度**：指定输出的最小字符数
- **精度**：对于浮点数指定小数位数，对于字符串指定最大字符数

```c
printf("%10d", 42);      // 宽度10，右对齐：       42
printf("%-10d", 42);     // 宽度10，左对齐：42
printf("%010d", 42);     // 宽度10，零填充：0000000042
printf("%.2f", 3.14159); // 精度2：3.14
printf("%10.2f", 3.14);  // 宽度10，精度2：      3.14
printf("%.5s", "hello world"); // 最多5个字符：hello
```

## 示例代码

### 基本用法

```c
#include <stdio.h>

int main()
{
    // 整数输出
    printf("%d\n", 42);           // 42
    printf("%5d\n", 42);          // "   42"
    printf("%-5d\n", 42);         // "42   "
    printf("%05d\n", 42);         // "00042"

    // 浮点数输出
    printf("%f\n", 3.14159);      // 3.141590
    printf("%.2f\n", 3.14159);    // 3.14
    printf("%10.2f\n", 3.14159);  // "      3.14"

    // 十六进制输出
    printf("%x\n", 255);          // ff
    printf("%X\n", 255);          // FF
    printf("%#x\n", 255);         // 0xff

    // 字符串输出
    printf("%s\n", "hello");      // hello
    printf("%10s\n", "hello");    // "     hello"
    printf("%.3s\n", "hello");    // hel

    return 0;
}
```

### snprintf 安全用法

```c
#include <stdio.h>

int main()
{
    char buffer[128];

    // 安全的字符串格式化
    int len = snprintf(buffer, sizeof(buffer),
                       "Name: %s, Age: %d, Score: %.2f",
                       "Alice", 25, 95.5);

    printf("Written %d characters: %s\n", len, buffer);

    return 0;
}
```

### sprintf 与 snprintf 的区别

```c
char buffer[10];

// 危险：可能导致缓冲区溢出
sprintf(buffer, "%s", "This is a very long string");

// 安全：最多写入9个字符 + '\0'
snprintf(buffer, sizeof(buffer), "%s", "This is a very long string");
// 结果：buffer = "This is a"
```

## 关键要点

- **printf**：输出到标准输出（屏幕）
- **fprintf**：输出到文件
- **sprintf**：输出到字符串（不安全）
- **snprintf**：输出到字符串（安全，推荐使用）
- **vprintf 系列**：用于实现自定义的可变参数函数
- 格式字符串攻击是常见的安全漏洞，永远不要将用户输入直接作为格式字符串
- `snprintf` 的返回值是"应该写入"的字符数（不包括 '\0'），可用于检测截断
