# 格式化输入函数

## 简介

在 Linux 环境下的 C/C++ 编程中，`scanf` 和 `sscanf` 是 C 语言标准库（cstdio）提供的、用于**格式化输入**的函数，它们是 `printf` 和 `sprintf` 的"逆操作"，用来从输入源（如标准输入或字符串）中解析和读取格式化的数据。

空格是默认分隔符。部分数据格式是可以自动正确分割的，但是小数和整数之间，需要手动分割（使用空格）。

---

## scanf 函数

### 函数原型

```c
int scanf(const char* format, ...);
```

`scanf` 的作用是从**标准输入设备**（通常是键盘或命令行）读取格式化的数据，并将其存储到指定的变量中。

### 基本用法

```c
#include <cstdio>
#include <iostream>

int main() {
    int age;
    char name[50];
    double pi;

    printf("Enter your name, age, and a value for pi (e.g., John 30 3.14): \n");

    // %s 用于读取字符串，%d 用于读取整数，%lf 用于读取 double 类型
    // 注意：变量前面需要加 & 符号来获取地址
    int result = scanf("%s %d %lf", name, &age, &pi);

    if (result == 3) { // 检查是否成功读取了所有3个数据项
        printf("Hello, %s!\n", name);
        printf("You are %d years old.\n", age);
        printf("Pi value read is: %lf\n", pi);
    } else {
        printf("Error: Could not read all data items.\n");
    }

    return 0;
}
```

## 格式说明符

| 占位符 | 数据类型 | 说明 | 示例 (C++) | 注意事项 |
|--------|----------|------|------------|----------|
| `%d` | int | 十进制整数，读取一个有符号十进制整数 | `int a; scanf("%d", &a);` | 最常用的整数占位符 |
| `%u` | unsigned int | 无符号十进制整数，读取一个非负整数 | `unsigned int b; scanf("%u", &b);` | 仅读取非负数 |
| `%c` | char | 单个字符，读取一个字符，包括空格、换行符 | `char ch; scanf("%c", &ch);` | `%c` 不会跳过前面的空白字符 |
| `%s` | char[] | 字符串，读取一个非空白字符序列，并在末尾自动添加空字符 `\0` | `char str[50]; scanf("%s", str);` | 数组名本身就是地址，不需要加 `&`。遇到空白字符（空格、换行符等）停止读取 |
| `%f` | float | 单精度浮点数，读取一个 float 类型数据 | `float f; scanf("%f", &f);` | 用于 float 类型 |
| `%lf` | double | 双精度浮点数，读取一个 double 类型数据 | `double d; scanf("%lf", &d);` | 与 printf 中的 `%f` 不同，printf 中 `%f` 即可用于 float 和 double，但 scanf 必须用 `%lf` |
| `%Lf` | long double | 长双精度浮点数，读取一个 long double 类型数据 | `long double ld; scanf("%Lf", &ld);` | 用于更精确的浮点数类型 |
| `%hd` | short | 短整型，用长度修饰符 h 修饰 `%d`，读取一个 short 类型 | `short sh; scanf("%hd", &sh);` | 用于 short 类型 |
| `%ld` | long | 长整型，用长度修饰符 l 修饰 `%d`，读取一个 long 类型 | `long l; scanf("%ld", &l);` | 用于 long 类型 |
| `%lld` | long long | 长长整型，用长度修饰符 ll 修饰 `%d`，读取一个 long long 类型 | `long long ll; scanf("%lld", &ll);` | 用于 long long 类型 |
| `%x` | int | 十六进制整数，读取一个以 0x 或 0X 开头的十六进制数 | `int hex_val; scanf("%x", &hex_val);` | 0x 前缀是可选的 |

## sscanf 函数

### 函数原型

```c
int sscanf(const char* str, const char* format, ...);
```

`sscanf` 的作用是从一个**已经存在的字符串**中读取格式化的数据，而不是从标准输入。这对于解析配置文件、日志文件或网络数据非常有用。

- **str**：源字符串，sscanf 将从这个字符串中解析数据
- **format**：格式化字符串，与 scanf 相同
- **...**：可变参数列表，指向要存储数据的变量的指针

### 示例代码

```c
int main() {
    const char* data_string = "User ID: 101, Status: active, Score: 95.5";

    int id;
    char status[20];
    float score;

    // 从 data_string 中解析数据
    // 注意：格式字符串中的 "User ID: " 和 ", Status: " 等都是用来匹配源字符串中的固定文本
    int result = sscanf(data_string, "User ID: %d, Status: %s, Score: %f", &id, status, &score);

    if (result == 3) {
        printf("Successfully parsed data:\n");
        printf("ID: %d\n", id);
        printf("Status: %s\n", status);
        printf("Score: %.1f\n", score);
    }
    else {
        printf("Error: Data format does not match.\n");
    }

    return 0;
}
```

## scanf 与 sscanf 的区别

| 特性 | scanf | sscanf |
|------|-------|--------|
| 数据来源 | 标准输入（键盘） | 字符串 |
| 典型用途 | 交互式输入 | 解析已有数据 |
| 安全性 | 相对较低（缓冲区溢出风险） | 相对较高（数据已在内存中） |

---

## 安全注意事项

1. **缓冲区溢出**：使用 `%s` 时，如果不限制读取长度，可能导致缓冲区溢出
   ```c
   char buf[10];
   scanf("%9s", buf);  // 限制最多读取9个字符
   ```

2. **返回值检查**：始终检查 scanf/sscanf 的返回值，确保成功读取了预期数量的数据项

3. **清空输入缓冲区**：scanf 读取后可能残留换行符，影响后续输入
   ```c
   scanf("%d", &num);
   while (getchar() != '\n');  // 清空缓冲区
   ```

---

## 关键要点

- `scanf` 从标准输入读取，`sscanf` 从字符串读取
- 变量前需要加 `&` 获取地址（数组名除外）
- `%lf` 用于 double 类型（与 printf 不同）
- 空格是默认分隔符
- 始终检查返回值以确保读取成功
- 使用宽度限制防止缓冲区溢出
