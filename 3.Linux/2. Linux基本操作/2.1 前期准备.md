# Linux 开发环境前期准备

本节介绍在 Linux 系统上进行 C/C++ 开发前的环境准备工作，包括编译器安装、SSH 服务配置以及 Visual Studio 远程开发设置。

---

## GCC 安装

在 Ubuntu/Linux 系统上安装 GCC 编译器的步骤：

| 步骤 | 操作 |
|------|------|
| 1 | 设置 root 密码：`sudo passwd root` |
| 2 | 登录 root 账户：`su` |
| 3 | 安装编译器：`apt-get install gcc g++` |
| 4 | 查看当前的 gcc g++ 版本 |
| 5 | 验证：`g++ -o test main.cpp` 然后 `./test` |

## SSH 服务安装与配置

SSH（Secure Shell）是一种加密的网络协议，用于在不安全的网络上安全地访问远程计算机。

### 安装 SSH

```bash
# 安装服务器程序
sudo apt install openssh-server

# 安装客户端程序
sudo apt install openssh-client
```

### 修改配置文件

SSH 配置文件位于 `/etc/ssh/sshd_config`，以下是常用配置项：

```
LoginGraceTime 2m
PermitRootLogin yes          # 允许 root 用户登录（红色标注，需要修改）
PubkeyAuthentication yes
PasswordAuthentication yes
ChallengeResponseAuthentication no
UsePAM yes
X11Forwarding yes
PrintMotd no
AcceptEnv LANG LC_*
Subsystem    sftp /usr/lib/openssh/sftp-server
```

> **注意**：`PermitRootLogin yes` 是需要修改的关键配置，它允许 root 用户通过 SSH 登录。

### 启动服务

```bash
# 在 Ubuntu 中启动 SSH 服务
sudo service ssh start

# 检查 sshd 是否开启
sudo ps -aux | grep sshd
```

### 设置开机自启（Windows WSL 环境）

在 Windows 启动目录下创建批处理文件：

路径：`C:\Users\{用户名}\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup`

创建文件 `1.bat`，内容为：
```batch
C:\Windows\System32\bash.exe -c "sudo service ssh start"
```

## Visual Studio 远程 Linux 开发配置

Visual Studio 支持通过 SSH 连接到 Linux 系统进行远程开发和调试。

### 配置步骤

#### 1. 打开连接管理器

在 Visual Studio 中：
- 菜单栏 → **工具(T)** → **选项(O)**
- 在左侧导航中找到：**跨平台** → **连接管理器**

#### 2. 添加远程连接

在"添加或移除到 Linux、Mac 或 Windows 等远程系统的 SSH 连接"对话框中：

| 字段 | 示例值 |
|------|--------|
| 主机名 | 127.0.0.1（或远程 Linux 的 IP 地址） |
| 端口 | 22 |
| 用户名 | root |
| 操作系统 | Ubuntu (x6) |
| 身份验证类型 | 密码 |
| 密码 | （输入对应的密码） |

#### 3. 配置项目属性

右键点击解决方案 → **属性(R)** → 进入 **LinuxConsole 属性页**：

在"配置属性"→"常规"中设置：

| 属性 | 值 |
|------|-----|
| 远程生成计算机 | `sername=root, port=22, authentication=Password` |
| 远程根目录 | `~/projects` |
| 远程项目目录 | `$(ProjectName)` |
| 配置类型 | 应用程序(.out) |

#### 4. 运行和调试

配置完成后，Visual Studio 会自动：
1. 通过 SSH 将源代码同步到远程 Linux 系统
2. 在远程系统上使用 GCC 编译代码
3. 在远程控制台中运行程序

示例程序：
```cpp
#include <cstdio>

int main()
{
    printf("%s 向你问好!\n", "LinuxConsole");
    return 0;
}
```

运行结果会显示在 Visual Studio 底部的 **Remote Console** 窗口中：
```
LinuxConsole 向你问好!
```

## 关键要点

- **GCC 安装**：使用 `apt-get install gcc g++` 安装编译器
- **SSH 配置**：修改 `/etc/ssh/sshd_config`，确保 `PermitRootLogin yes`
- **启动 SSH**：`sudo service ssh start`，使用 `ps -aux | grep sshd` 验证
- **VS 远程开发**：通过连接管理器配置 SSH 连接，支持远程编译和调试
