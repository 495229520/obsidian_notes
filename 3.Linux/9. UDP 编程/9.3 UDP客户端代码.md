# UDP 客户端代码

## 概述

UDP 客户端的实现比 [[8.3 TCP客户端|TCP 客户端]] 更加简单。由于 [[9.1 UDP原理|UDP 协议]] 的无连接特性，客户端不需要调用 `connect()` 函数来建立连接，可以直接使用 `sendto()` 发送数据。

根据 [[9.1 UDP原理#为什么客户端不用 bind？|UDP 原理]]，客户端通常**不需要显式调用 bind()**，操作系统会在首次调用 `sendto()` 时自动为其分配一个临时端口。

```
       UDP 客户端                              UDP 服务端
          │                                        │
   socket() │                                      │
          │                                        │
   设置服务器地址                                    │
          │                                        │
   sendto() ──────────── 数据 ────────────→ recvfrom()
          │                                        │
   recvfrom() ←─────────── 数据 ─────────── sendto()
          │                                        │
   close() │                                       │
```

---

## 代码流程

UDP 客户端的基本流程如下：

```
┌─────────────────────────────────────────┐
│           socket() 创建套接字            │
└─────────────────┬───────────────────────┘
                  │
                  ↓
┌─────────────────────────────────────────┐
│          设置目标服务器地址               │
└─────────────────┬───────────────────────┘
                  │
                  ↓
┌─────────────────────────────────────────┐
│    sendto() 发送数据到服务器              │
└─────────────────┬───────────────────────┘
                  │
                  ↓
┌─────────────────────────────────────────┐
│   recvfrom() 接收服务器响应               │
└─────────────────┬───────────────────────┘
                  │
                  ↓
┌─────────────────────────────────────────┐
│           close() 关闭套接字             │
└─────────────────────────────────────────┘
```

## 示例代码

以下是一个完整的 UDP 客户端示例，可与 [[9.2 UDP服务端代码|UDP 服务端]] 配合使用：

```c
#include <stdio.h>
#include <stdlib.h>
#include <string.h>
#include <unistd.h>
#include <arpa/inet.h>
#include <sys/socket.h>

#define BUFFER_SIZE 512

void handle_error(const char* msg) {
    perror(msg);
    exit(1);
}

int lession74(int argc, char* argv[])
{
    // 变量声明
    int client_sock;
    struct sockaddr_in serv_addr;
    socklen_t serv_len = sizeof(serv_addr);
    char message[BUFFER_SIZE] = "";

    // 1. 校验参数
    if (argc != 3) {
        printf("usage: %s <ip> <port>\n", argv[0]);
        handle_error("argument error!");
    }

    // 2. 创建套接字（UDP）
    client_sock = socket(PF_INET, SOCK_DGRAM, 0);  // SOCK_DGRAM 表示 UDP
    if (client_sock == -1) {
        handle_error("socket create failed!");
    }

    // 3. 设置服务器地址
    memset(&serv_addr, 0, sizeof(serv_addr));
    serv_addr.sin_family = AF_INET;                          // 设置为 IPv4
    serv_addr.sin_addr.s_addr = inet_addr(argv[1]);          // 需要发送到的 IP 地址
    serv_addr.sin_port = htons((short)atoi(argv[2]));        // 需要发送到的端口

    // 4. 通信循环
    while (1) {
        // 获取用户输入
        printf("input message (q to Quit): ");
        scanf("%s", message);

        // 检查退出条件
        if ((strcmp(message, "q\n") == 0) || strcmp(message, "Q\n") == 0) {
            break;
        }

        // 发送数据到服务器
        ssize_t len = sendto(client_sock, message, strlen(message), 0,
                             (struct sockaddr*)&serv_addr, serv_len);
        if (len == -1) {
            perror("sendto failed");
            continue;
        }

        // 清空缓冲区，准备接收响应
        memset(message, 0, len);

        // 接收服务器响应
        recvfrom(client_sock, message, sizeof(message), 0,
                 (struct sockaddr*)&serv_addr, &serv_len);

        printf("recv: %s\n", message);
    }

    // 5. 关闭套接字
    close(client_sock);
    return 0;
}

int main(int argc, char* argv[]) {
    lession74(argc, argv);
    return 0;
}
```

## 代码详解

### 1. 创建 UDP 套接字

```c
client_sock = socket(PF_INET, SOCK_DGRAM, 0);
```

- 使用 `SOCK_DGRAM` 创建 UDP 类型的 [[7.6 套接字#type（套接字类型）常用值|套接字]]
- 与 TCP 客户端使用的 `SOCK_STREAM` 不同

### 2. 设置服务器地址

```c
serv_addr.sin_family = AF_INET;
serv_addr.sin_addr.s_addr = inet_addr(argv[1]);  // 服务器 IP
serv_addr.sin_port = htons((short)atoi(argv[2])); // 服务器端口
```

- 使用 `inet_addr()` 将点分十进制 [[7.2 IP地址|IP]] 转换为网络字节序
- 使用 `htons()` 将 [[7.3 端口|端口号]] 转换为网络字节序

### 3. 发送数据

```c
sendto(client_sock, message, strlen(message), 0,
       (struct sockaddr*)&serv_addr, serv_len);
```

- 第一次调用 `sendto()` 时，操作系统会自动为客户端分配一个临时端口
- 无需事先调用 `connect()` 或 `bind()`

### 4. 接收响应

```c
recvfrom(client_sock, message, sizeof(message), 0,
         (struct sockaddr*)&serv_addr, &serv_len);
```

- 通过 `recvfrom()` 接收服务器的响应
- `serv_addr` 参数可用于验证响应来源

## UDP 客户端 vs TCP 客户端

| 对比项 | UDP 客户端 | TCP 客户端 |
|:---|:---|:---|
| 建立连接 | 不需要 `connect()` | 需要 `connect()` |
| 绑定端口 | 自动分配（无需 `bind()`） | 自动分配 |
| 发送数据 | `sendto()` 指定目标地址 | `write()` / `send()` |
| 接收数据 | `recvfrom()` | `read()` / `recv()` |
| 可靠性 | 不保证送达 | 保证送达 |

---

## 关键要点

1. **无需 connect()**：UDP 客户端直接使用 `sendto()` 发送数据，无需建立连接
2. **无需 bind()**：首次调用 `sendto()` 时系统自动分配临时端口
3. **每次指定地址**：`sendto()` 和 `recvfrom()` 都需要地址参数
4. **不保证可靠性**：发送的数据可能丢失，应用层需自行处理
