# 读取文件

本节介绍 C 标准库中的文件读取函数，包括 `fread()`、`fgetc()` 和 `fgets()`，以及文件内部指针的概念。

---

## 文件内部指针

每个打开的文件流都维护一个**文件位置指示器**（file position indicator），也称为**文件内部指针**。

### 指针特性

- 指针会记录上次读/写的位置。读写的位置是分开的，统一记录
- 每次读写，都可以改变文件中的位置。这可能会覆盖原内容的偏移记录
- 不管是读（`fread`、`fgets`）还是写（`fwrite`、`fprintf`），所有操作都使用同一个文件位置指示器
- 一个文件位置指针：一个变量存储，它既指向当前读取位置，又指向当前写入位置

### 指针行为示例

```c
FILE *fp;
char buffer[100];
const char* data_to_write = "1234567890";

// 以 "w+" 模式打开：可读写，清空原内容
fp = fopen("test.txt", "w+");

// 写入一个字符串并关闭
fputs("ABCDEFGHIJKLMNOP", fp);
fclose(fp);

// 以 "r+" 重新打开文件
fp = fopen("test.txt", "r+");

// 1. 读取 5 个字符
size_t bytes_read = fread(buffer, 1, 5, fp);
buffer[bytes_read] = '\0';
printf("After fread(), read '%s'. Current file position: %ld\n", buffer, ftell(fp));

// 此时指针在位置 5

// 2. 注意：这一步容易忽略
// 注意！这个位置有坑：在某些不增加换行符的文本上切换读/写时
// 可能需要 fseek(fp, 0, SEEK_CUR); 来"同步"读写指针
fseek(fp, 0, SEEK_SET);  // 回到文件开头

// 3. 向前读写 8 个字符
bytes_read = fread(buffer, 1, 8, fp);
buffer[bytes_read] = '\0';
printf("After fread(8), write '%s'. Current file position: %ld\n", buffer, ftell(fp));

// 4. 再次读取，从当前位置（位置 8）开始
fseek(fp, 0, SEEK_SET);  // 重置位置
fseek(fp, 8, SEEK_SET);  // 跳到位置 8
bytes_read = fread(buffer, 1, 5, fp);
buffer[bytes_read] = '\0';
printf("After seek(8) and fread(5): '%s'\n", buffer);

fclose(fp);
```

## fread() - 二进制读取

`fread()` 是一个**二进制读取函数**。它用于从文件中读取指定大小和数量的数据块，适合读取二进制文件或结构化数据。

### 函数原型

```c
#include <stdio.h>

size_t fread(void *buffer, size_t size, size_t count, FILE *stream);
```

### 参数说明

- **buffer**：一个指向内存区域的指针，读取的数据将存储到这里
- **size**：要读取的每个数据块的字节大小
- **count**：要读取的数据块数量
- **stream**：一个指向已打开文件的 `FILE*` 指针

### 返回值

- 成功：返回实际读取的数据块数量（可能小于 count，如果到达文件末尾或发生错误）
- 到达 EOF 或错误：返回值小于 count

### fread 的特点

- `fread()` 适用于读取数据的**块状结构**（比如结构体数组）
- 比 `fgetc()` 或 `fgets()` 更高效，因为可以一次性读取大量数据
- 对于二进制文件，`fread()` 是首选方法

### 示例代码

```c
#include <stdio.h>
#include <stdlib.h>
#include <string.h>

// 假设我们有一个结构体来存储数据
typedef struct {
    int id;
    float value;
} record;

int main() {
    FILE *fp;
    const char *text_data = "This is a simple text line.\n";
    record rec_out[2] = {{1, 10.5}, {2, 20.25}};
    record rec_in[2];
    char ch;
    char line_buffer[50];

    // 1. 创建并写入文件，以便后续读取
    fp = fopen("data.bin", "wb+");
    if (fp == NULL) {
        perror("fopen error");
        return 1;
    }

    // 写入结构体数据
    fwrite(rec_out, sizeof(record), 2, fp);
    // 写入文本数据
    fwrite(text_data, sizeof(char), strlen(text_data), fp);

    // 将文件指针移开头，以便读取
    rewind(fp);

    // 2. 使用 fread() 读取结构体数据
    if (fread(rec_in, sizeof(record), 2, fp) == 2) {
        printf("Record 1: ID=%d, Value=%.1f\n", rec_in[0].id, rec_in[0].value);
        printf("Record 2: ID=%d, Value=%.1f\n", rec_in[1].id, rec_in[1].value);
    } else {
        printf("fread failed to read all records.\n");
    }

    // 3. 使用 fgetc() 逐个字符读取
    printf("\n--- Using fgetc() to read text data char by char ---\n");
    printf("Reading the text line: ");
    while ((ch = fgetc(fp)) != EOF && ch != '\n') {
        printf("%c", ch);
    }
    printf("\n");

    // 将文件指针移回头，以便 fgets() 再次读取
    fseek(fp, sizeof(record) * 2, SEEK_SET);

    // 4. 使用 fgets() 读取整行
    printf("\n--- Using fgets() to read text line ---\n");
    if (fgets(line_buffer, sizeof(line_buffer), fp) != NULL) {
        printf("Line read by fgets(): %s", line_buffer);
    } else {
        printf("fgets failed.\n");
    }

    // 关闭文件
    fclose(fp);
    return 0;
}
```

### 运行与分析

1. 程序首先创建一个 data.bin 文件，并使用 `fwrite()` 写入两个 record 结构体和一段文本
2. `fread(rec_in, sizeof(record), 2, fp)` 会一次性读取文件中原来 2 个 record 大小的数据。这比单独读取每个字段高效得多
3. `fgetc(fp)` 在 fread 之后开始逐字节读取剩余数据。适合处理未知长度的数据
4. 为了演示 `fgets()`，我们使用 `fseek()` 将文件指针移到文本数据开头。`fgets(line_buffer, sizeof(line_buffer), fp)` 会读取一整行文本，并将其安全地存入 line_buffer，遇换行或达到缓冲区上限

## fgetc() - 读取单个字符

`fgetc()` 是最基本的字符读取函数，它从文件中读取**一个字符**。

### 函数原型

```c
#include <stdio.h>

int fgetc(FILE *stream);
```

### 参数说明

- **stream**：一个指向已打开文件的 `FILE*` 指针

### 返回值

- 成功：返回读取的字符（作为 `unsigned char` 转换为 `int`）
- 到达文件末尾或错误：返回 `EOF`

### fgetc 的特点

- `fgetc()` 每次只读取一个字符，通常用于逐字符处理文件内容
- 它在解析文件内容时非常灵活，适合读取长度不固定的数据
- 返回类型是 `int` 而非 `char`，以便能够区分 `EOF`（通常为 -1）和有效字符

### 示例：逐字符读取文件

```c
#include <stdio.h>

int main() {
    FILE *fp = fopen("example.txt", "r");
    if (fp == NULL) {
        perror("fopen failed");
        return 1;
    }

    int ch;
    printf("File content (char by char):\n");

    // 逐字符读取，直到 EOF
    while ((ch = fgetc(fp)) != EOF) {
        putchar(ch);  // 输出字符
    }

    fclose(fp);
    return 0;
}
```

## fgets() - 读取一行文本

`fgets()` 是一个**行读取函数**，它从文件中读取一行字符或指定数量的字符。

### 函数原型

```c
#include <stdio.h>

char *fgets(char *str, int n, FILE *stream);
```

### 参数说明

- **str**：一个字符数组或字符指针，用于存储读取的字符串
- **n**：要读取的最大字符数（包括 `'\0'`）。`fgets()` 最多读取 n-1 个字符，并在末尾添加一个空字符 `'\0'`
- **stream**：一个指向已打开文件的 `FILE*` 指针

### 返回值

- 成功：返回 `str` 指针
- 到达文件末尾且未读取任何字符，或发生错误：返回 `NULL`

### fgets 的行为

- 读取直到遇到**换行符**（`'\n'`）、**EOF** 或读取了 **n-1** 个字符
- 如果读取到换行符，换行符**会被包含**在字符串中
- 始终在末尾添加 `'\0'`

### 示例：逐行读取文件

```c
#include <stdio.h>

int main() {
    FILE *fp = fopen("example.txt", "r");
    if (fp == NULL) {
        perror("fopen failed");
        return 1;
    }

    char line[256];
    int line_number = 1;

    printf("File content (line by line):\n");

    // 逐行读取
    while (fgets(line, sizeof(line), fp) != NULL) {
        printf("Line %d: %s", line_number++, line);
    }

    fclose(fp);
    return 0;
}
```

### fgets 与 gets 的区别

| 特性 | fgets | gets（已废弃） |
|------|-------|----------------|
| 缓冲区大小 | 必须指定 | 不指定 |
| 安全性 | 安全（防止溢出） | 不安全（可能溢出） |
| 换行符 | 保留在字符串中 | 不保留 |
| 推荐度 | 推荐使用 | **不要使用**（C11 已移除） |

## 三种读取函数对比

| 函数 | 读取单位 | 适用场景 | 特点 |
|------|---------|---------|------|
| `fread()` | 数据块 | 二进制文件、结构体数组 | 高效，适合批量读取 |
| `fgetc()` | 单个字符 | 逐字符处理、解析文本 | 灵活，适合字符级操作 |
| `fgets()` | 一行文本 | 读取文本文件、配置文件 | 安全，自动处理换行 |

---

## 关键要点

- **文件内部指针**：所有读写操作共享同一个位置指示器，读/写操作会移动指针位置
- **fread()**：二进制读取，适合读取结构化数据和大块数据
- **fgetc()**：逐字符读取，返回 `int` 类型以区分 EOF
- **fgets()**：逐行读取，安全且会保留换行符，**推荐用于文本文件**
- 始终检查返回值以判断是否成功读取或到达文件末尾
