# 创建删除目录

本节介绍 Linux 系统中用于创建和删除目录的函数：`mkdir()` 和 `rmdir()`。

---

## mkdir() - 创建目录

`mkdir()` 函数用于创建一个新目录。

### 函数原型

```c
#include <sys/stat.h>
#include <sys/types.h>

int mkdir(const char *pathname, mode_t mode);
```

### 参数说明

- **pathname**：要创建的目录路径名
- **mode**：目录的[[3.5 权限控制函数#权限表示方式|权限模式]]（八进制数），实际权限会受到 `umask` 的影响

### 返回值

- 成功：返回 0
- 失败：返回 -1，并设置 errno

### 常见错误码

| errno | 说明 |
|-------|------|
| `EEXIST` | 目录已存在 |
| `ENOENT` | 路径中的某个目录不存在 |
| `EACCES` | 权限不足 |
| `ENOSPC` | 磁盘空间不足 |

---

## rmdir() - 删除空目录

`rmdir()` 函数用于删除一个**空目录**。这是 [[1.2 cp、rm、chmod、vim、gcc#rm 命令|rm 命令]]删除目录功能的编程接口。

### 函数原型

```c
#include <unistd.h>

int rmdir(const char *pathname);
```

### 参数说明

- **pathname**：要删除的目录路径名

### 返回值

- 成功：返回 0
- 失败：返回 -1，并设置 errno

### 注意事项

- `rmdir()` 只能删除**空目录**
- 如果目录非空，需要先递归删除其中的文件和子目录
- 要删除非空目录，可以使用 `nftw()` 函数遍历目录树后删除

### 常见错误码

| errno | 说明 |
|-------|------|
| `ENOTEMPTY` | 目录非空 |
| `ENOENT` | 目录不存在 |
| `EACCES` | 权限不足 |
| `EBUSY` | 目录正在被使用 |

---

## 示例代码

```c
#include <stdio.h>
#include <stdlib.h>
#include <sys/stat.h>
#include <sys/types.h>
#include <unistd.h>
#include <errno.h>
#include <string.h>

int main() {
    const char *dir_name = "test_dir";
    const char *file_name = "test_dir/file.txt";

    // 1. 使用 mkdir() 创建目录
    if (mkdir(dir_name, 0755) == -1) {
        if (errno == EEXIST) {
            printf("Directory '%s' already exists.\n", dir_name);
        } else {
            perror("mkdir failed");
            return 1;
        }
    } else {
        printf("Directory '%s' created successfully.\n", dir_name);
    }

    // 2. 尝试用 rmdir() 删除非空目录（会失败）
    // 首先在目录中创建一个文件
    FILE *fp = fopen(file_name, "w");
    if (fp != NULL) {
        fprintf(fp, "This is a test file.\n");
        fclose(fp);
        printf("File '%s' created.\n", file_name);
    }

    // 尝试删除非空目录
    printf("Attempting to delete non-empty directory '%s' with rmdir()...\n", dir_name);
    if (rmdir(dir_name) == -1) {
        printf("rmdir failed as expected: %s\n", strerror(errno));
    }

    // 3. 删除目录中的文件
    printf("Attempting to delete file '%s' with remove()...\n", file_name);
    if (remove(file_name) == -1) {
        perror("remove failed");
    } else {
        printf("File '%s' deleted successfully.\n", file_name);
    }

    // 4. 再次使用 rmdir() 删除空目录（会成功）
    printf("Attempting to delete empty directory '%s' with rmdir()...\n", dir_name);
    if (rmdir(dir_name) == -1) {
        perror("rmdir failed");
        return 1;
    }
    printf("Directory '%s' deleted successfully.\n", dir_name);

    return 0;
}
```

### 运行结果

```
Directory 'test_dir' created successfully.
File 'test_dir/file.txt' created.
Attempting to delete non-empty directory 'test_dir' with rmdir()...
rmdir failed as expected: Directory not empty
Attempting to delete file 'test_dir/file.txt' with remove()...
File 'test_dir/file.txt' deleted successfully.
Attempting to delete empty directory 'test_dir' with rmdir()...
Directory 'test_dir' deleted successfully.
```

### 分步解读

1. **mkdir("test_dir", 0755)**：创建一个名为 `test_dir` 的目录，权限为 `rwxr-xr-x`
2. 在目录中创建一个测试文件，使目录变为非空
3. **rmdir()** 尝试删除非空目录会失败，errno 被设置为 `ENOTEMPTY`
4. 使用 **remove()** 删除目录中的文件
5. 再次调用 **rmdir()** 删除空目录，这次成功

---

## 关键要点

- **mkdir()**：创建目录，权限受 `umask` 影响
- **rmdir()**：只能删除空目录
- 删除非空目录需要先递归删除其内容
- 始终检查返回值并处理错误
- 权限模式使用八进制数（如 `0755`）
