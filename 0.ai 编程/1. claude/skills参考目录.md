name: remote-ctrl-note

description: |

  远控系统项目专用笔记管理 Skill。当用户需要以下操作时使用此 Skill：

  - 为远控项目写笔记、更新笔记

  - 同步项目代码变化到笔记（通过 git diff 精准定位新增代码）

  - 分析远控系统代码并生成文档

  特性：
  - **Git 驱动**：自动读取 git log/diff，精准定位代码变更，节省 token
  - **与 note-creator 协作**：复用代码讲解规范和笔记模板
  - **关键代码详解**：技术栈、设计思路、参数说明缺一不可

  触发词：远控笔记、远控系统笔记、更新远控笔记、同步远控、远控项目、远控系统
