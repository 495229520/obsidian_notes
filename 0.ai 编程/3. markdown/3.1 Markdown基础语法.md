# Markdown 基础语法

Markdown 是一种轻量级标记语言，由 John Gruber 于 2004 年创建。它使用简单的纯文本格式编写文档，可以转换为 HTML 等多种格式。本笔记介绍 Markdown 的基础语法。

---

## 1. 标题 (Headings)

Markdown 使用 `#` 符号来创建标题，`#` 的数量代表标题级别（1-6 级）。

### 语法

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

### 效果

| 语法 | HTML 对应 | 用途说明 |
|------|-----------|----------|
| `# 标题` | `<h1>` | 文章主标题，一篇文档通常只有一个 |
| `## 标题` | `<h2>` | 主要章节标题 |
| `### 标题` | `<h3>` | 子章节标题 |
| `#### 标题` | `<h4>` | 小节标题 |

> **最佳实践**：标题层级应该递进使用，不要跳级（如直接从 `#` 跳到 `###`）。

## 2. 段落与换行 (Paragraphs & Line Breaks)

### 段落

两个段落之间需要空一行：

```markdown
这是第一段内容。

这是第二段内容。
```

### 换行

在行末添加两个或以上空格，然后回车，即可实现软换行：

```markdown
这是第一行
这是第二行（同一段落内换行）
```

也可以使用 `<br>` 标签强制换行。

## 3. 文本样式 (Text Styling)

### 基本样式

| 样式 | 语法 | 示例 | 渲染效果 |
|------|------|------|----------|
| **粗体** | `**文本**` 或 `__文本__` | `**重要**` | **重要** |
| *斜体* | `*文本*` 或 `_文本_` | `*强调*` | *强调* |
| ***粗斜体*** | `***文本***` | `***特别强调***` | ***特别强调*** |
| ~~删除线~~ | `~~文本~~` | `~~已删除~~` | ~~已删除~~ |
| `行内代码` | `` `代码` `` | `` `printf()` `` | `printf()` |

### 组合使用示例

```markdown
这是 **粗体**，这是 *斜体*，这是 ***粗斜体***。

你可以用 `inline code` 表示代码，用 ~~删除线~~ 表示废弃内容。
```

## 4. 列表 (Lists)

### 4.1 无序列表

使用 `-`、`*` 或 `+` 创建无序列表：

```markdown
- 项目一
- 项目二
  - 子项目 2.1
  - 子项目 2.2
- 项目三
```

效果：
- 项目一
- 项目二
  - 子项目 2.1
  - 子项目 2.2
- 项目三

### 4.2 有序列表

使用数字加 `.` 创建有序列表：

```markdown
1. 第一步
2. 第二步
3. 第三步
   1. 子步骤 3.1
   2. 子步骤 3.2
```

效果：
1. 第一步
2. 第二步
3. 第三步
   1. 子步骤 3.1
   2. 子步骤 3.2

### 4.3 任务列表

```markdown
- [x] 已完成的任务
- [ ] 未完成的任务
- [ ] 另一个待办事项
```

效果：
- [x] 已完成的任务
- [ ] 未完成的任务
- [ ] 另一个待办事项

## 5. 链接与图片 (Links & Images)

### 5.1 链接

#### 行内链接

```markdown
[显示文本](URL "可选标题")

示例：
[Google](https://www.google.com)
[带标题的链接](https://www.google.com "Google 首页")
```

#### 引用链接

```markdown
[显示文本][引用标识]

[引用标识]: URL "可选标题"

示例：
这是一个 [示例链接][example]。

[example]: https://www.example.com "示例网站"
```

### 5.2 图片

语法与链接类似，只需在前面加 `!`：

```markdown
![替代文本](图片URL "可选标题")

示例：
![Logo](./images/logo.png "公司 Logo")
```

### 5.3 图片链接（点击图片跳转）

```markdown
[![替代文本](图片URL)](链接URL)
```

## 6. 引用 (Blockquotes)

使用 `>` 创建引用块：

```markdown
> 这是一段引用文本。
> 可以跨多行。

> 这是另一段引用。
>
> > 这是嵌套引用。
```

效果：

> 这是一段引用文本。
> 可以跨多行。

> 这是另一段引用。
>
> > 这是嵌套引用。

---

## 7. 分隔线 (Horizontal Rules)

使用三个或以上的 `-`、`*` 或 `_` 创建分隔线：

```markdown
---
***
___
```

## 8. 转义字符 (Escaping)

如果需要显示 Markdown 语法中使用的特殊字符，使用反斜杠 `\` 转义：

```markdown
\*这不是斜体\*
\# 这不是标题
\[这不是链接\]
```

### 需要转义的常用字符

| 字符 | 名称 | 转义写法 |
|------|------|----------|
| `\` | 反斜杠 | `\\` |
| `` ` `` | 反引号 | `` \` `` |
| `*` | 星号 | `\*` |
| `_` | 下划线 | `\_` |
| `#` | 井号 | `\#` |
| `+` | 加号 | `\+` |
| `-` | 减号 | `\-` |
| `.` | 点 | `\.` |
| `!` | 感叹号 | `\!` |
| `[]` | 方括号 | `\[\]` |
| `()` | 圆括号 | `\(\)` |

---

## 总结

| 语法类型 | 常用符号 | 备注 |
|----------|----------|------|
| 标题 | `#` | 1-6 级，`#` 数量对应级别 |
| 粗体 | `**文本**` | 也可用 `__文本__` |
| 斜体 | `*文本*` | 也可用 `_文本_` |
| 列表 | `-` 或 `1.` | 无序/有序列表 |
| 链接 | `[文本](URL)` | 可加标题 |
| 图片 | `![alt](URL)` | 在链接前加 `!` |
| 引用 | `>` | 可嵌套 |
| 代码 | `` `代码` `` | 行内代码 |
| 分隔线 | `---` | 三个以上 |
