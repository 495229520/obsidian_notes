# Windows 系统编程：文件操作与注册表

> **注意**：本笔记内容仅用于学习 Windows API 和系统编程知识。了解这些技术有助于安全研究、防护软件开发和系统管理。

---

## WIN32_FIND_DATA 结构

`WIN32_FIND_DATA` 是 Windows API 中用于存储文件或目录信息的结构体，通常与 `FindFirstFile()` 和 `FindNextFile()` 函数配合使用。

### 结构定义

```cpp
typedef struct _WIN32_FIND_DATAW {
    DWORD    dwFileAttributes;           // 文件属性
    FILETIME ftCreationTime;             // 创建时间
    FILETIME ftLastAccessTime;           // 最后访问时间
    FILETIME ftLastWriteTime;            // 最后修改时间
    DWORD    nFileSizeHigh;              // 文件大小高32位
    DWORD    nFileSizeLow;               // 文件大小低32位
    DWORD    dwReserved0;                // 保留
    DWORD    dwReserved1;                // 保留
    WCHAR    cFileName[MAX_PATH];        // 文件名
    WCHAR    cAlternateFileName[14];     // 8.3 格式的短文件名
} WIN32_FIND_DATAW, *PWIN32_FIND_DATAW;
```

### 关键常量

| 常量 | 值 | 说明 |
|:---|:---|:---|
| `MAX_PATH` | 260 | Windows 路径的最大长度限制 |

## 文件查找示例

```cpp
#include <Windows.h>
#include <stdio.h>

void FindFiles(const char* path)
{
    WIN32_FIND_DATA findData;
    HANDLE hFind;

    // 查找指定路径下的所有文件
    hFind = FindFirstFile(path, &findData);
    if (hFind == INVALID_HANDLE_VALUE)
    {
        printf("FindFirstFile failed: %d\n", GetLastError());
        return;
    }

    do {
        printf("Found: %s\n", findData.cFileName);
    } while (FindNextFile(hFind, &findData));

    FindClose(hFind);
}
```

## 注册表操作

Windows 注册表是一个层次化的数据库，存储系统和应用程序的配置信息。

### 注册表根键

| 根键 | 缩写 | 说明 |
|:---|:---|:---|
| `HKEY_LOCAL_MACHINE` | HKLM | 本地计算机的配置信息 |
| `HKEY_CURRENT_USER` | HKCU | 当前用户的配置信息 |
| `HKEY_CLASSES_ROOT` | HKCR | 文件关联和 COM 对象信息 |
| `HKEY_USERS` | HKU | 所有用户的配置信息 |

### 常用注册表函数

| 函数 | 说明 |
|:---|:---|
| `RegOpenKeyEx()` | 打开注册表键 |
| `RegCreateKeyEx()` | 创建注册表键 |
| `RegSetValueEx()` | 设置注册表值 |
| `RegQueryValueEx()` | 查询注册表值 |
| `RegDeleteKey()` | 删除注册表键 |
| `RegDeleteValue()` | 删除注册表值 |
| `RegCloseKey()` | 关闭注册表键句柄 |

### 注册表值类型

| 类型常量 | 说明 |
|:---|:---|
| `REG_SZ` | 以 null 结尾的字符串 |
| `REG_DWORD` | 32 位整数 |
| `REG_BINARY` | 二进制数据 |
| `REG_MULTI_SZ` | 多字符串（以 null 分隔） |
| `REG_EXPAND_SZ` | 可展开的环境变量字符串 |

## 注册表操作示例

### 写入注册表

```cpp
#include <Windows.h>
#include <stdio.h>

BOOL WriteToRegistry(LPCSTR keyPath, LPCSTR valueName, LPCSTR data)
{
    HKEY hKey;
    LONG ret;

    // 打开或创建注册表键
    ret = RegOpenKeyEx(HKEY_LOCAL_MACHINE, keyPath, 0, KEY_WRITE, &hKey);
    if (ret != ERROR_SUCCESS)
    {
        // 如果键不存在，尝试创建
        ret = RegCreateKeyEx(HKEY_LOCAL_MACHINE, keyPath, 0, NULL,
                             REG_OPTION_NON_VOLATILE, KEY_WRITE, NULL, &hKey, NULL);
        if (ret != ERROR_SUCCESS)
        {
            printf("Failed to open/create key: %ld\n", ret);
            return FALSE;
        }
    }

    // 设置值
    ret = RegSetValueEx(hKey, valueName, 0, REG_SZ,
                        (const BYTE*)data, strlen(data) + 1);
    if (ret != ERROR_SUCCESS)
    {
        printf("Failed to set value: %ld\n", ret);
        RegCloseKey(hKey);
        return FALSE;
    }

    RegCloseKey(hKey);
    return TRUE;
}
```

### 读取注册表

```cpp
BOOL ReadFromRegistry(LPCSTR keyPath, LPCSTR valueName, LPSTR buffer, DWORD bufSize)
{
    HKEY hKey;
    LONG ret;
    DWORD type = REG_SZ;

    ret = RegOpenKeyEx(HKEY_LOCAL_MACHINE, keyPath, 0, KEY_READ, &hKey);
    if (ret != ERROR_SUCCESS)
    {
        printf("Failed to open key: %ld\n", ret);
        return FALSE;
    }

    ret = RegQueryValueEx(hKey, valueName, NULL, &type, (LPBYTE)buffer, &bufSize);
    if (ret != ERROR_SUCCESS)
    {
        printf("Failed to query value: %ld\n", ret);
        RegCloseKey(hKey);
        return FALSE;
    }

    RegCloseKey(hKey);
    return TRUE;
}
```

## 获取系统路径

```cpp
#include <Windows.h>
#include <stdio.h>

void GetPaths()
{
    char sysPath[MAX_PATH];
    char currentPath[MAX_PATH];

    // 获取系统目录 (如 C:\Windows\System32)
    GetSystemDirectory(sysPath, MAX_PATH);
    printf("System Directory: %s\n", sysPath);

    // 获取当前模块（exe）的完整路径
    GetModuleFileName(NULL, currentPath, MAX_PATH);
    printf("Current Module: %s\n", currentPath);
}
```

## 窗口操作

### 隐藏控制台窗口

```cpp
void HideConsoleWindow()
{
    // 获取当前控制台窗口句柄
    HWND hwnd = GetForegroundWindow();
    // 或使用: HWND hwnd = GetConsoleWindow();

    // 隐藏窗口
    ShowWindow(hwnd, SW_HIDE);
}
```

### ShowWindow 参数

| 参数 | 说明 |
|:---|:---|
| `SW_HIDE` | 隐藏窗口 |
| `SW_SHOW` | 显示窗口 |
| `SW_MINIMIZE` | 最小化窗口 |
| `SW_MAXIMIZE` | 最大化窗口 |
| `SW_RESTORE` | 恢复窗口 |

## 错误处理

Windows API 调用失败时，通常返回特定的错误值。可以使用 `GetLastError()` 获取详细错误码：

```cpp
DWORD err = GetLastError();
printf("Error code: %lu\n", err);

// 获取错误描述
char errMsg[256];
FormatMessage(FORMAT_MESSAGE_FROM_SYSTEM, NULL, err, 0, errMsg, 256, NULL);
printf("Error message: %s\n", errMsg);
```

## 关键要点

1. `WIN32_FIND_DATA` 配合 `FindFirstFile()`/`FindNextFile()` 用于遍历文件
2. 注册表操作需要正确的权限，修改 `HKEY_LOCAL_MACHINE` 通常需要管理员权限
3. `MAX_PATH` (260) 是 Windows 路径长度的传统限制
4. Windows API 错误处理使用 `GetLastError()` 获取错误码
5. 窗口操作使用句柄（HWND）来标识窗口
