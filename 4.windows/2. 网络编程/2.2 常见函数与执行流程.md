# Winsock 常见函数与执行流程

## 头文件与库

```cpp
#include <WinSock2.h>           // Winsock2 主头文件
#pragma comment(lib, "ws2_32.lib")  // 链接 Winsock 库
```

---

## 类型定义

| 类型 | 说明 |
|:---|:---|
| `SOCKET` | 套接字句柄类型 |
| `sockaddr` | 通用套接字地址结构 |
| `sockaddr_in` | IPv4 地址结构 |
| `WSADATA` | Winsock 初始化数据结构 |

---

## 地址族与套接字类型

### 地址族 (Address Family)

| 常量 | 说明 |
|:---|:---|
| `AF_INET` | IPv4 互联网协议 |
| `AF_INET6` | IPv6 互联网协议 |

### 套接字类型 (Socket Type)

| 常量 | 说明 |
|:---|:---|
| `SOCK_STREAM` | 面向连接的流式套接字（TCP） |
| `SOCK_DGRAM` | 面向消息的数据报套接字（UDP） |

---

## 地址结构体

### sockaddr_in 结构

```cpp
struct sockaddr_in {
    short          sin_family;    // 地址族（如 AF_INET）
    u_short        sin_port;      // 端口号（网络字节序）
    struct in_addr sin_addr;      // IP 地址
    char           sin_zero[8];   // 填充字节，保持与 sockaddr 大小一致
};
```

## TCP 服务端核心函数

### 1. 创建套接字 (Create Socket)

```cpp
SOCKET hServSock = socket(AF_INET, SOCK_STREAM, 0);
```

| 参数 | 说明 |
|:---|:---|
| `af` | 地址族，`AF_INET` 表示 IPv4 |
| `type` | 套接字类型，`SOCK_STREAM` 表示 TCP，`SOCK_DGRAM` 表示 UDP |
| `protocol` | 通常为 0，让系统自动选择合适的协议；`IPPROTO_TCP` 可明确指定 TCP，`IPPROTO_UDP` 指定 UDP |
| **返回值** | 成功返回套接字句柄；失败返回 `INVALID_SOCKET`，可用 `WSAGetLastError()` 获取错误码 |

### 2. 绑定套接字 (Bind Socket)

```cpp
int ret = bind(hServSock, (SOCKADDR*)&servAddr, sizeof(servAddr));
```

将套接字绑定到本地网络接口（绑定服务器将监听的 IP 地址和端口）。

| 参数 | 说明 |
|:---|:---|
| `socket` | 要绑定的套接字句柄 |
| `addr` | 指向 `sockaddr` 结构的指针（通常使用 `sockaddr_in` 并强制转换） |
| `addrlen` | 地址结构的大小 |
| **返回值** | 成功返回 0；失败返回 `SOCKET_ERROR` |

**地址配置说明：**

| 设置 | 说明 |
|:---|:---|
| `sin_addr.S_un.S_addr = htonl(INADDR_ANY)` | 绑定到本机所有可用的网络接口（监听所有 IP） |
| `sin_addr.S_un.S_addr = inet_addr("192.168.1.1")` | 绑定到指定 IP 地址（仅监听该 IP） |

### 3. 监听套接字 (Listen Socket)

```cpp
int ret = listen(hServSock, 5);
```

将套接字设置为监听模式，等待客户端的连接请求。

| 参数 | 说明 |
|:---|:---|
| `socket` | 已绑定的服务器套接字 |
| `backlog` | 等待连接队列的最大长度（允许排队的最大连接数），使用 `SOMAXCONN` 可设置为系统允许的最大值 |
| **返回值** | 成功返回 0；失败返回 `SOCKET_ERROR` |

### 4. 接受连接 (Accept Socket)

```cpp
SOCKET hClntSock = accept(hServSock, (SOCKADDR*)&clntAddr, &clntAddrLen);
```

**阻塞调用**：从监听队列中取出一个待处理的客户端连接请求，并创建一个新的套接字来与该客户端进行通信。

| 参数 | 说明 |
|:---|:---|
| `socket` | 处于监听状态的服务器套接字 |
| `addr` | 输出参数，用于接收连接的客户端的地址信息 |
| `addrlen` | 输入输出参数，传入地址结构的大小，返回实际填充的大小 |
| **返回值** | 成功返回一个新的套接字（用于与该客户端通信）；失败返回 `INVALID_SOCKET` |

## TCP 数据收发函数

### send - 发送数据

```cpp
int ret = send(hSocket, buf, len, flags);
```

| 参数 | 说明 |
|:---|:---|
| `socket` | 已连接的套接字 |
| `buf` | 指向要发送数据的缓冲区 |
| `len` | 要发送的数据长度 |
| `flags` | 通常为 0；可使用 `MSG_OOB` 发送带外数据等 |
| **返回值** | 成功返回实际发送的字节数；失败返回 `SOCKET_ERROR` |

### recv - 接收数据

```cpp
int ret = recv(hSocket, buf, len, flags);
```

| 参数 | 说明 |
|:---|:---|
| `socket` | 已连接的套接字 |
| `buf` | 指向用于存放接收数据的缓冲区 |
| `len` | 缓冲区大小 |
| `flags` | 通常为 0 |
| **返回值** | 成功返回实际接收的字节数；连接正常关闭返回 0；失败返回 `SOCKET_ERROR` |

## TCP 客户端核心函数

### 1. 创建套接字 (Create Socket)

```cpp
SOCKET hSocket = socket(AF_INET, SOCK_STREAM, 0);
```

与服务端相同，创建一个流式套接字。

### 2. 连接服务器 (Connect)

```cpp
int ret = connect(hSocket, (SOCKADDR*)&servAddr, sizeof(servAddr));
```

**阻塞调用**：向服务器发起连接请求，完成 [[7.5 TCP协议基础#TCP 三次握手（建立连接）|TCP 三次握手]] 过程。

| 参数 | 说明 |
|:---|:---|
| `socket` | 客户端套接字 |
| `name` | 指向服务器地址的 `sockaddr` 结构指针，通常使用 `inet_addr()` 将 IP 字符串转换为 `in_addr` |
| `namelen` | 地址结构的大小 |
| **返回值** | 成功返回 0；失败返回 `SOCKET_ERROR` |

## UDP 核心函数

UDP 是无连接的协议，不需要 `listen()` 和 `accept()`，数据通过 `sendto()` 和 `recvfrom()` 直接收发。

### sendto - 发送数据报

```cpp
int ret = sendto(hSocket, buf, len, flags, (SOCKADDR*)&destAddr, addrLen);
```

| 参数 | 说明 |
|:---|:---|
| `socket` | UDP 套接字 |
| `buf` | 要发送的数据缓冲区 |
| `len` | 数据长度 |
| `flags` | 通常为 0 |
| `to` | 目标地址结构指针 |
| `tolen` | 地址结构大小 |
| **返回值** | 成功返回发送的字节数；失败返回 `SOCKET_ERROR` |

### recvfrom - 接收数据报

```cpp
int ret = recvfrom(hSocket, buf, len, flags, (SOCKADDR*)&srcAddr, &addrLen);
```

| 参数 | 说明 |
|:---|:---|
| `socket` | UDP 套接字 |
| `buf` | 接收数据的缓冲区 |
| `len` | 缓冲区大小 |
| `flags` | 通常为 0 |
| `from` | 输出参数，用于存放发送方的地址信息 |
| `fromlen` | 输入输出参数，地址结构大小 |
| **返回值** | 成功返回接收的字节数；失败返回 `SOCKET_ERROR` |

## Winsock 初始化与清理

### WSAStartup - 初始化 Winsock

```cpp
WSADATA wsaData;
int err = WSAStartup(MAKEWORD(2, 2), &wsaData);
if (err != 0) {
    // 初始化失败
    return err;
}
```

**必须在调用任何 Winsock 函数之前调用此函数。**

| 参数 | 说明 |
|:---|:---|
| `wVersionRequested` | 请求的 Winsock 版本，`MAKEWORD(2, 2)` 表示 2.2 版本 |
| `lpWSAData` | 指向 `WSADATA` 结构的指针，用于接收 Winsock 实现的详细信息 |
| **返回值** | 成功返回 0；失败返回错误码 |

### WSACleanup - 清理 Winsock

```cpp
WSACleanup();
```

终止 Winsock 的使用，释放相关资源。应在程序退出前调用。

### closesocket - 关闭套接字

```cpp
closesocket(hSocket);
```

关闭套接字并释放相关资源。

---

## 关键要点

1. Windows 网络编程必须先调用 `WSAStartup()` 初始化，程序结束前调用 `WSACleanup()` 清理
2. TCP 服务端流程：`socket()` → `bind()` → `listen()` → `accept()` → `send()/recv()` → `closesocket()`
3. TCP 客户端流程：`socket()` → `connect()` → `send()/recv()` → `closesocket()`
4. UDP 不需要连接，直接使用 `sendto()` 和 `recvfrom()` 收发数据
5. 网络字节序使用大端序，需要使用 `htons()`、`htonl()`、`ntohs()`、`ntohl()` 进行转换
