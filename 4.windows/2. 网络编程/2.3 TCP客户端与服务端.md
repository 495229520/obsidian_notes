# TCP 客户端与服务端

## 通信流程概述

TCP 是面向连接的协议，通信前需要建立连接。下图展示了 TCP 服务端和客户端的完整通信流程：

```
         服务器端                              客户端
            │                                   │
      ┌─────┴─────┐                             │
      │ socket()  │ 创建套接字                   │
      └─────┬─────┘                             │
            │                                   │
      ┌─────┴─────┐                             │
      │  bind()   │ 绑定地址                     │
      └─────┬─────┘                             │
            │                                   │
      ┌─────┴─────┐                       ┌─────┴─────┐
      │ listen()  │ 进入监听               │ socket()  │ 创建套接字
      └─────┬─────┘                       └─────┬─────┘
            │                                   │
      ┌─────┴─────┐ 阻塞直到收到        ┌───────┴───────┐
      │ accept()  │←─────建立连接───────│   connect()   │
      └─────┬─────┘ 客户端连接请求       └───────┬───────┘
            │       TCP 三次握手                │
            │                                   │
      ┌─────┴─────┐                       ┌─────┴─────┐
      │  read()   │←───────数据请求───────│  write()  │
      └─────┬─────┘                       └─────┬─────┘
            │        处理请求                   │
      ┌─────┴─────┐                       ┌─────┴─────┐
      │  write()  │────────数据响应──────→│  read()   │
      └─────┬─────┘                       └─────┬─────┘
            │                                   │
      ┌─────┴─────┐                       ┌─────┴─────┐
      │  close()  │←──────关闭连接───────→│  close()  │
      └───────────┘                       └───────────┘
```

## 服务端代码

```cpp
#include <stdio.h>
#include <stdlib.h>
#include <WinSock2.h>
#include <iostream>
#pragma comment(lib, "ws2_32.lib")

using namespace std;

int main()
{
    // 1. 初始化 Winsock
    WORD wVersion;
    WSADATA wsaData;
    int err;

    wVersion = MAKEWORD(2, 2);  // 请求 Winsock 2.2 版本
    err = WSAStartup(wVersion, &wsaData);
    if (err != 0)
    {
        return err;
    }

    // 验证版本
    if (LOBYTE(wsaData.wVersion) != 2 || HIBYTE(wsaData.wVersion) != 2)
    {
        WSACleanup();
        return -1;
    }

    // 2. 创建套接字
    SOCKET sockSrv = socket(AF_INET, SOCK_STREAM, 0);
    if (INVALID_SOCKET == sockSrv)
    {
        printf("socket error = %d\n", GetLastError());
        return -1;
    }

    // 3. 绑定地址和端口
    SOCKADDR_IN addrSrv;
    addrSrv.sin_addr.S_un.S_addr = htonl(INADDR_ANY);  // 监听所有网卡
    addrSrv.sin_family = AF_INET;
    addrSrv.sin_port = htons(6000);  // 监听端口 6000

    if (SOCKET_ERROR == bind(sockSrv, (SOCKADDR*)&addrSrv, sizeof(SOCKADDR)))
    {
        printf("bind error = %d\n", GetLastError());
        return -1;
    }

    // 4. 开始监听
    if (SOCKET_ERROR == listen(sockSrv, 5))
    {
        printf("listen error = %d\n", GetLastError());
        return -1;
    }
    printf("Server is listening on port 6000...\n");

    // 5. 等待客户端连接
    SOCKADDR_IN addrCli;
    int len = sizeof(SOCKADDR);

    while (1)
    {
        // accept() 会阻塞，直到有客户端连接
        SOCKET sockConn = accept(sockSrv, (SOCKADDR*)&addrCli, &len);
        if (INVALID_SOCKET == sockConn)
        {
            printf("accept error = %d\n", GetLastError());
            continue;
        }

        // 6. 发送欢迎消息
        char sendBuf[] = "Welcome to Beijing!";
        send(sockConn, sendBuf, strlen(sendBuf) + 1, 0);

        // 7. 接收客户端消息
        char recvBuf[100] = { 0 };
        recv(sockConn, recvBuf, sizeof(recvBuf), 0);
        printf("Received from client: %s\n", recvBuf);

        // 8. 关闭与该客户端的连接
        closesocket(sockConn);
    }

    // 9. 关闭服务器套接字
    closesocket(sockSrv);
    WSACleanup();

    return 0;
}
```

## 客户端代码

```cpp
#include <stdio.h>
#include <stdlib.h>
#include <WinSock2.h>
#include <iostream>
#pragma comment(lib, "ws2_32.lib")

using namespace std;

int main()
{
    // 1. 初始化 Winsock
    WORD wVersion;
    WSADATA wsaData;
    int err;

    wVersion = MAKEWORD(2, 2);
    err = WSAStartup(wVersion, &wsaData);
    if (err != 0)
    {
        return err;
    }

    // 验证版本
    if (LOBYTE(wsaData.wVersion) != 2 || HIBYTE(wsaData.wVersion) != 2)
    {
        WSACleanup();
        return -1;
    }

    // 2. 创建套接字
    SOCKET sockCli = socket(AF_INET, SOCK_STREAM, 0);
    if (INVALID_SOCKET == sockCli)
    {
        printf("socket error = %d\n", GetLastError());
        return -1;
    }

    // 3. 设置服务器地址
    SOCKADDR_IN addrSrv;
    addrSrv.sin_addr.S_un.S_addr = inet_addr("127.0.0.1");  // 服务器 IP
    addrSrv.sin_family = AF_INET;
    addrSrv.sin_port = htons(6000);  // 服务器端口

    // 4. 连接服务器
    if (SOCKET_ERROR == connect(sockCli, (SOCKADDR*)&addrSrv, sizeof(SOCKADDR)))
    {
        printf("connect error = %d\n", GetLastError());
        return -1;
    }
    printf("Connected to server!\n");

    // 5. 接收服务器消息
    char recvBuf[100] = { 0 };
    recv(sockCli, recvBuf, sizeof(recvBuf), 0);
    printf("Server says: %s\n", recvBuf);

    // 6. 发送消息给服务器
    char sendBuf[] = "Hello from client!";
    send(sockCli, sendBuf, strlen(sendBuf) + 1, 0);

    // 7. 关闭套接字
    closesocket(sockCli);
    WSACleanup();

    system("pause");
    return 0;
}
```

## 收发数据的健壮实现

由于 TCP 是流式协议，`send()` 和 `recv()` 可能不会一次性发送/接收所有数据。以下是循环收发的健壮实现：

### MySocketRecv - 循环接收

```cpp
int MySocketRecv(int sock, char* buf, int dataSize)
{
    // 循环接收
    int numsRecvSoFar = 0;           // 目前收到的数据
    int numsRemainingToRecv = dataSize;  // 剩余要收的数据

    printf("enter MySocketRecv\n");

    while (1)
    {
        // bytesRead = 接收数据的长度
        int bytesRead = recv(sock, &buf[numsRecvSoFar], numsRemainingToRecv, 0);
        printf("###bytesRead = %d, numsRecvSoFar = %d, numsRemainingToRecv = %d\n",
               bytesRead, numsRecvSoFar, numsRemainingToRecv);

        if (bytesRead == numsRemainingToRecv)
        {
            return 0;  // 一次性接收完，不需要继续循环
        }
        else if (bytesRead > 0)
        {
            // numsRecvSoFar + numsRemainingToRecv 是等于需要接收的数据长度的
            numsRecvSoFar += bytesRead;
            numsRemainingToRecv -= bytesRead;
            continue;
        }
        else if ((bytesRead < 0) && (errno == EAGAIN))
        {
            continue;  // 暂时无数据，继续等待
        }
        else
        {
            return -1;  // 发生错误
        }
    }
}
```

### MySocketSend - 循环发送

```cpp
int MySocketSend(int socketNum, unsigned char* data, unsigned dataSize)
{
    unsigned numBytesSentSoFar = 0;
    unsigned numBytesRemainingToSend = dataSize;

    while (1)
    {
        int bytesSend = send(socketNum, (char const*)(&data[numBytesSentSoFar]),
                             numBytesRemainingToSend, 0 /*flags*/);

        if (bytesSend == numBytesRemainingToSend)
        {
            return 0;  // 发送完成
        }
        else if (bytesSend > 0)
        {
            numBytesSentSoFar += bytesSend;
            numBytesRemainingToSend -= bytesSend;
            continue;
        }
        else if ((bytesSend < 0) && (errno == 11))  // EAGAIN
        {
            continue;  // 缓冲区满，稍后重试
        }
        else
        {
            return -1;  // 发生错误
        }
    }
}
```

## Visual Studio 项目配置

### 方法一：多启动项目配置（推荐）

1. 在 Visual Studio 中，右键点击解决方案 (Solution)，选择 **"添加" → "新建项目"**，分别创建 Server 项目和 Client 项目
2. 右键点击解决方案，选择 **"设置启动项目"**
3. 选择 **"多个启动项目"**，将 Server 和 Client 都设置为 **"启动"**
4. 按 F5 运行，两个项目会同时启动

### 方法二：分别运行

1. 分别创建 Server 和 Client 两个独立的项目
2. 先启动 Server 项目（Ctrl + F5 或 **"调试" → "开始执行(不调试)"**）
3. 再启动 Client 项目

### 方法三：使用 exe 文件

1. 先将 Server 和 Client 分别生成 exe 文件（**"生成" → "生成解决方案"**）
2. 找到 Debug 或 Release 文件夹下的 exe 文件
3. 先运行 Server.exe，再运行 Client.exe

## 常见问题

### 1. 链接错误：无法解析的外部符号

确保添加了 Winsock 库链接：

```cpp
#pragma comment(lib, "ws2_32.lib")
```

或在项目属性中添加：**"链接器" → "输入" → "附加依赖项"** 中添加 `ws2_32.lib`

### 2. 端口被占用

如果提示 `bind error`，可能是端口已被占用。可以：
- 更换端口号
- 使用 `netstat -an` 查看端口占用情况
- 等待一段时间让系统释放端口（TIME_WAIT 状态）

### 3. 防火墙阻止连接

确保 Windows 防火墙允许程序通过网络通信。

## 关键要点

1. **服务端流程**：`WSAStartup()` → `socket()` → `bind()` → `listen()` → `accept()` → `send()/recv()` → `closesocket()` → `WSACleanup()`
2. **客户端流程**：`WSAStartup()` → `socket()` → `connect()` → `send()/recv()` → `closesocket()` → `WSACleanup()`
3. TCP 是流式协议，需要循环收发以确保数据完整
4. `accept()` 和 `recv()` 是阻塞调用，会等待数据到来
5. 网络字节序使用 `htons()`、`htonl()` 转换
