# 多线程群聊服务器案例

## 概述

本案例展示了一个基于 Windows 多线程和 `select` IO复用的群聊服务器实现。服务器能够接受多个客户端连接，并将任意客户端发送的消息广播给所有其他在线客户端。

---

## 技术要点

| 技术 | 说明 |
|------|------|
| **多线程** | 使用 `_beginthreadex` 创建线程处理客户端消息 |
| **select IO复用** | 使用 `select` 函数监控多个套接字的可读状态 |
| **TCP套接字** | 基于 TCP 协议的可靠通信 |
| **[[3.4 互斥量\|互斥量]]** | 保护共享的客户端套接字数组，防止竞态条件 |

## 服务器端实现

### 全局变量与宏定义

```cpp
#define BUF_SIZE 100
#define MAX_CLNT 256

// 客户端套接字管理
unsigned numOfClnt = 0;           // 当前连接的客户端数量
SOCKET clntSocks[MAX_CLNT];       // 存储所有客户端套接字
HANDLE hMutex;                    // 互斥量，保护共享资源
```

### 主函数结构

```cpp
int main(int argc, char* argv[])
{
    WSADATA wsaData;
    SOCKET hServSock, hClntSock;
    SOCKADDR_IN servAdr, clntAdr;
    int clntAdrSz;
    HANDLE hThread;

    // 初始化 Winsock
    WSAStartup(MAKEWORD(2, 2), &wsaData);

    // 创建互斥量
    hMutex = CreateMutex(NULL, FALSE, NULL);

    // 创建服务器套接字
    hServSock = socket(PF_INET, SOCK_STREAM, 0);

    // 绑定地址
    memset(&servAdr, 0, sizeof(servAdr));
    servAdr.sin_family = AF_INET;
    servAdr.sin_addr.s_addr = htonl(INADDR_ANY);
    servAdr.sin_port = htons(atoi(argv[1]));
    bind(hServSock, (SOCKADDR*)&servAdr, sizeof(servAdr));

    // 开始监听
    listen(hServSock, 5);

    while (1)
    {
        clntAdrSz = sizeof(clntAdr);
        hClntSock = accept(hServSock, (SOCKADDR*)&clntAdr, &clntAdrSz);

        // 添加新客户端到数组
        WaitForSingleObject(hMutex, INFINITE);
        clntSocks[numOfClnt++] = hClntSock;
        ReleaseMutex(hMutex);

        // 为新客户端创建处理线程
        hThread = (HANDLE)_beginthreadex(NULL, 0, HandleClnt,
                                          (void*)&hClntSock, 0, NULL);

        printf("Connected client IP: %s \n", inet_ntoa(clntAdr.sin_addr));
    }

    // 清理资源
    closesocket(hServSock);
    WSACleanup();
    return 0;
}
```

### 客户端处理线程

```cpp
unsigned WINAPI HandleClnt(void* arg)
{
    SOCKET hClntSock = *((SOCKET*)arg);
    int strLen = 0;
    char msg[BUF_SIZE];

    // 持续接收并广播消息
    while ((strLen = recv(hClntSock, msg, sizeof(msg), 0)) != 0)
        SendMsg(msg, strLen);  // 广播给所有客户端

    // 客户端断开，从数组中移除
    WaitForSingleObject(hMutex, INFINITE);
    for (int i = 0; i < numOfClnt; i++)
    {
        if (hClntSock == clntSocks[i])
        {
            // 将最后一个客户端移到当前位置
            while (i++ < numOfClnt - 1)
                clntSocks[i] = clntSocks[i + 1];
            break;
        }
    }
    numOfClnt--;
    ReleaseMutex(hMutex);

    closesocket(hClntSock);
    return 0;
}
```

### 消息广播函数

```cpp
void SendMsg(char* msg, int len)
{
    int i;
    WaitForSingleObject(hMutex, INFINITE);
    for (i = 0; i < numOfClnt; i++)
        send(clntSocks[i], msg, len, 0);
    ReleaseMutex(hMutex);
}
```

## 客户端实现

### 主函数结构

```cpp
int main(int argc, char* argv[])
{
    WSADATA wsaData;
    SOCKET hSock;
    SOCKADDR_IN servAdr;
    HANDLE hSndThread, hRcvThread;

    WSAStartup(MAKEWORD(2, 2), &wsaData);

    // 创建套接字并连接服务器
    hSock = socket(PF_INET, SOCK_STREAM, 0);
    memset(&servAdr, 0, sizeof(servAdr));
    servAdr.sin_family = AF_INET;
    servAdr.sin_addr.s_addr = inet_addr(argv[1]);
    servAdr.sin_port = htons(atoi(argv[2]));
    connect(hSock, (SOCKADDR*)&servAdr, sizeof(servAdr));

    // 创建发送和接收线程
    hSndThread = (HANDLE)_beginthreadex(NULL, 0, SendMsg, (void*)&hSock, 0, NULL);
    hRcvThread = (HANDLE)_beginthreadex(NULL, 0, RecvMsg, (void*)&hSock, 0, NULL);

    // 等待线程结束
    WaitForSingleObject(hSndThread, INFINITE);
    WaitForSingleObject(hRcvThread, INFINITE);

    closesocket(hSock);
    WSACleanup();
    return 0;
}
```

### 发送线程

```cpp
unsigned WINAPI SendMsg(void* arg)
{
    SOCKET hSock = *((SOCKET*)arg);
    char nameMsg[NAME_SIZE + BUF_SIZE];

    while (1)
    {
        fgets(msg, BUF_SIZE, stdin);
        if (!strcmp(msg, "q\n") || !strcmp(msg, "Q\n"))
        {
            closesocket(hSock);
            exit(0);
        }
        sprintf(nameMsg, "%s %s", name, msg);
        send(hSock, nameMsg, strlen(nameMsg), 0);
    }
    return 0;
}
```

### 接收线程

```cpp
unsigned WINAPI RecvMsg(void* arg)
{
    SOCKET hSock = *((SOCKET*)arg);
    char nameMsg[NAME_SIZE + BUF_SIZE];
    int strLen;

    while (1)
    {
        strLen = recv(hSock, nameMsg, NAME_SIZE + BUF_SIZE - 1, 0);
        if (strLen == -1)
            return -1;
        nameMsg[strLen] = 0;
        fputs(nameMsg, stdout);
    }
    return 0;
}
```

## 线程同步分析

### 为什么需要互斥量？

在本案例中，多个线程会同时访问共享资源：

1. **主线程**：添加新客户端到 `clntSocks` 数组
2. **HandleClnt 线程**：从 `clntSocks` 数组移除断开的客户端
3. **SendMsg 函数**：遍历 `clntSocks` 数组发送消息

如果没有[[3.4 互斥量|互斥量]]保护，可能出现：
- 数组越界访问
- 遗漏或重复发送消息
- 数据不一致

### 互斥量使用位置

```
┌─────────────────┐
│   主线程        │
│  accept()       │──┬──► WaitForSingleObject(hMutex)
│  添加客户端     │  │    clntSocks[numOfClnt++] = hClntSock
│                 │  │    ReleaseMutex(hMutex)
└─────────────────┘  │
                     │
┌─────────────────┐  │
│ HandleClnt线程  │  │
│  recv()         │──┼──► WaitForSingleObject(hMutex)
│  移除客户端     │  │    // 移除操作
│                 │  │    ReleaseMutex(hMutex)
└─────────────────┘  │
                     │
┌─────────────────┐  │
│   SendMsg()     │  │
│  广播消息       │──┴──► WaitForSingleObject(hMutex)
│                 │       // 遍历发送
│                 │       ReleaseMutex(hMutex)
└─────────────────┘
```

---

## 关键要点

1. **线程安全**：所有对 `clntSocks` 数组和 `numOfClnt` 的操作都必须在互斥量保护下进行

2. **客户端移除算法**：当客户端断开时，采用将最后一个元素移动到被删除位置的策略，保持数组紧凑

3. **双线程客户端**：客户端使用独立的发送和接收线程，实现全双工通信

4. **资源管理**：
   - 服务器为每个客户端创建独立线程
   - 客户端断开时正确关闭套接字并更新计数

## 运行示例

**服务器启动**：
```bash
server.exe 9190
```

**客户端连接**：
```bash
client.exe 127.0.0.1 9190
```

```
客户端A: Hello everyone!
客户端B: [客户端A] Hello everyone!
客户端C: [客户端A] Hello everyone!
```
