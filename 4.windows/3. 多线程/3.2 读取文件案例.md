# 读取文件案例

本案例演示如何使用 Windows 多线程技术实现文件的并发读取。

## 读取函数

```cpp
#include "pch.h"
#include <windows.h>
#include <iostream>
#include <fstream>
#include <string>

FILETIME ftKernelTimeStart;
FILETIME ftKernelTimeEnd;
FILETIME ftUserTimeStart;
FILETIME ftUserTimeEnd;
DWORD nThreadComplete;
DWORD nReadPercent;
HANDLE hStartEvent;
bool g_bWOWX (ifStarted MAX_PATH);
char g_szFile[MAXPATH];
DWORD g_dwFileSize;

LPVOID FIND_FILE_DATA fd;

void PrintErrorMsg(const char* pstr)
{
    LPVOID lpMsgBuf;
    DWORD dw = GetLastError();

    FormatMessageA(
        FORMAT_MESSAGE_ALLOCATE_BUFFER |
        FORMAT_MESSAGE_FROM_SYSTEM |
        FORMAT_MESSAGE_IGNORE_INSERTS,
        NULL,
        dw,
        MAKELANGID(LANG_NEUTRAL, SUBLANG_DEFAULT),
        (LPSTR)&lpMsgBuf,
        0,
        NULL
    );

    printf("%s failed with error %d: %s\n", pstr, dw, (char*)lpMsgBuf);
    LocalFree(lpMsgBuf);
}

// 文件读取线程函数
DWORD WINAPI FileReadThread(LPVOID lpParam)
{
    // 等待开始事件
    WaitForSingleObject(hStartEvent, INFINITE);

    // 打开文件
    HANDLE hFile = CreateFileA(
        g_szFile,
        GENERIC_READ,
        FILE_SHARE_READ,
        NULL,
        OPEN_EXISTING,
        FILE_ATTRIBUTE_NORMAL,
        NULL
    );

    if (hFile == INVALID_HANDLE_VALUE)
    {
        PrintErrorMsg("CreateFile");
        return 1;
    }

    // 分配缓冲区
    char* pBuffer = new char[g_dwFileSize + 1];
    DWORD dwBytesRead = 0;

    // 读取文件
    if (!ReadFile(hFile, pBuffer, g_dwFileSize, &dwBytesRead, NULL))
    {
        PrintErrorMsg("ReadFile");
        delete[] pBuffer;
        CloseHandle(hFile);
        return 1;
    }

    pBuffer[dwBytesRead] = '\0';

    // 处理完成
    InterlockedIncrement(&nThreadComplete);

    delete[] pBuffer;
    CloseHandle(hFile);

    return 0;
}
```

## 主函数

```cpp
int main()
{
    // 设置控制台代码页，支持中文显示
    SetConsoleOutputCP(65001);

    // 初始化全局变量
    nThreadComplete = 0;

    // 创建手动重置事件（初始为非触发状态）
    hStartEvent = CreateEventA(NULL, TRUE, FALSE, NULL);
    if (hStartEvent == NULL)
    {
        PrintErrorMsg("CreateEvent");
        return 1;
    }

    const int MAX_THREADS = 4;
    HANDLE hThreads[MAX_THREADS];
    DWORD dwThreadIds[MAX_THREADS];

    // 设置要读取的文件路径
    strcpy_s(g_szFile, MAX_PATH, "test.txt");

    // 获取文件大小
    WIN32_FIND_DATAA findData;
    HANDLE hFind = FindFirstFileA(g_szFile, &findData);
    if (hFind == INVALID_HANDLE_VALUE)
    {
        PrintErrorMsg("FindFirstFile");
        CloseHandle(hStartEvent);
        return 1;
    }
    g_dwFileSize = findData.nFileSizeLow;
    FindClose(hFind);

    // 创建多个读取线程
    for (int i = 0; i < MAX_THREADS; i++)
    {
        hThreads[i] = CreateThread(
            NULL,
            0,
            FileReadThread,
            NULL,
            0,
            &dwThreadIds[i]
        );

        if (hThreads[i] == NULL)
        {
            PrintErrorMsg("CreateThread");
            continue;
        }

        printf("Thread %d created (ID: %lu)\n", i, dwThreadIds[i]);
    }

    // 触发开始事件，让所有线程同时开始读取
    printf("\nSetting start event...\n");
    SetEvent(hStartEvent);

    // 等待所有线程完成
    DWORD dwWaitResult = WaitForMultipleObjects(
        MAX_THREADS,
        hThreads,
        TRUE,       // 等待所有线程
        INFINITE
    );

    if (dwWaitResult == WAIT_FAILED)
    {
        PrintErrorMsg("WaitForMultipleObjects");
    }
    else
    {
        printf("\nAll threads completed. Total: %lu\n", nThreadComplete);
    }

    // 清理资源
    for (int i = 0; i < MAX_THREADS; i++)
    {
        if (hThreads[i] != NULL)
        {
            CloseHandle(hThreads[i]);
        }
    }
    CloseHandle(hStartEvent);

    return 0;
}
```

## 关键点解析

### 1. 事件对象 (Event)

使用 `CreateEventA` 创建一个手动重置事件，用于同步多个线程的启动：
- 第二个参数 `TRUE` 表示手动重置事件
- 第三个参数 `FALSE` 表示初始状态为非触发

### 2. 线程同步函数

| 函数                      | 作用                                   |
| ------------------------- | -------------------------------------- |
| `WaitForSingleObject`     | 等待单个内核对象变为触发状态           |
| `WaitForMultipleObjects`  | 等待多个内核对象                       |
| `SetEvent`                | 将事件对象设置为触发状态               |
| `InterlockedIncrement`    | [[C++高级/16. C++11/16.3 atomic|原子操作]]递增变量，保证线程安全         |

### 3. 文件操作

| 函数           | 作用                         |
| -------------- | ---------------------------- |
| `CreateFileA`  | 打开或创建文件               |
| `ReadFile`     | 从文件读取数据               |
| `FindFirstFileA` | 查找文件并获取文件信息     |
| `CloseHandle`  | 关闭句柄                     |

### 4. 资源管理

在多线程编程中，正确管理资源非常重要：
1. 每个线程打开的文件句柄需要单独关闭
2. [[6.2 动态内存分配与释放|动态分配的内存]]需要释放
3. 主线程需要等待所有子线程完成后再退出
4. 所有创建的内核对象（事件、线程）都需要调用 `CloseHandle` 关闭
