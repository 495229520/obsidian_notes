# 5.1 UML 时序图入门：服务端核心调用流程

## 四个参与者

| 参与者 | 含义 | 对应代码 |
|--------|------|----------|
| **global** | 全局作用域的函数 | `main()`、`ExcuteCommand()`、`MakeDriverInfo()` 等 |
| **WinAPI** | Windows/C运行时 API | `WSAStartup`、`socket`、`bind`、`recv`、`send` 等 |
| **CPacket** | 协议封包类 | 构造函数（解析/封装）、`Data()`、`Size()` |
| **CServerSocket** | 服务端网络单例 | `getInstance()`、`InitSocket()`、`DealCommand()` 等 |

```mermaid
        Note over S,W: CServerSocket 构造函数内部
        S->>S: InitSockEnv()
        S->>W: WSAStartup(MAKEWORD(1,1), &data)
        W-->>S: 0 (成功)
        S->>W: socket(PF_INET, SOCK_STREAM, 0)
        W-->>S: m_sock (套接字描述符)
    end

    S-->>G: pserver (单例指针)

    G->>S: pserver->InitSocket()
    rect rgb(240, 255, 240)
        Note over S,W: InitSocket() 内部
        S->>W: htons(9527)
        W-->>S: 网络字节序端口
        S->>W: bind(m_sock, &serv_adr, ...)
        W-->>S: 0 (成功)
        S->>W: listen(m_sock, 1)
        W-->>S: 0 (成功)
    end
    S-->>G: true
```

### 阶段二：主循环 — 接受连接与解析命令

```mermaid
sequenceDiagram
    autonumber
    participant G as global
    participant S as CServerSocket
    participant P as CPacket
    participant W as WinAPI

    Note over G,W: ========== 阶段二：主循环（每次迭代） ==========

    loop while CServerSocket::getInstance() != NULL

        G->>S: pserver->AcceptClient()
        rect rgb(255, 248, 240)
            Note over S,W: AcceptClient() 内部
            S->>W: accept(m_sock, &client_adr, &cli_sz)
            Note right of W: 阻塞等待客户端连接
            W-->>S: m_client (客户端套接字)
        end
        S-->>G: true

        G->>S: pserver->DealCommand()
        rect rgb(255, 240, 245)
            Note over S,W: DealCommand() 内部
            S->>W: new char[BUFFER_SIZE]
            loop 循环接收直到完整包
                S->>W: recv(m_client, buffer+index, ...)
                W-->>S: len (接收字节数)
                S->>P: CPacket((BYTE*)buffer, len)
                Note over P: 解析构造函数
                Note right of P: 1. 搜索包头 0xFEFF
                Note right of P: 2. 提取 nLength (4B)
                Note right of P: 3. 提取 sCmd (2B)
                Note right of P: 4. 提取 strData (变长)
                Note right of P: 5. 提取并验证 sSum
                P-->>S: m_packet (解析后的包)
            end
            S->>W: memmove(buffer, ...) 移动剩余数据
            S->>W: delete[] buffer
        end
        S-->>G: m_packet.sCmd (命令ID)

        G->>S: pserver->GetPacket()
        S-->>G: m_packet 引用
        G->>G: ExcuteCommand(sCmd)
        Note right of G: switch(nCmd) 分发到具体处理函数

    end
```

### 阶段三：命令处理（以 Cmd 1 获取磁盘为例）

```mermaid
sequenceDiagram
    autonumber
    participant G as global
    participant S as CServerSocket
    participant P as CPacket
    participant W as WinAPI

    Note over G,W: ========== 阶段三：ExcuteCommand 命令分发 ==========

    G->>G: ExcuteCommand(1)
    G->>G: MakeDriverInfo()

    rect rgb(240, 255, 240)
        Note over G,W: MakeDriverInfo() 内部
        loop i = 1 to 26
            G->>W: _chdrive(i)
            W-->>G: 0(存在) / -1(不存在)
        end
        Note right of G: 拼接结果 "C,D,E"

        G->>P: CPacket(1, data, size)
        Note over P: 发送构造函数
        Note right of P: 1. sHead = 0xFEFF
        Note right of P: 2. nLength = size + 4
        Note right of P: 3. sCmd = 1
        Note right of P: 4. memcpy → strData
        Note right of P: 5. 计算 sSum 校验和
        P-->>G: pack 对象

        G->>S: getInstance()->Send(pack)
        S->>P: pack.Data()
        Note over P: 序列化为字节流
        Note right of P: [FEFF][Length][Cmd][Data][Sum]
        P-->>S: strOut.c_str()
        S->>P: pack.Size()
        P-->>S: nLength + 6
        S->>W: send(m_client, pData, nSize, 0)
        W-->>S: 发送字节数
        S-->>G: true
    end

    G->>S: pserver->CloseClient()
    S->>W: closesocket(m_client)
    W-->>S: 0
    Note right of S: m_client = INVALID_SOCKET
```

## 调用关系总结

### global → CServerSocket 的调用

| 调用点 | 函数 | 作用 |
|--------|------|------|
| `main()` | `getInstance()` | 获取/创建单例 |
| `main()` | `InitSocket()` | bind + listen |
| `main()` | `AcceptClient()` | 等待客户端连接 |
| `main()` | `DealCommand()` | 接收并解析命令 |
| `main()` | `GetPacket()` | 获取解析后的数据包 |
| `main()` | `CloseClient()` | 关闭客户端连接 |
| 各命令函数 | `Send(pack)` | 发送响应包 |
| 各命令函数 | `GetFilePath()` | 从包中提取路径 |
| `MouseEvent()` | `GetMouseEvent()` | 从包中提取鼠标事件 |

### CServerSocket → CPacket 的调用

| 调用点 | 函数 | 作用 |
|--------|------|------|
| `DealCommand()` | `CPacket(BYTE*, size_t&)` | 从原始字节解析包 |
| `Send(CPacket&)` | `pack.Data()` | 序列化为字节流 |
| `Send(CPacket&)` | `pack.Size()` | 获取总包大小 |

### CServerSocket → WinAPI 的调用

| 调用点 | API | 作用 |
|--------|-----|------|
| 构造函数 | `WSAStartup()` | 初始化 Winsock |
| 构造函数 | `socket()` | 创建套接字 |
| `InitSocket()` | `bind()` | 绑定地址 |
| `InitSocket()` | `listen()` | 开始监听 |
| `AcceptClient()` | `accept()` | 接受连接 |
| `DealCommand()` | `recv()` | 接收数据 |
| `Send()` | `send()` | 发送数据 |
| `CloseClient()` | `closesocket()` | 关闭连接 |
| 析构函数 | `WSACleanup()` | 清理 Winsock |

### global → WinAPI 的直接调用

> 命令处理函数中直接调用的 WinAPI，不经过 CServerSocket。

| 命令函数 | API | 作用 |
|----------|-----|------|
| `MakeDriverInfo()` | `_chdrive()` | 检测磁盘分区 |
| `MakeDirectoryInfo()` | `_chdir()` / `_findfirst()` / `_findnext()` | 遍历目录 |
| `RunFile()` | `ShellExecuteA()` | 执行文件 |
| `DownloadFile()` | `fopen_s()` / `fread()` / `fclose()` | 读取文件 |
| `MouseEvent()` | `SetCursorPos()` / `mouse_event()` | 控制鼠标 |
| `SendScreen()` | `GetDC()` / `BitBlt()` / `GlobalAlloc()` | 屏幕截图 |
| `DeleteLocalFile()` | `DeleteFileA()` | 删除文件 |
