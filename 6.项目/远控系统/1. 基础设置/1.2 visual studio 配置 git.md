## 3. 克隆git
![[Pasted image 20251223020118.png]]
![[Pasted image 20251223020122.png]]
*使用HTTP就不需要加载Putty密钥了
点击确认后会提醒你输入账号密码*

如果使用SSH(以git开头的网址)，就需要生成Putty密码
- **生成密钥：** 在开始菜单搜索并打开 PuTTYgen（安装 TortoiseGit 时会自动安装）。点击 Generate，鼠标在空白处乱晃生成随机数，然后点击 Save private key，保存为一个 .ppk 文件。
- **上传公钥：** 复制 PuTTYgen 界面上方那一大串以 ssh-rsa 开头的代码。登录你的 Git 网页，进入“个人设置” -> “SSH 密钥” (SSH Keys)，粘贴并添加。
- **加载密钥：** 回到 TortoiseGit 的克隆界面（你的第2张图），在“加载 Putty 密钥”后面，点击 ... 按钮，选择你刚才保存的那个 .ppk 文件。
