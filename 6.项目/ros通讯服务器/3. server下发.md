# Server 命令下发

## 概述

本笔记分析 MFMS 数据中台的**命令下发链路**，涵盖从 Qt 前端到设备驱动的完整调用流程。

**核心组件:**
- **MfmsCommandService** - Qt 侧命令服务，管理 ROS Service Client
- **FrAdapterServer** - ROS Service 服务端，接收命令并调用驱动
- **FrRobot** - 法奥机械臂驱动层，与设备 SDK 通信

```mermaid
flowchart LR
    A[MfmsCommandService<br/>Qt 前端] -->|ROS Service<br/>FrCmdInterface| B[FrAdapterServer<br/>ROS 节点]
    B -->|SDK/TCP<br/>8083端口| C[FrRobot<br/>设备驱动]
```

### Service Client 缓存机制

```cpp
template<typename ServiceT>
std::shared_ptr<rclcpp::Client<ServiceT>>
MfmsCommandService::getOrCreateServiceClient(const OnlineDeviceInfo& device)
{
    QString serviceName = device.rosServiceName;  // e.g., "/FR_01_cmd"

    auto it = serviceClients_.find(serviceName);
    if (it != serviceClients_.end()) {
        // 复用已有客户端
        return std::dynamic_pointer_cast<rclcpp::Client<ServiceT>>(it->second);
    }

    // 创建新客户端
    auto client = rosNode_->create_client<ServiceT>(serviceName.toStdString());
    serviceClients_[serviceName] = client;
    return client;
}
```

**设计要点:**
- 按设备 Service 名称缓存客户端，避免重复创建
- 使用 `rclcpp::ClientBase` 基类存储异构客户端
- 调用时通过 `dynamic_pointer_cast` 转换为具体类型

### 命令发送流程

```cpp
void MfmsCommandService::sendRobotMotion(const RobotMotionCommand& cmd)
{
    // 1. 查找设备信息
    OnlineDeviceInfo device = findDevice(cmd.deviceId);
    if (device.deviceId.isEmpty()) {
        emit errorOccurred(CommandServiceError::DEVICE_NOT_FOUND,
                          "设备不存在: " + cmd.deviceId);
        return;
    }

    // 2. 检查设备在线状态
    if (!checkDeviceOnline(cmd.deviceId)) {
        emit errorOccurred(CommandServiceError::DEVICE_OFFLINE,
                          "设备离线: " + cmd.deviceId);
        return;
    }

    // 3. 根据设备类型分发
    if (device.deviceType == "fr_robot") {
        sendFrRobotCommand(device, cmd);
    } else if (device.deviceType == "hs_robot") {
        sendHsRobotCommand(device, cmd);
    } else {
        emit errorOccurred(CommandServiceError::UNSUPPORTED_DEVICE_TYPE,
                          "不支持的设备类型: " + device.deviceType);
    }
}
```

### 异步 Service 调用

```cpp
void MfmsCommandService::sendFrRobotCommand(const OnlineDeviceInfo& device,
                                            const RobotMotionCommand& cmd)
{
    using FrCmd = com_interfaces::srv::FrCmdInterface;

    // 获取或创建 Service 客户端
    auto client = getOrCreateServiceClient<FrCmd>(device);

    // 等待 Service 可用
    if (!client->wait_for_service(std::chrono::seconds(1))) {
        emit robotMotionExecuted(cmd.deviceId, false,
                                CommandServiceError::SERVICE_CALL_FAILED,
                                "Service 不可用");
        return;
    }

    // 构建请求
    auto request = std::make_shared<FrCmd::Request>();
    request->id = cmd.mode == 0 ? MOVEAXIS : (cmd.useCartesian ? MOVEL : MOVEJ);
    request->joint_or_desc = cmd.useCartesian ? DESC : JOINT;
    for (int i = 0; i < 6; i++) {
        request->position[i] = cmd.useCartesian ?
            cmd.params[i + 1] : cmd.params[i + 7];
    }
    request->deg = cmd.params[0];  // 步长

    // 异步调用
    auto future = client->async_send_request(request,
        [this, deviceId = cmd.deviceId](auto future) {
            try {
                auto response = future.get();
                emit robotMotionExecuted(deviceId, response->err_code == 0,
                                        response->err_code,
                                        response->err_code == 0 ? "" : "执行失败");
            } catch (const std::exception& e) {
                emit robotMotionExecuted(deviceId, false,
                                        CommandServiceError::SERVICE_CALL_FAILED,
                                        QString::fromStdString(e.what()));
            }
        });
}
```

```cpp
FrAdapterServer::FrAdapterServer(const rclcpp::NodeOptions &options)
    : Node("FrAdapterServer", options)
{
    node_name_ = std::string(this->get_name());

    // 获取设备 IP 参数
    this->declare_parameter("device_ip", "192.168.58.2");
    this->get_parameter("device_ip", device_ip_);

    // 获取 FrRobot 实例 (单例管理)
    frRobot_ = FrRobotManager::instance().get(node_name_);
    frRobot_->IP = device_ip_;

    // 创建 Service: /<node_name>_cmd
    service_name_ = node_name_ + "_cmd";
    cmd_server_ = this->create_service<com_interfaces::srv::FrCmdInterface>(
        service_name_,
        std::bind(&FrAdapterServer::cmd_callback, this, _1, _2));
}
```

### Service 回调实现

```cpp
void FrAdapterServer::cmd_callback(
    std::shared_ptr<FrCmdInterface::Request> req,
    std::shared_ptr<FrCmdInterface::Response> res)
{
    int result = -1;

    switch (req->id)
    {
    case CONNECT:
        // 连接机械臂 (SDK RPC + TCP 8083)
        result = frRobot_->IPConnect(device_ip_);
        if (result == 0) {
            int tcp_result = frRobot_->TCPConnect(device_ip_);
            if (tcp_result != 0) {
                RCLCPP_WARN(get_logger(), "TCP端口连接失败 (8083)");
            }
        }
        res->err_code = result;
        break;

    case MOVEJ:
        // 关节运动
        if (req->joint_or_desc == JOINT) {
            for (int i = 0; i < 6; i++) {
                target_jpos_.jPos[i] = req->position[i];
            }
            result = frRobot_->MoveJ(target_jpos_);
        } else if (req->joint_or_desc == DESC) {
            target_dpos_.tran.x = req->position[0];
            target_dpos_.tran.y = req->position[1];
            target_dpos_.tran.z = req->position[2];
            target_dpos_.rpy.rx = req->position[3];
            target_dpos_.rpy.ry = req->position[4];
            target_dpos_.rpy.rz = req->position[5];
            result = frRobot_->MoveJ(target_dpos_);
        }
        res->err_code = result;
        break;

    case MOVEAXIS:
        // 单关节点动
        result = frRobot_->StartJOG(req->axid, req->dir, req->deg, 50, 80);
        res->err_code = result;
        break;

    case SETIO:
        // 设置数字输出
        if (req->input_or_output == D_OUTPUT) {
            result = frRobot_->SetDO(req->io_index, req->io_value);
        } else {
            RCLCPP_WARN(get_logger(), "无法设置 DI 口");
            result = -1;
        }
        res->err_code = result;
        break;

    case GETSTATE:
        // 获取完整状态
        frRobot_->GetConnectState(frRobot_->Connected);
        if (frRobot_->Connected) {
            frRobot_->GetRobotState(frRobot_->robotState);
            frRobot_->GetRobotSpeed(frRobot_->speed);

            ROBOT_STATE_PKG &state = frRobot_->robotState;
            res->pub_state.robot_name = node_name_;
            res->pub_state.robot_type = "法奥机器人";
            res->pub_state.robot_ip = device_ip_;
            res->pub_state.program_state = state.program_state;
            res->pub_state.main_code = state.main_code;
            res->pub_state.sub_code = state.sub_code;
            res->pub_state.robot_mode = state.robot_mode;
            res->pub_state.robot_speed = frRobot_->speed;

            for (int i = 0; i < 6; i++) {
                res->pub_state.jt_cur_pos[i] = state.jt_cur_pos[i];
                res->pub_state.tl_cur_pos[i] = state.tl_cur_pos[i];
            }

            res->pub_state.cl_dgt_output_h = state.cl_dgt_output_h;
            res->pub_state.cl_dgt_output_l = state.cl_dgt_output_l;
            res->pub_state.cl_dgt_input_h = state.cl_dgt_input_h;
            res->pub_state.cl_dgt_input_l = state.cl_dgt_input_l;
            res->pub_state.emergency_stop = state.EmergencyStop;
            res->err_code = 0;
        } else {
            res->err_code = -1;
        }
        break;

    // ... 其他命令处理
    }
}
```

### FrRobotManager 单例模式

```cpp
class FrRobotManager {
public:
    static FrRobotManager& instance() {
        static FrRobotManager inst;
        return inst;
    }

    std::shared_ptr<FrRobot> get(const std::string& name) {
        std::lock_guard<std::mutex> lock(mutex_);
        auto it = robots_.find(name);
        if (it != robots_.end()) {
            return it->second;
        }
        auto robot = std::make_shared<FrRobot>();
        robots_[name] = robot;
        return robot;
    }

    void remove(const std::string& name) {
        std::lock_guard<std::mutex> lock(mutex_);
        robots_.erase(name);
    }

private:
    FrRobotManager() = default;
    std::map<std::string, std::shared_ptr<FrRobot>> robots_;
    std::mutex mutex_;
};
```

**设计要点:**
- 多节点场景下同一设备名共享 FrRobot 实例
- 线程安全的 get/remove 操作
- 避免重复创建 SDK 连接

### 连接管理

```cpp
int FrRobot::IPConnect(const std::string& ip) {
    IP = ip;
    // SDK RPC 连接 (默认端口 8080)
    return robot.RPC(ip.c_str());
}

int FrRobot::TCPConnect(const std::string& ip) {
    // TCP 8083 端口用于实时状态获取
    // 返回 429 字节的 ROBOT_STATE_PKG 数据包
    return robot.TCPConnect(ip.c_str(), 8083);
}

void FrRobot::GetConnectState(int& state) {
    state = robot.GetConnectState();
    Connected = state;
}
```

## 关键要点

1. **分层架构**: Qt 命令服务 → ROS Service → 设备驱动，职责清晰
2. **Service Client 缓存**: 避免重复创建客户端，提升性能
3. **单例管理器**: FrRobotManager 确保同一设备只有一个驱动实例
4. **双端口通信**: SDK RPC (8080) 用于命令，TCP (8083) 用于状态
5. **异步调用**: 前端使用 `async_send_request` + 回调，避免阻塞 UI
