# 3. 无重复字符的最长子串

## 滑动窗口法

### 核心思想

滑动窗口适合处理“连续区间”问题。我们用两个指针 `left` 和 `right` 表示当前窗口 `[left, right]`：

- `right` 向右移动，负责扩张窗口
- 如果窗口不满足条件，就移动 `left` 缩小窗口
- 在窗口变化的过程中不断更新答案

> [!tip]
> 当题目对象是字符串或数组，并且要求的是连续的子串、子数组、最长区间、最短区间时，通常可以优先考虑滑动窗口。

### 常见特征

- 题目要求的是**子串**或**子数组**，而不是子序列
- 需要动态维护一个连续区间
- 区间是否合法，会随着元素加入或移出而改变

### 基本模版

```cpp
int left = 0;
for (int right = 0; right < n; ++right) {
    // 1. 扩张窗口：加入 a[right]

    while (窗口不合法) {
        // 2. 缩小窗口：移除 a[left]
        ++left;
    }

    // 3. 更新答案
}
```

---

## 题目描述

给定一个字符串 `s`，求其中不含重复字符的最长子串长度。

### 关键点

- 这里要求的是**子串**，必须连续
- 不是子序列，所以像 `"pwke"` 这样的结果不合法

### 示例

- `s = "abcabcbb"`，答案是 `3`
- `s = "bbbbb"`，答案是 `1`
- `s = "pwwkew"`，答案是 `3`

---

## 解题思路

我们用一个 `unordered_set<char>` 维护当前窗口中出现过的字符。

1. `right` 不断右移，准备把 `s[right]` 加入窗口
2. 如果 `s[right]` 已经在窗口中，说明出现了重复字符
3. 这时不断移动 `left`，并把 `s[left]` 从集合中删除，直到窗口重新合法
4. 再把 `s[right]` 插入集合，更新当前窗口长度

### 为什么时间复杂度是 O(n)

虽然代码里有 `for` 和 `while` 两层结构，但：

- `right` 只会从左到右走一遍
- `left` 也只会从左到右走一遍

所以每个字符最多进入窗口一次、移出窗口一次，总复杂度仍然是 `O(n)`。

### 过程示意

以 `s = "abcabcbb"` 为例：

```text
right = 0: 窗口 "a"       ans = 1
right = 1: 窗口 "ab"      ans = 2
right = 2: 窗口 "abc"     ans = 3
right = 3: 遇到重复 'a'，移动 left，窗口变为 "bca"
right = 4: 遇到重复 'b'，移动 left，窗口变为 "cab"
...
```

---

## 代码实现

```cpp
class Solution {
public:
    int lengthOfLongestSubstring(string s) {
        if (s.size() <= 1) {
            return s.size();
        }

        unordered_set<char> window;
        int left = 0, ans = 0;

        for (int right = 0; right < s.size(); ++right) {
            while (window.count(s[right])) {
                window.erase(s[left]);
                ++left;
            }

            window.insert(s[right]);
            ans = max(ans, right - left + 1);
        }

        return ans;
    }
};
```

### 代码分析

- `window`：记录当前窗口中已经出现过的字符
- `while (window.count(s[right]))`：如果当前字符重复，就持续缩小窗口
- `ans = max(ans, right - left + 1)`：每次窗口合法时更新答案

---

## 易错点

1. 题目求的是**最长长度**，不是具体子串内容
2. 题目要求的是**子串**，必须连续
3. 先缩小窗口，再插入当前字符
4. `left` 右移时，要同步把旧字符从 `window` 中删除

---

## 复杂度

- 时间复杂度：`O(n)`
- 空间复杂度：`O(k)`，`k` 为窗口内不同字符个数，最坏情况下可看作 `O(n)`
