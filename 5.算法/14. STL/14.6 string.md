# string 字符串容器

> 本笔记是对 [[8.1 string]] 的补充，主要讲解 string 在 STL 中的高级用法。

## C 风格字符串

```cpp
// char 数组
char charVARS;
char arr[10];
int i = 0;
// i == 'hello'
```

## C++ 风格字符串

```cpp
string str("hello");
string str = "hello";
```

## 两者区别

| 特性 | C 风格 | C++ 风格 |
|------|--------|----------|
| 初始化 | `char* s = "hello"` | `string s("hello")` 或 `string s = "hello"` |
| 获取长度 | `strlen(s)` | `s.length()` 或 `s.size()` |
| 字符串拼接 | `strcat(s1, s2)` | `s1 + s2` 或 `s1.append(s2)` |
| 字符串比较 | `strcmp(s1, s2)` | `s1 == s2` 或 `s1.compare(s2)` |
| 安全性 | 需要手动管理内存 | 自动管理，无溢出风险 |

## 常用方法

| 类型 | 函数 | 用法 | 功能 |
|------|------|------|------|

| | `push_back(ch)` | `s.push_back('a')` | 在尾部追加一个字符 |
| | `pop_back()` | `s.pop_back()` | 移除最后一个字符 |
| **append 系列** | `append(str)` | `s1.append(s2)` | 追加字符串 s2 到 s1 后面；等价于 `s1 += s2` |
| | `append(n, ch)` | `s.append(3, 'x')` | 追加 n 个字符 ch |
| | `append(str, pos, len)` | `s1.append(s2, 0, 3)` | 追加 s2 的子串 |
| **获取长度** | `length()` | `s.length()` | 返回字符串长度（与 `size()` 等价） |

| | `reserve(n)` | `s.reserve(100)` | 预留至少 n 个字符的空间 |
| **元素访问** | `operator[]` | `s[0]` | 访问指定位置字符（无边界检查） |

| | `back()` | `s.back()` | 返回尾字符 |

| | `erase(pos, len)` | `s.erase(0, 5)` | 删除从 pos 开始的 len 个字符 |

| | `replace(pos, len, str)` | `s.replace(0, 5, "Hi")` | 替换从 pos 开始的 len 个字符为 str |

| | `rend()` | `s.rend()` | 返回反向迭代器（指向第一个字符之前） |

## 字符串与数值转换

### std::stoi/stol/stoll/stof/stod 系列

将 `std::string` 转换为对应的数值类型：
- `stoi`：转 `int`
- `stol`：转 `long`
- `stoll`：转 `long long`
- `stof`：转 `float`
- `stod`：转 `double`

```cpp
std::string numStr = "12345";
int num = std::stoi(numStr);  // num = 12345

std::string floatStr = "3.14159";
double pi = std::stod(floatStr);  // pi = 3.14159
```

### std::to_string

将数值类型转为 `std::string`：

```cpp
int value = 42;
std::string str = std::to_string(value);  // str = "42"
```

## 字符串分割

### std::stringstream 配合 getline

```cpp
// 将 "apple,banana,orange,pear" 分割成一个 vector
#include <sstream>
#include <vector>
#include <string>

std::string input = "apple,banana,orange,pear";
std::stringstream ss(input);
std::string token;
std::vector<std::string> tokens;

while (std::getline(ss, token, ',')) {
    tokens.push_back(token);
}
// tokens = {"apple", "banana", "orange", "pear"}
```

### C++20 std::ranges 和 std::views::split

```cpp
#include <ranges>
#include <string_view>

std::string str = "Split these words into a vector";
auto words = str | std::views::split(' ');
for (auto word : words) {
    // 处理每个单词
}
```

## 字符串拼接

### 使用 std::ostringstream 高效拼接

```cpp
#include <sstream>

std::ostringstream oss;
oss << "Hello" << ", " << "World" << "!";
std::string result = oss.str();  // result = "Hello, World!"
```

### C++17 std::string::reserve 预分配

```cpp
std::string result;
result.reserve(1000);  // 预分配空间，减少重分配

for (int i = 0; i < 100; ++i) {
    result += std::to_string(i) + " ";
}
```

## str.erase() 删除指定字符

```cpp
int main() {
    std::string s = "Hello World! This is C++..."; // 去掉感叹号和省略号以外的标点

    // 找到 '!' 的位置并删除
    size_t pos = s.find('!');
    if (pos != std::string::npos) {
        s.erase(pos, 1);  // 删除 1 个字符
    }

    // 使用 erase-remove idiom 删除所有指定字符
    s.erase(std::remove(s.begin(), s.end(), '.'), s.end());
}
```

## str.remove_if 删除满足条件的字符

配合 `erase` 使用 `remove_if` 删除满足特定条件的字符：

```cpp
#include <algorithm>
#include <cctype>

std::string str = "Hello123World456";

// 删除所有数字
str.erase(
    std::remove_if(str.begin(), str.end(), ::isdigit),
    str.end()
);
// str = "HelloWorld"
```

**重要说明**：`std::remove_if` 不是真正删除元素，它只是将不符合条件的元素移到前面，并返回一个指向"逻辑结尾"的迭代器。需要配合 `erase` 才能真正删除元素。

## 关键要点

1. **string 是动态数组**：底层是动态分配的字符数组，自动管理内存
2. **与 C 风格兼容**：使用 `c_str()` 或 `data()` 获取 C 风格字符串
3. **预分配优化**：使用 `reserve()` 预分配空间可提高拼接性能
4. **边界检查**：`at()` 有边界检查，`[]` 没有
5. **迭代器支持**：支持所有标准迭代器操作
