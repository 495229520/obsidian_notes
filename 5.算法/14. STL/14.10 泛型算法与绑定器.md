# 14.10 泛型算法与绑定器

## 简介

C++ 的**泛型算法**是 C++ 标准库 (STL) 中提供的一系列[[10.1 函数模版|函数模板]]，它们可以对不同类型的容器和迭代器进行操作。这些算法之所以被称为"泛型"，是因为它们不依赖于特定的数据类型或容器，而是通过**迭代器**抽象来工作。

```cpp
泛型算法 = template + 迭代器 + 函数对象
```

### 核心特点

- **容器** (Containers)：泛型算法操作的数据集，比如 `std::vector`、`std::list`、`std::set` 等
- **迭代器** (Iterators)：泛型算法通过迭代器来访问容器中的元素，迭代器提供了一种独立于具体容器的关键
- **算法** (Algorithms)：实际执行任务的[[10.1 函数模版|函数模板]]，比如 `std::sort`、`std::find`、`std::for_each` 等

---

## 常见的泛型算法

### 非修改序列操作

不改变容器内容的算法：

- `std::find`：查找指定元素
- `std::count`：统计元素数量
- `std::for_each`：对序列中每个元素执行指定操作
- `std::min_element / std::max_element`：查找最小值或最大值

### 修改序列操作

可能会改变容器内容的算法：

- `std::sort`：对序列进行排序
- `std::copy`：将一个序列的元素复制到另一个序列
- `std::remove`：移除序列中特定值的元素（注意：它只将元素移到末尾，不会改变容器大小）
- `std::transform`：对序列中每个元素应用一个函数，并将结果保存到另一个位置

---

## 绑定器

**绑定器**（Binder）用于将[[14.9 函数对象|函数对象]]的某个参数绑定为固定值，从而将二元函数对象转换为一元函数对象。

### 绑定器类型

| 绑定器 | 作用 |
|--------|------|
| `bind1st` | 把二元函数对象的第一个参数 `operator()(a, b)` 的第一个参数绑定起来 |
| `bind2nd` | 把二元函数对象的第二个参数 `operator()(a, b)` 的第二个参数绑定起来 |

> **注意**：`bind1st` 和 `bind2nd` 在 C++11 中已被弃用，推荐使用 `std::bind`。

### 绑定器的工作原理

```
绑定器 + 二元函数对象 => 一元函数对象
```

---

## 案例

```cpp
int main()
{
    int arr[] = { 12, 4, 78, 9, 21, 43, 56, 52, 42, 31 };
    vector<int> vec(arr, arr + sizeof(arr) / sizeof(arr[0]));

    for (int v : vec)
    {
        cout << v << " ";
    }
    cout << endl;

    // 排序，默认从小到大
    sort(vec.begin(), vec.end());
    for (int v : vec)
    {
        cout << v << " ";
    }
    cout << endl;

    // 在有序容器中进行二分查找
    if (binary_search(vec.begin(), vec.end(), 21))
    {
        cout << "21存在" << endl;
    }

    // 普通查找方法
    auto it1 = find(vec.begin(), vec.end(), 21);
    if (it1 != vec.end())
    {
        cout << "find 21存在" << endl;
    }

    // 通过传入函数对象 greater，改变容器元素排序时的比较方式
    sort(vec.begin(), vec.end(), greater<int>());
    for (int v : vec)
    {
        cout << v << " ";
    }
    cout << endl;

    // 把 48 按序插入到 vector 容器中，找到第一个小于 48 的数字
    // find_if 需要的是一个一元函数对象
    // greater a > b  将 a 绑定为 48，因此使用 bind1st
    /*
    greater<int>() 是一个二元函数对象，它的 operator() 接受两个参数 a 和 b，并返回 a > b 的布尔结果。
    bind1st 是一个绑定器适配器，它将一个二元函数对象（greater<int>()）的第一个参数固定为一个固定值 48。
    因此，bind1st(greater<int>(), 48) 会创建一个新的函数对象。这个新的函数对象只接受一个参数 x，
    然后内部会调用 greater<int>(48, x)，其效果是判断 48 > x。
    最后，迭代器指向 vec 容器中第一个小于 48 的元素
    */
    auto it2 = find_if(vec.begin(), vec.end(), bind1st(greater<int>(), 48));
    // 下面也是一样的道理，因为是 less a < b，因此需要绑定第二个元素 b
    auto it3 = find_if(vec.begin(), vec.end(), bind2nd(less<int>(), 48));
    // 使用 lambda 表达式
    auto it4 = find_if(vec.begin(), vec.end(), [](int val)->bool { return val < 48; });

    vec.insert(it4, 48);
    for (int v : vec)
    {
        cout << v << " ";
    }
    cout << endl;

    // 使用泛型算法打印每一个偶数
    // for_each 可以遍历容器的所有元素，可以自行添加合适的函数对象对容器进行遍历
    for_each(vec.begin(), vec.end(),
        [](int val)->void
        {
            if (val % 2 == 0)
            {
                cout << val << " ";
            }
        });

    return 0;
}
```

## C++11 std::bind

C++11 引入了更强大的 `std::bind`，可以替代 `bind1st` 和 `bind2nd`：

```cpp
#include <functional>
using namespace std::placeholders;

// 绑定第一个参数为 48
auto pred = std::bind(std::greater<int>(), 48, _1);
auto it = find_if(vec.begin(), vec.end(), pred);

// 绑定第二个参数为 48
auto pred2 = std::bind(std::less<int>(), _1, 48);
auto it2 = find_if(vec.begin(), vec.end(), pred2);
```

其中 `_1`、`_2` 等是占位符，表示调用时传入的第 1、2 个参数。

---

## 关键要点

1. **泛型算法** = 模板 + 迭代器 + 函数对象
2. **泛型算法的参数接收的都是迭代器**
3. **泛型算法的参数可以接收函数对象**
4. **绑定器**：将二元函数对象转换为一元函数对象
5. **C++11 推荐使用 `std::bind`** 替代 `bind1st`/`bind2nd`
6. **Lambda 表达式**：现代 C++ 中更简洁的替代方案
