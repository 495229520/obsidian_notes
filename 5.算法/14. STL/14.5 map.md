# map 映射容器

## 简介

| 性质 | 解释 | map | multimap | unordered_map |
|------|------|-----|----------|---------------|
| **互异性** | 一个键 key 仅可以在映射中出现一次 | ✓ | ✗ | ✓ |
| **有序性** | 键是没有顺序的 | ✗ | ✗ | ✓ |

### 构造方法

```cpp
// 从 int 映射到 int
map<int, int> mp;                         // 键从小到大
map<int, int, greater<int>> mp;           // 键从大到小
```

### 常用方法

| 作用 | 用法 | 示例 |
|------|------|------|
| 插入、增加、修改元素 | `[key]=value` | `map[key]=value` |
| 删除元素 | `.erase(key)` | `map.erase(2);` |
| 查找元素，返回迭代器 | `.find(key)` | `auto it = map.find(1);` |
| 判断元素是否存在<br>找到返回 1，否则 0 | `.count(元素)` | `map.count(3);` |
| 清空 | `.clear()` | `map.clear()` |
| 判断是否为空 | `.empty()` | `map.empty()` |

### 遍历

```cpp
// 遍历，使用迭代器
for (map<int, int>::iterator it = mp.begin(); it != mp.end(); ++it)
    cout << it->first << it->second << endl;

// 底层也是使用的迭代器
for (auto& ele : mp)
    cout << ele.first << ele.second << endl;

// first 是 key，second 是 value
for (const auto& pair : myMap) {
    // pair.first 是 Key
    // pair.second 是 Value
    std::cout << "Key: " << pair.first << ", Value: " << pair.second << std::endl;
}
```

### 适用范围

求字符串的出现次数：

```cpp
map<string, int> myp;
vector<string> word;
word.push_back("abc");
word.push_back("abc");
word.push_back("abc");
word.push_back("bbc");
word.push_back("bbc");

for (int i = 0; i < word.size(); i++)
{
    myp[word[i]]++;
}

for (auto& ptr : myp)
{
    cout << ptr.first << ptr.second << endl;
}
```

## 有序哈希表 map

### 简介

`map` 是基于[[14.14 红黑树]]实现的关联容器，具有以下特点：

1. **有序性**：map 的排序是通过 key 来进行的，底层红黑树保证元素按 key 从小到大自动排序
2. **自动插入**：可以通过 `Map[key]` 查询元素，但是如果该元素不存在时，编译器会调用调用该类型的默认构造函数生成 value 然后插入
3. **键值对结构**：key 对应着 `Map->first`; value 对应着 `Map->second`
4. **高效查找**：由于红黑树的特性，查找、插入、删除的时间复杂度都是 **O(logn)**

### 案例

```cpp
class Student
{
public:
    Student(int id = 0, string name = "")
        : _id(id), _name(name) { }

    bool operator<(const Student& stu) const
    {
        return _id < stu._id;
    }

private:
    int _id;
    string _name;
    friend ostream& operator<<(ostream& out, const Student& stu);
};

ostream& operator<<(ostream& out, const Student& stu)
{
    out << "id:" << stu._id << " name:" << stu._name << endl;
    return out;
}

int main()
{
    // map 的 value 如果是自定义类型，那么需要有默认的构造函数
    map<int, Student> stuMap;
    stuMap.insert({ 1000, Student(1000, "张三") });
    stuMap.insert({ 1020, Student(1020, "李四") });
    stuMap.insert({ 1020, Student(1020, "王五") });

    // 红六井没有智能插入，只会打印张三和李四
    // 红六井只能打印张三和李四
    cout << stuMap.size() << endl;  // 3

    // 迭代器通过 key 删除集合元素
    stuMap.erase(1020);

    // 可以通过 Map[key] 查询元素，但是如果该元素不存在时，会调用调用该类型的默认构造函数生成 value 然后插入
    // stuMap[2000]  [2000, V()]  结果输出：(id:0 name:)
    cout << stuMap[1020] << endl;

    // 通过迭代器遍历集合
    auto it = stuMap.begin();
    for (; it != stuMap.end(); ++it)
    {
        cout << "key:" << it->first << " value:" << it->second << endl;
    }
    cout << endl;
}
```

## unordered_map 案例

```cpp
// [key, value]
struct pair
{
    first;   // => key
    second;  // => value
};

// unordered_map 不允许 key 重复
unordered_map<int, string> map1;
// 映射在 map 中对应的是 pair
// map1.insert(make_pair(1000, "张三"));
map1.insert({ 1010, "张四" });
map1.insert({ 1010, "王五" });
map1.insert({ 1020, "王五" });
map1.insert({ 1030, "周八" });

cout << map1.size() << endl;  // 3

// 通过迭代器遍历方式
auto it1 = map1.begin();
for (; it1 != map1.end(); ++it1)
{
    cout << "key:" << it1->first << " value:" << it1->second << endl;
}
cout << endl;

// 修改 key 重复的 value
// 增加 key 重复时是不合法操作，但是更改时，需要使用 [] 修改一个非存在键将执行的操作类型
map1[3000] = "神仙";
cout << map1[3000] << endl;

auto it1 = map1.find(1030);
if (it1 != map1.end())
{
    // key:1030  value:周八
    cout << "key:" << it1->first << " value:" << it1->second << endl;
}

// 处理去重大数据
const int ARR_LEN = 100;
int arr[ARR_LEN] = { 0 };
for (int i = 0; i < ARR_LEN; ++i)
{
    arr[i] = rand() % 20 + 1;
}

// 在上面 100 个整数中，找出哪些数字重复了，并统计重复的次数
unordered_map<int, int> map1;
for (int k : arr)
{
    auto it = map1.find(k);
    // 第一次出现
    if (it == map1.end())
    {
        map1.insert({ k, 1 });
    }
    else
    {
        // value 是重复的次数
        it->second++;
    }
}

// 浓缩为一段代码
/*
 * + 也叫 ++，加的是 value
 * map1[k]++;
*/

for (const pair<int, int>& p : map1)
{
    if (p.second > 1)
    {
        cout << "key:" << p.first << " count:" << p.second << endl;
    }
}

auto it = map1.begin();
for (; it != map1.end(); ++it)
{
    if (it->second > 1)
    {
        cout << "key:" << it->first << " count:" << it->second << endl;
    }
}

// 上面 10 个字整数中，把数字进行去重打印
unordered_set<int> set;
for (int v : arr)
{
    set.insert(v);
}
for (int v : set)
{
    cout << v << " ";
}
cout << endl;
cout << set.size() << " ";
```

## pair

`pair` 本质跟存储两个类型的 STRUCT 是一样的。

```cpp
pair<int, int> p1 = make_pair(1, 2);
pair<int, int> p2 = { 1, 2 };
if (p1 == p2)
    cout << "yes" << endl;
```

## 关键要点

| 容器 | 底层结构 | 有序性 | 键唯一性 | 查找复杂度 |
|------|----------|--------|----------|------------|
| `map` | [[14.14 红黑树]] | 有序 | 唯一 | O(logn) |
| `multimap` | [[14.14 红黑树]] | 有序 | 可重复 | O(logn) |
| `unordered_map` | 哈希表 | 无序 | 唯一 | 平均 O(1) |
