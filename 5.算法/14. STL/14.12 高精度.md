# 14.12 高精度

## 只有 C++ 需要用高精度

| 语言 | 需要高精度？ |
|------|-------------|
| A+B | 20% |
| A-B | 10% |
| A*B | `len(A)*len(B)>32%` |
| A/B | 10% |

> Java 和 Python 原生支持大整数运算，不需要手写高精度算法。

## 大整数的存储方式

### 存储原则

- 以字符串形式读入大整数
- **逆序存储**到 [[14.1 vector|vector]] 中：将大整数的低位存在数组的低位，高位存在数组的高位
- 原因：方便进位操作，当最高位有进位时，在数组末尾添加元素即可

### 示例

```cpp
string a = "12345";
// 存储为: A[0]=5, A[1]=4, A[2]=3, A[3]=2, A[4]=1
// 即逆序存储
```

## 高精度加法

### 思路

1. 逆序存储两个大整数 A 和 B
2. 从低位开始逐位相加
3. 处理进位：`t = A[i] + B[i] + 进位`，结果位 = `t % 10`，新进位 = `t / 10`
4. 最后若有进位，追加到结果末尾

### 代码模板

```cpp
#define _CRT_SECURE_NO_WARNINGS
#include <iostream>
#include <vector>

using namespace std;

const int N = 1e6 + 10;

// C 用来存储运算结果
// reverse 是已经反转好的数组
vector<int> add(vector<int>& A, vector<int>& B) {
    vector<int> C;
    int t = 0;  // 进位

    for (int i = 0; i < A.size() || i < B.size(); i++) {
        if (i < A.size()) t += A[i];
        if (i < B.size()) t += B[i];
        C.push_back(t % 10);
        t /= 10;
    }

    if (t) C.push_back(1);

    return C;
}

int main()
{
    string a, b;
    vector<int> A, B;

    cin >> a >> b;
    // 将 a 字符串的数据反过来存入 A vector 中（反转存储）
    for (int i = a.size() - 1; i >= 0; i--) A.push_back(a[i] - '0');
    for (int i = b.size() - 1; i >= 0; i--) B.push_back(b[i] - '0');

    auto C = add(A, B);

    // 结果也需要逆序输出
    for (int i = C.size() - 1; i >= 0; i--) printf("%d", C[i]);

    return 0;
}
```

## 高精度减法

### 比较大小

先比较两个数的大小，确保大数减小数：

```cpp
bool cmp(vector<int>& A, vector<int>& B) {
    if (A.size() != B.size()) return A.size() > B.size();
    for (int i = A.size() - 1; i >= 0; i--) {
        if (A[i] != B[i])
            return A[i] > B[i];
    }
    return true;  // 相等
}
```

### 代码模板

```cpp
// 保证 A >= B
vector<int> sub(vector<int>& A, vector<int>& B) {
    vector<int> C;
    int t = 0;  // 借位

    for (int i = 0; i < A.size(); i++) {
        t = A[i] - t;
        if (i < B.size()) t -= B[i];
        C.push_back((t + 10) % 10);
        t = t < 0 ? 1 : 0;
    }

    // 去除前导零
    while (C.size() > 1 && C.back() == 0) C.pop_back();

    return C;
}
```

## 高精度乘法（大数 × 小数）

### 代码模板

```cpp
// A 是一个大数，b 是一个小整数
vector<int> mul(vector<int>& A, int b) {
    vector<int> C;
    int t = 0;

    for (int i = 0; i < A.size() || t; i++) {
        if (i < A.size()) t += A[i] * b;
        C.push_back(t % 10);
        t /= 10;
    }

    // 去除前导零（处理乘以 0 的情况）
    while (C.size() > 1 && C.back() == 0) C.pop_back();

    return C;
}

int main() {
    string a;
    int b;

    cin >> a >> b;
    vector<int> A;
    for (int i = a.size() - 1; i >= 0; i--) A.push_back(a[i] - '0');

    auto C = mul(A, b);

    for (int i = C.size() - 1; i >= 0; i--) printf("%d", C[i]);

    return 0;
}
```

## 高精度除法（大数 ÷ 小数）

### 代码模板

```cpp
// r 是余数，通过引用返回
vector<int> div(vector<int>& A, int b, int& r) {
    vector<int> C;
    r = 0;

    // 除法从高位开始
    for (int i = A.size() - 1; i >= 0; i--) {
        r = r * 10 + A[i];
        C.push_back(r / b);
        r %= b;
    }

    // 结果需要反转（因为是从高位开始计算的）
    reverse(C.begin(), C.end());
    // 去除前导零
    while (C.size() > 1 && C.back() == 0) C.pop_back();

    return C;
}

int main() {
    string a;
    int b;

    cin >> a >> b;
    vector<int> A;
    // 虽然除法可以不用逆序，但为了和其他运算保持一致
    for (int i = a.size() - 1; i >= 0; i--) A.push_back(a[i] - '0');

    int r;
    auto C = div(A, b, r);

    for (int i = C.size() - 1; i >= 0; i--) printf("%d", C[i]);
    printf("\n%d", r);  // 输出余数

    return 0;
}
```

---

## 关键要点

1. **逆序存储**：低位存低索引，便于进位处理
2. **进位/借位**：用变量 `t` 统一处理
3. **去除前导零**：结果可能有多余的 0，需要去除
4. **大数比较**：先比长度，再从高位比数值
5. **除法特殊**：从高位开始运算，结果需要反转
