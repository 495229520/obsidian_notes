# 14.11 算法

## swap 交换

### 排序方式

| 方式 | 代码 |
|------|------|
| **默认（升序）** | `sort(arr.begin(), arr.end());` |
| **降序** | `sort(arr.begin(), arr.end(), greater<int>());` |

### 自定义排序规则

```cpp
// #include <algorithm>
vector<int> arr{ 1, 5, 4, 3, 9, 8, 2, 6, 7, 0 };
// 从小到大
sort(arr.begin(), arr.end());

// 从大到小
sort(arr.begin(), arr.end(), greater<int>());

// 自定义规则：按 pair 的第二位从小到大，第二位相同时第一位从大到小
bool cmp(const pair<int, int>& aa, const pair<int, int>& bb)
{
    // 第二位从小到大
    if (aa.second != bb.second)
        return aa.second < bb.second;
    // 如果第二位相同，第一位从大到小
    return aa.first > bb.first;
}
```

## 二分查找

### 简介

在已升序排序的元素中，应用**二分查找**检索指定元素，返回对应元素迭代器位置。找不到则返回尾迭代器。

### lower_bound 和 upper_bound

| 函数 | 作用 |
|------|------|
| `lower_bound()` | 寻找 ≥ x 的第一个元素的位置 |
| `upper_bound()` | 寻找 > x 的第一个元素的位置 |

### 怎么找 ≤ x / < x 的第一个元素呢？

- `> x` 的第一个元素的前一个元素（如果有）便是 `≤ x` 的第一个元素
- `≥ x` 的第一个元素的前一个元素（如果有）便是 `< x` 的第一个元素

### 获取索引

返回的是迭代器，如何转成下标索引呢？**减去头迭代器即可**。

```cpp
vector<int> arr{0, 1, 1, 1, 8, 9, 9};
vector<int>::iterator it = lower_bound(arr.begin(), arr.end(), 7);
int idx = it - arr.begin();
// idx = 4
```

### 示例

```cpp
vector<int> arr{0, 1, 1, 1, 8, 9, 9};

idx = lower_bound(arr.begin(), arr.end(), 7) - arr.begin();  // 4
idx = lower_bound(arr.begin(), arr.end(), 8) - arr.begin();  // 4
idx = upper_bound(arr.begin(), arr.end(), 7) - arr.begin();  // 4
idx = upper_bound(arr.begin(), arr.end(), 8) - arr.begin();  // 5
```

## 整数二分

### 二分的本质是边界

定义了某种性质，将整个区间一分为二，使左半边不满足这个性质，右半边是满足这个性质。二分法可以寻找这个性质的边界。

```
┌────────────────┬────────────────┐
│      l         │        r       │
│  不满足性质     │    满足性质     │
└────────────────┴────────────────┘
        ↑                  ↑
      红色边界点         绿色边界点
```

### 两种模板

**① 二分红色边界点**

```
mid = (l + r + 1) / 2

if (check(mid)) {
    true  => [mid, r]   l = mid
    false => [l, mid-1] r = mid - 1
}
```

**② 二分绿色边界点**

```
mid = (l + r) / 2

if (check(mid)) {
    true  => [l, mid]   r = mid
    false => [mid+1, r] l = mid + 1
}
```

### 操作步骤

| 步骤 | 说明 |
|------|------|
| 1 | 写 check 函数，如果 check 函数正确时 mid，错误是 mid-1，那么这时候 `mid = (l+r+1)/2` |
| 2 | 如果 check 函数正确时 mid，错误是 mid+1，那么这时候 `mid = (l+r)/2` |
| 总结 | 如果 l=mid，那么 mid 需要补上 +1；如果 r=mid，那么 mid 不用补上 +1 |

> **问题**：可能是无解的，但二分一定是有解的

### 代码模板：求 n 个数组的 x 值在第几位

```cpp
#define _CRT_SECURE_NO_WARNINGS
#include <iostream>

using namespace std;

const int N = 1e6 + 10;

int n, m;
int q[N];

int main()
{
    scanf("%d %d", &n, &m);
    for (int i = 0; i < n; i++) scanf("%d", &q[i]);

    while (m--) {
        int x;
        // 需要查找的对象
        scanf("%d", &x);

        int l = 0, r = n - 1;
        while (l < r) {
            int mid = l + r >> 1;
            // 找不到的时候 q[mid] >= x; r = mid;
            if (q[mid] >= x) r = mid;
            else l = mid + 1;
        }
        // 找不到的时候 q[l] 是不等于 x 的
        if (q[l] != x) cout << "-1 -1" << endl;
        else {
            cout << l << " ";

            int l = 0, r = n - 1;
            while (l < r) {
                int mid = l + r + 1 >> 1;
                if (q[mid] <= x) l = mid;
                else r = mid - 1;
            }
            cout << l << endl;
        }
    }

    return 0;
}
```

## 浮点数二分

### 与整数二分的区别

- 不需要考虑是否要 +1
- 需要考虑精度问题

### 例子：二分法求 x 的平方根

```cpp
#define _CRT_SECURE_NO_WARNINGS
#include <iostream>

using namespace std;

int main()
{
    double x;
    cin >> x;

    double l = 0, r = x;
    while (r - l > 1e-8) {
        double mid = (l + r) / 2;
        if (mid * mid >= x) r = mid;
        else l = mid;
    }

    printf("%lf\n", l);

    return 0;
}
```

## 快排

### 思想

```
┌────────────────┬────────────────┐
│     ≤x         │       ≥x       │
│                │                │
└────────────────┴────────────────┘
        ↑             ↑
        i  ────→    ←──── j
```

1. 确定分界点 x：左值、中间值、右值
2. 调整数值：使得所有小于等于 x 的数在左边，所有大于等于 x 的数在右边
3. 递归处理左右两级

### 实现方法

**暴力法**

1. 创建两个数组 a[] 和 b[]
2. 遍历 l~r，如果 q[i] < x，就将 q[i] 放入 a 数组中
3. 如果 q[i] > x，就将 q[i] 放入 b 数组中

**指针法**（推荐）

1. 创建一个指针从左往右数，当指针指着的数大于 x 的时候停止
2. 创建一个指针从右往左数，当指针指着的数小于 x 的时候停止
3. 当 i 和 j 都停止的时候交换两个指针指着的数，继续进行 1、2 步

### 模版代码

```cpp
#define _CRT_SECURE_NO_WARNINGS
#include <iostream>

using namespace std;

const int N = 1e6 + 10;
int n;
int q[N];

void quick_sort(int q[], int l, int r) {
    // 判断边界
    if (l >= r) return;

    // 确定分界点 x
    int x = q[l], i = l - 1, j = r + 1;
    while (i < j) {
        // i 指针向后移动，指导 q[i] > x
        do i++; while (q[i] < x);
        // j 指针向后移动，指导 q[j] < x
        do j--; while (q[j] > x);
        // 交换 i 和 j 指针
        if (i < j) swap(q[i], q[j]);
    }

    // 递归调用
    quick_sort(q, l, j);
    quick_sort(q, j + 1, r);
}

int main()
{
    scanf("%d", &n);
    for (int i = 0; i < n; i++) scanf("%d", &q[i]);

    quick_sort(q, 0, n-1);

    for (int i = 0; i < n; i++) printf("%d ", q[i]);

    return 0;
}
```

### 注意事项

在使用快排的时候，要注意一点就是当取 i 来作边界的时候即 `quick_sort(q, l, j-1)`，不可以使用左边界 `x=q[l]`。反之亦然，当使用 j 来作为边界的时候即 `quick_sort(q, l, j)`，就不可以使用右边界 `x=q[r]`。

## 归并排序

### 思想

```
┌─────────────────────────────────┐
│       left           right      │
└─────────────────────────────────┘
```

1. 以整个数组的中间数为分界线，分为左边和右边
2. 递归排序 left 和 right（此时左边和右边已经变成有序的数组了）
3. 归并两个有序的数组合并为一个有序的数组
   1. 比较两个数组的最小值
   2. 更小的那一个数放入新数组中，然后该指针向后移动一位
   3. 重复 1 和 2

### 代码模版

```cpp
#include <iostream>

using namespace std;

const int N = 1e6 + 10;

int n;
// q 是目标数组，tmp 是中间数组
int q[N], tmp[N];

void merge_sort(int q[], int l, int r) {
    if (l >= r) return;

    int mid = l + r >> 1;

    // 递归排序左右两边
    merge_sort(q, l, mid);
    merge_sort(q, mid + 1, r);

    // 归并数组
    // k 是新数组的下标
    int k = 0, i = l, j = mid + 1;
    while (i <= mid && j <= r)
        if (q[i] <= q[j]) tmp[k++] = q[i++];
        else tmp[k++] = q[j++];

    // 若有一边数组全部添加进了新的数组而另外一个数组还没有
    // 那么就将这个还没有排序进新的数组的数组一个一个元素的添加过去
    while (i <= mid) tmp[k++] = q[i++];
    while (j <= r) tmp[k++] = q[j++];

    // 将中间数组赋值给 q 数组
    for (i = l, j = 0; i <= r; i++, j++) q[i] = tmp[j];
}

int main()
{
    scanf("%d", &n);
    for (int i = 0; i < n; i++) scanf("%d", &q[i]);

    merge_sort(q, 0, n - 1);

    for (int i = 0; i < n; i++) printf("%d ", q[i]);

    return 0;
}
```

## 差分

### 简介

**整数前缀和的逆运算**

```
a1, a2, a3, a4, ...... an
b1, b2, b3, b4, ...... bi
```

使得：
- b1 = a1
- b2 = a2 - a1
- b3 = a3 - a2
- b4 = a4 - a3
- ...
- bn = an - a(n-1)

对 b 数组进行一次前缀和就可以求出 a 数组。

### 一维差分

**问题描述**：

当 b[i] 数组加上 c 时区间 [i, n] 所有数都加 c（只是加上 c，其他区间的数不变的情况）。
只需要在 b[i] 加 c，然后在 b[i+1] 减 c 就可以了。

> 可以把 b[i] 看成 a[i] 中所有数都加 c，而 b[i+1] 减去 c 就把 [i+1, n] 的范围撤销了，那么最后剩余的范围是 [i, i+1) 也就是 [i]。

**代码**

```cpp
#define _CRT_SECURE_NO_WARNINGS
#include <iostream>
#include <vector>

using namespace std;

const int N = 1e6 + 10;

vector<int> add(vector<int>& A, vector<int>& B) {
    vector<int> C;
    int t = 0;

    for (int i = 0; i < A.size() || i < B.size(); i++) {
        if (i < A.size()) t += A[i];
        if (i < B.size()) t += B[i];
        C.push_back(t % 10);
        t /= 10;
    }

    if (t) C.push_back(1);

    return C;
}

int main()
{
    string a, b;
    vector<int> A, B;

    cin >> a >> b;
    for (int i = a.size() - 1; i >= 0; i--) A.push_back(a[i] - '0');
    for (int i = b.size() - 1; i >= 0; i--) B.push_back(b[i] - '0');

    auto C = add(A, B);

    for (int i = C.size() - 1; i >= 0; i--) printf("%d", C[i]);

    return 0;
}
```

### 二维差分

输入一个 n 行 m 列的整数矩阵，再输入 q 个操作，每个操作包含五个整数 x1, y1, x2, y2, c，表示一个子矩阵的左上角坐标和右下角坐标 x, y。

**输入格式**

- 输入包含整数 n, m, q
- 接下来行，每行包含 m 个整数，表示整数矩阵
- 接下来行，每行包含 5 个整数 x1, y1, x2, y2, c 表示一个操作

**输出格式**

共 m 行，每行输出一个询问的结果

**解析**

从 x1y1 到 x2y2 的区间中的数据全部加 c。

```cpp
#define _CRT_SECURE_NO_WARNINGS
#include <iostream>

using namespace std;

const int N = 1010;

int n, m, q;
int a[N][N], b[N][N];

void insert(int x1, int y1, int x2, int y2, int c) {
    b[x1][y1] += c;
    b[x2 + 1][y1] -= c;
    b[x1][y2 + 1] -= c;
    b[x2 + 1][y2 + 1] += c;
}

int main()
{
    scanf("%d%d%d", &n, &m, &q);
    for (int i = 1; i <= n; i++)
        for (int j = 1; j <= m; j++)
            scanf("%d", &a[i][j]);

    for (int i = 1; i <= n; i++)
        for (int j = 1; j <= m; j++)
            insert(i, j, i, j, a[i][j]);

    while (q--) {
        int x1, y1, x2, y2, c;
        cin >> x1 >> y1 >> x2 >> y2 >> c;
        insert(x1, y1, x2, y2, c);
    }

    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= m; j++)
            b[i][j] += b[i - 1][j] + b[i][j - 1] - b[i - 1][j - 1];
    }

    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= m; j++) {
            printf("%d ", b[i][j]);
        }
        puts(" ");
    }

    return 0;
}
```

## 关键要点

1. **排序**：`sort` 默认升序，可用 `greater<T>()` 降序
2. **二分查找**：`lower_bound` 找 ≥x，`upper_bound` 找 >x
3. **快排**：分治思想，平均 O(n log n)
4. **归并排序**：稳定排序，O(n log n)
5. **差分**：前缀和的逆运算，用于区间修改
