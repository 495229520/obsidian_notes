# 14.13 前缀和

## 一维前缀和

### 问题描述

- 输出一个长度为 n 的整数序列
- 接下来输入 m 个询问，每个询问输入一对 l, r
- 对于每个询问，输出原序列中从第 l 个数到第 r 个数的和

**输入格式**

- 第一行包含两个整数 n 和 m
- 第二行包含 n 个整数，表示整数数列
- 接下来 m 行，每行包含两个整数 l 和 r

**输出格式**

共 m 行，每行输出一个询问的结果

### 代码实现

```cpp
using namespace std;

const int N = 1e6 + 10;

int n, m;
int a[N], s[N];

int main() {
    scanf("%d%d", &n, &m);
    for (int i = 1; i <= n; i++) scanf("%d", &a[i]);

    // 前缀和的初始化
    for (int i = 1; i <= n; i++) s[i] = s[i - 1] + a[i];

    while (m--) {
        int l, r;
        scanf("%d%d", &l, &r);
        // 区间和的计算
        printf("%d\n", s[r] - s[l - 1]);
    }

    return 0;
}
```

### 原理

```
sum[r] = a[1] + a[2] + a[3] + a[l-1] + a[l] + a[l+1] ...... a[r];
sum[l - 1] = a[1] + a[2] + a[3] + a[l - 1];
sum[r] - sum[l - 1] = a[l] + a[l+1] + ...... + a[r];
```

### 图解

```
sum[r]   = a[1]+a[2]+a[3]+a[l-1]+a[l]+a[l+1]......a[r]
              ↓   ↓   ↓     ↓
sum[l-1] = a[1]+a[2]+a[3]+a[l-1]

sum[r] - sum[l-1] = a[l]+a[l+1]......a[r]
```

> **关键**：前缀和数组 `s[i]` 表示原数组 `a[1]` 到 `a[i]` 的和，区间 `[l, r]` 的和为 `s[r] - s[l-1]`。

## 二维前缀和

### 图解

```
┌───────────────────────────┐
│                           │
│         S(i,j-1)          │
│                           │
├───────────┬───────────────┤
│           │               │
│  S(i-1,j) │     S(i,j)    │
│           │               │
└───────────┴───────────────┘
```

### 公式推导

**前缀和计算**：

```
S(i,j) = S(i-1,j) + S(i,j-1) - S(i-1,j-1) + a(i,j)
```

**区间和计算**：

对于左上角 (x1, y1) 到右下角 (x2, y2) 的子矩阵：

```
sum = S(x2,y2) - S(x1-1,y2) - S(x2,y1-1) + S(x1-1,y1-1)
```

### 代码实现

```cpp
const int N = 1e4 + 10;

int n, m, q;
int a[N][N], s[N][N];

int main() {
    // 输入 n 行 m 列，q 代表要计算几次区间内的和
    scanf("%d%d%d", &n, &m, &q);
    for (int i = 1; i <= n; i++)
        for (int j = 1; j <= m; j++)
            scanf("%d", &a[i][j]);

    // 前缀和的初始化
    for (int i = 1; i <= n; i++)
        for (int j = 1; j <= m; j++)
            s[i][j] = s[i - 1][j] + s[i][j - 1] - s[i - 1][j - 1] + a[i][j];  // 求前缀和

    while (q--) {
        int x1, y1, x2, y2;
        scanf("%d%d%d%d", &x1, &y1, &x2, &y2);
        // 区间和的计算，这里的区间是 [x1-x2][y1-y2]
        printf("%d\n", s[x2][y2] - s[x1 - 1][y2] - s[x2][y1 - 1] + s[x1 - 1][y1 - 1]);
    }

    return 0;
}
```

## 前缀和的应用场景

| 场景 | 说明 |
|------|------|
| 区间求和 | O(1) 时间查询任意区间和 |
| 子矩阵求和 | 二维前缀和的典型应用 |
| 与差分配合 | [[14.11 算法#差分|差分]]是前缀和的逆运算 |

---

## 关键要点

1. **一维前缀和**：`s[i] = s[i-1] + a[i]`，区间和 `[l,r] = s[r] - s[l-1]`
2. **二维前缀和**：利用容斥原理计算
3. **下标从 1 开始**：便于处理边界条件，避免 `s[0]` 的特殊处理
4. **时间复杂度**：预处理 O(n)，查询 O(1)
