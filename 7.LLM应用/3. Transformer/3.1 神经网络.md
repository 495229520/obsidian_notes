## 什么是神经网络？

你可以把神经网络看成一个可学习的函数。早期方法希望为问题写出精确规则，但现实任务往往很复杂，难以手工建模。联结主义的核心思路是：不强行写出精确公式，而是让模型从数据中学习一个近似映射。线性关系表达能力有限，于是我们引入非线性激活函数。

通过线性变化和非线性激活函数的不断组合形成了神经网络，神经网络有输入层、隐藏层和输出层。

这就是从”函数”到”神经网络”的关系。后面要做的，就是通过训练求出较合适的参数 $W$ 和 $b$。

### 更准确的数学表示

每一层的计算过程：

**第一层隐藏层**：
$$z^{(1)} = W^{(1)} \cdot x + b^{(1)}$$
$$a^{(1)} = \sigma(z^{(1)})$$

**第二层隐藏层**：
$$z^{(2)} = W^{(2)} \cdot a^{(1)} + b^{(2)}$$
$$a^{(2)} = \sigma(z^{(2)})$$

**输出层**：
$$z^{(3)} = W^{(3)} \cdot a^{(2)} + b^{(3)}$$
$$\hat{y} = g(z^{(3)})$$

其中：
- $W^{(l)}$ 是第 $l$ 层的权重矩阵
- $b^{(l)}$ 是第 $l$ 层的偏置向量
- $\sigma()$ 是激活函数（如ReLU、Sigmoid等）
- $g()$ 是输出层激活函数（回归常用恒等映射，二分类常用 Sigmoid，多分类常用 Softmax）
- $z^{(l)}$ 是线性变换的结果（未激活）
- $a^{(l)}$ 是激活后的结果

## 损失函数与优化问题

### 什么是损失函数？

好的 $W$ 和 $b$ 需要让模型尽可能贴近真实值。那“拟合得好”该如何衡量？

损失函数（Loss Function）是一个关于W和B的函数。它表示真实值与预测值之间的误差，目标就是让这个误差降到最低。

**常见的损失函数**：

- **均方误差（MSE）**：$L = \frac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2$，用于回归问题
- **交叉熵（Cross Entropy）**：用于分类问题，衡量预测概率分布与真实分布的差异
- **绝对值误差（MAE）**：$L = \frac{1}{n}\sum_{i=1}^{n}|y_i - \hat{y}_i|$

**损失函数的作用**：

损失函数 $L(W, B)$ 把"预测好坏"这个定性的概念转化成了一个**可优化的数值目标**。我们的任务就是找到使 $L(W, B)$ 最小的 $W$ 和 $B$。

### 从一元到多元的优化问题

![[Pasted image 20260214215132.png]]

**一元函数的情况**：对于函数 $f(x)$，令 $\frac{df}{dx} = 0$ 可以得到驻点候选，再结合其他条件判断是否为极值点。

**多元函数的情况**：对于损失函数 $L(W, B)$，我们有成千上万个参数。此时需要用**偏导数**来分别计算每个参数对损失函数的影响。

## 深入理解偏导数与梯度

### 为什么需要偏导数？

对于一元方程，可以令导数为 0 求出驻点（即候选的最优解）；但对于神经网络这种高维非凸问题，无法直接解方程，因此需要用**梯度下降**等迭代优化算法来逐步逼近最优的 W 和 B。

二元函数求 W 和 B 又该怎么办呢？答案是——**求偏导**。

### 偏导数的计算方法

求偏导数（Partial Derivative）听起来很高端，但如果你已经会做最基本的导数，求偏导其实只需记住一句话：**"锁死"其他变量，只看其中一个。**

假设你有一个函数 $f(x, y)$，它同时受 $x$ 和 $y$ 的影响。

#### 1. 核心口诀

- **对 $x$ 求偏导**：把 $y$ 看成一个**固定的数字**（比如 5 或 100），然后正常求导。
    
- **对 $y$ 求偏导**：把 $x$ 看成一个**固定的数字**，然后正常求导。
    

---

#### 2. 举个例子

假设函数是：

$$f(x, y) = x^2 + 3xy + y^2$$

#### 第一步：对 $x$ 求偏导（记作 $\frac{\partial f}{\partial x}$ 或 $f_x$）

这时候，我们要把 $y$ 看作常数：

- $x^2$ 的导数是 $2x$。
    
- $3xy$ 中，$3$ 和 $y$ 都是常数，所以 $3xy$ 就像 $5x$ 一样，导数是 $3y$。
    
- $y^2$ 是纯常数（因为它不含 $x$），常数的导数是 **0**。
    
    **结果：** $\frac{\partial f}{\partial x} = 2x + 3y$
    

#### 第二步：对 $y$ 求偏导（记作 $\frac{\partial f}{\partial y}$ 或 $f_y$）

这时候，我们要把 $x$ 看作常数：

- $x^2$ 是纯常数，导数是 **0**。
    
- $3xy$ 中，$3$ 和 $x$ 都是常数，导数是 $3x$。
    
- $y^2$ 的导数是 $2y$。
    
    **结果：** $\frac{\partial f}{\partial y} = 3x + 2y$
    

---

#### 3. 几何直观理解

想象你在爬一座山，山的高度由 $f(x, y)$ 决定。

- **对 $x$ 求偏导**：相当于你被困在了一根平行于 $x$ 轴的钢丝上，你只能前后走，不能左右偏。此时你感受到的坡度（斜率）就是对 $x$ 的偏导。
    
- **对 $y$ 求偏导**：同理，你只能沿着 $y$ 轴方向移动，此时的坡度就是对 $y$ 的偏导。
    

---

#### 4. 常见误区

最容易出错的地方是**复合函数**。

例如：$f(x, y) = \sin(xy)$

- 对 $x$ 求偏导：外层是 $\sin$ 变 $\cos$，内层 $xy$ 对 $x$ 求导得到 $y$。
    
- 结果：$y \cdot \cos(xy)$。

**更多常见的复合函数求偏导：**

| 函数 | 对 $x$ 求偏导 | 链式法则拆解 |
|------|--------------|------------|
| $f = e^{x^2 + y}$ | $2x \cdot e^{x^2 + y}$ | 外层 $e^u$ 导数不变，内层 $x^2+y$ 对 $x$ 导数为 $2x$ |
| $f = \ln(x^2 y)$ | $\frac{2x}{x^2 y} \cdot y = \frac{2}{x}$ | 外层 $\ln u$ 变 $\frac{1}{u}$，内层 $x^2 y$ 对 $x$ 导数为 $2xy$ |
| $f = (x + y^2)^3$ | $3(x + y^2)^2$ | 外层幂函数降幂，内层 $x+y^2$ 对 $x$ 导数为 $1$ |
| $f = e^{xy^2}$ | $y^2 \cdot e^{xy^2}$ | 外层 $e^u$ 不变，内层 $xy^2$ 对 $x$ 导数为 $y^2$ |

> [!tip] 链式法则口诀
> **"由外向内，逐层求导，再乘起来。"**
>
> 对于 $f(g(x, y))$，对 $x$ 求偏导的公式：
> $$\frac{\partial f}{\partial x} = f'(g) \cdot \frac{\partial g}{\partial x}$$

**易错点对比：**

$f(x, y) = e^{xy}$

- 对 $x$ 求偏导：$y \cdot e^{xy}$（$y$ 是常数，被"带下来"）
- 对 $y$ 求偏导：$x \cdot e^{xy}$（$x$ 是常数，被"带下来"）
- **错误写法**：$e^{xy}$（忘记对内层求导，这是最高频的错误）

> [!summary] 计算方法小结
> 求偏导就是一场**”选择性失明”**：求谁的导，眼里就只有谁，其他的统统当成冷冰冰的常数处理即可。

### 偏导数的本质

求偏导的本质，可以用一句话概括：**在多元世界里，控制变量，寻找局部的影响力。**

如果你觉得这个定义太抽象，我们可以从**”视角”**、**”变化”**和**”降维”**三个维度来彻底拆解它的本质。

---

#### 1. 它是”控制变量法”的数学表达

作为工程背景的你，一定熟悉“控制变量”。求偏导的本质就是：**假设世界上的其他因素都静止不动，只看某一个因素的变化对结果造成了多大的冲击。**

- **普通导数 $f'(x)$**：是一维的，只有一条路可以走。
    
- **偏导数 $\frac{\partial f}{\partial x}$**：是多维的。当你对 $x$ 求偏导时，本质上是你手动**“冻结”**了 $y, z$ 等其他所有轴。在这一瞬间，复杂的多维函数坍缩成了一条一维的曲线。
    

---

#### 2. 它是”切线”的降维打击

在几何上，普通导数是曲线的**切线斜率**。而求偏导的本质是**切片的斜率**。

想象你面前有一个波浪形的 3D 地形图（函数 $f(x, y)$）：

1. 你拿一把菜刀，沿着平行于 $x$ 轴的方向，“垂直”地切下一刀。
    
2. 切开后的断面是一条 **2D 的曲线**。
    
3. 这条断面上某一点的**切线斜率**，就是对 $x$ 的偏导数。
    

---

#### 3. 它是”局部影响力”的量化

在物理和工程中，偏导数的本质是**灵敏度（Sensitivity）**。

假设你正在写一个 ROS 机器人的控制算法，机器人的位置 $P$ 取决于左轮速度 $v_1$ 和右轮速度 $v_2$。

- $\frac{\partial P}{\partial v_1}$ 的本质是：如果你把左轮速度稍微加一点点（$v_2$ 保持不变），机器人的位置会发生多大的偏移？
    
- 如果偏导数值很大，说明这个变量对结果的影响非常**灵敏**；如果偏导数为 0，说明这个变量在这一刻对结果**毫无贡献**。
    

---

#### 4. 它是”梯度”的零件

这是最关键的一点：**偏导数是个体，梯度是集体。**

- **偏导数**告诉你：沿着坐标轴方向（东、西、南、北）走，坡度是多少。
    
- **梯度**（所有偏导数组成的向量）告诉你：如果你可以随意旋转身体，朝哪个方向走坡度最陡。
    

---

#### 总结：本质到底是什么？

求偏导的本质是**”局部线性化”**。

复杂的函数曲面在宏观上弯弯曲曲，难以捉摸；但当我们求偏导时，我们是在极小的局部范围内，把这个曲面看作是一个**平坦的滑梯**。

- **对于 C++ 工程师：** 它是你优化算法时，确定每个 `double` 变量调整权重的“步进方向”。
    
- **对于数学家：** 它是多维空间向一维空间的投影。
    
- **对于 AI：** 它是每一个神经元参数（$w$ 和 $b$）在犯错时，应该承担的那部分“责任”。
    

所以，当你求偏导时，你其实是在问：**"如果全世界都停下，只有我动一点点，结果会变多少？"**

## 梯度与梯度下降

### 什么是梯度？

**梯度（Gradient）** 是所有偏导数组成的向量。对于损失函数 $L(W, B)$：

$$\nabla L = \left\{\frac{\partial L}{\partial W^{(l)}}, \frac{\partial L}{\partial b^{(l)}}\right\}_{l=1}^{L}$$

**逐符号拆解：**

| 符号 | 含义 |
|------|------|
| $\nabla L$ | 损失函数 $L$ 的梯度，$\nabla$（nabla）是梯度算子 |
| $\frac{\partial L}{\partial W^{(l)}}$ | 损失函数对第 $l$ 层权重矩阵的偏导数 |
| $\frac{\partial L}{\partial b^{(l)}}$ | 损失函数对第 $l$ 层偏置向量的偏导数 |
| $\{\cdots\}_{l=1}^{L}$ | 从第 1 层到第 $L$ 层，把所有层的偏导数收集在一起 |

**这个公式在说什么？**

假设网络有 3 层，展开就是：

$$\nabla L = \left\{\frac{\partial L}{\partial W^{(1)}}, \frac{\partial L}{\partial b^{(1)}}, \frac{\partial L}{\partial W^{(2)}}, \frac{\partial L}{\partial b^{(2)}}, \frac{\partial L}{\partial W^{(3)}}, \frac{\partial L}{\partial b^{(3)}}\right\}$$

每一项回答一个问题：**"如果我微调第 $l$ 层的某个参数，损失会变大还是变小？变多少？"** 把所有层、所有参数的答案打包在一起，就是梯度。

**梯度的含义**：
- 每个分量告诉你该参数对损失的影响有多大
- 梯度的方向指向**损失函数增长最快的方向**
- 梯度的大小表示**增长的陡峭程度**

### 为什么要梯度下降？

我们的目标是**最小化损失函数** $L(W, B)$。如果我们沿着梯度方向走，损失函数会增大；如果我们沿着**梯度的反方向**走，损失函数会减小。

这就是 **梯度下降（Gradient Descent）** 的核心思想：

$$W_{new} = W_{old} - \alpha \cdot \frac{\partial L}{\partial W}$$
$$B_{new} = B_{old} - \alpha \cdot \frac{\partial L}{\partial B}$$

其中 $\alpha$ 是**学习率**，控制我们每次移动的步长。

![[Pasted image 20260214220233.png]]

**直观理解**：想象你在山上，想要走到最低点。最聪明的办法不是一步走到底，而是每次看看脚下坡度最陡的地方（梯度），然后沿着下坡方向走一小步。重复很多次，就能到达低谷。

**学习率的作用**：
- 学习率太小：下降太慢，训练时间长
- 学习率太大：可能会跳过最优点，甚至让损失增加
- 学习率合适：以适当速度找到最优点

## 前向传播与反向传播

现在问题来了：**怎样高效地计算损失函数对每个参数的偏导数？**

对于深层神经网络，直接计算偏导数非常困难，因为损失函数是由**多层嵌套的函数复合**而成的。这就需要用到**链式法则**。

### 前向传播（Forward Pass）

前向传播就是我们之前讲的：把数据从输入层送到输出层，逐层计算激活值。

**以一个简单的3层网络为例**：

给定输入 $x$，权重 $W^{(1)}, W^{(2)}, W^{(3)}$，偏差 $b^{(1)}, b^{(2)}, b^{(3)}$

**第1层**：
$$z^{(1)} = W^{(1)} x + b^{(1)}$$
$$a^{(1)} = \sigma(z^{(1)})$$

**第2层**：
$$z^{(2)} = W^{(2)} a^{(1)} + b^{(2)}$$
$$a^{(2)} = \sigma(z^{(2)})$$

**第3层（输出）**：
$$z^{(3)} = W^{(3)} a^{(2)} + b^{(3)}$$
$$\hat{y} = g(z^{(3)})$$

**计算损失**：
$$L = \text{LossFunction}(y, \hat{y})$$

其中 $y$ 是真实标签，$\hat{y}$ 是我们的预测。

### 链式法则的核心

要计算 $\frac{\partial L}{\partial W^{(1)}}$，我们需要追踪 $L$ 如何通过每一层传递回去。

**关键观察**：$L$ 依赖于 $\hat{y}$，$\hat{y}$ 依赖于 $z^{(3)}$，$z^{(3)}$ 依赖于 $a^{(2)}$，... 最终才依赖于 $W^{(1)}$。

这就是**链式法则**的应用：

$$\frac{\partial L}{\partial W^{(1)}} = \frac{\partial L}{\partial \hat{y}} \cdot \frac{\partial \hat{y}}{\partial z^{(3)}} \cdot \frac{\partial z^{(3)}}{\partial a^{(2)}} \cdot \frac{\partial a^{(2)}}{\partial z^{(2)}} \cdot \frac{\partial z^{(2)}}{\partial a^{(1)}} \cdot \frac{\partial a^{(1)}}{\partial z^{(1)}} \cdot \frac{\partial z^{(1)}}{\partial W^{(1)}}$$

**这很复杂对吧？**但反向传播利用了一个聪明的技巧：**从后往前逐层计算**。

### 反向传播（Backward Pass）

反向传播的核心思想：**复用中间计算结果，避免重复计算**。

我们定义 **误差项** $\delta$ 来表示损失函数对某个中间量的梯度：

**第3层（输出层）**：

$$\delta^{(3)} = \frac{\partial L}{\partial z^{(3)}} = \frac{\partial L}{\partial \hat{y}} \cdot \frac{\partial \hat{y}}{\partial z^{(3)}}$$

对于输出层，$\frac{\partial L}{\partial \hat{y}}$ 取决于具体的损失函数。例如，对于均方误差：

$$\frac{\partial L}{\partial \hat{y}_i} = \frac{2}{n}(\hat{y}_i - y_i)$$

而 $\frac{\partial \hat{y}}{\partial z^{(3)}}$ 取决于输出层激活函数 $g$。

**计算第3层的参数梯度**：

$$\frac{\partial L}{\partial W^{(3)}} = \delta^{(3)} (a^{(2)})^T$$
$$\frac{\partial L}{\partial b^{(3)}} = \delta^{(3)}$$

如果按 mini-batch 实现，偏置梯度需要在 batch 维上求和（或平均）。

**传播到第2层**：

现在我们需要计算 $\delta^{(2)}$。使用链式法则：

$$\delta^{(2)} = \frac{\partial L}{\partial z^{(2)}} = \frac{\partial L}{\partial z^{(3)}} \cdot \frac{\partial z^{(3)}}{\partial a^{(2)}} \cdot \frac{\partial a^{(2)}}{\partial z^{(2)}}$$

$$= ((W^{(3)})^T \delta^{(3)}) \odot \sigma'(z^{(2)})$$

其中 $\sigma'(z)$ 是激活函数的导数。

**计算第2层的参数梯度**：

$$\frac{\partial L}{\partial W^{(2)}} = \delta^{(2)} (a^{(1)})^T$$
$$\frac{\partial L}{\partial b^{(2)}} = \delta^{(2)}$$

**传播到第1层**：

$$\delta^{(1)} = ((W^{(2)})^T \delta^{(2)}) \odot \sigma'(z^{(1)})$$

$$\frac{\partial L}{\partial W^{(1)}} = \delta^{(1)} x^T$$
$$\frac{\partial L}{\partial b^{(1)}} = \delta^{(1)}$$

### 反向传播的精妙之处

反向传播之所以高效，是因为它**自下而上地计算**，每一层都复用了前一层的结果：

```
前向传播:  x → z⁽¹⁾ → a⁽¹⁾ → z⁽²⁾ → a⁽²⁾ → z⁽³⁾ → ŷ → L

反向传播:  L ← δ⁽³⁾ ← δ⁽²⁾ ← δ⁽¹⁾ ← 梯度更新
```

**效率对比**：
- 如果对每个参数**独立**地从头用链式法则求偏导：每个参数都要从输出层走到该参数所在层，大量中间结果被重复计算
- 使用反向传播：从后往前一次扫过整个网络，所有中间梯度 $\delta^{(l)}$ 被**复用**，一次反向传播即可求出全部参数的梯度

一次前向传播 + 一次反向传播的计算量都与网络的**边数（参数数量）**同阶，即 $O(E)$，其中 $E$ 是网络中所有权重的总数。反向传播的核心优势不是降低渐进复杂度，而是**避免了重复计算**——把 naive 方法中 $O(E \times L)$ 的总开销降到了 $O(E)$（$L$ 为网络深度）。

### 一次训练迭代（One Training Step）

**输入**：
- 一批样本：$\{(x^{(1)}, y^{(1)}), (x^{(2)}, y^{(2)}), ..., (x^{(m)}, y^{(m)})\}$
- 当前的参数：$W, B$
- 学习率：$\alpha$

**步骤1：前向传播**

对每个样本计算：
1. 通过所有隐藏层计算激活值：$a^{(0)} = x$，然后对每层 $l=1,2,...,L$：
   - $z^{(l)} = W^{(l)} a^{(l-1)} + b^{(l)}$
   - $a^{(l)} = \sigma(z^{(l)})$

2. 计算损失：$L = \text{LossFunction}(y, a^{(L)})$

3. 对一个batch的所有样本，计算平均损失

**步骤2：反向传播**

1. 计算输出层的误差：$\delta^{(L)} = \frac{\partial L}{\partial z^{(L)}}$

2. 从后往前逐层反向传播：对 $l = L-1, L-2, ..., 1$：
   - $\delta^{(l)} = ((W^{(l+1)})^T \delta^{(l+1)}) \odot \sigma'(z^{(l)})$
   - 其中 $\odot$ 表示逐元素乘法（Hadamard积）

3. 计算参数梯度：
   - $\frac{\partial L}{\partial W^{(l)}} = \delta^{(l)} (a^{(l-1)})^T$
   - $\frac{\partial L}{\partial b^{(l)}} = \delta^{(l)}$

**步骤3：参数更新**

对所有参数执行梯度下降：

$$W^{(l)} \leftarrow W^{(l)} - \alpha \cdot \frac{\partial L}{\partial W^{(l)}}$$
$$b^{(l)} \leftarrow b^{(l)} - \alpha \cdot \frac{\partial L}{\partial b^{(l)}}$$

### 可视化流程图

```
┌─────────────────────────────────────────────────────────┐
│                    一次训练迭代                           │
└─────────────────────────────────────────────────────────┘
                            │
                            ▼
        ┌─────────────────────────────────┐
        │  前向传播（Forward Pass）         │
        │  计算所有激活值和损失              │
        │  x → a⁽¹⁾ → a⁽²⁾ → ... → L        │
        └─────────────────────────────────┘
                            │
                            ▼
        ┌─────────────────────────────────┐
        │  反向传播（Backward Pass）        │
        │  计算所有梯度                     │
        │  L → δ⁽ᴸ⁾ → δ⁽ᴸ⁻¹⁾ → δ⁽¹⁾            │ 
        │  ↓ ↓ ↓ → ∂L/∂W, ∂L/∂b           │
        └─────────────────────────────────┘
                            │
                            ▼
        ┌─────────────────────────────────┐
        │  参数更新（Parameter Update）     │
        │  W ← W - α·∂L/∂W                │
        │  b ← b - α·∂L/∂b                │
        └─────────────────────────────────┘
```

### 完整的训练算法（伪代码）

```
Algorithm: Mini-batch Stochastic Gradient Descent (SGD)
Input: Training data, initial parameters W and b, learning rate α
Output: Trained parameters W and b

# 约定：一个 mini-batch 按列堆叠
# a[l] shape = (n_l, batch_size), W[l] shape = (n_l, n_{l-1})

while not converged:
    for each mini-batch in training set:
        # 前向传播
        a[0] = x_batch
        for l = 1 to L:
            z[l] = W[l] @ a[l-1] + b[l]
            a[l] = activation(z[l])

        # 计算损失
        loss = compute_loss(y_batch, a[L])

        # 反向传播
        delta[L] = compute_output_delta(y_batch, a[L])
        for l = L-1 down to 1:
            delta[l] = (W[l+1].T @ delta[l+1]) * activation_derivative(z[l])

        # 参数梯度
        for l = 1 to L:
            dW[l] = (delta[l] @ a[l-1].T) / batch_size
            db[l] = sum(delta[l], axis=1, keepdims=True) / batch_size

        # 梯度下降
        for l = 1 to L:
            W[l] = W[l] - α * dW[l]
            b[l] = b[l] - α * db[l]

    # 验证集评估、学习率调整等...
```

## 总结：神经网络训练的本质

现在你应该理解了：

1. **神经网络是什么**：由多层线性变换和非线性激活函数组合而成的复杂函数

2. **损失函数是什么**：衡量预测与真实值的偏差，是我们要最小化的目标

3. **梯度是什么**：损失函数对参数的偏导数，指向损失增大的方向

4. **梯度下降为什么有效**：沿着梯度的反方向走，可以逐步降低损失

5. **反向传播为什么高效**：通过链式法则和动态规划思想，高效地计算所有参数的梯度

6. **一次训练的完整流程**：前向传播计算损失 → 反向传播计算梯度 → 梯度下降更新参数

**关键直觉**：
- **前向传播**就是函数求值
- **反向传播**就是微积分链式法则的应用
- **梯度下降**就是数值优化

这三个步骤的组合，就构成了现代深度学习的基础。
