## 环境准备

### 1. 安装 Python

**Windows**：

1. 访问 [python.org](https://www.python.org/downloads/)
2. 下载最新稳定版（推荐 3.11+）
3. **重要**：安装时勾选 "Add Python to PATH"

```bash
# 验证安装
python --version
# 应输出: Python 3.11.x
```

**常见问题**：如果提示 `python` 不是内部命令，说明没有添加到 PATH，需要手动配置环境变量。

### 2. 安装 VS Code

1. 访问 [code.visualstudio.com](https://code.visualstudio.com/)
2. 下载并安装
3. 安装 Python 扩展：
   - 打开 VS Code
   - 按 `Ctrl+Shift+X` 打开扩展面板
   - 搜索 "Python"，安装 Microsoft 官方扩展

### 3. 安装 Git

1. 访问 [git-scm.com](https://git-scm.com/)
2. 下载并安装（Windows 用户选择默认选项即可）

```bash
# 验证安装
git --version
# 应输出: git version 2.x.x

# 配置用户信息（首次使用必须）
git config --global user.name "你的名字"
git config --global user.email "你的邮箱"
```

### 验证环境

创建一个测试文件，确保环境正常：

```bash
# 创建工作目录
mkdir python-learning
cd python-learning

# 创建测试文件
echo print("Hello, Python!") > test.py

# 运行
python test.py
# 应输出: Hello, Python!
```

## 核心概念

### 什么是 Python？

**定义**：Python 是一门高级编程语言，以简洁易读著称，广泛用于 Web 开发、数据分析、人工智能等领域。

> **Guido van Rossum**（Python 创始人）："Python 的设计哲学是代码可读性很重要。"

#### 为什么选择 Python 学习 LLM？

- **问题**：LLM 开发涉及大量库（PyTorch、Transformers、LangChain），需要一门"胶水语言"把它们串起来
- **解决**：Python 生态最丰富，几乎所有 AI/ML 工具都优先支持 Python

#### Python 程序的基本结构

```python
# 这是注释，解释代码用途
# Python 用缩进表示代码块，不用大括号

print("Hello, World!")  # 这是一条语句
```

**关键点**：
- `#` 开头的是注释，Python 会忽略
- `print()` 是内置函数，用于输出
- 每条语句独占一行（不需要分号）

## 代码实践

### 脚本1：hello.py - 最简单的输出

**目标**：理解 `print()` 函数的基本用法

```python
# hello.py - 你的第一个 Python 程序

print("Hello, World!")
print("你好，Python！")
print("=" * 20)  # 打印20个等号
```

**关键点**：
- `print()` 可以输出字符串（用引号包裹的文本）
- 字符串可以用单引号 `'...'` 或双引号 `"..."`
- `*` 可以重复字符串

**运行方式**：

```bash
python hello.py
```

**预期输出**：

```
Hello, World!
你好，Python！
====================
```

### 脚本2：greeting.py - 获取用户输入

**目标**：学习 `input()` 函数，实现交互

```python
# greeting.py - 交互式问候程序

# input() 会暂停程序，等待用户输入
name = input("请输入你的名字: ")

# 使用 f-string 格式化输出（Python 3.6+）
print(f"你好，{name}！欢迎学习 Python！")

# 也可以用字符串拼接
print("你好，" + name + "！")
```

**关键点**：
- `input("提示文字")` 显示提示并等待用户输入
- `name = ...` 把输入的值存到变量 `name` 中
- `f"...{变量}..."` 是格式化字符串，可以嵌入变量
- `+` 可以拼接字符串

**运行方式**：

```bash
python greeting.py
```

**预期输出**：

```
请输入你的名字: 小明
你好，小明！欢迎学习 Python！
你好，小明！
```

### 脚本3：calc.py - 简单计算器

**目标**：学习基本运算和类型转换

```python
# calc.py - 简单计算器

print("=== 简单计算器 ===")

# 获取两个数字
# 注意：input() 返回的是字符串，需要转换为数字
num1 = input("请输入第一个数: ")
num2 = input("请输入第二个数: ")

# 转换为浮点数（可以处理小数）
a = float(num1)
b = float(num2)

# 四则运算
print(f"\n计算结果：")
print(f"{a} + {b} = {a + b}")
print(f"{a} - {b} = {a - b}")
print(f"{a} * {b} = {a * b}")
print(f"{a} / {b} = {a / b}")

# 额外运算
print(f"\n额外运算：")
print(f"{a} 的 {b} 次方 = {a ** b}")
print(f"{a} 整除 {b} = {a // b}")
print(f"{a} 取余 {b} = {a % b}")
```

**关键点**：
- `input()` 返回的永远是**字符串**，不能直接做数学运算
- `float()` 把字符串转换为浮点数（小数）
- `int()` 把字符串转换为整数
- Python 运算符：`+` `-` `*` `/`（除法）`**`（幂）`//`（整除）`%`（取余）

**运行方式**：

```bash
python calc.py
```

**预期输出**：

```
=== 简单计算器 ===
请输入第一个数: 10
请输入第二个数: 3

计算结果：
10.0 + 3.0 = 13.0
10.0 - 3.0 = 7.0
10.0 * 3.0 = 30.0
10.0 / 3.0 = 3.3333333333333335

额外运算：
10.0 的 3.0 次方 = 1000.0
10.0 整除 3.0 = 3.0
10.0 取余 3.0 = 1.0
```

## 踩坑记录

### 问题1：`python` 命令找不到

**症状**：

```
'python' 不是内部或外部命令，也不是可运行的程序
```

**原因**：Python 没有添加到系统 PATH 环境变量

**解决方案**：
1. 重新运行 Python 安装程序，勾选 "Add Python to PATH"
2. 或手动添加：`C:\Users\你的用户名\AppData\Local\Programs\Python\Python3x\` 到系统 PATH

### 问题2：中文乱码

**症状**：运行包含中文的脚本时显示乱码

**原因**：文件编码不是 UTF-8

**解决方案**：
1. VS Code 右下角点击编码，选择 "Save with Encoding" → UTF-8
2. 或在文件开头添加：`# -*- coding: utf-8 -*-`

### 问题3：类型错误

**症状**：

```python
num = input("输入数字: ")
result = num + 10  # TypeError!
```

**原因**：`input()` 返回字符串，不能和数字相加

**解决方案**：

```python
num = input("输入数字: ")
result = int(num) + 10  # 先转换为整数
```

## 最佳实践

| 推荐做法 | 避免做法 | 原因 |
|----------|----------|------|
| 用 f-string 格式化 `f"{var}"` | 用 `+` 拼接多个变量 | f-string 更清晰，性能更好 |
| 变量名用小写+下划线 `user_name` | 用驼峰 `userName` | Python 惯例（PEP 8） |
| 写注释说明代码意图 | 不写任何注释 | 方便以后理解 |
| 及时保存并运行测试 | 写完一大段再运行 | 更容易定位错误 |
