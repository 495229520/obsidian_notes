## 1. 引入：从范数到内积

### 为什么需要内积？

在 [[第03次课-范数空间笔记|上一讲]] 中，我们学习了赋范线性空间——在线性空间上定义范数 $\|\mathbf{x}\|$，从而衡量元素的"大小"。但范数空间有一个局限：**它只能衡量"长度"，无法衡量两个元素之间的"角度"和"正交性"**。

| 概念 | 能做什么 | 不能做什么 |
|------|----------|------------|
| 距离 $d(x,y)$ | 衡量两元素间的远近 | 衡量大小、角度 |
| 范数 $\|x\|$ | 衡量单个元素的大小 | 衡量角度、正交性 |
| 内积 $(x,y)$ | 衡量大小 + 角度 + 正交性 | — |

在 $\mathbb{R}^n$ 中，我们熟悉的**向量点积**就是内积的一个例子：

$$\vec{a} \cdot \vec{b} = |\vec{a}||\vec{b}|\cos\theta$$

内积告诉我们：两个向量有多"相似"（夹角小 → 内积大），是否"垂直"（内积为零）。

**核心关系**：有了内积，就能自然地定义范数：

$$\|x\| = \sqrt{(x, x)}$$

> 内积比范数更"强"：每个内积都能诱导出范数，但不是每个范数都能由内积诱导。

## 2. 内积空间的定义

### 2.1 内积的定义

> **定义1（内积）**：设 $U$ 是数域 $K$（$K = \mathbb{R}$ 或 $\mathbb{C}$）上的线性空间。若存在一个二元泛函 $(x, y): U \times U \to K$，满足以下三条公理，则称 $(x, y)$ 为 $U$ 上的**内积**，称 $(U, (\cdot, \cdot))$ 为**内积空间**。

| 公理 | 表达式 | 直观含义 |
|------|--------|----------|
| **共轭对称性** | $(x, y) = \overline{(y, x)}$ | 交换顺序取共轭（实数情况下退化为对称性） |
| **第一变元线性** | $(\alpha x + \beta y, z) = \alpha(x, z) + \beta(y, z)$ | 第一个变元满足线性性 |
| **正定性** | $(x, x) \geq 0$，且 $(x, x) = 0 \Leftrightarrow x = \mathbf{0}$ | 自身的内积非负，且仅零元素为零 |

**与范数公理的对比**：

| | 范数公理 | 内积公理 |
|---|---------|---------|
| 衡量对象 | 单个元素的"大小" | 两个元素的"关系" |
| 核心公理数 | 3条（正定、齐次、三角） | 3条（共轭对称、线性、正定） |
| 能否定义角度 | 不能 | 能 |

### 2.2 第二变元的共轭线性

内积关于**第一变元是线性的**，但关于**第二变元是共轭线性的**：

由公理直接推导：

$$\begin{aligned}
(x, \alpha y + \beta z) &= \overline{(\alpha y + \beta z, x)} \quad \text{（共轭对称性）} \\
&= \overline{\alpha(y, x) + \beta(z, x)} \quad \text{（第一变元线性）} \\
&= \bar{\alpha}\overline{(y, x)} + \bar{\beta}\overline{(z, x)} \\
&= \bar{\alpha}(x, y) + \bar{\beta}(x, z)
\end{aligned}$$

> **注意**：当 $K = \mathbb{R}$ 时，$\bar{\alpha} = \alpha$，此时内积关于两个变元都是线性的（双线性）。只有在复数域上才需要区分"线性"和"共轭线性"。

### 2.3 实内积空间 vs 复内积空间

| 类型 | 数域 $K$ | 共轭对称性退化为 | 第二变元性质 |
|------|---------|----------------|------------|
| **实内积空间** | $\mathbb{R}$ | $(x, y) = (y, x)$（对称性） | 线性（双线性） |
| **复内积空间** | $\mathbb{C}$ | $(x, y) = \overline{(y, x)}$ | 共轭线性 |

> 通常在数值分析课程中，如不特别说明，讨论的是**实内积空间**。

## 3. 内积空间的例子

### 例1：$\mathbb{R}^n$ 上的内积

设 $\mathbf{x} = (x_1, \ldots, x_n)^T$，$\mathbf{y} = (y_1, \ldots, y_n)^T \in \mathbb{R}^n$，定义：

$$(\mathbf{x}, \mathbf{y}) = \sum_{i=1}^{n} x_i y_i$$

**验证三条公理**：

| 公理 | 验证 |
|------|------|
| 对称性 | $\sum x_i y_i = \sum y_i x_i$ ✓ |
| 线性性 | $\sum (\alpha x_i + \beta z_i) y_i = \alpha \sum x_i y_i + \beta \sum z_i y_i$ ✓ |
| 正定性 | $\sum x_i^2 \geq 0$，且等于 $0 \Leftrightarrow$ 所有 $x_i = 0$ ✓ |

这就是我们熟悉的**向量点积**（dot product）。

### 例1'：$\mathbb{C}^n$ 上的内积

在复向量空间 $\mathbb{C}^n$ 中：

$$(\mathbf{x}, \mathbf{y}) = \sum_{i=1}^{n} x_i \overline{y_i}$$

> 注意第二个变元取了**复共轭** $\overline{y_i}$，这保证了 $(x, x) = \sum |x_i|^2 \geq 0$。

### 例2：$L^2[a,b]$ 上的内积

若 $U = L^2[a,b]$（**平方可积**实值函数按“几乎处处相等”划分后的等价类空间），定义：

$$\boxed{(x, y) = \int_a^b x(t) y(t) \, dt}$$

**验证三条公理**：

| 公理 | 验证 |
|------|------|
| 对称性 | $\int_a^b x(t)y(t) dt = \int_a^b y(t)x(t) dt$ ✓ |
| 线性性 | $\int_a^b [\alpha x(t) + \beta z(t)] y(t) dt = \alpha \int_a^b x(t)y(t) dt + \beta \int_a^b z(t)y(t) dt$ ✓（积分线性性） |
| 正定性 | $\int_a^b x^2(t) dt \geq 0$，且等于 $0 \Leftrightarrow x = 0$（即该等价类几乎处处为零）✓ |

> **关键理解**：函数空间的内积是通过**积分**实现的——把有限维的"求和" $\sum x_i y_i$ 推广为无限维的"积分" $\int x(t)y(t)dt$。详见 [[微积分基础回顾#12. 函数内积与正交性|微积分基础：函数内积]]。

### 例3：$C[a,b]$ 中的不同定义

在连续函数空间 $C[a,b]$ 中：

| 定义 | 是否是内积 | 原因 |
|------|:--------:|------|
| $(x, y) = \int_a^b x(t)y(t) dt$ | ✓ | 满足三条公理 |
| $(x, y) = \max_{t \in [a,b]} x(t)y(t)$ | ✗ | **不满足**内积公理（不满足线性性和正定性） |

> **注意**：$C[a,b]$ 上用积分定义的内积是合法的，但 $\max$ 定义不是。

## 4. Cauchy-Schwarz 不等式（内积空间版）

### 4.1 定理与证明

> **性质1（Cauchy-Schwarz 不等式）**：在内积空间 $U$ 中，$\forall x, y \in U$：
>
> $$\boxed{|(x, y)| \leq \sqrt{(x, x)} \cdot \sqrt{(y, y)}}$$
>
> 即 $|(x, y)| \leq \|x\| \cdot \|y\|$

**证明**：

对任意 $\lambda \in K$，由内积的正定性：

$$(x - \lambda y, \, x - \lambda y) \geq 0$$

展开（利用线性性和共轭线性性）：

$$(x, x) - \lambda(y, x) - \bar{\lambda}(x, y) + |\lambda|^2 (y, y) \geq 0$$

**关键步骤**：取 $\lambda = \dfrac{(x, y)}{(y, y)}$（假设 $y \neq \mathbf{0}$），代入得：

$$(x, x) - \frac{|(x, y)|^2}{(y, y)} \geq 0$$

即：

$$|(x, y)|^2 \leq (x, x) \cdot (y, y)$$

两边开方得 $|(x, y)| \leq \|x\| \cdot \|y\|$。$\square$

> **等号成立条件**：$x$ 与 $y$ **线性相关**，即 $x = \alpha y$ 对某个 $\alpha \in K$。

### 4.2 Cauchy-Schwarz 不等式的具体形式

不同空间的内积给出不同形式的 Cauchy-Schwarz 不等式：

**【1】$\mathbb{R}^n$ 中**（离散形式）：

$$\left(\sum_{i=1}^{n} x_i y_i\right)^2 \leq \left(\sum_{i=1}^{n} x_i^2\right)\left(\sum_{i=1}^{n} y_i^2\right)$$

> 这正是 [[微积分基础回顾#9.1 Cauchy-Schwarz 不等式|微积分基础回顾]] 中用判别式法证明的形式。

**【2】$C[a,b]$ 或 $L^2[a,b]$ 中**（积分形式）：

$$\left(\int_a^b f(t)g(t) \, dt\right)^2 \leq \int_a^b f^2(t) \, dt \cdot \int_a^b g^2(t) \, dt$$

> 详见 [[微积分基础回顾#12.3 Cauchy-Schwarz 积分不等式|微积分基础：Cauchy-Schwarz 积分不等式]]。

**【3】矩阵内积形式**：

以矩阵 $A, B \in \mathbb{R}^{n \times n}$ 的 Frobenius 内积 $(A, B) = \sum_{i,j} a_{ij} b_{ij}$ 为例：

$$\left(\sum_{i=1}^{n}\sum_{j=1}^{n} a_{ij} b_{ij}\right)^2 \leq \left(\sum_{i,j} a_{ij}^2\right)\left(\sum_{i,j} b_{ij}^2\right)$$

**【4】加权内积形式**：

若 $A \in \mathbb{R}^{n \times n}$ 是正定对称矩阵，定义加权内积 $(x, y) = \sum_{i,j} a_{ij} x_j y_i$，则：

$$\left(\sum_{i,j} a_{ij} x_j y_i\right)^2 \leq \left(\sum_{i,j} a_{ij} x_j x_i\right)\left(\sum_{i,j} a_{ij} y_j y_i\right)$$

## 5. 内积诱导范数

### 5.1 核心定理

> **性质2**：在内积空间 $U$ 中，定义 $\|x\| = \sqrt{(x, x)}$，则 $\|x\|$ 满足范数的三条公理。

**验证**：

| 公理 | 验证 |
|------|------|
| **正定性** | $(x, x) \geq 0$，$(x, x) = 0 \Leftrightarrow x = \mathbf{0}$ ✓（内积公理直接保证） |
| **齐次性** | $\|\alpha x\| = \sqrt{(\alpha x, \alpha x)} = \sqrt{|\alpha|^2 (x, x)} = |\alpha| \|x\|$ ✓ |
| **三角不等式** | 需要利用 Cauchy-Schwarz 不等式（见下） |

**三角不等式的证明**：

$$\|x + y\|^2 = (x + y, x + y) = (x, x) + (x, y) + (y, x) + (y, y)$$

$$= \|x\|^2 + 2\text{Re}(x, y) + \|y\|^2$$

由 $\text{Re}(x, y) \leq |(x, y)| \leq \|x\| \cdot \|y\|$（Cauchy-Schwarz），得：

$$\|x + y\|^2 \leq \|x\|^2 + 2\|x\|\|y\| + \|y\|^2 = (\|x\| + \|y\|)^2$$

两边开方：$\|x + y\| \leq \|x\| + \|y\|$ $\square$

### 5.2 具体的诱导范数

| 空间 | 内积 | 诱导的范数 |
|------|------|-----------|
| $\mathbb{R}^n$ | $(x, y) = \sum x_i y_i$ | $\|x\|_2 = \sqrt{\sum x_i^2}$（欧氏范数） |
| $L^2[a,b]$ | $(x, y) = \int_a^b x(t)y(t) dt$ | $\|x\| = \sqrt{\int_a^b x^2(t) dt}$ |

> **重要结论**：内积空间一定是赋范线性空间（在其诱导的范数下）。

## 6. 三大空间的关系

### 6.1 层次递进

$$\text{内积空间} \subset \text{赋范线性空间} \subset \text{距离空间}$$

具体来说：

| 层次 | 结构 | 方向 |
|------|------|------|
| 内积 → 范数 | $\|x\| = \sqrt{(x, x)}$ | 每个内积都能诱导范数 ✓ |
| 范数 → 距离 | $d(x, y) = \|x - y\|$ | 每个范数都能诱导距离 ✓ |
| 距离 → 范数 | 不一定 | 有些距离不能由范数诱导 ✗ |
| 范数 → 内积 | 不一定 | 有些范数不能由内积诱导 ✗ |

### 6.2 不能由内积诱导的范数举例

$C[a,b]$ 上的 $\max$ 范数 $\|x\| = \max_{t \in [a,b]} |x(t)|$ **不能**由任何内积诱导。

$\mathbb{R}^n$ 上的 $1$-范数和 $\infty$-范数**也不能**由内积诱导。

**那么，如何判断一个范数能否由内积诱导？** 答案就是**平行四边形公式**。

## 7. 平行四边形公式

### 7.1 定理

> **性质3（平行四边形公式）**：在内积空间 $U$ 中，按内积导出的范数满足：
>
> $$\boxed{\|x + y\|^2 + \|x - y\|^2 = 2(\|x\|^2 + \|y\|^2)}$$

**证明**：

$$\|x + y\|^2 = (x+y, x+y) = (x,x) + (x,y) + (y,x) + (y,y)$$

$$\|x - y\|^2 = (x-y, x-y) = (x,x) - (x,y) - (y,x) + (y,y)$$

两式相加，交叉项 $(x,y) + (y,x)$ 和 $-(x,y) - (y,x)$ **恰好抵消**：

$$\|x+y\|^2 + \|x-y\|^2 = 2(x,x) + 2(y,y) = 2(\|x\|^2 + \|y\|^2) \quad \square$$

### 7.2 几何意义

在 $\mathbb{R}^2$ 中，$\|x+y\|$ 和 $\|x-y\|$ 分别是平行四边形的两条**对角线**长度，$\|x\|$ 和 $\|y\|$ 是两条**边**长度：

$$\text{两对角线平方和} = 2 \times \text{两边平方和}$$

这就是几何中的**平行四边形对角线定理**。详见 [[微积分基础回顾#12.4 平行四边形公式的几何背景|微积分基础：平行四边形公式]]。

### 7.3 反例：$C[0,1]$ 的 $\max$ 范数不满足平行四边形公式

取 $x(t) = 1$，$y(t) = t$，$t \in [0, 1]$：

| 函数 | $\max$ 范数 |
|------|------------|
| $x(t) = 1$ | $\|x\| = \max(1) = 1$ |
| $y(t) = t$ | $\|y\| = \max(t) = 1$ |
| $x(t) + y(t) = 1 + t$ | $\|x+y\| = \max(1+t) = 2$ |
| $x(t) - y(t) = 1 - t$ | $\|x-y\| = \max(1-t) = 1$ |

验证：

$$\|x+y\|^2 + \|x-y\|^2 = 4 + 1 = 5$$

$$2(\|x\|^2 + \|y\|^2) = 2(1 + 1) = 4$$

$$5 \neq 4 \quad \Rightarrow \quad \text{不满足平行四边形公式}$$

因此 $C[a,b]$ 按 $\max$ 范数**不是**内积空间。

> **课后练习**：在 $\mathbb{R}^2$ 中举反例说明 $1$-范数和 $\infty$-范数也不满足平行四边形公式。

## 8. 判别定理：范数何时由内积诱导？

### 8.1 定理（充要条件）

> **判别定理**：赋范线性空间 $X$ 的**给定范数**满足平行四边形公式
>
> $$\|x + y\|^2 + \|x - y\|^2 = 2(\|x\|^2 + \|y\|^2)$$
>
> $\Longleftrightarrow$ 该范数可以由某个内积诱导。

### 8.2 由范数构造内积（极化恒等式）

**实空间**的情况：若范数满足平行四边形公式，定义：

$$\boxed{(x, y) = \frac{1}{4}\left(\|x + y\|^2 - \|x - y\|^2\right)}$$

可以验证此定义满足内积的三条公理。

**复空间**的情况：

$$(x, y) = \frac{1}{4}\left(\|x + y\|^2 - \|x - y\|^2\right) + \frac{i}{4}\left(\|x + iy\|^2 - \|x - iy\|^2\right)$$

> **核心结论**：**一个给定范数是否来自某个内积，关键判据就是它是否满足平行四边形公式。**

### 8.3 为什么叫"极化恒等式"？

在内积空间中，已知 $\|x\| = \sqrt{(x,x)}$，极化恒等式反过来从范数"恢复"了内积：

$$\|x+y\|^2 = \|x\|^2 + 2(x,y) + \|y\|^2$$

$$\|x-y\|^2 = \|x\|^2 - 2(x,y) + \|y\|^2$$

两式相减：

$$\|x+y\|^2 - \|x-y\|^2 = 4(x,y)$$

$$\therefore (x, y) = \frac{1}{4}(\|x+y\|^2 - \|x-y\|^2)$$

## 9. Hilbert 空间

### 9.1 定义

> **定义**：**完备的**内积空间称为 **Hilbert 空间**，记作 $H$。
>
> 即：内积空间 $U$ 按范数 $\|x\| = \sqrt{(x, x)}$ 成为 Banach 空间时，$U$ 就是 Hilbert 空间。

| 概念 | 定义 | 关系 |
|------|------|------|
| 内积空间 | 线性空间 + 内积 | 基本结构 |
| Hilbert 空间 | **完备的**内积空间 | 更强条件 |

对比 [[第03次课-范数空间笔记#7.4 完备性与 Banach 空间|Banach 空间]]：

| | 出发点 | 完备化后 |
|---|--------|---------|
| 赋范线性空间 + 完备 | = **Banach 空间** |
| 内积空间 + 完备 | = **Hilbert 空间** |

### 9.2 内积的连续性

> **性质4**：在内积空间 $U$ 中，内积 $(x, y)$ 是两个变元 $x, y$ 的**连续泛函**。

即当 $x_n \to x$，$y_n \to y$（按范数收敛）时：

$$(x_n, y_n) \to (x, y)$$

等价地：

$$\lim_{n \to \infty} (x_n, y_n) = \left(\lim_{n \to \infty} x_n, \lim_{n \to \infty} y_n\right) = (x, y)$$

> **实用意义**：内积运算和极限运算可以交换顺序。这在数值计算中非常重要——逼近函数的内积可以用来近似原函数的内积。

### 9.3 常见 Hilbert 空间

| 空间 | 内积 | 诱导范数 | 是否完备 | 是否 Hilbert |
|------|------|---------|:--------:|:-----------:|
| $\mathbb{R}^n$ | $\sum x_i y_i$ | $\sqrt{\sum x_i^2}$ | ✓ | ✓ |
| $L^2[a,b]$ | $\int_a^b x(t)y(t)dt$ | $\sqrt{\int_a^b x^2(t)dt}$ | ✓ | ✓ |
| $l^2$ | $\sum_{i=1}^{\infty} x_i y_i$ | $\sqrt{\sum_{i=1}^{\infty} x_i^2}$ | ✓ | ✓ |
| $(C[a,b], \int xy\,dt)$ | $\int_a^b x(t)y(t)dt$ | $\sqrt{\int_a^b x^2(t)dt}$ | ✗ | ✗ |

> **关键区别**：$C[a,b]$ 在积分内积下**不是**完备的（存在连续函数的 Cauchy 列收敛到不连续函数），因此不是 Hilbert 空间。而 $L^2[a,b]$ 恰好是 $C[a,b]$ 在该范数下的**完备化**。

### 9.4 $C[a,b]$ 的不同身份

| 范数选择 | 诱导距离 | 完备？ | 空间类型 |
|---------|---------|:------:|---------|
| $\|x\| = \max_{t} |x(t)|$ | $d = \max |x(t) - y(t)|$ | ✓ | Banach 空间（但**不是**内积空间） |
| $\|x\| = \sqrt{\int x^2(t)dt}$（积分范数） | $d = \sqrt{\int (x-y)^2 dt}$ | ✗ | 内积空间（但**不是** Hilbert 空间） |

> 同一个集合 $C[a,b]$，配上不同的范数，空间性质截然不同！

## 10. 正交与正交分解

### 10.1 正交的定义

设 $U$ 是内积空间，$x, y \in U$，$M, N \subset U$ 是集合。

| 概念 | 定义 | 记号 |
|------|------|------|
| $x$ 与 $y$ **正交** | $(x, y) = 0$ | $x \perp y$ |
| $x$ 与集合 $N$ **正交** | $\forall y \in N$，$(x, y) = 0$ | $x \perp N$ |
| 集合 $M$ 与 $N$ **正交** | $\forall x \in M, \forall y \in N$，$(x, y) = 0$ | $M \perp N$ |

> **零元素是唯一一个与所有元素都正交的元素**：若 $\forall y \in U$，$(x, y) = 0$，取 $y = x$ 得 $(x, x) = 0$，故 $x = \mathbf{0}$。

### 10.2 正交补

> **定义**：$U$ 中与集合 $M$ 正交的所有元素的全体称为 $M$ 的**正交补**，记作 $M^\perp$：
>
> $$M^\perp = \{y \in U \mid (y, x) = 0, \; \forall x \in M\}$$

**$M^\perp$ 的重要性质**：

| 性质 | 内容 | 证明要点 |
|------|------|---------|
| $M^\perp$ 是**线性子空间** | 若 $x, y \in M^\perp$，则 $\alpha x + \beta y \in M^\perp$ | $(\alpha x + \beta y, z) = \alpha(x, z) + \beta(y, z) = 0$ |
| $M^\perp$ 是**闭**子空间 | 若 $x_n \in M^\perp$ 且 $x_n \to x$，则 $x \in M^\perp$ | $(x, z) = \lim(x_n, z) = \lim 0 = 0$（内积连续性） |

### 10.3 正交分解

> **定义**：设 $M$ 是 $U$ 的线性子空间，$x \in U$。若存在 $x_0 \in M$，$x_1 \in M^\perp$，使得
>
> $$x = x_0 + x_1$$
>
> 则称 $x_0$ 为 $x$ 在 $M$ 上的**正交投影**，上式称为 $x$ 的**正交分解**。

**几何图像**（$\mathbb{R}^3$ 中的类比）：

```
        x
       /|
      / |
     /  | x₁ (⊥ M)
    /   |
   -----x₀-------→ M
   正交投影
```

$x_0$ 是 $x$ 在子空间 $M$ 上的"影子"，$x_1 = x - x_0$ 是垂直于 $M$ 的分量。

## 11. 正交分解的性质

### 11.1 商高定理（勾股定理推广）

> **性质**：在内积空间中，若 $x \perp y$，则：
>
> $$\|x + y\|^2 = \|x\|^2 + \|y\|^2$$

**证明**：

$$\|x + y\|^2 = (x + y, x + y) = (x, x) + \underbrace{(x, y)}_{=0} + \underbrace{(y, x)}_{=0} + (y, y) = \|x\|^2 + \|y\|^2 \quad \square$$

> 这正是**勾股定理**在内积空间中的推广——直角三角形中斜边的平方等于两直角边的平方和。

### 11.2 最佳逼近定理

> **性质**：设 $x_0$ 是 $x$ 在 $M$ 上的正交投影，则 $x_0$ 是 $M$ 中**最接近** $x$ 的元素：
>
> $$\|x - x_0\| = \min_{y \in M} \|x - y\|$$

**证明**：对任意 $y \in M$，设 $x_1 = x - x_0 \in M^\perp$，则：

$$\|x - y\|^2 = \|x - x_0 + x_0 - y\|^2 = \|\underbrace{x_1}_{\in M^\perp} + \underbrace{(x_0 - y)}_{\in M}\|^2$$

由 $x_1 \perp (x_0 - y)$（因为 $x_1 \in M^\perp$，$x_0 - y \in M$），利用商高定理：

$$\|x - y\|^2 = \|x_1\|^2 + \|x_0 - y\|^2 \geq \|x_1\|^2 = \|x - x_0\|^2$$

等号在 $y = x_0$ 时取到。$\square$

> **核心结论**：在内积空间中，**正交投影 = 最佳逼近元**。最佳逼近问题就是正交投影问题。

## 12. 投影定理

### 12.1 定理

> **投影定理**：设 $M$ 是 Hilbert 空间 $H$ 中的**闭线性子空间**，则 $\forall x \in H$，必存在唯一的 $x_0 \in M$，$x_1 \in M^\perp$，使得：
>
> $$x = x_0 + x_1$$

**条件总结**：

| 条件 | 原因 |
|------|------|
| $H$ 是 Hilbert 空间（完备） | 保证最小化序列收敛 |
| $M$ 是**闭**子空间 | 保证极限点在 $M$ 内 |
| $M$ 是**线性**子空间 | 保证投影的唯一性 |

> **推广**：若 $M$ 是内积空间 $U$ 的**完备**子空间，定理结论仍成立。
>
> **注意**：任何有限维赋范线性空间一定是完备的。因此有限维子空间的投影总是存在的。

## 13. 正交投影的计算

### 13.1 问题设置

设 $U$ 为完备的内积空间，$x_1, x_2, \ldots, x_n$ 是 $U$ 中**线性无关**的向量。

对于 $x \in U$，$M = \text{span}\{x_1, x_2, \ldots, x_n\}$，求 $x$ 在 $M$ 中的正交投影 $x_0$。

### 13.2 推导过程

**Step 1**：$x_0 \in M$，所以可以表示为基的线性组合：

$$x_0 = \sum_{i=1}^{n} a_i x_i$$

**Step 2**：正交条件 $x - x_0 \perp M$，即 $\forall j = 1, 2, \ldots, n$：

$$(x - x_0, x_j) = 0$$

**Step 3**：展开：

$$\left(x - \sum_{i=1}^{n} a_i x_i, \, x_j\right) = 0$$

$$(x, x_j) = \sum_{i=1}^{n} a_i (x_i, x_j), \quad j = 1, 2, \ldots, n$$

### 13.3 Gram 矩阵方程

写成矩阵形式：

$$\underbrace{\begin{pmatrix}
(x_1, x_1) & (x_2, x_1) & \cdots & (x_n, x_1) \\
(x_1, x_2) & (x_2, x_2) & \cdots & (x_n, x_2) \\
\vdots & \vdots & \ddots & \vdots \\
(x_1, x_n) & (x_2, x_n) & \cdots & (x_n, x_n)
\end{pmatrix}}_{\text{Gram 矩阵 } A}
\begin{pmatrix} a_1 \\ a_2 \\ \vdots \\ a_n \end{pmatrix}
= \begin{pmatrix} (x, x_1) \\ (x, x_2) \\ \vdots \\ (x, x_n) \end{pmatrix}$$

即 $A\mathbf{a} = \mathbf{b}$，其中：

- $A_{ji} = (x_i, x_j)$ —— **Gram 矩阵**
- $b_j = (x, x_j)$ —— 右端向量

> **因为 $x_1, x_2, \ldots, x_n$ 线性无关，所以 Gram 矩阵 $A$ 可逆**，方程有唯一解。

### 13.4 特殊情况：正交系

**(1) 若 $\{x_1, \ldots, x_n\}$ 是正交系**，即 $(x_i, x_j) = \begin{cases} c_i > 0, & i = j \\ 0, & i \neq j \end{cases}$

则 $A$ 是**对角矩阵**，直接求解：

$$a_i = \frac{(x, x_i)}{(x_i, x_i)}, \quad i = 1, 2, \ldots, n$$

正交投影：

$$\boxed{x_0 = \sum_{i=1}^{n} \frac{(x, x_i)}{(x_i, x_i)} x_i}$$

**(2) 若 $\{x_1, \ldots, x_n\}$ 是规范正交系**，即 $(x_i, x_j) = \delta_{ij} = \begin{cases} 1, & i = j \\ 0, & i \neq j \end{cases}$

则 $A$ 是**单位矩阵**，更加简单：

$$a_i = (x, x_i), \quad i = 1, 2, \ldots, n$$

正交投影：

$$\boxed{x_0 = \sum_{i=1}^{n} (x, x_i) x_i}$$

> **这就是 Fourier 展开的有限维版本！** 系数 $(x, x_i)$ 就是 $x$ 在基向量 $x_i$ 方向上的"投影长度"。

### 13.5 Gram-Schmidt 正交化

上面看到，当基是正交的时，Gram 矩阵退化为对角阵，投影计算极大简化。那如何从**一般的线性无关组**出发，构造出**正交组**？答案就是 **Gram-Schmidt 正交化**。

> **算法**：给定内积空间中的线性无关组 $\{x_1, x_2, \ldots, x_n\}$，构造正交组 $\{e_1, e_2, \ldots, e_n\}$：
>
> $$e_1 = x_1$$
>
> $$e_k = x_k - \sum_{i=1}^{k-1} \frac{(x_k, e_i)}{(e_i, e_i)} e_i, \quad k = 2, 3, \ldots, n$$

**直觉理解**：每一步从 $x_k$ 中**减去它在已有正交基上的投影**，剩下的就是垂直于所有已有基的分量。

**验证正交性**：对于 $j < k$：

$$(e_k, e_j) = \left(x_k - \sum_{i=1}^{k-1} \frac{(x_k, e_i)}{(e_i, e_i)} e_i, \; e_j\right) = (x_k, e_j) - \frac{(x_k, e_j)}{(e_j, e_j)}(e_j, e_j) = 0 \quad \checkmark$$

**规范化**：若还需要规范正交系（模长为1），则令：

$$\hat{e}_k = \frac{e_k}{\|e_k\|} = \frac{e_k}{\sqrt{(e_k, e_k)}}$$

**具体例子**（$\mathbb{R}^3$ 中）：

设 $x_1 = (1, 1, 0)^T$，$x_2 = (1, 0, 1)^T$，用标准内积。

$$e_1 = x_1 = (1, 1, 0)^T$$

$$e_2 = x_2 - \frac{(x_2, e_1)}{(e_1, e_1)} e_1 = (1, 0, 1)^T - \frac{1}{2}(1, 1, 0)^T = \left(\frac{1}{2}, -\frac{1}{2}, 1\right)^T$$

验证：$(e_1, e_2) = \frac{1}{2} - \frac{1}{2} + 0 = 0$ ✓

> **在课程中的用处**：
> - 第08-09次课：正交多项式（Legendre、Chebyshev）本质上就是对 $\{1, x, x^2, \ldots\}$ 在相应内积下做 Gram-Schmidt 正交化的结果
> - 理解为什么正交基能极大简化最佳逼近的计算——Gram 矩阵从满矩阵变成对角矩阵

### 13.6 Bessel 不等式

Gram-Schmidt 正交化给出了正交基，而 Bessel 不等式给出了**Fourier 系数的有界性保证**。

> **性质**：设 $\{e_1, e_2, \ldots, e_n\}$ 是内积空间 $U$ 中的**规范正交系**，则对任意 $x \in U$：
>
> $$\boxed{\sum_{i=1}^{n} |(x, e_i)|^2 \leq \|x\|^2}$$

**证明**：设 $x_0 = \sum_{i=1}^{n}(x, e_i)e_i$ 是 $x$ 在 $M = \text{span}\{e_1, \ldots, e_n\}$ 上的正交投影，则：

$$0 \leq \|x - x_0\|^2 = \|x\|^2 - \sum_{i=1}^{n}|(x, e_i)|^2$$

移项即得。

> **意义**：Fourier 系数的平方和不会超过信号的总能量，保证了正交展开的合理性。

## 14. 不动点原理

> 本节进入 §1.4，是理论准备的最后一部分。

### 14.1 不动点的定义

> **定义**：设映射 $G: D \subset \mathbb{R}^n \to \mathbb{R}^n$。若 $x^* \in D$ 满足 $G(x^*) = x^*$，则称 $x^*$ 为 $G$ 的**不动点**。

**动机**：求解方程 $F(x) = 0$ 可以改写为等价形式 $x = G(x)$，则方程的解就是 $G$ 的不动点。

**例子**：

| 映射 | 不动点 |
|------|--------|
| $T: \mathbb{R} \to \mathbb{R}$，$Tx = x^2$ | $x = 0$ 和 $x = 1$（$x^2 = x$ 的解） |
| $T: \mathbb{R}^2 \to \mathbb{R}^2$，$T(x, y) = (x, 0)$ | $x$ 轴上所有点 $(x, 0)$ |
| 旋转变换 $T$（旋转角 $\theta \in (0, 2\pi)$） | 仅原点 $(0, 0)$ |
| 平移变换 $Tx = x + b$（$b \neq 0$） | **无**不动点 |

### 14.2 迭代法求不动点

**核心思想**：任取初始点 $x_0$，构造迭代序列：

$$x_{k+1} = G(x_k), \quad k = 0, 1, 2, \ldots$$

即：$x_0 \xrightarrow{G} x_1 \xrightarrow{G} x_2 \xrightarrow{G} x_3 \xrightarrow{G} \cdots$

如果 $x_k \to x^*$（序列收敛），则 $x^*$ 就是 $G$ 的不动点。

**需要解决三个问题**：

1. **存在性与唯一性**：不动点是否存在？是否唯一？
2. **收敛性**：迭代序列是否收敛到不动点？
3. **误差分析**：收敛速度如何？需要迭代多少次？

### 14.3 压缩映射

> **定义**：设映射 $G: D \subset \mathbb{R}^n \to \mathbb{R}^n$。若存在实数 $\alpha \in (0, 1)$，使得 $\forall x, y \in D$：
>
> $$\|G(x) - G(y)\| \leq \alpha \|x - y\|$$
>
> 则称 $G$ 在 $D$ 上是**压缩映射**，$\alpha$ 称为**压缩系数**。

**直观理解**：压缩映射使任意两点的距离**缩小**至少 $\alpha$ 倍。每次迭代后，点对之间越来越近。

**相关概念对比**：

| 映射类型 | 条件 | 关系 |
|---------|------|------|
| 压缩映射 | $\|G(x) - G(y)\| \leq \alpha\|x-y\|$，$\alpha < 1$ | 最强 |
| 严格非膨胀映射 | $\|G(x) - G(y)\| < \|x-y\|$（$x \neq y$） | 中间 |
| 非膨胀映射 | $\|G(x) - G(y)\| \leq \|x-y\|$ | 最弱 |

> 压缩映射 $\Rightarrow$ 严格非膨胀 $\Rightarrow$ 非膨胀 $\Rightarrow$ 连续。反之不成立。

### 14.4 压缩映射原理（Banach 不动点定理）

> **定理1**：设 $G: \mathbb{R}^n \to \mathbb{R}^n$ 为 $\mathbb{R}^n$ 上的**压缩映射**，则 $G$ 在 $\mathbb{R}^n$ 中有**唯一**的不动点。

**证明**：

**第一步：构造 Cauchy 列**

任取 $x_0 \in \mathbb{R}^n$，令 $x_{k+1} = G(x_k)$。

$$\|x_{k+1} - x_k\| = \|G(x_k) - G(x_{k-1})\| \leq \alpha \|x_k - x_{k-1}\| \leq \cdots \leq \alpha^k \|x_1 - x_0\|$$

对于 $p \geq 1$：

$$\|x_{k+p} - x_k\| \leq \|x_{k+p} - x_{k+p-1}\| + \cdots + \|x_{k+1} - x_k\|$$

$$\leq (\alpha^{k+p-1} + \cdots + \alpha^k)\|x_1 - x_0\| = \alpha^k \cdot \frac{1 - \alpha^p}{1 - \alpha} \|x_1 - x_0\|$$

因为 $\alpha < 1$，当 $k \to \infty$ 时 $\alpha^k \to 0$，所以 $\{x_k\}$ 是 **Cauchy 列**。

**第二步：由完备性得到收敛性**

$\mathbb{R}^n$ 是完备的，所以 $\exists x^* \in \mathbb{R}^n$ 使得 $x_k \to x^*$。

**第三步：证明 $x^*$ 是不动点**

由 $G$ 的连续性（压缩映射必连续）：

$$x^* = \lim_{k \to \infty} x_{k+1} = \lim_{k \to \infty} G(x_k) = G\left(\lim_{k \to \infty} x_k\right) = G(x^*)$$

**第四步：证明唯一性**

设 $y^*$ 也是不动点，则：

$$\|x^* - y^*\| = \|G(x^*) - G(y^*)\| \leq \alpha \|x^* - y^*\|$$

因为 $\alpha < 1$，必须 $\|x^* - y^*\| = 0$，即 $x^* = y^*$。$\square$

> **定理的价值**：
> - **存在性**：不动点一定存在
> - **唯一性**：不动点是唯一的
> - **构造性**：给出了求不动点的**具体方法**（迭代法），而不仅仅是抽象的存在性证明
> - **注意**：定理要求在**全空间** $\mathbb{R}^n$ 上压缩，这个条件较强。实际上有更弱条件的不动点理论（见教材 P21 定理 1.4.2~1.4.5）

## 15. 知识结构总结

### 15.1 空间层次关系（完整版）

```
集合（Set）
 └── 距离空间（Metric Space）          ← 加距离函数 d(x,y)
      ├── 完备距离空间                  ← 加完备性
      └── 赋范线性空间（Normed Space）  ← 加线性结构 + 范数 ||x||
           ├── Banach 空间              ← 加完备性
           └── 内积空间                 ← 加内积 ⟨x,y⟩（范数满足平行四边形公式）
                └── Hilbert 空间        ← 加完备性
```

每一层结构更丰富，能做的事情更多：

| 层次 | 新增能力 |
|------|---------|
| 距离空间 | 收敛、Cauchy 列、完备性 |
| 赋范线性空间 | 元素大小、线性运算、范数等价性 |
| 内积空间 | **角度**、**正交性**、**投影** |
| Hilbert 空间 | 投影定理、正交分解保证存在唯一 |

| $(\mathbb{R}^n, \|\cdot\|_1)$ | ✓ | ✓ | ✗ | ✗ |
