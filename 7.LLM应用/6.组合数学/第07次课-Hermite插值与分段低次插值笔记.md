## 1. 引言：从函数值插值到带导数的插值

### 1.1 前面学的插值方法的局限

[[第05次课-拉格朗日插值笔记|Lagrange 插值]]和 [[第06次课-牛顿插值法笔记|Newton 插值]]都只利用了函数在节点处的**函数值** $f(x_i)$。但在很多实际场景中，我们同时还知道**导数值** $f'(x_i)$：

| 场景 | 已知信息 | 单纯用函数值插值的问题 |
|------|---------|---------------------|
| 物理实验 | 位移 $f(x_i)$ 和速度 $f'(x_i)$ | 浪费了速度数据 |
| 工程曲线设计 | 曲线通过指定点且切线方向确定 | 不能保证切线方向 |
| 数值微分方程 | 函数值和导数值同时给出 | 精度不够 |

### 1.2 本讲的两大主题

| 主题 | 核心思想 | 解决的问题 |
|------|---------|-----------|
| **Hermite 插值** | 同时匹配函数值**和**导数值 | 利用更多信息提高精度 |
| **分段低次插值** | 将区间分段，每段用低次多项式 | 避免高次多项式的 Runge 现象 |

## 2. Hermite 插值的定义

### 2.1 问题的精确定义

> **定义（Hermite 插值）**：已知函数 $f(x)$ 在 $n+1$ 个互异节点 $x_0, x_1, \ldots, x_n$ 处的**函数值和一阶导数值**：
>
> $$f(x_i) = y_i, \quad f'(x_i) = m_i, \quad i = 0, 1, \ldots, n$$
>
> 求一个次数不超过 $2n+1$ 的多项式 $H_{2n+1}(x)$，使得：
>
> $$H_{2n+1}(x_i) = y_i, \quad H'_{2n+1}(x_i) = m_i, \quad i = 0, 1, \ldots, n$$

### 2.2 条件计数

| 项目 | 个数 |
|------|:----:|
| 函数值条件 $H(x_i) = y_i$ | $n+1$ 个 |
| 导数值条件 $H'(x_i) = m_i$ | $n+1$ 个 |
| **总条件数** | $2(n+1) = 2n+2$ 个 |
| 多项式系数 $a_0, a_1, \ldots, a_{2n+1}$ | $2n+2$ 个 |

条件数 = 未知数个数，恰好可以唯一确定多项式。

### 2.3 与 Lagrange 插值的对比

| | Lagrange 插值 | Hermite 插值 |
|---|-------------|-------------|
| **已知信息** | 函数值 $f(x_i)$ | 函数值 $f(x_i)$ + 导数值 $f'(x_i)$ |
| **条件数** | $n+1$ | $2(n+1)$ |
| **多项式次数** | $\leq n$ | $\leq 2n+1$ |
| **精度** | $O(h^{n+1})$ | $O(h^{2n+2})$（同样节点数下精度更高） |
| **曲线光滑性** | 只过数据点 | 过数据点且**切线方向**也匹配 |

## 3. Hermite 插值多项式的构造

### 3.1 基函数法的思路

类似 Lagrange 插值用基函数 $l_k(x)$ 构造，Hermite 插值需要**两组**基函数：

$$\boxed{H_{2n+1}(x) = \sum_{j=0}^{n} \left[ y_j \cdot \alpha_j(x) + m_j \cdot \beta_j(x) \right]}$$

其中 $\alpha_j(x)$ 和 $\beta_j(x)$ 满足以下条件：

**$\alpha_j(x)$（对应函数值的基函数）**：

$$\alpha_j(x_i) = \delta_{ji}, \quad \alpha_j'(x_i) = 0 \quad (i, j = 0, 1, \ldots, n)$$

**$\beta_j(x)$（对应导数值的基函数）**：

$$\beta_j(x_i) = 0, \quad \beta_j'(x_i) = \delta_{ji} \quad (i, j = 0, 1, \ldots, n)$$

> **直觉理解**：
> - $\alpha_j(x)$ 在节点 $x_j$ 处"打开"函数值（值为1），同时不影响任何节点处的导数（导数为0）
> - $\beta_j(x)$ 在节点 $x_j$ 处"打开"导数值（导数为1），同时不影响任何节点处的函数值（值为0）

### 3.2 构造 $\alpha_j(x)$ 和 $\beta_j(x)$

设 $l_j(x)$ 是 [[第05次课-拉格朗日插值笔记#6. $n$ 次 Lagrange 插值多项式（一般公式）|Lagrange 基函数]]：

$$l_j(x) = \prod_{\substack{i=0 \\ i \neq j}}^{n} \frac{x - x_i}{x_j - x_i}$$

**关键观察**：$l_j(x_i) = \delta_{ji}$，且 $l_j(x)$ 在 $x_i$（$i \neq j$）处有零点。

**构造 $\beta_j(x)$**（较简单）：

$$\boxed{\beta_j(x) = (x - x_j) \cdot [l_j(x)]^2}$$

**验证**：
- $\beta_j(x_i) = (x_i - x_j)[l_j(x_i)]^2$：当 $i \neq j$ 时 $l_j(x_i) = 0$，所以 $\beta_j(x_i) = 0$；当 $i = j$ 时 $(x_j - x_j) = 0$，所以 $\beta_j(x_j) = 0$ ✓
- $\beta_j'(x) = [l_j(x)]^2 + 2(x - x_j)l_j(x)l_j'(x)$，所以 $\beta_j'(x_j) = [l_j(x_j)]^2 = 1$ ✓
- $\beta_j'(x_i) = 0 + 0 = 0$（$i \neq j$ 时，$l_j(x_i) = 0$） ✓

**构造 $\alpha_j(x)$**：

设 $\alpha_j(x) = (ax + b)[l_j(x)]^2$，由 $\alpha_j(x_j) = 1$ 和 $\alpha_j'(x_j) = 0$ 确定 $a, b$。

经过计算（利用 $l_j(x_j) = 1$ 和求导后令 $x = x_j$）：

$$\boxed{\alpha_j(x) = [1 - 2l_j'(x_j)(x - x_j)] \cdot [l_j(x)]^2}$$

其中 $l_j'(x_j) = \sum_{\substack{i=0 \\ i \neq j}}^{n} \frac{1}{x_j - x_i}$。

**验证**：
- $\alpha_j(x_j) = [1 - 0] \cdot 1 = 1$ ✓
- $\alpha_j(x_i) = [\cdots] \cdot [l_j(x_i)]^2 = 0$（$i \neq j$） ✓
- $\alpha_j'(x_j) = -2l_j'(x_j) \cdot 1 + 1 \cdot 2l_j'(x_j) = 0$ ✓

## 4. Hermite 插值的唯一性

> **定理1**：满足插值条件的 $2n+1$ 次 Hermite 插值多项式是**唯一**的。

**证明**：

设 $H_{2n+1}(x)$ 和 $\overline{H}_{2n+1}(x)$ 都满足插值条件。令：

$$\varphi(x) = H_{2n+1}(x) - \overline{H}_{2n+1}(x)$$

则 $\varphi(x)$ 是次数不超过 $2n+1$ 的多项式，且：

$$\varphi(x_i) = 0, \quad \varphi'(x_i) = 0, \quad i = 0, 1, \ldots, n$$

每个节点 $x_i$ 都是 $\varphi(x)$ 的**二重零点**（函数值和导数都为零）。

$\varphi(x)$ 共有 $2(n+1) = 2n+2$ 个零点（计重数），但 $\varphi$ 的次数 $\leq 2n+1$。

次数不超过 $2n+1$ 的多项式最多有 $2n+1$ 个零点（除非恒为零）。

$2n+2 > 2n+1$，所以 $\varphi(x) \equiv 0$，即 $H_{2n+1}(x) = \overline{H}_{2n+1}(x)$。$\square$

## 5. Hermite 插值的余项

### 5.1 一般余项公式

> **定理2**：设 $f(x)$ 在 $[a, b]$ 上有 $2n+2$ 阶连续导数，$H_{2n+1}(x)$ 是 $n+1$ 个节点的 Hermite 插值多项式，则：
>
> $$\boxed{R_{2n+1}(x) = f(x) - H_{2n+1}(x) = \frac{f^{(2n+2)}(\xi)}{(2n+2)!} \cdot \omega_{n+1}^2(x)}$$
>
> 其中 $\xi \in (a, b)$，$\omega_{n+1}(x) = \prod_{i=0}^{n}(x - x_i)$。

### 5.2 与 Lagrange 余项的对比

| | Lagrange 余项 | Hermite 余项 |
|---|-------------|-------------|
| **公式** | $\frac{f^{(n+1)}(\xi)}{(n+1)!} \omega_{n+1}(x)$ | $\frac{f^{(2n+2)}(\xi)}{(2n+2)!} \omega_{n+1}^2(x)$ |
| **导数阶数** | $n+1$ | $2n+2$（翻倍） |
| **$\omega$ 的幂次** | 1次方 | **2次方**（平方） |
| **精度** | $O(h^{n+1})$ | $O(h^{2n+2})$（翻倍） |

**关键观察**：$\omega_{n+1}^2(x)$ 的出现是因为每个节点都是余项的**二重零点**（函数值和导数都匹配）。

### 5.3 证明思路

证明方法与 [[第05次课-拉格朗日插值笔记#7.3 证明（详细版）|Lagrange 余项证明]]完全类似——辅助函数法 + 反复 Rolle 定理。

关键区别在于零点计数：

| 步骤 | Lagrange | Hermite |
|------|----------|---------|
| $\varphi(t)$ 的零点 | $n+2$ 个（$n+1$ 个节点 + 1个 $x$） | $2n+3$ 个（每个节点二重 + 1个 $x$） |
| 应用 Rolle 次数 | $n+1$ | $2n+2$ |
| 最终导数阶数 | $\varphi^{(n+1)}(\xi) = 0$ | $\varphi^{(2n+2)}(\xi) = 0$ |

## 6. 两点三次 Hermite 插值（重点）

这是 Hermite 插值中**最常用**的情况——只有两个节点，用三次多项式。

### 6.1 问题设置

**已知**：两个节点 $x_0, x_1$ 处的函数值和导数值：

$$f(x_0) = f_0, \quad f(x_1) = f_1, \quad f'(x_0) = f_0', \quad f'(x_1) = f_1'$$

**求**：三次多项式 $H_3(x)$，满足：

$$H_3(x_0) = f_0, \quad H_3(x_1) = f_1, \quad H_3'(x_0) = f_0', \quad H_3'(x_1) = f_1'$$

4个条件确定3次多项式的4个系数，恰好。

### 6.2 公式

$$\boxed{H_3(x) = f_0 \cdot \alpha_0(x) + f_1 \cdot \alpha_1(x) + f_0' \cdot \beta_0(x) + f_1' \cdot \beta_1(x)}$$

四个基函数为：

$$\alpha_0(x) = \left(1 + 2\frac{x - x_0}{x_1 - x_0}\right)\left(\frac{x - x_1}{x_0 - x_1}\right)^2$$

$$\alpha_1(x) = \left(1 + 2\frac{x - x_1}{x_0 - x_1}\right)\left(\frac{x - x_0}{x_1 - x_0}\right)^2$$

$$\beta_0(x) = (x - x_0)\left(\frac{x - x_1}{x_0 - x_1}\right)^2$$

$$\beta_1(x) = (x - x_1)\left(\frac{x - x_0}{x_1 - x_0}\right)^2$$

### 6.3 基函数的推导

以 $\alpha_0(x)$ 为例。

**Step 1**：Lagrange 基函数

$$l_0(x) = \frac{x - x_1}{x_0 - x_1}, \qquad l_1(x) = \frac{x - x_0}{x_1 - x_0}$$

**Step 2**：$\alpha_0(x)$ 必须在 $x_1$ 处为零 → 含因子 $(x - x_1)^2$

设 $\alpha_0(x) = (ax + b)(x - x_1)^2 = (ax + b)[l_0(x)]^2 (x_0 - x_1)^2$

实际上用一般公式 $\alpha_0(x) = [1 - 2l_0'(x_0)(x - x_0)][l_0(x)]^2$：

$$l_0'(x) = \frac{1}{x_0 - x_1}, \quad l_0'(x_0) = \frac{1}{x_0 - x_1}$$

$$\alpha_0(x) = \left[1 - \frac{2}{x_0 - x_1}(x - x_0)\right] \cdot \left(\frac{x - x_1}{x_0 - x_1}\right)^2 = \left(1 + \frac{2(x - x_0)}{x_1 - x_0}\right)\left(\frac{x - x_1}{x_0 - x_1}\right)^2$$

### 6.4 基函数验证表

| 基函数 | $x = x_0$ 处值 | $x = x_0$ 处导数 | $x = x_1$ 处值 | $x = x_1$ 处导数 |
|--------|:-------------:|:---------------:|:-------------:|:---------------:|
| $\alpha_0(x)$ | 1 | 0 | 0 | 0 |
| $\alpha_1(x)$ | 0 | 0 | 1 | 0 |
| $\beta_0(x)$ | 0 | 1 | 0 | 0 |
| $\beta_1(x)$ | 0 | 0 | 0 | 1 |

每个基函数只在特定位置"打开"一个条件，其余全部为零——这保证了插值条件自动满足。

### 6.5 几何理解

```
y
│       ·───────·       ← H₃(x)：过两点，切线方向也匹配
│      /·        \·
│     / ↗          ↘
│    · f'₀           · f'₁   ← 两端切线斜率已知
│
└────────────────────────── x
     x₀                x₁
```

三次 Hermite 插值**同时保证**：
- 曲线通过 $(x_0, f_0)$ 和 $(x_1, f_1)$
- 两端的切线斜率分别为 $f_0'$ 和 $f_1'$

## 7. 两点三次 Hermite 插值的余项

### 7.1 余项公式

$$\boxed{R_3(x) = f(x) - H_3(x) = \frac{f^{(4)}(\xi)}{4!}(x - x_0)^2(x - x_1)^2}$$

其中 $\xi \in (x_0, x_1)$。

**注意**：$(x - x_0)^2(x - x_1)^2 \geq 0$，所以余项的符号完全由 $f^{(4)}(\xi)$ 决定。

### 7.2 余项推导（详细版）

**Step 1**：固定 $x \neq x_0, x_1$，定义：

$$K(x) = \frac{R_3(x)}{(x - x_0)^2(x - x_1)^2}$$

**Step 2**：构造辅助函数

$$\varphi(t) = f(t) - H_3(t) - K(x)(t - x_0)^2(t - x_1)^2$$

**Step 3**：$\varphi(t)$ 的零点分析

| 零点 | 类型 | 原因 |
|------|------|------|
| $t = x_0$ | **二重零点** | $\varphi(x_0) = 0$ 且 $\varphi'(x_0) = 0$（因为 $H_3$ 和 $K \cdot (\cdots)^2$ 在 $x_0$ 处都匹配函数值和导数） |
| $t = x_1$ | **二重零点** | 同理 |
| $t = x$ | 单零点 | 由 $K(x)$ 的定义保证 |

共计 $2 + 2 + 1 = 5$ 个零点。

**Step 4**：反复应用 [[微积分基础回顾#15. Rolle 定理与反复应用|Rolle 定理]]

| 导数阶 | 零点数 | 说明 |
|:------:|:------:|------|
| $\varphi(t)$ | $\geq 5$ | 包含二重零点 |
| $\varphi'(t)$ | $\geq 4$ | 相邻零点之间各一个 |
| $\varphi''(t)$ | $\geq 3$ | |
| $\varphi'''(t)$ | $\geq 2$ | |
| $\varphi^{(4)}(t)$ | $\geq 1$ | 存在 $\xi$ 使得 $\varphi^{(4)}(\xi) = 0$ |

**Step 5**：计算 $\varphi^{(4)}$

$$\varphi^{(4)}(t) = f^{(4)}(t) - H_3^{(4)}(t) - K(x) \cdot [(t-x_0)^2(t-x_1)^2]^{(4)}$$

- $H_3$ 是3次多项式 → $H_3^{(4)} = 0$
- $(t-x_0)^2(t-x_1)^2$ 展开后最高次项为 $t^4$ → 其4阶导数 = $4!$

$$\varphi^{(4)}(\xi) = f^{(4)}(\xi) - K(x) \cdot 4! = 0$$

$$K(x) = \frac{f^{(4)}(\xi)}{4!}$$

代回：$R_3(x) = K(x)(x - x_0)^2(x - x_1)^2 = \frac{f^{(4)}(\xi)}{4!}(x - x_0)^2(x - x_1)^2$ $\square$

## 8. 计算示例

### 8.1 例1：基函数法

已知 $f(x)$ 在节点 $x_0 = 1, x_1 = 2$ 处的信息：

| | $x_0 = 1$ | $x_1 = 2$ |
|---|:---------:|:---------:|
| $f(x_i)$ | 2 | 3 |
| $f'(x_i)$ | 0 | $-1$ |

求两点三次 Hermite 插值多项式 $H_3(x)$，以及 $f(1.5)$ 和 $f(1.7)$ 的近似值。

**解**：

**Step 1**：Lagrange 基函数

$$l_0(x) = \frac{x - 2}{1 - 2} = -(x - 2) = 2 - x, \qquad l_1(x) = \frac{x - 1}{2 - 1} = x - 1$$

**Step 2**：计算四个基函数

$$\alpha_0(x) = \left(1 + 2\frac{x-1}{2-1}\right)(2-x)^2 = (2x - 1)(2-x)^2$$

$$\alpha_1(x) = \left(1 + 2\frac{x-2}{1-2}\right)(x-1)^2 = (5 - 2x)(x-1)^2$$

$$\beta_0(x) = (x-1)(2-x)^2$$

$$\beta_1(x) = (x-2)(x-1)^2$$

**Step 3**：代入公式

$$H_3(x) = 2 \cdot (2x-1)(2-x)^2 + 3 \cdot (5-2x)(x-1)^2 + 0 \cdot \beta_0 + (-1) \cdot (x-2)(x-1)^2$$

展开化简：

$$H_3(x) = 2(2x-1)(x^2 - 4x + 4) + 3(5-2x)(x^2 - 2x + 1) + (-1)(x-2)(x^2 - 2x + 1)$$

$$= 2(2x^3 - 9x^2 + 12x - 4) + 3(-2x^3 + 9x^2 - 12x + 5) + (-x^3 + 4x^2 - 5x + 2)$$

$$= (4 - 6 - 1)x^3 + (-18 + 27 + 4)x^2 + (24 - 36 - 5)x + (-8 + 15 + 2)$$

$$= -3x^3 + 13x^2 - 17x + 9$$

**验证**：
- $H_3(1) = -3 + 13 - 17 + 9 = 2$ ✓
- $H_3(2) = -24 + 52 - 34 + 9 = 3$ ✓
- $H_3'(x) = -9x^2 + 26x - 17$，$H_3'(1) = -9 + 26 - 17 = 0$ ✓
- $H_3'(2) = -36 + 52 - 17 = -1$ ✓

**Step 4**：计算近似值

$$f(1.5) \approx H_3(1.5) = -3(3.375) + 13(2.25) - 17(1.5) + 9 = -10.125 + 29.25 - 25.5 + 9 = 2.625$$

$$f(1.7) \approx H_3(1.7) = -3(4.913) + 13(2.89) - 17(1.7) + 9 = -14.739 + 37.57 - 28.9 + 9 = 2.931$$

### 8.2 例1（Newton 型重节点法）

同一个问题，用 [[第06次课-牛顿插值法笔记#9. 重节点 Newton 插值公式|重节点 Newton 插值]]方法。

**节点序列**（含重节点）：$x_0 = 1, x_0 = 1, x_1 = 2, x_1 = 2$

**差商表**：

| 节点 | $f$ 值 | 一阶差商 | 二阶差商 | 三阶差商 |
|:----:|:------:|:-------:|:-------:|:-------:|
| 1 | 2 | | | |
| 1 | 2 | $f[1,1] = f'(1) = 0$ | | |
| 2 | 3 | $f[1,2] = \frac{3-2}{1} = 1$ | $\frac{1-0}{2-1} = 1$ | |
| 2 | 3 | $f[2,2] = f'(2) = -1$ | $\frac{-1-1}{2-1} = -2$ | $\frac{-2-1}{2-1} = -3$ |

> 重节点差商：$f[x_i, x_i] = f'(x_i)$（参见 [[第06次课-牛顿插值法笔记#9.2 重节点差商的概念|重节点差商]]）

**Newton 插值公式**：

$$H_3(x) = 2 + 0 \cdot (x-1) + 1 \cdot (x-1)^2 + (-3)(x-1)^2(x-2)$$

展开：

$$= 2 + (x-1)^2 - 3(x-1)^2(x-2)$$

$$= 2 + (x^2 - 2x + 1) - 3(x^3 - 4x^2 + 5x - 2)$$

$$= 2 + x^2 - 2x + 1 - 3x^3 + 12x^2 - 15x + 6$$

$$= -3x^3 + 13x^2 - 17x + 9$$

与基函数法的结果**完全一致** ✓

> **Newton 型方法的优势**：不需要推导基函数，只需列差商表，计算更快捷。

### 8.3 例2：带不完全导数的 Hermite 插值

已知 $f(x) \in C^4[a, b]$，$f(-1) = -1$，$f(2) = 9$，$f(3) = 23$，$f'(4) = 3$。

求次数不超过三次的插值多项式。

**解**：这是**不完全导数**的情况——不是每个节点都有导数信息。

**方法**：先用前三点做二次 Lagrange（或 Newton）插值 $L_2(x)$，再补上导数条件。

**Step 1**：二次插值 $L_2(x)$

用 $(-1, -1), (2, 9), (3, 23)$ 三点：

$$L_2(x) = \frac{1}{3}(8x^2 + 2x - 9)$$

（通过 Lagrange 或 Newton 方法均可得到）

**Step 2**：设 $H(x) = L_2(x) + A(x+1)(x-2)(x-3)$

增加一项含 $(x+1)(x-2)(x-3)$ 的项——这保证新增项在前三个节点处为零，不破坏已有的插值条件。

**Step 3**：利用导数条件 $H'(4) = 3$ 确定 $A$

计算 $L_2'(x)$ 和 $(x+1)(x-2)(x-3)$ 的导数在 $x = 4$ 处的值，解出 $A$。

$$H'(4) = L_2'(4) + A \cdot [(x+1)(x-2)(x-3)]'|_{x=4} = 3$$

由

$$L_2'(x) = \frac{16x + 2}{3}, \quad L_2'(4) = 22$$

以及

$$g(x) = (x+1)(x-2)(x-3) = x^3 - 4x^2 + x + 6, \quad g'(4) = 17$$

可得

$$22 + 17A = 3 \quad \Longrightarrow \quad A = -\frac{19}{17}$$

$$H(x) = \frac{1}{3}(8x^2 + 2x - 9) - \frac{19}{17}(x+1)(x-2)(x-3)$$

### 8.4 例3：高阶重节点

已知数据表：

| $x$ | 0 | 0 | 1 | 1 | 1 |
|-----|---|---|---|---|---|
| 信息 | $f(0) = 3$ | $f'(0) = 4$ | $f(1) = 5$ | $f'(1) = 6$ | $f''(1) = 7$ |

$x = 0$ 是**二重节点**，$x = 1$ 是**三重节点**，共5个条件 → 4次多项式。

**重节点差商表**：

| 节点 | $f$ 值 | 一阶 | 二阶 | 三阶 | 四阶 |
|:----:|:------:|:----:|:----:|:----:|:----:|
| 0 | 3 | | | | |
| 0 | 3 | $f'(0) = 4$ | | | |
| 1 | 5 | $\frac{5-3}{1} = 2$ | $\frac{2-4}{1} = -2$ | | |
| 1 | 5 | $f'(1) = 6$ | $\frac{6-2}{1} = 4$ | $\frac{4-(-2)}{1} = 6$ | |
| 1 | 5 | $f'(1) = 6$ | $\frac{f''(1)}{2!} = \frac{7}{2}$ | $\frac{7/2 - 4}{1} = -\frac{1}{2}$ | $\frac{-1/2 - 6}{1} = -\frac{13}{2}$ |

> 三重节点的差商：$f[x_i, x_i, x_i] = \frac{f''(x_i)}{2!}$

**Newton 插值多项式**：

$$H_4(x) = 3 + 4x + (-2)x^2 + 6x^2(x-1) + \left(-\frac{13}{2}\right)x^2(x-1)^2$$

$$= 3 + 4x - 2x^2 + 6x^2(x-1) - \frac{13}{2}x^2(x-1)^2$$

## 9. Runge 现象：高次插值的警告

### 9.1 Runge 的发现

1901年，德国数学家 Runge 研究了函数：

$$f(x) = \frac{1}{1 + x^2}, \quad x \in [-5, 5]$$

用等距节点做 Lagrange 插值，令人惊讶的发现：

| 插值次数 $n$ | 插值效果 |
|:------------:|---------|
| 4 | 中心附近尚可，端点有偏差 |
| 6 | 中心好转，但端点**振荡加剧** |
| 10 | 中心很好，但端点**剧烈振荡** |
| $n \to \infty$ | 在 $|x| > 3.63$ 处**不收敛** |

> **两种常见形式的等价性**：部分教材使用 $f(t) = \frac{1}{1 + 25t^2}$ 在 $[-1, 1]$ 上的等距插值来演示 Runge 现象。两者通过线性变换 $x = 5t$（即 $t = x/5$）等价：
>
> $$\frac{1}{1 + x^2}\bigg|_{x \in [-5,5]} \quad \xleftrightarrow{x = 5t} \quad \frac{1}{1 + 25t^2}\bigg|_{t \in [-1,1]}$$
>
> 选择 $[-1, 1]$ 上的形式可以更方便地与 Chebyshev 节点等标准区间上的方法对比。

### 9.2 为什么会发生 Runge 现象？

插值余项：

$$R_n(x) = \frac{f^{(n+1)}(\xi)}{(n+1)!} \omega_{n+1}(x)$$

**两个因素的竞赛**：

| 因素 | 随 $n$ 增大的趋势 | 效果 |
|------|:-----------------:|------|
| $(n+1)!$（分母） | 快速增大 | 有利于减小误差 |
| $f^{(n+1)}(\xi)$（分子） | 对某些函数**更快速**增大 | 可能导致误差增大 |
| $\omega_{n+1}(x)$（节点多项式） | 在端点附近剧烈振荡 | 端点误差恶化 |

对于 $f(x) = \frac{1}{1+x^2}$，其高阶导数在 $[-5, 5]$ 上增长极快，超过了 $(n+1)!$ 的衰减速度。

### 9.3 Runge 现象的教训

$$\boxed{\text{增大插值多项式次数 } n \text{ 不一定能提高精度！}}$$

**解决方案**：不用一个高次多项式覆盖整个区间，而是**分段**使用低次多项式。

## 10. 分段线性插值

### 10.1 核心思想

将区间 $[a, b]$ 分成 $n$ 个小段 $[x_k, x_{k+1}]$，在每段上用**一次多项式**（直线）连接相邻数据点。

```
y
│     ·         ← f(x) 原函数
│    · \·
│   /    \·
│  ·      ·\·
│ /        \ ·
│·          \·   ← I(x) 分段线性插值（折线）
└───────────────── x
  x₀  x₁  x₂  x₃
```

### 10.2 公式

在每个子区间 $[x_k, x_{k+1}]$ 上：

$$\boxed{I(x) = y_k \cdot \frac{x_{k+1} - x}{x_{k+1} - x_k} + y_{k+1} \cdot \frac{x - x_k}{x_{k+1} - x_k}, \quad x \in [x_k, x_{k+1}]}$$

这就是连接 $(x_k, y_k)$ 和 $(x_{k+1}, y_{k+1})$ 的直线段。

### 10.3 基函数表示

分段线性插值可以用**帽子函数**（hat functions）表示：

$$I(x) = \sum_{j=0}^{n} y_j l_j(x)$$

其中帽子函数 $l_j(x)$：

$$l_j(x) = \begin{cases}
\dfrac{x - x_{j-1}}{x_j - x_{j-1}}, & x \in [x_{j-1}, x_j] \\[6pt]
\dfrac{x_{j+1} - x}{x_{j+1} - x_j}, & x \in [x_j, x_{j+1}] \\[6pt]
0, & \text{其他}
\end{cases}$$

```
l_j(x)
 1│    /\
  │   /  \
  │  /    \
  │ /      \
 0│/________\______
  x_{j-1}  x_j  x_{j+1}
```

**帽子函数的性质**：$l_j(x_i) = \delta_{ij}$，与 Lagrange 基函数性质相同，但只在相邻区间上非零。

**边界节点的特殊处理**：上述定义适用于内部节点 $j = 1, 2, \ldots, n-1$。对于端点：

- $j = 0$（左端点）：$l_0(x)$ 只有**右半边**，在 $[x_0, x_1]$ 上从 1 线性下降到 0
- $j = n$（右端点）：$l_n(x)$ 只有**左半边**，在 $[x_{n-1}, x_n]$ 上从 0 线性上升到 1

```
l_0(x)                l_n(x)
1│\                      /│1
 │ \                    / │
 │  \                  /  │
 │   \                /   │
0│____\____      ____/____|0
 x_0  x_1       x_{n-1}  x_n
```

### 10.4 误差估计

> 设 $f \in C^2[a, b]$，$M = \max_{a \leq x \leq b}|f''(x)|$，$h = \max_{0 \leq k \leq n-1}(x_{k+1} - x_k)$（最大步长），则：
>
> $$\boxed{|R(x)| = |f(x) - I(x)| \leq \frac{1}{8}Mh^2}$$

**理解**：
- 误差与 $h^2$ 成正比——步长缩小一半，误差减小到 $\frac{1}{4}$
- $M = \max|f''|$：函数曲率越大（弯曲越厉害），直线逼近误差越大
- $\frac{1}{8}$ 是常数因子

**证明思路**：在 $[x_k, x_{k+1}]$ 上，$I(x)$ 是线性插值，由一次插值余项：

$$R(x) = \frac{f''(\xi)}{2!}(x - x_k)(x - x_{k+1})$$

而由 [[微积分基础回顾#3.4 闭区间上二次函数的极值估计|二次函数极值引理]]，$|(x - x_k)(x - x_{k+1})| \leq \frac{h_k^2}{4}$（在区间中点取最大值），所以 $|R(x)| \leq \frac{M}{2} \cdot \frac{h^2}{4} = \frac{Mh^2}{8}$。

### 10.5 优缺点

| 优点 | 缺点 |
|------|------|
| 计算简单 | 在节点处**不可导**（尖点） |
| 在节点处连续（$C^0$） | 曲线不光滑 |
| 保证收敛于 $f(x)$ | 精度不高（只有 $O(h^2)$） |
| 避免了 Runge 现象 | 不适合要求光滑的应用 |

> 详见 [[微积分基础回顾#18. 函数的光滑度与 $C^k$ 连续性|微积分基础：函数的光滑度]]。

## 11. 分段三次 Hermite 插值

### 11.1 动机

分段线性插值的主要缺陷是**不光滑**（$C^0$ 但非 $C^1$）。为了在节点处也有光滑的过渡，使用分段三次 Hermite 插值。

### 11.2 定义

在每个子区间 $[x_k, x_{k+1}]$ 上，使用 **两点三次 Hermite 插值**：

$$\boxed{H(x) = y_k \cdot \alpha_k(x) + y_{k+1} \cdot \alpha_{k+1}(x) + y_k' \cdot \beta_k(x) + y_{k+1}' \cdot \beta_{k+1}(x), \quad x \in [x_k, x_{k+1}]}$$

其中四个基函数（在 $[x_k, x_{k+1}]$ 上）：

$$\alpha_k(x) = \left(1 + 2\frac{x - x_k}{x_{k+1} - x_k}\right)\left(\frac{x - x_{k+1}}{x_k - x_{k+1}}\right)^2$$

$$\alpha_{k+1}(x) = \left(1 + 2\frac{x - x_{k+1}}{x_k - x_{k+1}}\right)\left(\frac{x - x_k}{x_{k+1} - x_k}\right)^2$$

$$\beta_k(x) = (x - x_k)\left(\frac{x - x_{k+1}}{x_k - x_{k+1}}\right)^2$$

$$\beta_{k+1}(x) = (x - x_{k+1})\left(\frac{x - x_k}{x_{k+1} - x_k}\right)^2$$

### 11.3 关键性质

| 性质 | 分段线性 | 分段三次 Hermite | 三次样条 |
|------|:-------:|:---------------:|:-------:|
| **连续性** | $C^0$（连续） | $C^1$（一阶导数连续） | $C^2$（二阶导数连续） |
| **节点处表现** | 可能有尖点 | 光滑 | 更光滑 |
| **需要的信息** | 函数值 | 函数值 + 导数值 | 函数值（导数由方程组确定） |
| **精度** | $O(h^2)$ | $O(h^4)$ | $O(h^4)$ |

### 11.4 精确误差上界

在每个子区间 $[x_k, x_{k+1}]$ 上，由两点三次 Hermite 余项公式：

$$R_3(x) = \frac{f^{(4)}(\xi)}{4!}(x - x_k)^2(x - x_{k+1})^2$$

需要估计 $|(x - x_k)^2(x - x_{k+1})^2|$ 在 $[x_k, x_{k+1}]$ 上的最大值。

令 $g(x) = (x - x_k)^2(x - x_{k+1})^2 = [(x - x_k)(x - x_{k+1})]^2$。由 [[微积分基础回顾#3.4 闭区间上二次函数的极值估计|二次函数极值引理]]：

$$|(x - x_k)(x - x_{k+1})| \leq \frac{h_k^2}{4}$$

两边平方：

$$|(x - x_k)^2(x - x_{k+1})^2| \leq \frac{h_k^4}{16}$$

因此：

$$|R_3(x)| \leq \frac{\max|f^{(4)}|}{24} \cdot \frac{h^4}{16} = \frac{1}{384}\max_{[a,b]}|f^{(4)}(x)| \cdot h^4$$

> **分段三次 Hermite 插值的精确误差上界**：
>
> $$\boxed{|R(x)| = |f(x) - H(x)| \leq \frac{1}{384}\max_{[a,b]}|f^{(4)}(x)| \cdot h^4}$$

**与分段线性插值的精度对比**：

| 方法 | 误差上界 | 步长减半后误差变为 |
|:----:|:-------:|:---------------:|
| 分段线性 | $\frac{1}{8}Mh^2$ | 原来的 $\frac{1}{4}$ |
| 分段三次 Hermite | $\frac{1}{384}M_4 h^4$ | 原来的 $\frac{1}{16}$ |

> 分段三次 Hermite 的收敛速度是分段线性的**平方**——步长减半，精度提升 16 倍而非 4 倍。

### 11.6 为什么 $C^1$ 连续？

在节点 $x_k$ 处，左右两段 Hermite 插值的交界：

- **左段**（$[x_{k-1}, x_k]$）在 $x_k$ 处的值 = $y_k$，导数 = $y_k'$
- **右段**（$[x_k, x_{k+1}]$）在 $x_k$ 处的值 = $y_k$，导数 = $y_k'$

两段的**函数值和导数值**在 $x_k$ 处一致 → $H(x)$ 在 $x_k$ 处连续且一阶可导。

> 但二阶导数一般**不连续**——左右两段的 $H''(x_k^-)$ 和 $H''(x_k^+)$ 通常不同。

### 11.7 局限性

| 局限 | 说明 |
|------|------|
| **需要导数数据** | 实际问题中导数值可能不可用 |
| **仅 $C^1$ 光滑** | 对于机翼形线设计等需要 $C^2$ 光滑的应用不够 |

> 进一步改进 → **三次样条插值**（§2.6）：不需要给定导数值，通过求解方程组确定导数，保证 $C^2$ 光滑。

## 12. 各种插值方法总结对比

### 12.1 全局多项式插值

| 方法 | 基函数 | 利用的信息 | 多项式次数 | 继承性 |
|------|--------|----------|:----------:|:------:|
| Lagrange | $l_k(x)$ | 函数值 | $n$ | 无 |
| Newton | $\prod(x-x_j)$ | 函数值（差商） | $n$ | **有** |
| Hermite | $\alpha_j, \beta_j$ | 函数值 + 导数值 | $2n+1$ | — |

**共同问题**：节点多时，高次多项式可能出现 Runge 现象。

### 12.2 分段低次插值

| 方法 | 每段次数 | 光滑度 | 精度 | 需要信息 |
|------|:--------:|:------:|:----:|---------|
| 分段线性 | 1 | $C^0$ | $O(h^2)$ | 函数值 |
| 分段三次 Hermite | 3 | $C^1$ | $O(h^4)$ | 函数值 + 导数值 |
| 三次样条 | 3 | $C^2$ | $O(h^4)$ | 函数值 |

**共同优点**：避免 Runge 现象，保证收敛。

### 12.3 精度层级

$$\underbrace{C^0 \text{（分段线性）}}_{\text{连续但有尖点}} \subset \underbrace{C^1 \text{（分段 Hermite）}}_{\text{光滑}} \subset \underbrace{C^2 \text{（样条）}}_{\text{曲率连续}}$$

## 13. 知识结构总结

### 13.1 概念关系图

```
Lagrange / Newton 插值（只用函数值）
    │
    ├── 增加导数信息 → Hermite 插值（函数值 + 导数值，精度翻倍）
    │       ├── 一般 (2n+1) 次 Hermite
    │       ├── 两点三次 Hermite（最常用）
    │       └── Newton 型重节点方法（差商表构造）
    │
    └── 高次多项式 → Runge 现象（端点振荡、不收敛）
            │
            ↓ 解决方案
        分段低次插值
            ├── 分段线性（C⁰，简单但有尖点）
            ├── 分段三次 Hermite（C¹，光滑，需要导数）
            └── 三次样条（C²，最光滑，不需要导数）→ 下一讲
```
