## 1. 引入：为什么需要距离空间？

### 从高等数学到泛函分析的推广

| 高等数学 | 泛函分析 |
|----------|----------|
| 研究对象：**函数** | 研究对象：**算子、泛函**（空间到空间的映射） |
| 基本工具：**极限** | 基本工具：**极限**（推广到一般空间） |
| 距离：**定义极限** | 首先引入度量工具——**距离** |

在高等数学中，我们处理的是实数之间的距离 $|x - y|$，以此定义极限、连续等概念。但泛函分析的研究对象更加广泛——可以是向量、函数、矩阵、信号、图像等。为了在这些抽象空间中也能谈论"两个元素有多近"，我们需要**把距离的概念推广到任意非空集合上**。

**核心思路**：

1. 在每一类空间上定义合适的**距离**
2. 用距离定义**极限**
3. 建立相应的收敛理论

## 2. 距离空间的定义

### 2.1 从平面距离出发

在平面上，两点 $X$ 和 $Y$ 之间的欧氏距离为：

$$d(X, Y) = \sqrt{(x_1 - y_1)^2 + (x_2 - y_2)^2}$$

这个距离满足三条基本性质：

| 性质 | 表达式 |
|------|--------|
| 1. 非负性 | $d(X, Y) \geq 0$ |
| 2. 对称性 | $d(X, Y) = d(Y, X)$ |
| 3. 三角不等式 | $d(X, Z) \leq d(X, Y) + d(Y, Z)$ |

**推广思路**：只要在任意非空集合上定义一个满足这三条性质的"度量"，就构成了距离空间。

### 2.2 距离空间的严格定义

> **定义（距离空间）**：设 $X$ 是非空集合，如果对于 $\forall x, y \in X$，对应唯一的非负实数 $d(x, y)$，且满足以下三条**距离公理**：
>
> **(1) 非负性**：$d(x, y) \geq 0$，且 $d(x, y) = 0$ 当且仅当 $x = y$
>
> **(2) 对称性**：$d(x, y) = d(y, x)$
>
> **(3) 三角不等式**：$d(x, y) \leq d(x, z) + d(z, y)$，$\forall x, y, z \in X$
>
> 则称 $d(x, y)$ 为元素 $x$ 与 $y$ 之间的**距离**，称 $X$ 为**距离空间**（或度量空间），记作 $(X, d)$ 或 $X$。

**两个核心要素**：非空集合 + 距离

**数学本质**：距离 $d(\cdot, \cdot)$ 是集合 $X \times X$（笛卡尔积空间）到实数集 $\mathbb{R}$ 上的**二元泛函**：

$$d(\cdot, \cdot): X \times X \to \mathbb{R}$$

### 2.3 对距离公理的逐条理解

#### 公理(1)：非负性

$$d(x, y) \geq 0, \quad d(x, y) = 0 \iff x = y$$

- **直觉**：两个不同元素之间的距离必须是正数；只有同一个元素到自身的距离才是0
- **作用**：保证距离有意义——如果不同元素距离可以为0，就无法区分它们
- 这实际上包含两个方向：$x = y \Rightarrow d(x,y) = 0$（同一性）和 $d(x,y) = 0 \Rightarrow x = y$（可分性）

#### 公理(2)：对称性

$$d(x, y) = d(y, x)$$

- **直觉**：从 $x$ 到 $y$ 的距离 = 从 $y$ 到 $x$ 的距离
- **注意**：现实中有些"距离"不满足对称性（如上坡与下坡的难度不同），但数学中的距离要求对称

#### 公理(3)：三角不等式

$$d(x, y) \leq d(x, z) + d(z, y), \quad \forall x, y, z \in X$$

- **直觉**：两点之间直接走不会比经过第三点绕行更远——"两边之和大于第三边"
- **作用**：这是最重要的公理，它赋予了空间"几何结构"，是证明收敛性定理的关键工具
- **验证技巧**：验证某个函数是否为距离时，非负性和对称性通常容易验证，**三角不等式往往是难点**

> **重要提示**：同一集合上可以定义不同的距离，从而构成**不同的距离空间**。

## 3. 常用不等式（证明三角不等式的工具）

在验证各种距离空间的三角不等式时，经常需要用到以下不等式。

### 3.1 Hölder（赫尔德）不等式

**离散形式**：

$$\sum_{i=1}^{n} |a_i b_i| \leq \left(\sum_{i=1}^{n} |a_i|^p\right)^{1/p} \left(\sum_{i=1}^{n} |b_i|^q\right)^{1/q}$$

其中 $a_i, b_i$ 是实数或复数，$p > 0, q > 0$，且满足 $\frac{1}{p} + \frac{1}{q} = 1$（共轭指数）。

**理解**：
- 当 $p = q = 2$ 时，Hölder不等式退化为Cauchy不等式
- 左边是"混合项"的求和，右边用两个"纯项"分别求和后再相乘来控制
- 共轭条件 $\frac{1}{p} + \frac{1}{q} = 1$ 保证了不等号成立（可由Young不等式 $ab \leq \frac{a^p}{p} + \frac{b^q}{q}$ 推导）

### 3.2 Cauchy（柯西）不等式

**离散形式**（Cauchy-Schwarz不等式）：

$$\left(\sum_{i=1}^{n} a_i b_i\right)^2 \leq \left(\sum_{i=1}^{n} a_i^2\right) \left(\sum_{i=1}^{n} b_i^2\right)$$

**积分形式**：

$$\left(\int_a^b f(x)g(x) \, dx\right)^2 \leq \int_a^b f^2(x) \, dx \cdot \int_a^b g^2(x) \, dx$$

（要求 $f(x), g(x)$ 在 $[a, b]$ 上平方可积）

**理解**：
- 这是 Hölder 不等式在 $p = q = 2$ 时的特例
- **几何意义**：内积的绝对值不超过两个向量模的乘积，即 $|\langle a, b \rangle| \leq \|a\| \cdot \|b\|$
- 等号成立条件：$a_i = \lambda b_i$（即两个向量平行/共线时）

### 3.3 Minkowski（闵可夫斯基）不等式

**离散形式**：

$$\left(\sum_{i=1}^{n} |a_i + b_i|^k\right)^{1/k} \leq \left(\sum_{i=1}^{n} |a_i|^k\right)^{1/k} + \left(\sum_{i=1}^{n} |b_i|^k\right)^{1/k}$$

其中 $k \geq 1$，$a_i, b_i$ 是实数或复数。

**积分形式**：

$$\left(\int_a^b |f(x) + g(x)|^k \, dx\right)^{1/k} \leq \left(\int_a^b |f(x)|^k \, dx\right)^{1/k} + \left(\int_a^b |g(x)|^k \, dx\right)^{1/k}$$

其中 $f(x), g(x)$ 是 $[a, b]$ 上的连续函数，$k \geq 1$。

**理解**：
- **本质就是三角不等式**！将 $a_i$ 看作 $x_i - z_i$，$b_i$ 看作 $z_i - y_i$，则 $a_i + b_i = x_i - y_i$
- 左边 = $d(x, y)$，右边 = $d(x, z) + d(z, y)$，正好是三角不等式的形式
- 因此 Minkowski 不等式是证明 $\mathbb{R}^n$、$L^p$、$l^p$ 等空间满足距离公理的核心工具

## 4. 距离空间的典型例子

### 例1：实数集 $\mathbb{R}^1$ 上的不同距离

#### ① 标准距离（是距离空间 ✅）

$$d(x, y) = |x - y|$$

验证三条公理均满足，故 $(\mathbb{R}^1, d)$ 是距离空间，即**通常意义下的欧氏空间**。

#### ② 变换距离（是距离空间 ✅）

$$d_1(x, y) = \frac{|x - y|}{1 + |x - y|}$$

验证三条公理均满足。$(\mathbb{R}^1, d_1)$ 也是距离空间。

**验证三角不等式的技巧**：

令 $f(t) = \frac{t}{1 + t}$（$t \geq 0$），则 $f'(t) = \frac{1}{(1+t)^2} > 0$，故 $f$ **单调递增**。

由标准距离的三角不等式 $|x - y| \leq |x - z| + |z - y|$，再利用 $f$ 单调递增：

$$\frac{|x - y|}{1 + |x - y|} \leq \frac{|x - z| + |z - y|}{1 + |x - z| + |z - y|} \leq \frac{|x - z|}{1 + |x - z|} + \frac{|z - y|}{1 + |z - y|}$$

最后一步不等式用到了 $\frac{a + b}{1 + a + b} \leq \frac{a}{1 + a} + \frac{b}{1 + b}$（$a, b \geq 0$）。

> **这个距离的特点**：无论 $x, y$ 相距多远，$d_1(x, y) < 1$ 恒成立。它把无穷大的实数轴"压缩"到了有界范围内。

#### ③ 平方距离（不是距离空间 ❌）

$$d_2(x, y) = (x - y)^2$$

**反例**：取 $x = 3, y = 2, z = x = 3$（即 $z = x$），则：

$$d_2(x, y) = (3 - 2)^2 = 1$$

但取另一组：$x = 0, z = 1, y = 3$：

$$d_2(x, y) = 9, \quad d_2(x, z) + d_2(z, y) = 1 + 4 = 5$$

$d_2(x, y) = 9 > 5 = d_2(x, z) + d_2(z, y)$，**三角不等式不满足**！

> **教训**：直觉上"距离的平方"似乎也能度量远近，但它违反三角不等式，因此不是合法的距离。

### 例2：$n$ 维欧氏空间 $\mathbb{R}^n$

设 $x = (x_1, x_2, \ldots, x_n)$，$y = (y_1, y_2, \ldots, y_n) \in \mathbb{R}^n$。

#### 欧氏距离（标准距离）

$$d_2(x, y) = \sqrt{\sum_{i=1}^{n} (x_i - y_i)^2}$$

- 当 $n = 1$ 时：$d(x, y) = |x - y|$（退化为实数轴上的距离）
- 当 $n = 2$ 时：$d(x, y) = \sqrt{(x_1 - y_1)^2 + (x_2 - y_2)^2}$（平面上的直线距离）

**三角不等式的证明**（核心！利用 Cauchy-Schwarz 不等式）：

令 $a_i = x_i - z_i$，$b_i = z_i - y_i$，则 $a_i + b_i = x_i - y_i$。

$$d_2(x, y) = \left[\sum_{i=1}^{n} (x_i - y_i)^2\right]^{1/2} = \left[\sum_{i=1}^{n} (a_i + b_i)^2\right]^{1/2}$$

展开平方：

$$= \left[\sum a_i^2 + 2\sum a_i b_i + \sum b_i^2\right]^{1/2}$$

由 Cauchy-Schwarz 不等式：$\sum a_i b_i \leq \left(\sum a_i^2\right)^{1/2}\left(\sum b_i^2\right)^{1/2}$，因此：

$$\leq \left[\sum a_i^2 + 2\left(\sum a_i^2\right)^{1/2}\left(\sum b_i^2\right)^{1/2} + \sum b_i^2\right]^{1/2}$$

注意到 $A^2 + 2AB + B^2 = (A + B)^2$，令 $A = (\sum a_i^2)^{1/2}$，$B = (\sum b_i^2)^{1/2}$：

$$= \left[\left(\sum a_i^2\right)^{1/2} + \left(\sum b_i^2\right)^{1/2}\right] = d_2(x, z) + d_2(z, y)$$

#### $\mathbb{R}^n$ 上的其他常见距离

| 名称 | 定义 | 几何意义 |
|------|------|----------|
| **曼哈顿距离** $d_1$ | $d_1(x,y) = \sum_{i=1}^{n} \|x_i - y_i\|$ | 折线/阶梯距离 |
| **切比雪夫距离** $d_\infty$ | $d_\infty(x,y) = \max_{1 \leq i \leq n} \|x_i - y_i\|$ | 棋盘距离，常用于图像处理 |
| **闵可夫斯基距离** $d_p$ | $d_p(x,y) = \left(\sum_{i=1}^{n} \|x_i - y_i\|^p\right)^{1/p}$，$p \geq 1$ | 以上距离的统一推广 |

**注意**：$d_\infty(x,y) = \min_{1 \leq i \leq n} |x_i - y_i|$ **不是距离**！

> **闵可夫斯基距离的理解**：从误差角度看，$d_p$ 可理解为所有分量误差的 $p$ 次方累加——所有分量对误差的贡献是一样的。$p$ 越大，最大分量误差的权重越高；当 $p \to \infty$ 时，$d_p \to d_\infty$。

#### 加权距离

$$d_p(x, y) = \left(\sum_{i=1}^{n} \alpha_i |x_i - y_i|^p\right)^{1/p}, \quad p \geq 1$$

其中 $\alpha_i$ 是权重。权重越大，该分量的近似精度要求越高，对应的误差越小。

#### $\mathbb{R}^2$ 中不同距离的几何直觉

以 $\mathbb{R}^2$ 为例，到原点距离为1的点构成的图形：

| 距离 | 等距线方程 | 几何形状 |
|------|-----------|----------|
| $d_2(x, o) = \sqrt{x_1^2 + x_2^2} = 1$ | $x_1^2 + x_2^2 = 1$ | **圆** |
| $d_1(x, o) = \|x_1\| + \|x_2\| = 1$ | $\|x_1\| + \|x_2\| = 1$ | **菱形**（旋转45°的正方形） |
| $d_\infty(x, o) = \max(\|x_1\|, \|x_2\|) = 1$ | $\max(\|x_1\|, \|x_2\|) = 1$ | **正方形** |

> 三种距离在同一个集合 $\mathbb{R}^2$ 上定义了三种不同的"等距面"，直观展示了**同一集合、不同距离 → 不同的几何结构**。

### 例3：连续函数空间 $C[a, b]$

$C[a, b]$ 表示定义在 $[a, b]$ 上的**所有连续函数**的集合。

$$d(x, y) = \max_{t \in [a, b]} |x(t) - y(t)|$$

**几何意义**：两条连续曲线上沿**竖直方向**最远的两点之间的距离。

**三角不等式的证明**：

$$\forall t \in [a, b]: \quad |x(t) - y(t)| = |x(t) - z(t) + z(t) - y(t)| \leq |x(t) - z(t)| + |z(t) - y(t)|$$

对右边分别取上界：

$$|x(t) - z(t)| \leq \max_{t} |x(t) - z(t)| = d(x, z)$$

$$|z(t) - y(t)| \leq \max_{t} |z(t) - y(t)| = d(z, y)$$

因此 $|x(t) - y(t)| \leq d(x, z) + d(z, y)$ 对所有 $t$ 成立，两边取 $\max$：

$$d(x, y) = \max_{t} |x(t) - y(t)| \leq d(x, z) + d(z, y) \qquad \square$$

### 例4：$p$ 方可积空间 $L^p[a, b]$

$L^p[a, b]$（$p \geq 1$）表示 $[a, b]$ 上 $p$ 方可积函数按“几乎处处相等”划分后的**等价类集合**：

$$L^p[a, b] = \left\{x(t) \;\bigg|\; \int_a^b |x(t)|^p \, dt < \infty\right\}\Big/\sim$$

定义距离：

$$d(x, y) = \left(\int_a^b |x(t) - y(t)|^p \, dt\right)^{1/p}$$

> **严格说明**：若把元素看成“函数本身”，可能出现 $d(x,y)=0$ 但 $x \neq y$（它们只是在零测集外相同）。把几乎处处相等的函数视为同一等价类后，上式才是严格意义上的距离。

- 当 $p = 2$ 时，$L^2[a, b]$ 称为**平方可积空间**
- 物理上，$L^2$ 中的函数代表**能量有限的信号**

> 三角不等式的验证需要用到 Minkowski 不等式的积分形式。

### 例5：$p$ 方可和空间 $l^p$

$l^p$（$p \geq 1$）是所有 $p$ 方可和的**无穷数列**构成的集合：

$$l^p = \left\{x = \{x_i\} \;\bigg|\; \sum_{i=1}^{\infty} |x_i|^p < \infty\right\}$$

对于 $x = \{x_i\}, y = \{y_i\} \in l^p$，定义距离：

$$d(x, y) = \left(\sum_{i=1}^{\infty} |x_i - y_i|^p\right)^{1/p}$$

- 当 $p = 2$ 时，$l^2$ 称为**平方可和的距离空间**

> 三角不等式的验证同样需要用到 Minkowski 不等式的离散形式。

### 各距离空间的应用场景

| 距离空间 | 距离公式 | 应用场景 |
|----------|----------|----------|
| $l^2$ | $d(x,y) = \sqrt{\sum_{i=1}^{n}(x_i - y_i)^2}$ | **离散采样序列**的误差度量 |
| $L^2[a,b]$ | $d(x,y) = \sqrt{\int_a^b(x(t)-y(t))^2 dt}$ | **连续函数**的误差度量 |

> 在数值分析中，插值法、最佳平方逼近、最佳一致逼近、最小二乘法等方法，本质上都是在不同的距离空间中寻找"最近的近似"。

## 5. 距离空间中的拓扑概念

### 5.1 邻域

> **定义**：设 $(X, d)$ 是距离空间，$x \in X$，$\varepsilon > 0$，则子集
>
> $$O(x, \varepsilon) = \{y \in X \mid d(x, y) < \varepsilon\}$$
>
> 称为 $x$ 的 **$\varepsilon$-邻域**。

**直觉**：以 $x$ 为中心、$\varepsilon$ 为半径的开球，**包含中心点 $x$ 本身**。

若去掉中心点，即 $O(x,\varepsilon)\setminus\{x\}$，则称为 $x$ 的**去心邻域**。

### 5.2 内点与开集

> **内点**：设 $A \subseteq X$，若存在 $O(x, \varepsilon) \subseteq A$，则称 $x$ 是 $A$ 的**内点**。
>
> **开集**：若 $A$ 中**所有的点都是内点**，则称 $A$ 是**开集**。

**直觉**：$A$ 是开集意味着 $A$ 中每个点周围都有一小片区域完全包含在 $A$ 内，没有"边界点"。

**具体例子**（$\mathbb{R}^1$ 中，标准距离）：

| 集合 | 是否开集 | 原因 |
|------|:-------:|------|
| $(0, 1)$ | ✓ | 每个点 $x \in (0,1)$ 都能找到 $\varepsilon$（取 $\varepsilon = \min(x, 1-x)$）使 $(x-\varepsilon, x+\varepsilon) \subset (0,1)$ |
| $[0, 1]$ | ✗ | 端点 $0$ 的任意邻域都包含负数，"溢出"了 $[0,1]$ |
| $(0, 1]$ | ✗ | 端点 $1$ 不是内点 |
| $\mathbb{R}$ | ✓ | 每个点的任意邻域都在 $\mathbb{R}$ 内 |
| $\varnothing$ | ✓ | 空集中没有点需要检验，条件"空成立" |

### 5.3 闭集

> **闭集**：设 $E$ 是一个集合，$A \subseteq E$，若 $A$ 的补集 $E \setminus A$ 为开集，则称 $A$ 为 $E$ 中的**闭集**。

**直觉**：闭集包含了自己所有的"边界点"。

**具体例子**（$\mathbb{R}^1$ 中，标准距离）：

| 集合 | 是否闭集 | 原因 |
|------|:-------:|------|
| $[0, 1]$ | ✓ | 补集 $(-\infty, 0) \cup (1, +\infty)$ 是开集 |
| $(0, 1)$ | ✗ | 补集 $(-\infty, 0] \cup [1, +\infty)$ 不是开集（$0$ 不是补集的内点） |
| $(0, 1]$ | ✗ | 既不开也不闭——$0$ 是极限点但不在集合内（不闭），$1$ 不是内点（不开） |
| $\mathbb{R}$ | ✓ | 补集 $\varnothing$ 是开集 |

> **注意**："开"和"闭"不是互斥的。$\mathbb{R}$ 和 $\varnothing$ 都**既是开集又是闭集**（称为**闭开集**）。

### 5.4 极限点（聚点）与导集

> **极限点**：设 $E$ 是距离空间，$A \subseteq E$，$x_0 \in E$。若在 $x_0$ 的**任意去心邻域** $O(x_0, \varepsilon)\setminus\{x_0\}$ 内都含有属于 $A$ 的点，则称 $x_0$ 为 $A$ 的一个**极限点**（或聚点）。
>
> $A$ 的所有极限点的全体称为 $A$ 的**导集**，记作 $A'$。

### 5.5 有界集

> **有界集**：设 $A \subseteq E$，若 $\exists x_0 \in E$ 和有限数 $r > 0$，使得 $\forall x \in A$ 都有 $d(x, x_0) \leq r$，则称 $A$ 是距离空间 $E$ 中的**有界集**。

**直觉**：$A$ 能被某个有限大小的球包住。

## 6. 收敛的概念

### 6.1 收敛点列的定义

> **定义**：设 $X$ 是距离空间，$\{x_n\}$ 是 $X$ 中的点列，$x \in X$。若
>
> $$\forall \varepsilon > 0, \; \exists N, \; \text{当} \; n > N \; \text{时}, \; d(x_n, x) < \varepsilon$$
>
> （即 $n \to \infty$ 时，$d(x_n, x) \to 0$）
>
> 则称点列 $\{x_n\}$ 在 $X$ 中**按距离收敛于 $x$**，记作 $\lim_{n \to \infty} x_n = x$ 或 $x_n \to x \; (n \to \infty)$。

**理解**：在不同的距离空间中，"收敛"的具体含义取决于所定义的距离。例如：

| 空间 | 收敛的含义 |
|------|-----------|
| $\mathbb{R}^n$ | 每个分量都收敛 |
| $C[a,b]$ | **一致收敛**（函数列在整个区间上均匀趋近） |
| $L^2[a,b]$ | **均方收敛**（能量意义下趋近） |

### 6.2 收敛点列的性质

**性质1（极限唯一性）**：在距离空间 $X$ 中，收敛点列 $\{x_n\}$ 的极限点是**唯一的**。

**性质2（有界性）**：在距离空间 $X$ 中的收敛点列 $\{x_n\}$ **必有界**。

**性质3（距离的连续性）**：在距离空间 $X$ 中，距离 $d(x, y)$ 是两个变元 $x, y$ 的**连续泛函**。即当 $x_n \to x_0$，$y_n \to y_0$ 时：

$$d(x_n, y_n) \to d(x_0, y_0) \quad (n \to \infty)$$

> 性质3的重要意义：它保证了"极限"与"距离"可以交换顺序，是后续理论推导的基础。

## 7. Cauchy 点列与完备性

### 7.1 Cauchy 点列的定义

> **定义**：设 $\{x_n\}$ 是距离空间 $X$ 中的一个点列，若
>
> $$\forall \varepsilon > 0, \; \exists N, \; \text{当} \; n, m > N \; \text{时}, \; d(x_n, x_m) < \varepsilon$$
>
> （即 $n, m \to \infty$ 时，$d(x_n, x_m) \to 0$）
>
> 则称 $\{x_n\}$ 为**基本点列**或 **Cauchy 点列**。

**与收敛的区别**：

| | 收敛点列 | Cauchy 点列 |
|--|---------|------------|
| 判定条件 | $d(x_n, x) \to 0$（需要知道极限 $x$） | $d(x_n, x_m) \to 0$（不需要知道极限） |
| 含义 | 点列趋向某个确定的点 | 点列中的点互相越来越近 |

### 7.2 收敛与 Cauchy 的关系

**性质4**：

- **收敛 $\Rightarrow$ Cauchy**：若 $\{x_n\}$ 收敛，则 $\{x_n\}$ 一定是 Cauchy 点列

  证明：设 $x_n \to x$，则 $d(x_n, x_m) \leq d(x_n, x) + d(x, x_m) \to 0$

- **Cauchy $\not\Rightarrow$ 收敛**：反之不一定成立！

#### 反例1：有理数空间 $\mathbb{Q}$

在有理数空间 $\mathbb{Q}$（按欧氏距离）中，点列：

$$1, \; 1.4, \; 1.41, \; 1.414, \; 1.4142, \; \ldots$$

这个点列逼近 $\sqrt{2}$，它是 $\mathbb{Q}$ 中的 Cauchy 点列，但 $\sqrt{2} \notin \mathbb{Q}$，因此在 $\mathbb{Q}$ 中**不收敛**。

> **直觉**：$\mathbb{Q}$ "有洞"——点列想要收敛到的那个点不在空间里。

#### 反例2：开区间 $(0, 1)$

设 $X = (0, 1)$，按标准距离 $d(x, y) = |x - y|$。点列 $x_n = \frac{1}{n+1}$ 是 $X$ 中的 Cauchy 列，但极限值 $0 \notin (0, 1)$，因此在 $X$ 中**不收敛**。

### 7.3 完备性的定义

> **定义（完备距离空间）**：$X$ 为距离空间，若 $X$ 中的**任一 Cauchy 点列都在 $X$ 中收敛**，则称 $X$ 是**完备的距离空间**。

**理解**：完备性意味着空间中"没有洞"——所有"应该存在的极限点"都在空间内。

> **结论**：在完备的距离空间中，收敛点列与 Cauchy 点列**等价**：
>
> $$\{x_n\} \text{ 收敛} \iff \{x_n\} \text{ 是 Cauchy 点列}$$

### 7.4 完备性的典型例子

| 空间 | 距离 | 是否完备 |
|------|------|----------|
| $\mathbb{R}^n$ | 欧氏距离 | ✅ 完备 |
| $\mathbb{Q}$ | 欧氏距离 | ❌ 不完备 |
| $C[a,b]$ | $d(x,y) = \max_{t \in [a,b]}\|x(t) - y(t)\|$ | ✅ 完备 |
| $C[a,b]$ | $d(x,y) = \int_a^b \|x(t) - y(t)\| \, dt$ | ❌ 不完备 |
| $C[a,b]$ | $d(x,y) = \left(\int_a^b \|x(t) - y(t)\|^2 \, dt\right)^{1/2}$ | ❌ 不完备 |
| $L^2[a,b]$ | $d(x,y) = \left(\int_a^b \|x(t) - y(t)\|^2 \, dt\right)^{1/2}$ | ✅ 完备 |

> **关键观察**：
> - **同一个集合**（如 $C[a,b]$）在不同距离下，完备性可能不同！
> - $C[a,b]$ 在最大值距离下完备，但在积分距离下不完备——因为按积分距离收敛（均方收敛）的连续函数列，其极限可能不连续，"跑出"了 $C[a,b]$ 这个空间
> - $L^2[a,b]$ 包含了更多函数（不要求连续），因此"补上了洞"，变成了完备空间
> - 不完备的空间可以**完备化**（类似于把 $\mathbb{Q}$ 完备化得到 $\mathbb{R}$）

## 8. 总结与展望

### 本节知识脉络

```
距离空间（度量空间）
├── 定义：非空集合 + 满足三条公理的距离
│   ├── 非负性
│   ├── 对称性
│   └── 三角不等式 ← Hölder/Cauchy/Minkowski 不等式
├── 典型例子
│   ├── R^n（欧氏/曼哈顿/切比雪夫/闵可夫斯基距离）
│   ├── C[a,b]（连续函数空间）
│   ├── L^p[a,b]（p方可积空间）
│   └── l^p（p方可和空间）
├── 拓扑概念
│   ├── 邻域、内点、开集、闭集
│   ├── 极限点（聚点）、导集
│   └── 有界集
├── 收敛理论
│   ├── 按距离收敛的定义
│   ├── 性质：唯一性、有界性、距离连续性
│   └── Cauchy 点列
└── 完备性
    ├── 完备 ⟺ 每个Cauchy列都收敛
    └── 不完备空间可以完备化
```

### 距离空间的局限性

距离空间**只有距离结构**，存在以下限制：

1. **无代数结构**——不能对元素进行加减运算
2. **无法衡量元素的大小**——只能衡量两个元素之间的远近
3. 应用受到限制

> **下一步**：引入**赋范线性空间**，在距离结构的基础上增加代数结构（线性运算）和范数（元素的大小），从而获得更丰富的理论工具。
