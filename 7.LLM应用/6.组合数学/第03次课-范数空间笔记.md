## 1. 引入：从距离到范数

### 为什么需要范数？

在 [[第02次课-距离空间笔记|上一讲]] 中，我们学习了距离空间——在任意集合上定义距离函数 $d(x,y)$，从而建立收敛理论。但距离空间有一个局限：**它只能衡量两个元素之间的"远近"，无法衡量单个元素的"大小"**。

| 概念 | 能做什么 | 不能做什么 |
|------|----------|------------|
| 距离 $d(x,y)$ | 衡量两元素间的远近 | 衡量单个元素的"大小" |
| 范数 $\|x\|$ | 衡量单个元素的"长度/大小" | — |

**核心关系**：有了范数，就能自然地定义距离：

$$d(x, y) = \|x - y\|$$

> 范数比距离更"强"：每个范数都能诱导出距离，但不是每个距离都能由范数诱导。

## 2. 向量范数

### 2.1 向量范数的定义

> **定义1（向量范数）**：设 $n$ 维向量 $\mathbf{x} = (x_1, x_2, \ldots, x_n)^T \in \mathbb{R}^n$，若对应某个非负实数 $\|\mathbf{x}\|$ 满足以下三个条件，则称 $\|\mathbf{x}\|$ 是 $\mathbb{R}^n$ 上的一个**向量范数**：

| 公理 | 表达式 | 直观含义 |
|------|--------|----------|
| **正定性** | $\|\mathbf{x}\| \geq 0$，且 $\|\mathbf{x}\| = 0 \Leftrightarrow \mathbf{x} = \mathbf{0}$ | 长度非负，只有零向量长度为0 |
| **正齐性** | $\lVert\alpha \mathbf{x}\rVert = \lvert\alpha\rvert \cdot \lVert\mathbf{x}\rVert$ | 伸缩向量，长度等比例变化 |
| **三角不等式** | $\|\mathbf{x} + \mathbf{y}\| \leq \|\mathbf{x}\| + \|\mathbf{y}\|$ | 两边之和大于第三边 |

**与距离公理的对比**：

| | 距离公理 | 范数公理 |
|---|---------|---------|
| 公理1 | 非负性 + 同一性 | **正定性**（等价） |
| 公理2 | 对称性 $d(x,y) = d(y,x)$ | **正齐性**（更强，包含了对称性） |
| 公理3 | 三角不等式 | 三角不等式（相同） |

> **为什么正齐性更强？** 正齐性意味着 $\|-\mathbf{x}\| = |-1| \cdot \|\mathbf{x}\| = \|\mathbf{x}\|$，这自然保证了对称性：$\|\mathbf{x} - \mathbf{y}\| = \|(-1)(\mathbf{y} - \mathbf{x})\| = \|\mathbf{y} - \mathbf{x}\|$。而且正齐性还给出了**缩放关系**，这是距离公理所没有的。

### 2.2 三种常用向量范数

设 $n$ 维向量 $\mathbf{x} = (x_1, x_2, \ldots, x_n)^T \in \mathbb{R}^n$：

| 名称 | 符号 | 定义 | 直观含义 |
|------|------|------|----------|
| **$\infty$-范数**（最大范数） | $\lVert\mathbf{x}\rVert_\infty$ | $\displaystyle\max_{1 \leq i \leq n} \lvert x_i \rvert$ | 分量绝对值的最大值 |
| **$1$-范数**（曼哈顿范数） | $\lVert\mathbf{x}\rVert_1$ | $\displaystyle\sum_{i=1}^{n} \lvert x_i \rvert$ | 各分量绝对值之和 |
| **$2$-范数**（欧氏范数） | $\lVert\mathbf{x}\rVert_2$ | $\displaystyle\left(\sum_{i=1}^{n} x_i^2\right)^{1/2}$ | 通常意义下的向量长度 |

**用范数度量误差**：

$$d(\mathbf{x}, \mathbf{y}) = \|\mathbf{x} - \mathbf{y}\|$$

选用不同范数，就得到不同的误差度量方式。

### 2.3 验证 $\infty$-范数满足三条公理

**验证 $\|\mathbf{x}\|_\infty = \max_{1 \leq i \leq n} |x_i|$ 是向量范数**：

**① 正定性**：

$$\|\mathbf{x}\|_\infty = \max_{1 \leq i \leq n} |x_i| \geq 0$$

且 $\|\mathbf{x}\|_\infty = 0 \Leftrightarrow \max_{1 \leq i \leq n} |x_i| = 0 \Leftrightarrow |x_i| = 0 \text{ for all } i \Leftrightarrow \mathbf{x} = \mathbf{0}$

**② 正齐性**：

$$\|\alpha \mathbf{x}\|_\infty = \max_{1 \leq i \leq n} |\alpha x_i| = \max_{1 \leq i \leq n} (|\alpha| \cdot |x_i|) = |\alpha| \cdot \max_{1 \leq i \leq n} |x_i| = |\alpha| \cdot \|\mathbf{x}\|_\infty$$

**③ 三角不等式**：

$$\|\mathbf{x} + \mathbf{y}\|_\infty = \max_{1 \leq i \leq n} |x_i + y_i| \leq \max_{1 \leq i \leq n} (|x_i| + |y_i|)$$

$$\leq \max_{1 \leq i \leq n} |x_i| + \max_{1 \leq i \leq n} |y_i| = \|\mathbf{x}\|_\infty + \|\mathbf{y}\|_\infty$$

> **关键技巧**：三角不等式的证明用了两步放大：先用绝对值三角不等式 $|x_i + y_i| \leq |x_i| + |y_i|$，再用"部分之和 ≤ 各自最大值之和"。

### 2.4 验证 $2$-范数的三角不等式

**验证 $\|\mathbf{x}\|_2 = \left(\sum_{i=1}^{n} x_i^2\right)^{1/2}$ 的三角不等式**：

$$\|\mathbf{x} + \mathbf{y}\|_2^2 = \sum_{i=1}^{n} (x_i + y_i)^2 = \sum_{i=1}^{n} x_i^2 + 2\sum_{i=1}^{n} x_i y_i + \sum_{i=1}^{n} y_i^2$$

关键一步：利用 **Cauchy-Schwarz 不等式**：

$$\sum_{i=1}^{n} x_i y_i \leq \left(\sum_{i=1}^{n} x_i^2\right)^{1/2} \left(\sum_{i=1}^{n} y_i^2\right)^{1/2}$$

因此：

$$\|\mathbf{x} + \mathbf{y}\|_2^2 \leq \|\mathbf{x}\|_2^2 + 2\|\mathbf{x}\|_2 \|\mathbf{y}\|_2 + \|\mathbf{y}\|_2^2 = (\|\mathbf{x}\|_2 + \|\mathbf{y}\|_2)^2$$

两边开方，得 $\|\mathbf{x} + \mathbf{y}\|_2 \leq \|\mathbf{x}\|_2 + \|\mathbf{y}\|_2$。

### 2.5 三种范数的几何意义

在 $\mathbb{R}^2$ 中，**单位球**（范数等于1的所有点的集合）展示了不同范数的几何特征：

| 范数 | 单位球形状 | 图形描述 |
|------|-----------|----------|
| $\lVert\mathbf{x}\rVert_1 = 1$ | 正方形（旋转45°） | $\lvert x_1\rvert + \lvert x_2\rvert = 1$，顶点在坐标轴上 |
| $\lVert\mathbf{x}\rVert_2 = 1$ | 圆 | $x_1^2 + x_2^2 = 1$ |
| $\lVert\mathbf{x}\rVert_\infty = 1$ | 正方形（边平行于坐标轴） | $\max(\lvert x_1\rvert, \lvert x_2\rvert) = 1$ |

## 3. 范数的重要性质

### 3.1 范数的连续性

> **定理1（范数的连续性）**：$\|\mathbf{x}\|$ 是 $\mathbb{R}^n$ 上任一种向量范数，则 $\|\mathbf{x}\|$ 是向量 $\mathbf{x}$ 的分量 $(x_1, x_2, \ldots, x_n)$ 的**连续函数**。

**证明**：设 $\mathbf{x}_n \to \mathbf{x}$，则：

$$|\|\mathbf{x}_n\| - \|\mathbf{x}\|| \leq \|\mathbf{x}_n - \mathbf{x}\|$$

这是由范数三角不等式推导出的**反三角不等式**：

$$\|\mathbf{x}\| = \|(\mathbf{x} - \mathbf{y}) + \mathbf{y}\| \leq \|\mathbf{x} - \mathbf{y}\| + \|\mathbf{y}\|$$

$$\Rightarrow \|\mathbf{x}\| - \|\mathbf{y}\| \leq \|\mathbf{x} - \mathbf{y}\|$$

对称地，$\|\mathbf{y}\| - \|\mathbf{x}\| \leq \|\mathbf{y} - \mathbf{x}\| = \|\mathbf{x} - \mathbf{y}\|$

所以 $|\|\mathbf{x}\| - \|\mathbf{y}\|| \leq \|\mathbf{x} - \mathbf{y}\|$

因此当 $\mathbf{x}_n \to \mathbf{x}$ 时，$\|\mathbf{x}_n - \mathbf{x}\| \to 0$，故 $\|\mathbf{x}_n\| \to \|\mathbf{x}\|$。

> **实用意义**：范数连续意味着"微小的输入扰动只会导致微小的范数变化"，这是数值分析中**误差分析**的理论基础。

### 3.2 向量范数的等价性

> **定义2（范数等价）**：设 $\|\cdot\|_s$ 和 $\|\cdot\|_t$ 是 $\mathbb{R}^n$ 上两种范数。若 $\|\mathbf{x}_n\|_s \to 0$ 等价于 $\|\mathbf{x}_n\|_t \to 0$（即在一种范数下收敛必然在另一种范数下也收敛），则称两种范数**等价**。

**等价的充分条件**：若存在常数 $C_1, C_2 > 0$，使得

$$C_1 \|\mathbf{x}\|_s \leq \|\mathbf{x}\|_t \leq C_2 \|\mathbf{x}\|_s, \quad \forall \mathbf{x} \in \mathbb{R}^n$$

则 $\|\cdot\|_s$ 与 $\|\cdot\|_t$ 等价。

> **定理2（向量范数的等价性）**：$\mathbb{R}^n$ 上**任意两种**向量范数都是等价的。

**这意味着什么？** 在有限维空间中，不管选用哪种范数来研究收敛性，**结论都是一样的**——序列在一种范数下收敛，必然在其他所有范数下也收敛。

### 3.3 例：证明 $\|\cdot\|_\infty$ 与 $\|\cdot\|_1$ 等价

只需找到 $C_1, C_2 > 0$ 使得 $C_1\|\mathbf{x}\|_\infty \leq \|\mathbf{x}\|_1 \leq C_2\|\mathbf{x}\|_\infty$：

**左侧不等式**：

$$\|\mathbf{x}\|_1 = \sum_{i=1}^{n} |x_i| \geq \max_{1 \leq i \leq n} |x_i| = \|\mathbf{x}\|_\infty$$

所以 $C_1 = 1$。

**右侧不等式**：

$$\|\mathbf{x}\|_1 = \sum_{i=1}^{n} |x_i| \leq n \cdot \max_{1 \leq i \leq n} |x_i| = n \|\mathbf{x}\|_\infty$$

所以 $C_2 = n$。

综合：$\|\mathbf{x}\|_\infty \leq \|\mathbf{x}\|_1 \leq n \|\mathbf{x}\|_\infty$

> **注意**：范数等价具有**传递性**——若 $\|\cdot\|_s$ 等价于 $\|\cdot\|_t$，$\|\cdot\|_t$ 等价于 $\|\cdot\|_r$，则 $\|\cdot\|_s$ 等价于 $\|\cdot\|_r$。

### 3.4 向量序列的收敛性

> **定义3（向量序列收敛）**：设 $\{\mathbf{x}^{(k)}\}$ 为 $\mathbb{R}^n$ 中的向量序列，$\mathbf{x}^* \in \mathbb{R}^n$。若
>
> $$\lim_{k \to \infty} x_i^{(k)} = x_i^*, \quad i = 1, 2, \ldots, n$$
>
> 则称 $\{\mathbf{x}^{(k)}\}$ **按坐标收敛**于 $\mathbf{x}^*$，记作 $\lim_{k \to \infty} \mathbf{x}^{(k)} = \mathbf{x}^*$。

> **定理3**：$\lim_{k \to \infty} \mathbf{x}^{(k)} = \mathbf{x}^*$ 的充要条件是：对**任意一种**范数，$\|\mathbf{x}^{(k)} - \mathbf{x}^*\| \to 0$。

**证明要点**：

利用范数等价性：存在 $C_1, C_2 > 0$ 使得

$$C_1 \|\mathbf{x}^{(k)} - \mathbf{x}^*\|_s \leq \|\mathbf{x}^{(k)} - \mathbf{x}^*\|_t \leq C_2 \|\mathbf{x}^{(k)} - \mathbf{x}^*\|_s$$

由夹逼定理，一种范数下趋于0，另一种也趋于0。

> **意义**：这给了我们选择范数的自由——在实际计算中可以选最方便的范数来判断收敛。

## 4. 矩阵范数

### 4.1 矩阵范数的定义

> **定义4（矩阵范数）**：设 $n$ 阶方阵 $A = (a_{ij})_{n \times n} \in \mathbb{R}^{n \times n}$，若对应非负实数 $\|A\|$ 满足以下**四条**条件，则称 $\|A\|$ 为 $n$ 阶矩阵 $A$ 的范数：

| 公理 | 表达式 | 说明 |
|------|--------|------|
| **正定性** | $\|A\| \geq 0$，且 $\|A\| = 0 \Leftrightarrow A = O$ | 与向量范数相同 |
| **正齐性** | $\lVert\alpha A\rVert = \lvert\alpha\rvert \cdot \lVert A\rVert$ | 与向量范数相同 |
| **三角不等式** | $\|A + B\| \leq \|A\| + \|B\|$ | 与向量范数相同 |
| **乘积不等式** | $\|AB\| \leq \|A\| \cdot \|B\|$ | ⬅ **新增公理** |

**关键区别**：矩阵范数比向量范数**多了一条公理——乘积不等式**（也称相容性条件或次可乘性）。

> **为什么需要乘积不等式？** 因为矩阵可以相乘，在误差分析中经常需要估计乘积的范数。乘积不等式保证了 $\|AB\|$ 可以用 $\|A\|$ 和 $\|B\|$ 来估计上界。

### 4.2 Frobenius 范数（F-范数）

定义 $A = (a_{ij})_{n \times n}$ 的 **Frobenius 范数**：

$$\|A\|_F = \left(\sum_{i=1}^{n} \sum_{j=1}^{n} a_{ij}^2\right)^{1/2}$$

**理解方式**：把矩阵 $A$ 的 $n^2$ 个元素**拉成一个 $n^2$ 维向量**，然后取该向量的 $2$-范数。

可以验证 F-范数满足矩阵范数的所有四条公理。

> **注意**：F-范数**不是**由任何向量范数诱导的算子范数。特别地，$\|I\|_F = \sqrt{n} \neq 1$（$n$ 阶单位矩阵），因此定理7（$\|B\| < 1 \Rightarrow I-B$ 可逆）**不能直接使用 F-范数**。F-范数虽然是矩阵范数，但它不具备算子范数特有的性质 $\|I\| = 1$。

### 4.3 矩阵范数与向量范数的相容性

> **定义5（相容性）**：若对所有 $A \in \mathbb{R}^{n \times n}$ 及 $\mathbf{x} \in \mathbb{R}^n$，都有
>
> $$\|A\mathbf{x}\| \leq \|A\| \cdot \|\mathbf{x}\|$$
>
> 则称矩阵范数 $\|A\|$ 与向量范数 $\|\mathbf{x}\|$ **相容**。

**意义**：相容性保证了"用矩阵作用于向量后，结果的范数不超过矩阵范数乘以向量范数"。这在误差传播分析中至关重要——如果 $\mathbf{x}$ 有误差 $\delta\mathbf{x}$，则 $\|A \cdot \delta\mathbf{x}\| \leq \|A\| \cdot \|\delta\mathbf{x}\|$。

### 4.4 算子范数（诱导范数）

> **定义6（算子范数）**：给定向量范数 $\|\cdot\|_v$，定义矩阵的**算子范数**：
>
> $$\|A\| = \max_{\mathbf{x} \neq \mathbf{0}} \frac{\|A\mathbf{x}\|_v}{\|\mathbf{x}\|_v}$$

**直观含义**：算子范数衡量矩阵 $A$ 作为线性变换能"放大"向量的**最大倍数**。

**两个重要性质**：
1. 算子范数**满足矩阵范数的四条公理**
2. 算子范数**与对应的向量范数自动相容**

### 4.5 三种常用矩阵算子范数

> **定理4**：设 $A \in \mathbb{R}^{n \times n}$，$\mathbf{x} \in \mathbb{R}^n$，则：

| 名称 | 定义 | 计算公式 | 记忆口诀 |
|------|------|----------|----------|
| **$\infty$-范数（行范数）** | $\displaystyle\max_{\mathbf{x} \neq 0} \frac{\lVert A\mathbf{x}\rVert_\infty}{\lVert\mathbf{x}\rVert_\infty}$ | $\displaystyle\max_{1 \leq i \leq n} \sum_{j=1}^{n} \lvert a_{ij}\rvert$ | 每行绝对值之和取最大 |
| **$1$-范数（列范数）** | $\displaystyle\max_{\mathbf{x} \neq 0} \frac{\lVert A\mathbf{x}\rVert_1}{\lVert\mathbf{x}\rVert_1}$ | $\displaystyle\max_{1 \leq j \leq n} \sum_{i=1}^{n} \lvert a_{ij}\rvert$ | 每列绝对值之和取最大 |
| **$2$-范数（谱范数）** | $\displaystyle\max_{\mathbf{x} \neq 0} \frac{\|A\mathbf{x}\|_2}{\|\mathbf{x}\|_2}$ | $\sqrt{\lambda_{\max}(A^T A)}$ | $A^TA$ 最大特征值的平方根 |

> **记忆技巧**：$\infty$-范数是**行**范数，$1$-范数是**列**范数——注意"行"和"列"与直觉可能相反，需要特别记忆。

### 4.6 行范数的证明

**证明** $\|A\|_\infty = \max_{1 \leq i \leq n} \sum_{j=1}^{n} |a_{ij}|$：

设 $\|\mathbf{x}\|_\infty \leq 1$（即 $|x_j| \leq 1$），则 $A\mathbf{x}$ 的第 $i$ 个分量为：

$$(A\mathbf{x})_i = \sum_{j=1}^{n} a_{ij} x_j$$

取绝对值：

$$|(A\mathbf{x})_i| \leq \sum_{j=1}^{n} |a_{ij}| \cdot |x_j| \leq \sum_{j=1}^{n} |a_{ij}|$$

因此：

$$\|A\mathbf{x}\|_\infty = \max_{1 \leq i \leq n} |(A\mathbf{x})_i| \leq \max_{1 \leq i \leq n} \sum_{j=1}^{n} |a_{ij}|$$

等号可达：取特殊向量 $x_j = \text{sign}(a_{i_0 j})$（其中 $i_0$ 是使行和最大的那一行），此时 $\|\mathbf{x}\|_\infty = 1$ 且等号成立。

### 4.7 计算实例

设 $A = \begin{pmatrix} 1 & -3 \\ -1 & 3 \end{pmatrix}$，计算 $A$ 的各种范数：

**$\infty$-范数（行范数）**：

$$\|A\|_\infty = \max\{|1| + |-3|, \; |-1| + |3|\} = \max\{4, 4\} = 4$$

**$1$-范数（列范数）**：

$$\|A\|_1 = \max\{|1| + |-1|, \; |-3| + |3|\} = \max\{2, 6\} = 6$$

**F-范数**：

$$\|A\|_F = \sqrt{1^2 + (-3)^2 + (-1)^2 + 3^2} = \sqrt{1 + 9 + 1 + 9} = \sqrt{20} = 2\sqrt{5}$$

**$2$-范数（谱范数）**：

先计算 $A^T A$：

$$A^T A = \begin{pmatrix} 1 & -1 \\ -3 & 3 \end{pmatrix} \begin{pmatrix} 1 & -3 \\ -1 & 3 \end{pmatrix} = \begin{pmatrix} 2 & -6 \\ -6 & 18 \end{pmatrix}$$

> 注：这里 $A^T = \begin{pmatrix} 1 & -1 \\ -3 & 3 \end{pmatrix}$

求 $A^T A$ 的特征值：

$$\det(A^T A - \lambda I) = (2 - \lambda)(18 - \lambda) - 36 = \lambda^2 - 20\lambda = \lambda(\lambda - 20) = 0$$

$$\lambda_1 = 0, \quad \lambda_2 = 20$$

（验算：展开 $(2-\lambda)(18-\lambda) - 36 = 36 - 2\lambda - 18\lambda + \lambda^2 - 36 = \lambda^2 - 20\lambda$）

因此：

$$\|A\|_2 = \sqrt{\lambda_{\max}} = \sqrt{20} = 2\sqrt{5}$$

> **观察**：本例中 $\|A\|_F = \|A\|_2 = 2\sqrt{5}$，但这只是巧合，一般情况下 $\|A\|_F \neq \|A\|_2$。

## 5. 谱半径与相关定理

### 5.1 谱半径的定义

> **定义7（谱半径）**：设 $A \in \mathbb{R}^{n \times n}$ 的特征值为 $\lambda_1, \lambda_2, \ldots, \lambda_n$，定义
>
> $$\rho(A) = \max_{1 \leq i \leq n} |\lambda_i|$$
>
> 为矩阵 $A$ 的**谱半径**。

谱半径就是**所有特征值的模的最大值**。

### 5.2 谱半径与范数的关系

> **定理5**：对任意矩阵范数 $\|\cdot\|$，有 $\rho(A) \leq \|A\|$。

**证明**：设 $\lambda_i$ 是 $A$ 的任意特征值，$\mathbf{u}_i$ 是对应特征向量（$\mathbf{u}_i \neq \mathbf{0}$），则：

$$A\mathbf{u}_i = \lambda_i \mathbf{u}_i$$

$$|\lambda_i| \cdot \|\mathbf{u}_i\| = \|\lambda_i \mathbf{u}_i\| = \|A\mathbf{u}_i\| \leq \|A\| \cdot \|\mathbf{u}_i\|$$

因为 $\|\mathbf{u}_i\| \neq 0$，两边除以 $\|\mathbf{u}_i\|$，得 $|\lambda_i| \leq \|A\|$。

对所有特征值取最大：$\rho(A) = \max |\lambda_i| \leq \|A\|$。

**意义**：谱半径是矩阵范数的**下界**——任何范数都不小于谱半径。

> **应用预告（迭代法收敛条件）**：在后续课程中，迭代法 $\mathbf{x}^{(k+1)} = B\mathbf{x}^{(k)} + \mathbf{g}$ 收敛的**充要条件**是 $\rho(B) < 1$。由于 $\rho(B) \leq \|B\|$，若能验证某种范数下 $\|B\| < 1$，就能直接推出收敛。反之，即使所有范数 $\|B\| \geq 1$，只要 $\rho(B) < 1$，迭代仍然收敛——谱半径才是判定收敛的**精确准则**。

### 5.3 对称矩阵的谱范数

> **定理6**：若 $A \in \mathbb{R}^{n \times n}$ 是**对称矩阵**，则 $\|A\|_2 = \rho(A)$。

**证明**：对称矩阵满足 $A^T = A$，因此：

$$A^T A = A^2$$

$A^T A$ 的特征值为 $\lambda_i^2$（$\lambda_i$ 是 $A$ 的特征值），所以：

$$\|A\|_2 = \sqrt{\lambda_{\max}(A^T A)} = \sqrt{\max_{i} \lambda_i^2} = \max_i |\lambda_i| = \rho(A)$$

> **注意**：此定理仅对**对称矩阵**成立。一般矩阵的 $2$-范数 $\neq \rho(A)$。

### 5.4 矩阵可逆性的判定

> **定理7**：设 $\|\cdot\|$ 是由某个向量范数诱导的**算子范数**。若 $\|B\| < 1$，则 $I - B$ 是**非奇异矩阵**（可逆），且
>
> $$\frac{1}{1 + \|B\|} \leq \|(I - B)^{-1}\| \leq \frac{1}{1 - \|B\|}$$

**证明**（反证法证非奇异性）：

若 $I - B$ 奇异，则存在非零向量 $\mathbf{x}_0$ 使得 $(I - B)\mathbf{x}_0 = \mathbf{0}$，即 $\mathbf{x}_0 = B\mathbf{x}_0$。

$$\|\mathbf{x}_0\| = \|B\mathbf{x}_0\| \leq \|B\| \cdot \|\mathbf{x}_0\|$$

因为 $\mathbf{x}_0 \neq \mathbf{0}$，所以 $\|B\| \geq 1$，与 $\|B\| < 1$ 矛盾。故 $I - B$ 非奇异。

**说明**：这里必须使用诱导范数，因为它满足 $\|I\| = 1$。像 Frobenius 范数这类一般矩阵范数不一定满足这一点，因此不能直接套用本结论。

**上界证明**：

由 $(I - B)^{-1} = I + B(I - B)^{-1}$：

$$\|(I - B)^{-1}\| = \|I + B(I-B)^{-1}\| \leq 1 + \|B\| \cdot \|(I-B)^{-1}\|$$

设 $M = \|(I - B)^{-1}\|$，则 $M \leq 1 + \|B\| \cdot M$，即 $M(1 - \|B\|) \leq 1$：

$$\|(I - B)^{-1}\| \leq \frac{1}{1 - \|B\|}$$

**下界证明**：

由

$$I = (I-B)(I-B)^{-1}$$

可得

$$1 = \|I\| \leq \|I-B\| \cdot \|(I-B)^{-1}\| \leq (1+\|B\|)\cdot \|(I-B)^{-1}\|$$

从而

$$\|(I-B)^{-1}\| \geq \frac{1}{1+\|B\|}$$

> **应用场景**：当矩阵 $A$ 接近单位矩阵（即 $A = I - B$ 且 $B$ 很小）时，可以用此定理判断 $A$ 可逆并估计 $A^{-1}$ 的范数。这在**迭代法**的收敛性分析中非常重要。

## 6. 赋范线性空间

### 6.1 线性空间回顾

> **定义（线性空间）**：设 $E$ 是非空集合，$K$ 是实数域（或复数域）。在 $E$ 中定义两种运算：
> - **加法**：$\forall x, y \in E$，存在唯一 $z \in E$，记作 $z = x + y$
> - **数乘**：$\forall \alpha \in K, x \in E$，存在唯一 $\alpha x \in E$
>
> 若满足以下**八条运算规律**，则称 $E$ 是 $K$ 上的**线性空间**（向量空间）。

**八条运算规律**：

| 序号 | 加法规律 | 序号 | 数乘规律 |
|------|----------|------|----------|
| (1) | $x + y = y + x$（交换律） | (5) | $(\alpha\beta)x = \alpha(\beta x)$（结合律） |
| (2) | $(x+y)+z = x+(y+z)$（结合律） | (6) | $1 \cdot x = x$，$0 \cdot x = \mathbf{0}$（单位元） |
| (3) | 存在零元素 $\mathbf{0}$，$x + \mathbf{0} = x$ | (7) | $(\alpha+\beta)x = \alpha x + \beta x$（分配律1） |
| (4) | 存在负元素 $-x$，$x + (-x) = \mathbf{0}$ | (8) | $\alpha(x+y) = \alpha x + \alpha y$（分配律2） |

### 6.2 线性空间的例子

| 例 | 空间 | 是否线性空间 | 说明 |
|----|------|:--------:|------|
| 例1 | $\mathbb{R}^n$（$n$ 维向量全体） | ✓ | 通常意义下加法、数乘 |
| 例2 | $P_n(x)$（次数 $\leq n$ 的多项式全体） | ✓ | 两个 $\leq n$ 次多项式之和仍 $\leq n$ 次 |
| 例3 | $Q_n(x)$（次数 $= n$ 的多项式全体） | ✗ | 两个 $n$ 次多项式相加后次数可能 $< n$，不封闭 |
| 例4 | $C[a,b]$、$L^2[a,b]$ | ✓ | 函数的加法 $(x+y)(t) = x(t) + y(t)$，数乘 $(\alpha x)(t) = \alpha \cdot x(t)$ |

> **例3 不封闭的反例**：$p(x) = x^2 + x$ 和 $q(x) = -x^2 + 1$ 都是 $Q_2(x)$ 中的元素（2次多项式），但 $p(x) + q(x) = x + 1$ 是1次多项式，不属于 $Q_2(x)$。

### 6.3 赋范线性空间的定义

> **定义（赋范线性空间）**：设 $E$ 是实数域上的线性空间。若对 $\forall x \in E$，按一定规则对应非负实数 $\|x\|$，且满足以下三个**范数公理**：
>
> | 公理 | 表达式 |
> |------|--------|
> | **正定性** | $\|x\| \geq 0$，且 $\|x\| = 0 \Leftrightarrow x = \mathbf{0}$ |
> | **齐次性** | $\lVert\alpha x\rVert = \lvert\alpha\rvert \cdot \lVert x\rVert$ |
> | **三角不等式** | $\|x + y\| \leq \|x\| + \|y\|, \quad \forall x, y \in E$ |
>
> 则称 $\|x\|$ 为 $x$ 的**范数**，称 $(E, \|\cdot\|)$ 为**赋范线性空间**。

**赋范线性空间 = 线性空间 + 范数**

### 6.4 利用范数定义距离

在赋范线性空间中，可以自然地定义距离：

$$d(x, y) = \|x - y\|$$

**例1**：在 $\mathbb{R}^n$ 上，不同范数诱导不同距离：

| 范数 | 诱导的距离 |
|------|-----------|
| $\lVert\mathbf{x}\rVert_\infty = \max \lvert x_i\rvert$ | $d_\infty(\mathbf{x}, \mathbf{y}) = \max_{i} \lvert x_i - y_i\rvert$ |
| $\lVert\mathbf{x}\rVert_1 = \sum \lvert x_i\rvert$ | $d_1(\mathbf{x}, \mathbf{y}) = \sum_{i} \lvert x_i - y_i\rvert$ |
| $\|\mathbf{x}\|_2 = (\sum x_i^2)^{1/2}$ | $d_2(\mathbf{x}, \mathbf{y}) = (\sum (x_i - y_i)^2)^{1/2}$ |

**例2**：在 $C[a,b]$ 上，$\|x\| = \max_{t \in [a,b]} |x(t)|$ 诱导距离 $d(x,y) = \max_{t} |x(t) - y(t)|$。

> **重要结论**：赋范线性空间通过范数定义距离后，**自动成为距离空间**。因此赋范线性空间是比距离空间更"丰富"的结构——它同时具有**线性结构**（加法、数乘）和**度量结构**（距离）。

## 7. 赋范线性空间中的收敛与完备性

### 7.1 收敛点列与 Cauchy 点列

> **定义1（收敛点列）**：设 $E$ 是赋范线性空间，$\{x_n\} \subset E$，$x \in E$。若
>
> $$\|x_n - x\| \to 0 \quad (n \to \infty)$$
>
> 则称 $\{x_n\}$ 在 $E$ 中**收敛**于 $x$，记作 $\lim_{n \to \infty} x_n = x$。

> **定义2（Cauchy 点列）**：若
>
> $$\forall \varepsilon > 0, \; \exists N, \; \forall n > N, \; \forall p \in \mathbb{Z}^+: \quad \|x_{n+p} - x_n\| < \varepsilon$$
>
> 则称 $\{x_n\}$ 为 $E$ 中的 **Cauchy 点列**（基本列）。

### 7.2 收敛点列的基本性质

| 性质 | 内容 |
|------|------|
| **性质1（唯一性）** | 收敛点列的极限点是**唯一**的 |
| **性质2（有界性）** | 收敛点列必定**有界** |
| **性质3（范数连续性）** | $x_n \to x_0$，$y_n \to y_0$ 时，$\|x_n - y_n\| \to \|x_0 - y_0\|$ |

**性质3的详细说明**：$\|x - y\|$ 是 $x, y$ 两个变元的**连续泛函**。即当 $x_n \to x_0, y_n \to y_0$ 时：

$$\|x_n - y_n\| \to \|x_0 - y_0\|$$

### 7.3 收敛与 Cauchy 的关系

> **性质4**：在赋范线性空间 $(E, \|\cdot\|)$ 中：
> - **收敛 $\Rightarrow$ Cauchy**：收敛点列一定是 Cauchy 点列
> - **Cauchy $\not\Rightarrow$ 收敛**：Cauchy 点列不一定收敛

**收敛 $\Rightarrow$ Cauchy 的证明**：

设 $x_n \to x$，则 $\|x_n - x\| \to 0$。对于任意 $p \in \mathbb{Z}^+$：

$$\|x_{n+p} - x_n\| = \|(x_{n+p} - x) - (x_n - x)\| \leq \|x_{n+p} - x\| + \|x_n - x\| \to 0$$

**Cauchy $\not\Rightarrow$ 收敛的反例**：

设 $X = P[0,1]$ 表示区间 $[0,1]$ 上的所有多项式，赋予一致范数

$$\|p\|_\infty = \max_{t \in [0,1]} |p(t)|$$

则 $X$ 是赋范线性空间。考虑多项式列

$$p_n(t) = \sum_{k=0}^{n} \frac{t^k}{k!}$$

由尾项估计

$$\|p_{n+p}-p_n\|_\infty \leq \sum_{k=n+1}^{n+p}\frac{1}{k!} \to 0$$

知 $\{p_n\}$ 在一致范数下是 Cauchy 列；但它一致收敛到 $e^t$，而 $e^t$ **不是多项式**，因此极限不属于 $X$。所以 $X$ 中存在 Cauchy 列不收敛。

### 7.4 完备性与 Banach 空间

> **定义（完备性）**：若赋范线性空间 $X$ 中**每个 Cauchy 点列都收敛**（极限点在 $X$ 内），则称 $X$ 是**完备的**赋范线性空间，也称为 **Banach 空间**。

| 概念 | 定义 | 关系 |
|------|------|------|
| 赋范线性空间 | 线性空间 + 范数 | 基本结构 |
| Banach 空间 | 完备的赋范线性空间 | 更强条件 |

> **在 Banach 空间中**：收敛点列 $\Leftrightarrow$ Cauchy 点列（等价！）

**实用意义**：在 Banach 空间中，判断点列是否收敛，**不需要事先知道极限值**——只需验证它是 Cauchy 列即可。这对于**迭代算法的收敛性证明**至关重要。

## 8. 知识结构总结

### 8.1 空间层次关系

```
集合（Set）
 └── 距离空间（Metric Space）            ← 加距离函数 d(x,y)
      └── 赋范线性空间（Normed Space）   ← 加线性结构 + 范数 ||x||
           ├── Banach 空间               ← 再加完备性
           └── 内积空间                  ← 再加内积 ⟨x,y⟩（内积可诱导范数）
                └── Hilbert 空间         ← 完备的内积空间
```

**关系说明**：
- 每个内积空间都自然是赋范线性空间。
- 每个 Hilbert 空间同时是 Banach 空间。
- Banach 空间与内积空间之间**没有一般性的包含关系**，不能简单串成一条链。

### 8.2 三种范数对照总表

| | 向量范数 | 矩阵算子范数 |
|---|---------|------------|
| **$\infty$-范数** | $\max_i \lvert x_i\rvert$ | $\max_i \sum_j \lvert a_{ij}\rvert$（**行**范数） |
| **$1$-范数** | $\sum_i \lvert x_i\rvert$ | $\max_j \sum_i \lvert a_{ij}\rvert$（**列**范数） |
| **$2$-范数** | $(\sum_i x_i^2)^{1/2}$ | $\sqrt{\lambda_{\max}(A^TA)}$（**谱**范数） |

### 8.3 核心定理速查

| 定理 | 内容 | 应用 |
|------|------|------|
| 定理1 | 范数是连续函数 | 误差分析的理论基础 |
| 定理2 | $\mathbb{R}^n$ 上所有向量范数等价 | 收敛性与范数选择无关 |
| 定理3 | 向量收敛等价于在任意范数下收敛 | 选择方便的范数判断收敛 |
| 定理4 | 三种矩阵算子范数的计算公式 | 实际计算矩阵范数 |
| 定理5 | $\rho(A) \leq \|A\|$ | 谱半径是范数的下界 |
| 定理6 | 对称矩阵：$\rho(A) = \|A\|_2$ | 简化对称矩阵的谱范数计算 |
| 定理7 | 诱导范数下：$\|B\| < 1 \Rightarrow I-B$ 可逆 | 迭代法收敛条件 |
