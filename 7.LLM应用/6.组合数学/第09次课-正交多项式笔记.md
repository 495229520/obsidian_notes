## 1. 引言：为什么需要正交多项式？

### 1.1 上一讲遗留的问题

在 [[第08次课-最佳逼近与最佳平方逼近笔记|第08次课]] 中，我们知道：

- 用 $\{1, x, x^2, \ldots, x^n\}$ 作为基做最佳平方逼近，Gram 矩阵是 **Hilbert 矩阵**（极度病态）
- 用**正交基**做逼近，Gram 矩阵是**对角矩阵**（完美良态）
- 每个 Fourier 系数可以**独立计算**：$\alpha_i^* = \frac{(f, \varphi_i)}{(\varphi_i, \varphi_i)}$

> **本讲的核心任务**：具体构造这些正交多项式。

### 1.2 两大正交多项式族

| 名称 | 区间 | 权函数 $\rho(x)$ | 核心特征 |
|------|:----:|:--------------:|---------|
| **Legendre 多项式** | $[-1, 1]$ | $1$ | 最"自然"的正交多项式 |
| **Chebyshev 多项式** | $[-1, 1]$ | $\frac{1}{\sqrt{1 - x^2}}$ | 与三角函数密切相关 |

### 1.3 为什么区间都是 $[-1, 1]$？

任意区间 $[a, b]$ 都可以通过**线性变换**映射到 $[-1, 1]$：

$$\boxed{t = \frac{2x - a - b}{b - a}} \quad \Longleftrightarrow \quad x = \frac{a + b}{2} + \frac{b - a}{2} t$$

- $x = a$ 时 $t = -1$，$x = b$ 时 $t = 1$
- 所以只需要在 $[-1, 1]$ 上建立理论，再变换到任意区间

## 2. Legendre 多项式

### 2.1 Rodrigues 公式（定义）

> **定义（Legendre 多项式）**：
>
> $$\boxed{P_n(x) = \frac{1}{2^n n!} \frac{d^n}{dx^n}\left[(x^2 - 1)^n\right], \quad n = 0, 1, 2, \ldots}$$
>
> 这称为 **Rodrigues 公式**。

**逐项理解**：

| 部分 | 含义 |
|------|------|
| $(x^2 - 1)^n$ | 起点：一个 $2n$ 次多项式 |
| $\frac{d^n}{dx^n}$ | 对它求 $n$ 阶导数 → 变成 $n$ 次多项式 |
| $\frac{1}{2^n n!}$ | 归一化系数，使 $P_n(1) = 1$ |

**为什么求 $n$ 阶导数后变成 $n$ 次多项式？**

$(x^2 - 1)^n$ 展开后最高次项是 $x^{2n}$。对 $2n$ 次多项式求 $n$ 阶导数，次数降低 $n$，得到 $n$ 次多项式。

> 理解 Rodrigues 公式需要 [[微积分基础回顾#24. Leibniz 求导法则（乘积的高阶导数）|Leibniz 高阶导数法则]] 的知识。

### 2.2 具体计算

**$n = 0$**：

$$P_0(x) = \frac{1}{1 \cdot 1}(x^2 - 1)^0 = 1$$

**$n = 1$**：

$$P_1(x) = \frac{1}{2 \cdot 1} \frac{d}{dx}(x^2 - 1) = \frac{1}{2} \cdot 2x = x$$

**$n = 2$**：

$$P_2(x) = \frac{1}{4 \cdot 2} \frac{d^2}{dx^2}(x^2 - 1)^2 = \frac{1}{8} \frac{d^2}{dx^2}(x^4 - 2x^2 + 1)$$

$$= \frac{1}{8}(12x^2 - 4) = \frac{1}{2}(3x^2 - 1)$$

**$n = 3$**：

$$(x^2 - 1)^3 = x^6 - 3x^4 + 3x^2 - 1$$

等等，让我仔细算：

- $\frac{d^3}{dx^3}(x^6) = 6 \cdot 5 \cdot 4 \cdot x^3 = 120x^3$
- $\frac{d^3}{dx^3}(-3x^4) = -3 \cdot 4 \cdot 3 \cdot 2 \cdot x = -72x$
- $\frac{d^3}{dx^3}(3x^2) = 0$
- $\frac{d^3}{dx^3}(-1) = 0$

$$\frac{d^3}{dx^3}(x^2 - 1)^3 = 120x^3 - 72x$$

$$P_3(x) = \frac{1}{2^3 \cdot 3!}(120x^3 - 72x) = \frac{1}{48}(120x^3 - 72x) = \frac{1}{2}(5x^3 - 3x)$$

### 2.3 前几个 Legendre 多项式

| $n$ | $P_n(x)$ | 最高次系数 $a_n$ |
|:---:|----------|:---------------:|
| 0 | $1$ | $1$ |
| 1 | $x$ | $1$ |
| 2 | $\frac{1}{2}(3x^2 - 1)$ | $\frac{3}{2}$ |
| 3 | $\frac{1}{2}(5x^3 - 3x)$ | $\frac{5}{2}$ |
| 4 | $\frac{1}{8}(35x^4 - 30x^2 + 3)$ | $\frac{35}{8}$ |
| 5 | $\frac{1}{8}(63x^5 - 70x^3 + 15x)$ | $\frac{63}{8}$ |

最高次系数的一般公式：

$$a_n = \frac{(2n)!}{2^n (n!)^2}$$

> **验证**：$a_2 = \frac{4!}{4 \cdot 4} = \frac{24}{16} = \frac{3}{2}$ ✓，$a_3 = \frac{6!}{8 \cdot 36} = \frac{720}{288} = \frac{5}{2}$ ✓

### 2.4 正交性（核心性质）

> **定理1**：Legendre 多项式在 $[-1, 1]$ 上关于权函数 $\rho(x) = 1$ 正交：
>
> $$(P_n, P_m) = \int_{-1}^{1} P_n(x) P_m(x) \, dx = \begin{cases} 0, & m \neq n \\[4pt] \dfrac{2}{2n + 1}, & m = n \end{cases}$$

**范数**：

$$\|P_n\| = \sqrt{(P_n, P_n)} = \sqrt{\frac{2}{2n + 1}}$$

| $n$ | $\|P_n\|^2 = \frac{2}{2n+1}$ | $\|P_n\|$ |
|:---:|:---------------------------:|:---------:|
| 0 | $2$ | $\sqrt{2}$ |
| 1 | $\frac{2}{3}$ | $\sqrt{\frac{2}{3}}$ |
| 2 | $\frac{2}{5}$ | $\sqrt{\frac{2}{5}}$ |
| 3 | $\frac{2}{7}$ | $\sqrt{\frac{2}{7}}$ |

**正交性的证明思路**（当 $m \neq n$ 时）：

利用 Rodrigues 公式和**反复分部积分**（参见 [[微积分基础回顾#26. 反复分部积分与正交性证明|微积分基础]]）：

$$(P_n, P_m) = \frac{1}{2^n n!} \int_{-1}^{1} \frac{d^n}{dx^n}[(x^2-1)^n] \cdot P_m(x) \, dx$$

对 $\frac{d^n}{dx^n}[(x^2-1)^n]$ 反复分部积分 $n$ 次，每次将一个导数"转移"到 $P_m(x)$ 上。

关键事实：$(x^2 - 1)^n$ 在 $x = \pm 1$ 处有 $n$ 重零点，所以每次分部积分的边界项都为零。

经过 $n$ 次分部积分后：

$$(P_n, P_m) = \frac{(-1)^n}{2^n n!} \int_{-1}^{1} (x^2-1)^n \cdot P_m^{(n)}(x) \, dx$$

若 $m < n$，$P_m(x)$ 是 $m$ 次多项式，$P_m^{(n)} = 0$（求导次数 > 多项式次数），积分为零。

若 $m > n$，交换 $m, n$ 的角色，同理为零。

### 2.5 奇偶性

> **性质**：$P_n(-x) = (-1)^n P_n(x)$

即：**$n$ 为偶数时 $P_n$ 是偶函数，$n$ 为奇数时 $P_n$ 是奇函数**。

**验证**：

- $P_0(x) = 1$：偶函数 ✓
- $P_1(x) = x$：奇函数 ✓
- $P_2(x) = \frac{1}{2}(3x^2 - 1)$：偶函数 ✓
- $P_3(x) = \frac{1}{2}(5x^3 - 3x)$：奇函数 ✓

**意义**：当 $f(x)$ 是偶函数时，$(f, P_n) = 0$ 对所有奇数 $n$ 成立（偶 $\times$ 奇 = 奇函数，在对称区间上积分为零）。这可以**减少一半的计算量**。

> 参见 [[微积分基础回顾#21. 对称区间上奇偶函数的积分|微积分基础：奇偶函数积分]]。

### 2.6 三项递推关系

> **定理2（递推公式）**：
>
> $$\boxed{P_{n+1}(x) = \frac{2n + 1}{n + 1} \, x \, P_n(x) - \frac{n}{n + 1} \, P_{n-1}(x), \quad n = 1, 2, \ldots}$$
>
> 初始值：$P_0(x) = 1$，$P_1(x) = x$。

**为什么叫"三项递推"？** 因为 $P_{n+1}$ 由 $P_n$ 和 $P_{n-1}$ 两项（加上 $P_{n+1}$ 本身共三项）决定。

**实际意义**：不需要用 Rodrigues 公式逐个计算高阶导数，只需用递推关系**从低到高依次生成**。

**验证**（$n = 1$，即由 $P_0, P_1$ 推出 $P_2$）：

$$P_2(x) = \frac{3}{2} x \cdot P_1(x) - \frac{1}{2} P_0(x) = \frac{3}{2}x \cdot x - \frac{1}{2} \cdot 1 = \frac{3x^2 - 1}{2} \quad \checkmark$$

**验证**（$n = 2$，即由 $P_1, P_2$ 推出 $P_3$）：

$$P_3(x) = \frac{5}{3} x \cdot P_2(x) - \frac{2}{3} P_1(x) = \frac{5}{3}x \cdot \frac{3x^2 - 1}{2} - \frac{2}{3}x = \frac{5x(3x^2-1)}{6} - \frac{2x}{3}$$

$$= \frac{15x^3 - 5x}{6} - \frac{4x}{6} = \frac{15x^3 - 9x}{6} = \frac{5x^3 - 3x}{2} \quad \checkmark$$

### 2.7 递推公式的推导

**核心思想**：$xP_n(x)$ 是 $n+1$ 次多项式，可以用 $P_0, P_1, \ldots, P_{n+1}$ 展开。

$$xP_n(x) = \sum_{k=0}^{n+1} c_k P_k(x)$$

由正交性，$c_k = \frac{(xP_n, P_k)}{(P_k, P_k)}$。

**关键观察**：$(xP_n, P_k) = \int_{-1}^{1} x P_n(x) P_k(x) dx$。

$xP_k(x)$ 是 $k+1$ 次多项式，当 $k + 1 < n$（即 $k \leq n - 2$）时，$xP_k$ 的次数 $< n$，与 $P_n$ 正交，所以 $c_k = 0$。

因此只有 $c_{n-1}, c_n, c_{n+1}$ 可能非零——**只剩三项**。

进一步利用 $P_n$ 的奇偶性可证 $c_n = 0$（$xP_n$ 的奇偶性与 $P_n$ 相反，正交性）。

最终得到递推关系中只涉及 $P_{n+1}, P_n, P_{n-1}$。

### 2.8 子空间等价性

> **定理3**：$\text{span}\{P_0, P_1, \ldots, P_n\} = \text{span}\{1, x, x^2, \ldots, x^n\}$

**白话**：Legendre 多项式 $P_0, \ldots, P_n$ 张成的空间，与幂函数 $1, x, \ldots, x^n$ 张成的空间**完全相同**——都是全体不超过 $n$ 次的多项式空间 $\Pi_n$。

**证明**：

- $P_k(x)$ 是 $k$ 次多项式（$k = 0, 1, \ldots, n$），所以 $P_k \in \text{span}\{1, x, \ldots, x^n\}$
- $P_0, P_1, \ldots, P_n$ 两两正交 → 线性无关 → 它们是 $n+1$ 维空间的一组基

因此两组基张成相同的空间。

**意义**：用 $\{P_0, \ldots, P_n\}$ 做最佳平方逼近，与用 $\{1, x, \ldots, x^n\}$ 做逼近，**搜索的函数空间完全一样**，只是基不同。正交基的计算更稳定。

## 3. 用 Legendre 多项式做最佳平方逼近

### 3.1 方法总结

**问题**：求 $f(x)$ 在 $[-1, 1]$ 上的 $n$ 次多项式最佳平方逼近 $S^*(x)$。

**步骤**：

| 步骤 | 操作 | 公式 |
|:----:|------|------|
| 1 | 计算 Fourier 系数 | $\alpha_j = \frac{(f, P_j)}{(P_j, P_j)} = \frac{2j+1}{2}\int_{-1}^{1} f(x) P_j(x) dx$ |
| 2 | 构造逼近多项式 | $S^*(x) = \sum_{j=0}^{n} \alpha_j P_j(x)$ |
| 3 | 计算误差（可选） | $\|f - S^*\|_2^2 = (f, f) - \sum_{j=0}^{n} \alpha_j (f, P_j)$ |

> **注意**：由于 $(P_j, P_j) = \frac{2}{2j+1}$，所以 $\alpha_j = \frac{2j+1}{2}(f, P_j)$。

### 3.2 利用奇偶性简化计算

| $f(x)$ 的类型 | $(f, P_j) = 0$ 的条件 | 需要计算的系数 |
|:------------:|:--------------------:|:------------:|
| 偶函数 | $j$ 为奇数 | 只需 $\alpha_0, \alpha_2, \alpha_4, \ldots$ |
| 奇函数 | $j$ 为偶数 | 只需 $\alpha_1, \alpha_3, \alpha_5, \ldots$ |
| 一般函数 | 无自动为零的 | 全部计算 |

**原因**：偶函数 $\times$ 奇函数 = 奇函数，在 $[-1, 1]$ 上积分为零。

### 3.3 一般区间 $[a, b]$ 的处理

若 $f(x)$ 定义在 $[a, b]$ 上而非 $[-1, 1]$：

**Step 1**：变量替换

$$x = \frac{a + b}{2} + \frac{b - a}{2} t, \quad t \in [-1, 1]$$

$$f(x) = f\left(\frac{a + b}{2} + \frac{b - a}{2} t\right) = g(t)$$

**Step 2**：对 $g(t)$ 在 $[-1, 1]$ 上用 Legendre 多项式逼近

**Step 3**：将结果代回 $x$

$$t = \frac{2x - a - b}{b - a}$$

## 4. 计算示例（Legendre）

### 4.1 例1：$f(x) = \sin\frac{\pi x}{2}$ 在 $[-1, 1]$ 上的三次逼近

**问题**：用 $P_0, P_1, P_2, P_3$ 做 Fourier 展开。

**分析**：$\sin\frac{\pi x}{2}$ 是**奇函数**（$\sin\frac{\pi(-x)}{2} = -\sin\frac{\pi x}{2}$）。

因此 $(f, P_j) = 0$ 对偶数 $j$ 成立，即 $\alpha_0 = \alpha_2 = 0$。

**Step 1**：计算 $(P_j, P_j)$

$$(P_0, P_0) = 2, \quad (P_1, P_1) = \frac{2}{3}, \quad (P_2, P_2) = \frac{2}{5}, \quad (P_3, P_3) = \frac{2}{7}$$

**Step 2**：计算非零的 Fourier 系数

$$(f, P_1) = \int_{-1}^{1} x \sin\frac{\pi x}{2} dx$$

> 被积函数 $x \sin\frac{\pi x}{2}$ 是偶函数（奇 × 奇 = 偶），所以：

$$= 2\int_0^1 x \sin\frac{\pi x}{2} dx$$

用分部积分（参见 [[微积分基础回顾#6.1 分部积分法（Integration by Parts）|分部积分]]）：

令 $u = x$，$dv = \sin\frac{\pi x}{2}dx$，则 $du = dx$，$v = -\frac{2}{\pi}\cos\frac{\pi x}{2}$

$$= 2\left\{\left[-\frac{2x}{\pi}\cos\frac{\pi x}{2}\right]_0^1 + \frac{2}{\pi}\int_0^1 \cos\frac{\pi x}{2}dx\right\}$$

$$= 2\left\{0 + \frac{2}{\pi}\left[\frac{2}{\pi}\sin\frac{\pi x}{2}\right]_0^1\right\} = 2 \cdot \frac{2}{\pi} \cdot \frac{2}{\pi} = \frac{8}{\pi^2}$$

$$\alpha_1 = \frac{(f, P_1)}{(P_1, P_1)} = \frac{8/\pi^2}{2/3} = \frac{12}{\pi^2} \approx 1.2159$$

$$(f, P_3) = \int_{-1}^{1} \frac{1}{2}(5x^3 - 3x) \sin\frac{\pi x}{2} dx$$

经过类似的分部积分计算（被积函数也是偶函数，利用 $= 2\int_0^1 \ldots$）：

$$(f, P_3) = \frac{48(\pi^2 - 10)}{\pi^4}$$

$$\alpha_3 = \frac{(f, P_3)}{(P_3, P_3)} = \frac{48(\pi^2 - 10)/\pi^4}{2/7} = \frac{168(\pi^2 - 10)}{\pi^4} \approx -0.0402$$

**Step 3**：最佳逼近多项式

$$\boxed{S^*(x) = \alpha_1 P_1(x) + \alpha_3 P_3(x) = \frac{12}{\pi^2} x + \frac{168(\pi^2 - 10)}{\pi^4} \cdot \frac{1}{2}(5x^3 - 3x)}$$

展开为标准形式：

$$S^*(x) \approx 1.5532\,x - 0.5622\,x^3$$

**直觉检验**：$\sin\frac{\pi x}{2} \approx \frac{\pi}{2}x \approx 1.5708\,x$ 在 $x$ 较小时。逼近结果 $1.5532\,x$ 与之接近，合理。

### 4.2 例2：$y = \arctan x$ 在 $[0, 1]$ 上的线性逼近

**问题**：$f(x) = \arctan x$，$x \in [0, 1]$，用一次多项式逼近。

**Step 1**：区间变换 $[0, 1] \to [-1, 1]$

$$x = \frac{0 + 1}{2} + \frac{1 - 0}{2}t = \frac{t + 1}{2}, \quad t \in [-1, 1]$$

$$g(t) = \arctan\frac{t + 1}{2}$$

**Step 2**：用 $P_0(t) = 1$，$P_1(t) = t$ 逼近 $g(t)$

$$(P_0, P_0) = 2, \quad (P_1, P_1) = \frac{2}{3}$$

$$(g, P_0) = \int_{-1}^{1} \arctan\frac{t+1}{2} \, dt = \frac{\pi}{2} - \ln 2$$

$$(g, P_1) = \int_{-1}^{1} t \cdot \arctan\frac{t+1}{2} \, dt = \frac{\pi}{2} - 2 + \ln 2$$

> 这两个积分都需要分部积分来计算（参见 [[微积分基础回顾#6.1 分部积分法（Integration by Parts）|分部积分法]]），其中 $\frac{d}{dx}\arctan x = \frac{1}{1+x^2}$。

**Step 3**：Fourier 系数

$$\alpha_0 = \frac{(g, P_0)}{(P_0, P_0)} = \frac{\pi/2 - \ln 2}{2}$$

$$\alpha_1 = \frac{(g, P_1)}{(P_1, P_1)} = \frac{3}{2}\left(\frac{\pi}{2} - 2 + \ln 2\right)$$

**Step 4**：$g(t)$ 的逼近

$$g^*(t) = \alpha_0 + \alpha_1 t$$

**Step 5**：代回 $x$（$t = 2x - 1$）

$$y^* = \arctan x \approx \left(\frac{\pi}{4} - \frac{1}{2}\ln 2\right) + \frac{3}{2}\left(\frac{\pi}{2} - 2 + \ln 2\right)(2x - 1)$$

## 5. Chebyshev 多项式

### 5.1 定义

> **定义（Chebyshev 多项式）**：
>
> $$\boxed{T_n(x) = \cos(n \cdot \arccos x), \quad -1 \leq x \leq 1}$$

**初看**：这个定义看起来很奇怪——一个"多项式"居然由三角函数定义？

**关键理解**：令 $x = \cos\theta$（$\theta \in [0, \pi]$），则：

$$T_n(\cos\theta) = \cos(n\theta)$$

也就是说，$T_n(x)$ 在做三角换元后就是 $\cos(n\theta)$——一个简单的三角函数。

> 理解这个换元需要 [[微积分基础回顾#25. 三角换元与反三角函数|三角换元]] 的知识。

### 5.2 为什么 $\cos(n\theta)$ 是 $\cos\theta$ 的多项式？

由三角恒等式（参见 [[微积分基础回顾#27. 三角函数的积化和差与和差化积|积化和差公式]]）：

$$\cos(n+1)\theta = 2\cos\theta \cdot \cos(n\theta) - \cos(n-1)\theta$$

令 $x = \cos\theta$：

$$T_{n+1}(x) = 2x \cdot T_n(x) - T_{n-1}(x)$$

从 $T_0(x) = 1$，$T_1(x) = x$ 出发，每一步乘以 $x$ 再减去上一项——**始终是多项式**！

### 5.3 前几个 Chebyshev 多项式

| $n$ | $T_n(x)$ | 最高次项 |
|:---:|----------|:-------:|
| 0 | $1$ | $1$ |
| 1 | $x$ | $x$ |
| 2 | $2x^2 - 1$ | $2x^2$ |
| 3 | $4x^3 - 3x$ | $4x^3$ |
| 4 | $8x^4 - 8x^2 + 1$ | $8x^4$ |
| 5 | $16x^5 - 20x^3 + 5x$ | $16x^5$ |
| 6 | $32x^6 - 48x^4 + 18x^2 - 1$ | $32x^6$ |

**最高次系数**：$T_n(x)$ 的最高次系数为 $2^{n-1}$（$n \geq 1$）。

**首一化 Chebyshev 多项式**（最高次系数为1）：

$$\hat{T}_n(x) = \frac{1}{2^{n-1}} T_n(x), \quad n \geq 1$$

### 5.4 递推关系的推导

从三角恒等式 $\cos(A + B) + \cos(A - B) = 2\cos A \cos B$ 出发。

令 $A = n\theta$，$B = \theta$：

$$\cos(n+1)\theta + \cos(n-1)\theta = 2\cos\theta \cdot \cos(n\theta)$$

$$\cos(n+1)\theta = 2\cos\theta \cdot \cos(n\theta) - \cos(n-1)\theta$$

用 $x = \cos\theta$ 替换：

$$\boxed{T_{n+1}(x) = 2x \cdot T_n(x) - T_{n-1}(x), \quad n = 1, 2, \ldots}$$

**验证**（$n = 1$）：

$$T_2(x) = 2x \cdot T_1(x) - T_0(x) = 2x \cdot x - 1 = 2x^2 - 1 \quad \checkmark$$

**验证**（$n = 2$）：

$$T_3(x) = 2x \cdot T_2(x) - T_1(x) = 2x(2x^2 - 1) - x = 4x^3 - 3x \quad \checkmark$$

### 5.5 与 Legendre 递推的对比

| 特征 | Legendre | Chebyshev |
|------|----------|-----------|
| **递推公式** | $P_{n+1} = \frac{2n+1}{n+1}xP_n - \frac{n}{n+1}P_{n-1}$ | $T_{n+1} = 2xT_n - T_{n-1}$ |
| **$x$ 的系数** | $\frac{2n+1}{n+1}$（依赖 $n$） | **恒为 $2$**（更简单） |
| **$P_{n-1}/T_{n-1}$ 的系数** | $\frac{n}{n+1}$（依赖 $n$） | **恒为 $1$** |
| **初值** | $P_0 = 1$，$P_1 = x$ | $T_0 = 1$，$T_1 = x$ |

Chebyshev 递推更简单，系数不依赖 $n$。

### 5.6 奇偶性

$$T_n(-x) = \cos(n \cdot \arccos(-x)) = \cos(n(\pi - \arccos x)) = \cos(n\pi - n\arccos x)$$

$$= (-1)^n \cos(n\arccos x) = (-1)^n T_n(x)$$

即 $T_n(-x) = (-1)^n T_n(x)$——与 Legendre 多项式相同的奇偶规律。

### 5.7 正交性（核心性质）

> **定理4**：Chebyshev 多项式在 $[-1, 1]$ 上关于权函数 $\rho(x) = \frac{1}{\sqrt{1 - x^2}}$ 正交：
>
> $$(T_n, T_m) = \int_{-1}^{1} \frac{T_n(x) T_m(x)}{\sqrt{1 - x^2}} \, dx = \begin{cases} 0, & n \neq m \\[4pt] \pi, & n = m = 0 \\[4pt] \dfrac{\pi}{2}, & n = m \neq 0 \end{cases}$$

### 5.8 正交性的证明

做三角换元 $x = \cos\theta$，则 $dx = -\sin\theta \, d\theta$，且 $\sqrt{1 - x^2} = \sin\theta$。

当 $x$ 从 $-1$ 到 $1$ 时，$\theta$ 从 $\pi$ 到 $0$：

$$(T_n, T_m) = \int_\pi^0 \frac{\cos(n\theta) \cos(m\theta)}{\sin\theta} (-\sin\theta) \, d\theta = \int_0^\pi \cos(n\theta) \cos(m\theta) \, d\theta$$

> 注意：$\frac{1}{\sqrt{1-x^2}}$ 和 $dx$ 中的 $\sin\theta$ **恰好约掉**！这就是权函数 $\frac{1}{\sqrt{1-x^2}}$ 的妙处——它使得三角换元后变成最简单的三角积分。

现在利用**积化和差公式**（参见 [[微积分基础回顾#27. 三角函数的积化和差与和差化积|微积分基础]]）：

$$\cos(n\theta)\cos(m\theta) = \frac{1}{2}[\cos(n-m)\theta + \cos(n+m)\theta]$$

**情况1**：$n \neq m$

$$\int_0^\pi \cos(n\theta)\cos(m\theta)d\theta = \frac{1}{2}\left[\frac{\sin(n-m)\theta}{n-m} + \frac{\sin(n+m)\theta}{n+m}\right]_0^\pi$$

当 $n \neq m$ 时，$n - m$ 和 $n + m$ 都是非零整数，$\sin(k\pi) = 0$，所以积分 $= 0$。

**情况2**：$n = m \neq 0$

$$\int_0^\pi \cos^2(n\theta) d\theta = \frac{1}{2}\int_0^\pi [1 + \cos(2n\theta)] d\theta = \frac{1}{2}\left[\theta + \frac{\sin(2n\theta)}{2n}\right]_0^\pi = \frac{\pi}{2}$$

**情况3**：$n = m = 0$

$$\int_0^\pi 1 \, d\theta = \pi \qquad \square$$

### 5.9 为什么 Chebyshev 用权函数 $\frac{1}{\sqrt{1-x^2}}$？

直觉上，$\frac{1}{\sqrt{1-x^2}}$ 在端点 $x = \pm 1$ 附近趋于无穷大。这意味着**端点处的误差被赋予更高的权重**。

回忆 Runge 现象：等距插值的误差在**端点处最严重**。Chebyshev 逼近通过加大端点权重，恰好平衡了这一问题。

## 6. 用 Chebyshev 多项式做最佳平方逼近

### 6.1 方法总结

**内积**（带权）：

$$(f, g) = \int_{-1}^{1} \frac{f(x)g(x)}{\sqrt{1 - x^2}} \, dx$$

**Fourier 系数**：

$$\alpha_0 = \frac{(f, T_0)}{(T_0, T_0)} = \frac{1}{\pi}\int_{-1}^{1} \frac{f(x)}{\sqrt{1-x^2}} dx$$

$$\alpha_j = \frac{(f, T_j)}{(T_j, T_j)} = \frac{2}{\pi}\int_{-1}^{1} \frac{f(x)T_j(x)}{\sqrt{1-x^2}} dx, \quad j = 1, 2, \ldots$$

**最佳逼近多项式**：

$$S^*(x) = \sum_{j=0}^{n} \alpha_j T_j(x)$$

### 6.2 幂函数与 Chebyshev 多项式的互相转换

在实际计算中，经常需要将结果从 Chebyshev 形式转换为标准多项式形式，或反过来。

**幂函数 → Chebyshev 展开**：

| $x^k$ | Chebyshev 展开 |
|:-----:|----------------|
| $1$ | $T_0$ |
| $x$ | $T_1$ |
| $x^2$ | $\frac{1}{2}[T_0 + T_2]$ |
| $x^3$ | $\frac{1}{4}[3T_1 + T_3]$ |
| $x^4$ | $\frac{1}{8}[3T_0 + 4T_2 + T_4]$ |
| $x^5$ | $\frac{1}{16}[10T_1 + 5T_3 + T_5]$ |
| $x^6$ | $\frac{1}{32}[10T_0 + 15T_2 + 6T_4 + T_6]$ |

> 这些转换公式可以由 $T_n(x) = \cos(n\arccos x)$ 和倍角公式导出。例如 $x^2 = \cos^2\theta = \frac{1 + \cos 2\theta}{2} = \frac{T_0 + T_2}{2}$。

## 7. 计算示例（Chebyshev）

### 7.1 例3：$f(x) = \sqrt{1 - x^2}$ 在 $[-1, 1]$ 上的二次逼近

**问题**：在 $[-1, 1]$ 上，以 $\rho(x) = \frac{1}{\sqrt{1-x^2}}$ 为权，用 $M = \text{span}\{T_0, T_1, T_2\}$ 逼近 $f(x) = \sqrt{1-x^2}$。

**分析**：$f(x) = \sqrt{1-x^2}$ 是**偶函数**，所以 $(f, T_1) = 0$（$T_1 = x$ 是奇函数）。

**Step 1**：计算内积

$$(T_0, T_0) = \pi, \quad (T_1, T_1) = \frac{\pi}{2}, \quad (T_2, T_2) = \frac{\pi}{2}$$

$$(f, T_0) = \int_{-1}^{1} \frac{\sqrt{1-x^2}}{\sqrt{1-x^2}} dx = \int_{-1}^{1} 1 \, dx = 2$$

$$(f, T_1) = \int_{-1}^{1} \frac{x\sqrt{1-x^2}}{\sqrt{1-x^2}} dx = \int_{-1}^{1} x \, dx = 0 \quad \text{（奇函数）}$$

$$(f, T_2) = \int_{-1}^{1} \frac{(2x^2 - 1)\sqrt{1-x^2}}{\sqrt{1-x^2}} dx = \int_{-1}^{1} (2x^2 - 1) dx$$

$$= 2 \cdot \frac{2}{3} - 2 = \frac{4}{3} - 2 = -\frac{2}{3}$$

**Step 2**：Fourier 系数

$$\alpha_0 = \frac{2}{\pi}, \quad \alpha_1 = 0, \quad \alpha_2 = \frac{-2/3}{\pi/2} = -\frac{4}{3\pi}$$

**Step 3**：最佳逼近多项式

$$S^*(x) = \frac{2}{\pi} T_0 + 0 \cdot T_1 - \frac{4}{3\pi} T_2 = \frac{2}{\pi} - \frac{4}{3\pi}(2x^2 - 1)$$

$$= \frac{2}{\pi} + \frac{4}{3\pi} - \frac{8}{3\pi}x^2 = \frac{10}{3\pi} - \frac{8}{3\pi}x^2$$

即 $a = -\frac{8}{3\pi}$，$b = 0$，$c = \frac{10}{3\pi}$。

**Step 4**：计算误差

$$\|f - S^*\|_2^2 = (f, f) - \alpha_0(f, T_0) - \alpha_2(f, T_2)$$

$$(f, f) = \int_{-1}^{1} \frac{(\sqrt{1-x^2})^2}{\sqrt{1-x^2}} dx = \int_{-1}^{1} \sqrt{1-x^2} \, dx = \frac{\pi}{2}$$

> $\int_{-1}^{1}\sqrt{1-x^2}dx$ 是上半单位圆的面积 $= \frac{\pi}{2}$。

$$\|f - S^*\|_2^2 = \frac{\pi}{2} - \frac{2}{\pi} \cdot 2 - \left(-\frac{4}{3\pi}\right)\left(-\frac{2}{3}\right) = \frac{\pi}{2} - \frac{4}{\pi} - \frac{8}{9\pi}$$

$$= \frac{\pi}{2} - \frac{44}{9\pi} \approx 0.0146$$

### 7.2 例3 的另一种解法：用普通基 $\{1, x, x^2\}$

同一个问题也可以用 $\{1, x, x^2\}$ 作为基，建立法方程组（3×3矩阵），解出来结果完全相同。

课件给出了用普通基的 Gram 矩阵（注意权函数 $\rho(x) = \frac{1}{\sqrt{1-x^2}}$）：

| $(\varphi_i, \varphi_j)$ | $\varphi_0 = 1$ | $\varphi_1 = x$ | $\varphi_2 = x^2$ |
|:---:|:---:|:---:|:---:|
| $\varphi_0 = 1$ | $\pi$ | $0$ | $\frac{\pi}{2}$ |
| $\varphi_1 = x$ | $0$ | $\frac{\pi}{2}$ | $0$ |
| $\varphi_2 = x^2$ | $\frac{\pi}{2}$ | $0$ | $\frac{3\pi}{8}$ |

> 这里用到了 $\int_{-1}^{1} \frac{x^k}{\sqrt{1-x^2}}dx$ 的计算（参见 [[微积分基础回顾#28. 常用带权三角积分|常用带权积分]]）。

两种方法得到的结果完全一致，但 Chebyshev 正交基方法**无需解方程组**。

## 8. 其他正交多项式族（简介）

除了 Legendre 和 Chebyshev 以外，还有几族重要的正交多项式：

### 8.1 第二类 Chebyshev 多项式

| 属性 | 内容 |
|------|------|
| **区间** | $[-1, 1]$ |
| **权函数** | $\rho(x) = \sqrt{1 - x^2}$ |
| **定义** | $U_n(x) = \frac{\sin[(n+1)\arccos x]}{\sqrt{1-x^2}}$ |
| **递推** | $U_{n+1}(x) = 2xU_n(x) - U_{n-1}(x)$ |
| **初值** | $U_0 = 1$，$U_1 = 2x$ |

### 8.2 Laguerre 多项式

| 属性 | 内容 |
|------|------|
| **区间** | $[0, +\infty)$ |
| **权函数** | $\rho(x) = e^{-x}$ |
| **Rodrigues 公式** | $L_n(x) = e^x \frac{d^n}{dx^n}(x^n e^{-x})$ |
| **正交性** | $\int_0^\infty L_n(x)L_m(x)e^{-x}dx = \begin{cases} (n!)^2, & m=n \\ 0, & m \neq n\end{cases}$ |
| **递推** | $L_{n+1}(x) = (1 + 2n - x)L_n(x) - n^2 L_{n-1}(x)$ |
| **初值** | $L_0 = 1$，$L_1 = 1 - x$ |

### 8.3 Hermite 多项式（与 Hermite 插值不同！）

| 属性 | 内容 |
|------|------|
| **区间** | $(-\infty, +\infty)$ |
| **权函数** | $\rho(x) = e^{-x^2}$ |
| **Rodrigues 公式** | $H_n(x) = (-1)^n e^{x^2} \frac{d^n}{dx^n}(e^{-x^2})$ |
| **正交性** | $\int_{-\infty}^{\infty} H_n(x)H_m(x)e^{-x^2}dx = \begin{cases} 2^n n!\sqrt{\pi}, & m=n \\ 0, & m \neq n\end{cases}$ |
| **递推** | $H_{n+1}(x) = 2xH_n(x) - 2nH_{n-1}(x)$ |
| **初值** | $H_0 = 1$，$H_1 = 2x$ |

### 8.4 四族正交多项式的对比

| 特征 | Legendre | Chebyshev (I) | Laguerre | Hermite |
|------|----------|-------------|----------|---------|
| **区间** | $[-1,1]$ | $[-1,1]$ | $[0,\infty)$ | $(-\infty,\infty)$ |
| **权** | $1$ | $\frac{1}{\sqrt{1-x^2}}$ | $e^{-x}$ | $e^{-x^2}$ |
| **递推 $x$ 系数** | $\frac{2n+1}{n+1}$ | $2$ | $1+2n-x$ | $2$ |
| **应用场景** | 标准逼近 | 等波纹逼近 | 半无穷区间 | 全实轴 |

### 8.5 正交多项式的共同结构

所有正交多项式族都满足以下**三条共同性质**：

1. **正交性**：$(P_n, P_m) = 0$（$n \neq m$，关于各自的权函数）
2. **三项递推**：$P_{n+1}$ 由 $xP_n$ 和 $P_{n-1}$ 线性组合得到
3. **$n$ 次多项式 $P_n$ 恰有 $n$ 个实零点**，且都在区间内部

### 9.2 核心公式速查

| 公式 | Legendre | Chebyshev |
|------|----------|-----------|
